# SC-4: Competition and coexistence of SC and nematic order in 2DEG
**arXiv:** 2604.18777 | **Authors:** Lopes, do Vale, Barci (UERJ) | **Year:** 2026

## 核心方法论
在二维电子气中引入竞争的 BCS 配对（s 波 + d 波）与四极前向散射（向列序），通过耦合自洽方程的数值求解，获得零温和有限温度相图。

### 模型细节
- 哈密顿量 = 自由电子 + SC 配对（V_s + V_d f_{kk'}）+ 四极前向散射（f₂ f_{kk'}）
- 5 个序参量：Δ_s (s 波), Δ_d^c, Δ_d^s (d 波两个分量), Δ_n^c, Δ_n^s (向列两个分量)
- 均场解耦 → Bogoliubov-de Gennes 对角化
- 准粒子能量 E_k(θ) = sqrt[(ε_k - μ(θ))² + Δ_k²(θ)]，其中 μ(θ) 依赖于向列序

### 关键相图结果
1. **零温：d 波 vs 向列 = 一阶转变**（强竞争，无共存）
2. **零温：s 波 + 向列 = 共存相**（各向异性 Fermi 面 + 均匀 SC gap）
3. **有限温度：** 四极相互作用促进额外 SC 分量出现 → s+d+nematic 三阶共存
4. **关键对称性：** d 波和向列序具有相同的旋转对称性（l=2）→ 直接竞争

### 物理洞察
- 向列序使 Fermi 面各向异性化 → μ(θ) 随角度变化
- s 波 gap 是各向同性的，可以"容忍"各向异性 Fermi 面
- d 波 gap 的节点结构与向列序的角度依赖冲突 → 一阶转变

## 启发的新想法
★ **中期方向：** 这个 s 波 + 向列共存的理论框架可以应用于氢化物。H₃S 在高压下有 s 波配对，但如果存在隐藏的向列涨落，可能产生混合 s+d 态。检验方法：计算 H₃S 的四极 Pomeranchuk 不稳定性参数 f₂。

## 与研究的关联
- 交织序的理论框架 → 对应今夜 kagome 论文中实验观测到的 CDW+SC 共存
- 连续旋转对称下的向列（不同于 Ising 向列）= 有 Goldstone 模 → 涨落效应更强
- 提供了一个计算 SC+nematic 相图的最小模型，可推广到多带系统

## 数值/公式
- 耦合常数：g_s = V_s N_F, g_d = V_d N_F, g_n = f₂ N_F（N_F = 费米面态密度）
- 几何因子：f_{kk'} = cos(2[θ_k - θ_{k'}])
- 相变类型：d 波↔向列 = 一阶，s 波 + 向列 = 二阶（共存相边界）
- 5 个耦合自洽方程同时求解
