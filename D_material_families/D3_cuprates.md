## 1. 物理定义与核心思想

### 1.1 铜氧化物的结构定义

**[定义 — 铜氧化物超导体]**

```
铜氧化物超导体（Cuprate Superconductors）：
  定义：含 CuO₂ 平面的层状钙钛矿型（或类钙钛矿）化合物
  化学特征：
    - Cu²⁺（d⁹，自旋 S=1/2）+ 平面内 4 个 O²⁻
    - d_x²-y² 轨道主导（沿 Cu-O-Cu 方向）[定理，晶体场分析]
    - CuO₂ 平面被 "charge reservoir" 层夹住
      → charge reservoir 层：La/Ba/Sr/Hg/Tl/Bi 氧化物
      → 通过成分控制引入空穴或电子掺杂

  结构示意（层状堆叠，ASCII）：
    ─────────────── charge reservoir 层（例：BaO-HgO-BaO）
    ─ CuO₂ 平面 ─  ← 超导在此发生
    ─────────────── charge reservoir 层
    ─ CuO₂ 平面 ─
    ─────────────── charge reservoir 层
    （n = CuO₂ 层数，n=1,2,3 为常见，n↑ → Tc↑ 至 n=3 最优）

  Tc 范围（[定理，实验确认]）：
    常压块材：30K（LSCO）~ 134K（Hg-1223）
    高压：Hg-1223 达 164K [Chu 1994 Nature 365, 323]
    压力淬火（2026）：Hg-1223 151K [Deng 2026 PNAS DOI:10.1073/pnas.2536178123]
```

### 1.2 为什么 CuO₂ 平面是高 Tc 的核心？

**[观察 + 猜想 — 高 Tc 的来源]**

```
核心物理逻辑（[观察/猜想，40 年高置信度但非严格定理]）：

1. Cu²⁺ 的 d⁹ 构型（半满 d 壳层）：
   → 强 Coulomb 排斥 U >> 跳跃积分 t（Hubbard U/t >> 1）
   → 母态是 Mott-Hubbard 绝缘体（而非金属），与 BCS 前提矛盾 [定理，B14 关联]

2. 超交换 J（AFM 耦合）：
   J ≈ 4t²/U（超交换，来自 Anderson 1950/1959）[定理，二阶微扰]
   铜氧化物典型值：J ≈ 125–145 meV（LSCO, YBCO）
   [来源：Coldea 2001 PRL 86, 5377 — LSCO 中子散射 J ≈ 132±4 meV]
   对比：声子频率 ω_ph ≈ 20–80 meV（比 J 低 2-5 倍）
   → J 大 → 可能的强配对"胶水"（如果 J 驱动配对）[猜想，待验证]

3. CuO₂ 层数效应（[观察]）：
   n=1：LSCO Tc≈38K，NCCO Tc≈24K
   n=2：YBCO Tc≈92K，Bi-2212 Tc≈91K
   n=3：Hg-1223 Tc≈134K（**最优**）
   n=4+：Tc 开始下降（层间耦合效应？）[观察，原因待确认]
   → n=3 是"Goldilocks"层数：层间耦合足够强，但不过强

4. CuO₂ 平面内的 d_x²-y² 主导轨道 → d-wave 配对的几何基础 [定理，C3 关联]
   节点沿 (1,1,0) 和 (1,-1,0) 方向（= k_x±k_y = 0）
   → 与 AFM 超交换 Q=(π,π) 的方向 完全一致 → 两者耦合最强 [猜想，B3 关联]
```

### 1.3 掺杂相图的物理逻辑

**[定理/观察 — 铜氧化物最重要的物理结构]**

```
母态（x = 0，未掺杂）：[定理，实验确认]
  Mott 绝缘体 + 长程 AFM 序（Néel 温度 TN）
  LSCO：TN ≈ 325K [来源：Johnston 1989 Phys. Rev. B，来源待精确验证]
  YBCO（YBa₂Cu₃O₆）：TN ≈ 415K [来源待验证]
  → 注意：TN >> Tc，说明磁性比超导"更强"

空穴掺杂相图（主流体系，p 型）：[观察，来源：Lee-Nagaosa-Wen 2006 RMP 78, 17]
  ──────────────────────────────────────────────────────
  p=0      | Mott 绝缘体 | AFM（TN ~ 300K）
  0 < p < 0.05 | AFM 消退 | SC 开始出现？（争议区：AFM+SC 共存？）
  0.05 < p < 0.12 | 欠掺杂（underdoped）| AFM 消失，赝能隙 T* 出现
  p ≈ 0.12 | 欠掺杂-最优交叉 | SC 穹快速上升
  p ≈ 0.16 | **最优掺杂（optimal）** | **Tc 最高**
  0.16 < p < 0.27 | 过掺杂（overdoped）| Tc 下降，赝能隙消失，趋向 Fermi 液体
  p > 0.27 | 重度过掺杂 | 非超导金属（LSCO, Tl-2201 的过掺杂终点）
  ──────────────────────────────────────────────────────

电子掺杂相图（n 型，NCCO, PCCO）：[观察]
  与空穴掺杂不对称！
  → AFM 范围更宽（TN 在 n > 0 仍然很高）
  → SC 穹更窄，最优掺杂 n ≈ 0.14–0.17
  → T* 行为不同（欠掺杂端的 T* 更像 AFM 涨落而非赝能隙）
  → 配对对称性可能是 d-wave（但证据比空穴掺杂薄弱）[猜想]

  ⚠️ 反谄媚（重要）：
  任何声称"已理解铜氧化物相图"的理论
  必须同时解释 n 型（NCCO）和 p 型（YBCO/BSCCO）的不对称性
  → 大多数主流理论（RVB、AFM 涨落）在 n 型上预测有偏差 [观察]
```

## 2. 历史关键节点（1986-2026 铜氧化物发现史）

**[历史 — 铜氧化物超导从发现到 2026 年最新进展]**

```
1986 Bednorz & Müller（IBM 苏黎世）：
  发现 La₂₋ₓBaₓCuO₄（LBCO），Tc ≈ 35K（最初 ~30K，后确认 35K）
  [来源：Bednorz & Müller 1986 Z. Phys. B 64, 189]
  ★ Nobel 物理学奖 1987（史上最快 Nobel，发现到颁奖仅约 13 个月）
  重要性：
  - 打破了 A15 Nb₃Ge 23.2K 的 Tc 纪录
  - 首次证明氧化物可以超导（主流当时认为氧化物是绝缘体）
  - 引发全球铜氧化物研究热潮（论文数在 1987 年爆炸性增长）

1987 Wu, Ashburn, Torng, Hor, Meng, Gao, Huang, Wang, Chu（UT Houston/Alabama）：
  发现 YBa₂Cu₃O₇（YBCO），Tc = 92K — 首次突破液氮温区（77K）！
  [来源：Wu et al. 1987 PRL 58, 908]
  重要性：
  - 液氮（廉价，$0.5/升）替代液氦（昂贵，$5/升）→ 应用门槛大降
  - 92K >> 77K（液氮沸点），首次实现液氮冷却超导
  - 全球 50+ 实验室在 1987 年 2-3 月同时复现

1987 Anderson（普林斯顿）：
  提出 RVB 理论（Resonating Valence Bond）[来源：Anderson 1987 Science 235, 1196]
  → 掺杂 Mott 绝缘体中的预成 singlet 对 → 相位相干 → SC
  → 开创了非 BCS 强关联超导理论框架 [见 B9 综述]

1987 Maeda, Tanaka, Fukutomi, Asano（NEC, 日本）：
  发现 Bi₂Sr₂CaCu₂O₈（Bi-2212/BSCCO），Tc ≈ 85K
  [来源：Maeda et al. 1988 Jpn. J. Appl. Phys. 27, L209，来源待精确验证]
  重要性：
  - BSCCO 易于解理（层间范德华力弱）→ 成为 ARPES 和 STM 研究首选材料
  - 至今大量铜氧化物实验知识来自 BSCCO

1988 Sheng & Hermann（Houston）：
  Tl 系 Tl₂Ba₂Ca₂Cu₃O₁₀（Tl-2223），Tc ≈ 125K
  [来源：Sheng & Hermann 1988 Nature 332, 138，来源待精确验证]
  → 继续推高 Tc 上限，但 Tl 有毒，制备困难

1993 Schilling, Cantoni, Guo, Ott（ETH Zürich）：
  发现 HgBa₂Ca₂Cu₃O₈₊δ（Hg-1223），Tc = 134K — 常压铜氧化物 Tc 记录！
  [来源：Schilling et al. 1993 Nature 363, 56]
  → 至今（2026）仍是常压下铜氧化物（化学合成）的最高 Tc 记录（直至 2026）
  → Hg 有毒，制备要求极高真空封管

1993 Hardy, Bonn, Morgan, Liang, Zhang（UBC 不列颠哥伦比亚大学）：
  YBCO 穿透深度 λ(T) 线性温度依赖 → d-wave 节点确认！
  [来源：Hardy et al. 1993 PRL 70, 3999]
  λ(0) ≈ 150 nm，dλ/dT|T→0 ≈ 3.7 Å/K
  → 在 s-wave 中 dλ/dT → 0（指数激活），线性是节点的标志 [见 C10 关联]

1994 Tsuei, Kirtley, Chi, Yu-Jahnes, Gupta, Shaw, Sun, Ketchen（IBM Watson）：
  三角结（tricrystal）相位干涉实验 → d-wave 判决性证据！
  [来源：Tsuei et al. 1994 PRL 73, 593]
  设计：三个晶粒拼成特殊几何 → 如果 d-wave，中间节点处有 π 相移 → 产生 Φ₀/2 自发磁通
  测量：SQUID 扫描成像，在预测位置确认 Φ₀/2 磁通 [定理层级，实验无争议]
  → 从此 YBCO 的 d_x²-y² 波配对被视为"已确认"

1994 Chu, Gao, Chen, Huang, Meng, Xue（Houston/Science University of Hong Kong）：
  Hg-1223 在 ~30 GPa 高压下 Tc = 164K — 铜氧化物高压最高 Tc！
  [来源：Gao et al. 1994 Phys. Rev. B 50, 4260；Chu et al. 1993 Nature 365, 323]
  ⚠️ 注：部分引用为 Chu 1993，需核查确切来源
  → 164K 至今（2026）仍是铜氧化物绝对 Tc 最高记录（高压）

2003 Damascelli, Hussain, Shen（Stanford）：
  铜氧化物 ARPES 综述 [来源：Damascelli et al. 2003 RMP 75, 473]
  → 系统梳理 BSCCO/YBCO 的 Fermi 面、赝能隙、Fermi arc
  → 确立 ARPES 在铜氧化物研究中的核心地位

2006 Lee, Nagaosa, Wen（MIT/Stanford）：
  铜氧化物理论综述 [来源：Lee, Nagaosa, Wen 2006 RMP 78, 17]
  → 从 t-J 模型 + RVB + U(1)/Z₂ 规范场理论梳理铜氧化物物理
  → 至今是铜氧化物理论的权威综述

2026 Deng, 陈大凯等（中国）：
  Hg-1223 压力淬火（quench after pressure treatment），常压 Tc = 151K
  [来源：Deng et al. 2026 PNAS DOI:10.1073/pnas.2536178123]
  方法：在高压下稳定 Hg-1223 的结构，然后快速卸压（淬火）→ 保留高压态结构在常压
  重要性：
  - 常压稳定（不需要维持高压）Tc = 151K — 远高于此前常压记录 134K
  - 突破"压力淬火"技术的新路线
  - 机制尚不清楚：是晶格常数变化？是氧含量变化？是新型结构相？[观察，待研究]
  ⚠️ 数据确认：此为 2026 年最新结果，已发表于 PNAS；重现性和机制待进一步确认
```

## 4. 掺杂相图详解

### 4.1 空穴掺杂（p 型，主流）

**[观察 — 铜氧化物相图，核心物理结构]**

```
完整 p-type 相图（基于 Lee-Nagaosa-Wen 2006 + Keimer 2015 Nature 518, 179）：

  温度 T
  ▲
  | TN↗ AFM 绝缘体
  |  ╲
T*↗   ╲  赝能隙 T*（欠掺杂区）
  |    ╲╲
Tc dome↗ ╲╲╲
  |  SC 穹  ╲╲
  |         ╲╲
  ──────────────────────→ p（空穴掺杂浓度）
     0    0.05  0.16  0.27

  各区域物理特征：
  ─────────────────────────────────────────────────────
  p = 0             | Mott 绝缘体，AFM（TN ~ 300K，LSCO）
  0 < p < 0.05      | AFM → SC 过渡区（争议：AFM + SC 共存？互相排斥？）
  0.05 < p < 0.12   | 欠掺杂（UD）：赝能隙 T* 开始，Tc 上升
  p ≈ 0.12–0.16     | 欠掺杂-最优过渡：Tc 快速上升，赝能隙减弱
  p ≈ 0.16          | **最优掺杂（OP）**：Tc 最高（YBCO 92K，Hg-1223 134K）
  0.16 < p < 0.27   | 过掺杂（OD）：Tc 下降，赝能隙消失，趋向 Fermi 液体
  p > 0.27          | 非超导金属（LSCO, Tl-2201 测量终点）
  ─────────────────────────────────────────────────────

  关键温度标度（[观察，LSCO 数据]）：
  TN(p=0) ≈ 325K [来源：Johnston 1989，来源待验证]
  T*(p=0.10) ≈ 200–300K（赝能隙起始，随掺杂减小）[来源待验证]
  Tc(p=0.16) ≈ 38K（LSCO），92K（YBCO），134K（Hg-1223）
  T*(p=0.16) ≈ Tc 附近（赝能隙在最优掺杂消失或继续？争议！）

  重要争议（[猜想，未解决]）：
  (1) T* 线：是 SC 预成对（preformed pairs）的开始温度？
             还是竞争序（CDW/PDW）的能量标度？
             还是 Mott 物理残留的 crossover？
             → 三种解释各有支持者，没有判决性实验 [猜想层级]
  (2) QCP 问题：p ≈ 0.19 处是否有隐藏量子临界点（QCP）？
             → 连接 strange metal 行为（ρ ∝ T）的来源 [猜想]
  (3) AFM + SC 共存：p < 0.05 时是真共存还是空间相分离？[争议]
```

### 4.2 电子掺杂（n 型）的不对称性

**[观察 — 重要但常被忽视的相图不对称性]**

```
n 型（NCCO Nd₂₋ₓCeₓCuO₄，PCCO Pr₂₋ₓCeₓCuO₄）相图特征：[观察]

  (1) AFM 范围更宽：TN 在 n ≈ 0.12 附近才消失（比 p 型在 p ≈ 0.05 消失晚得多）
  (2) SC 穹更窄：最优掺杂 n ≈ 0.14–0.17，SC 穹宽度比 p 型窄约 1.5×
  (3) SC 与 AFM 的共存区域更宽（争议）
  (4) T* 行为不同：n 型的赝能隙特征更接近 AFM 涨落，而非 p 型的复杂竞争序

  ⚠️ 反谄媚（核心）：
  n 型和 p 型相图的不对称性是任何完整理论必须解释的约束
  → RVB 框架（B9）主要为 p 型设计，对 n 型的解释有困难 [猜想]
  → AFM 涨落框架（B3）对 n 型的定量预测与实验有一定偏差 [来源待验证]
  → 没有一个理论完美解释两种掺杂的不对称性
```

## 5. 多体物理特征（铜氧化物的"谜题"集合）

### 5.1 赝能隙（Pseudogap）

**[观察 — H3 关联]**

```
赝能隙现象（[观察，实验确认，起源是猜想]）：
  在 T* > Tc 时，欠掺杂铜氧化物在反节点方向 ((π,0) 附近) 出现部分能隙
  → ARPES 测量：反节点谱函数 A(k,ω) 在费米能级处被压低
  → 在节点方向 ((π,π)/2 附近) 能隙不存在 → Fermi arc（见下节）

  典型实验数据（BSCCO 欠掺杂，T ≈ 0.7Tc）：
  赝能隙大小 Δ_PG ≈ 50–100 meV（比 SC 能隙 Δ_SC ≈ 25–30 meV 更大）
  [来源：Damascelli 2003 RMP 75, 473 及引用文献，来源待精确核查]

  赝能隙的竞争解释（[猜想层级，无判决性实验]）：
  A. 预成对（Preformed pairs）：SC 配对在 T* 就形成，T < Tc 才相位相干
     支持证据：Nernst 效应在 T* 以上仍有涡旋信号 [Xu 2000 Nature，来源待验证]
     反例：比热没有显示 T* 处有配对能量的大量消失 [来源待验证]

  B. 竞争序（Competing order）：赝能隙来自 CDW/SDW/PDW 等竞争序
     支持证据：欠掺杂区有 CDW（X 射线/中子散射确认）[Keimer 2015，来源待验证]
     反例：CDW 的有序温度与 T* 并不完全匹配 [来源待验证]

  C. Mottness 残留：赝能隙是 Mott 绝缘体的高能量尺度 J 在掺杂态的残留
     支持证据：T* 与 J 的量级一致 [观察]
     反例：难以与 Fermi arc 的低能结构调和 [来源待验证]

  [判决：无。三种解释都有实验支持和反例。]
```

### 5.2 Fermi Arc

**[观察 — H4 关联]**

```
Fermi arc 现象（[观察，ARPES 确认]）：
  在欠掺杂区 T > Tc，ARPES 显示：
  - 节点方向 ((π/2,π/2))：保留正常金属态的 Fermi 面片段（arc）
  - 反节点方向 ((π,0))：打开赝能隙，没有 Fermi 面

  温度依赖（[观察]）：
  T >> T*：完整 Fermi 面（圆弧连接成完整 Fermi 面圆）
  T* > T > Tc：Fermi arc（arc 长度随 T 减小而收缩）
  T < Tc：arc 上的节点开放 → d-wave SC 节点（arc 消失，d-wave 能隙在节点方向 = 0）

  理论诠释（[猜想，争议]）：
  解释 A：Fermi arc 是赝能隙打开后残留的 Fermi 面片段 → 即"真实 arc"
  解释 B：Fermi arc 是完整 Fermi 口袋（pocket）的背面，因矩阵元为零而看不见
           → 即"幽灵 Fermi 面"[猜想，量子振荡实验部分支持]
  量子振荡实验（[观察]）：
  欠掺杂 YBCO 的 Shubnikov-de Haas 振荡显示 Fermi 口袋（而非大 Fermi 面）
  → 支持 AFM 或 CDW 折叠的小 Fermi 口袋解释 [Doiron-Leyraud 2007 Nature，来源待验证]
  → 但这与 ARPES 看到的 Fermi arc 的协调仍有争议 [观察]
```

### 5.3 Strange Metal / Linear-T 电阻率

**[观察 — A8, H5 关联]**

```
Strange metal 特征（[观察，广泛确认]）：
  最优掺杂附近（p ≈ 0.16）：
  ρ(T) ≈ ρ₀ + A·T，在极宽温度范围（Tc ~ 800K）线性 [观察]
  [来源：Gurvitch & Fiory 1987 PRL 59, 1337 for YBCO，来源待验证]

  Planckian 散射（[猜想，有实验支持]）：
  1/τ ≈ α · k_B T / ℏ，α ≈ 1（Planckian 极限）
  → 散射速率达到量子力学允许的最大值（Mott-Ioffe-Regel 极限附近）

  重要性：
  - 这是 BCS/Fermi 液体理论的完全失效：FL 预测 ρ ∝ T²（Fermi liquid）
  - strange metal 是铜氧化物最难解的特征之一
  - 与 Paper A 框架（Eliashberg）：完全超出框架范围 [定理]
```

```
铜氧化物的配对机制在 2026 年仍然没有被解决。
以下每一个理论框架都有：
```

### 6.1 RVB（B9 关联）

```
Anderson RVB（Resonating Valence Bond）[猜想]：
  提出：Anderson 1987 Science 235, 1196
  核心思想：
  → 未掺杂 Cu²⁺（S=1/2）自旋 = 二维反铁磁 Heisenberg 模型
  → 自旋单重态（singlet）可以"共振"（RVB liquid），不打破平移对称性
  → 掺杂 → 注入空穴（holons）→ 破坏 RVB 液态的静态 AFM
  → 自旋单重态的相位相干 → SC

  理论发展：
  → Lee-Nagaosa-Wen 2006 RMP 78, 17（t-J 模型 + U(1)/Z₂ 规范场框架）
  → 预测：d-wave 配对自然涌现（[猜想，数值一致]）
  → 预测：Fermi arc（赝能隙作为"spinon 谱"）[猜想，与 ARPES 定性一致]

  支持证据：
  - d-wave 配对（定性）：RVB 自然给出 d_x²-y² 能隙 [数值计算支持，猜想层级]
  - 掺杂相图的定性描述（AFM → SC → 金属）[猜想层级]
  - 强关联 → 需要超越 BCS 框架：这是事实 [定理，Mottness 关联]

  失败和反例：
  - 定量失败：t-J 模型数值计算（大规模量子 Monte Carlo）给出的 Tc 和能隙大小
    与实验仍有定量差距 [来源待验证]
  - n 型相图：RVB 对电子掺杂的预测不如 p 型准确 [观察]
  - BCS 能隙比 2Δ/kTc（铜氧化物约 5–8）远高于弱耦合 3.52：
    RVB 可以给出高配对，但定量仍不精确 [猜想]
  - 在最过掺杂端（趋向 Fermi 液体）：RVB 框架如何平滑过渡？[争议]
```

### 6.2 AFM 自旋涨落（B3 关联）

```
AFM 自旋涨落介导 d-wave 配对 [猜想]：
  核心：Berk-Schrieffer（1966）+ Scalapino（1986）+ Pines 团队
  → Q=(π,π) 处的自旋涨落（反铁磁涨落）在费米面附近引发有效吸引
  → 有效相互作用 V(q,ω) 在 q=(π,π) 时最强
  → d_x²-y² 的节点方向 = (1,1,0) 与 Q=(π,π) 完美配合 → 最强配对

  理论：t-J 模型 + 自旋涨落交换 →
    Δ(k) ∝ cos(k_x) - cos(k_y)（d_x²-y² 形式）[猜想，数值计算支持]
    [来源：Monthoux-Pines 1992 PRL 69, 961，来源待验证]

  支持证据：
  - d-wave 配对形式与自旋涨落 Q 向量的几何关系 [观察/猜想]
  - 中子散射中的磁共振峰（resonance peak）：在 Tc 以下在 (π,π)/(π,0) 附近出现
    [来源：Rossat-Mignod 1991 for YBCO，来源待验证]
    → 解释为 SC 能隙对 Q=(π,π) 涨落的反馈 [猜想]
  - 铁基超导（D4）的 s± 配对也是 AFM 涨落框架的成功案例（但 q 不同）[见 D4 综述]

  失败和反例：
  - Tc 定量：AFM 涨落模型给出的 Tc 依赖于自旋涨落谱的细节，定量结果有大范围散射
    [来源待验证，数值计算综述：Scalapino 2012 RMP 84, 1383]
  - 强关联失效：铜氧化物是强关联体系，微扰论（散射展开）对 J ~ t 的情况失效
    → AFM 涨落框架本质上是准粒子微扰论，但铜氧化物准粒子概念本身就不明确 [定理，B14 关联]
  - 过掺杂端：AFM 涨落随过掺杂减弱，但 SC 还维持 → 两者的"胶水"可能不同 [观察]
```

### 6.3 其他机制

```
其他竞争理论（均为 [猜想] 层级）：

声子耦合（弱残留）：
  铜氧化物中有确定的声子模式（ARPES 测量的 "kink" 结构在 40–70 meV）
  [来源：Lanzara 2001 Nature 412, 510]
  但声子耦合强度（λ_ph ≈ 0.2–0.5）太小，无法单独解释 Tc ≈ 90K [观察]
  → 声子可能是"助攻"（减弱赝能隙效应？），但不是"主力" [猜想]

量子临界涨落（QCF）：
  p ≈ 0.19 处可能有隐藏 QCP → 量子临界涨落 → strange metal + SC [猜想]
  → 与 AdS/CFT 全息对偶框架连接 [猜想，理论模型，非铜氧化物特有]

Bi 层界面声子（FeSe/STO 类比）：
  铜氧化物中的 charge reservoir 层声子是否类似 FeSe/SrTiO₃ 中的界面声子？[猜想]
  → 目前无强力证据 [来源待验证]

激子机制（B7）：
  铜氧化物中激子是否可能？→ 强 Coulomb 屏蔽 → 激子不稳定，证据极弱 [猜想，失败]

关键困难总结（[定理/观察]）：
  (1) 铜氧化物是强关联体系：微扰理论失效
  (2) 精确数值方法（大规模 quantum Monte Carlo）计算 Hubbard/t-J 模型 → 
      费米符号问题（sign problem）使 2D 强关联计算极难 [定理]
  (3) DMFT（Dynamical Mean Field Theory）：可以处理强 Coulomb，但忽略空间关联
  → 以上困难导致：没有任何方法可以从第一性原理精确计算铜氧化物的 Tc
```

## 7. 与 Paper A 框架的边界（为何铜氧化物超出 Eliashberg 框架）

**[定理 — Paper A 框架的适用边界]**

```
Paper A 框架假设（多通道 Eliashberg，见 B2/B12）：
  ✅ 可辨认的准粒子（Fermi liquid 或准 FL）
  ✅ 声子或可准粒子化的玻色涨落介导配对
  ✅ 弱到中等耦合（λ ≲ 2，Migdal 定理有效）

铜氧化物的每一条都违反：

(1) 准粒子破坏（[定理，B14 Mottness 关联]）：
    母态是 Mott 绝缘体，掺杂后 Mottness 残留
    → 下 Hubbard 带与上 Hubbard 带分离 → 单粒子谱函数非 Fermi 液体
    → ARPES 显示：铜氧化物的准粒子权重 Z < 0.1（远小于金属的 Z ≈ 1）
    [来源：Damascelli 2003 RMP，来源待验证]
    → Fermi 液体基本假设失效 [定理]

(2) Strange metal（非准粒子行为）：
    ρ ∝ T（全温度范围）违反 Fermi 液体的 ρ ∝ T² [观察]
    → 表明不存在长寿命准粒子 → Eliashberg 框架失效 [定理，Paper A 边界]

(3) 配对机制非声子（[猜想，高置信度]）：
    J ≈ 130 meV（超交换）远大于声子频率 ω_ph ≈ 30–70 meV
    → 配对"胶水"更可能是 J（磁性），而非声子
    → 即使有声子贡献，λ_ph 也不足以解释 Tc ≈ 90K
    → Eliashberg 的 λ 概念（声子耦合）在此意义不明 [猜想]

结论（[定理，Paper A 适用性边界]）：
  铜氧化物是 Paper A（多通道 Eliashberg 框架）的 **对照组** 和 **边界定义材料**
  → Paper A 对铜氧化物不适用
  → 铜氧化物需要超越准粒子框架的全新理论（RVB 规范场？AdS/CFT？）
  → 这不是 Paper A 的缺陷，而是理论的适用边界：明确边界是科学严谨的一部分

Paper E 轨道选择性（C3 关联）：
  铜氧化物的 d_x²-y² 轨道主导是"强轨道选择"的典范
  → 但这是 Mott 物理的自然结果（d_x²-y² 与 O 的 σ 键最强，Mott 绝缘体中首先 Mott 化）
  → Paper E 框架（轨道选择性多通道 Eliashberg）也不适用于强 Mott 材料：
    需要从 t-J 模型出发，而非 Eliashberg
```

## 9. 关键反谄媚

### 9.1 配对机制 40 年争议：没有答案，这是科学事实

```
[观察/定理 — 必须诚实陈述的科学现实]

1986 年至今（2026 年），全球数以千计的物理学家投入铜氧化物研究
发表论文超过 10 万篇（Web of Science 数据，来源待精确核查）
然而，铜氧化物超导的配对机制依然是未解谜题。

主要"竞争"理论的核心问题：
  RVB（Anderson）：数学框架优美，但 Tc 定量计算困难，与 n 型相图不完全一致
  AFM 涨落（Scalapino-Pines）：解释 d-wave 对称性成功，但强关联失效问题未解
  CDW/PDW（Fradkin-Kivelson）：可以解释竞争序，但能否驱动高 Tc？不清楚
  声子（残留）：贡献可能存在但肯定不是主导 [猜想，高置信度]
  量子临界（Sachdev 等）：数学框架深刻，但从"第一性原理"到 Tc 的桥梁未建立

  "最强共识"（[观察，非定理]）：
  ① d-wave 配对确认（C3）——这是少数确认的"定理级"实验结果
  ② 磁性（AFM/RVB）在配对中有重要角色——但程度和机制不清楚
  ③ BCS 声子机制不能单独解释高 Tc——这也是共识
  ④ 其他一切都是争议

结论：任何声称"已知道铜氧化物配对机制"的论文
      必须给出：新的判决性实验 + 同时解释 n 型和 p 型相图的 + 从第一性原理定量算出 Tc
      满足这三条的论文在 2026 年不存在。
```

### 9.2 Tc 134K/164K/151K 的意义边界

```
[观察 — 数字的真实意义]

"134K 是常温超导的 45%"（300K 的 45%）：
  这是事实，但这意味着从 134K 到 300K 仍有巨大的物理鸿沟
  没有理论预测铜氧化物体系在常压下能超过 200K

"164K 高压记录"：
  需要 ~30 GPa（相当于地球下地幔压力）
  这在技术上使用金刚石对顶砧（DAC），不能工程应用
  → 与氢化物（D6）高压记录类似，均非工程可用

"2026 Deng 151K 压力淬火"：
  这是重要的科学突破（突破常压记录约 17K）
  但机制尚不清楚 → 是否可控合成？是否可重现？是否可规模化？
  2026 年 4 月，这仍是早期结果，需要更多独立重复实验
  ⚠️ PREDICTION vs RETRODICTION 标注：
    151K 是实验测量值（已知数据），任何理论解释均为 RETRODICTION

高 Tc 铜氧化物的实用困难（[观察]）：
  Hg/Tl 有毒 → 制备需要特殊安全措施
  YBCO 薄膜带材：制备成本高（约 $100–200/kA·m，来源待精确）
  BSCCO 导线：Bi-2223 银包套带材，Jc 较低，机械强度差
  铜氧化物各向异性（ξ_c < c）→ 晶界弱链（weak links）→ 高 Jc 技术困难
```

### 9.3 镍酸盐不是"第二个铜氧化物"（尚未确认）

```
[猜想 — D5 边界声明]

2023 年镍酸盐（La₃Ni₂O₇）高压 Tc ≈ 80K 引发关注
有人称镍酸盐是"类铜氧化物"的新体系

  镍（d⁸）而非铜（d⁹）→ 自旋/轨道结构不同（Ni: 两轨道 d_x²-y² + d_z²）
  → 是 singlet + triplet 混合？还是 s±？目前不清楚 [猜想]
  配对对称性未确认（2026 年仍有争议）
→ 在 D5 综述中详细处理；D3 边界：铜氧化物 ≠ 镍酸盐
```

## 11. 与 MECE 体系的关联分析

```
C3（d-wave 配对）：
  铜氧化物是 d_x²-y² 配对的 **唯一判决性确认体系**
  → Hardy 1993 λ(T) 线性 + Tsuei-Kirtley 1994 三角结 → d-wave 定理化 [定理]
  → D3 是 C3 的核心材料基础；C3 是 D3 最确定的实验结论

B9（RVB）：
  铜氧化物是 RVB 理论的 **唯一实验对象**（至今）
  → Anderson 1987 提出 RVB 就是为了解释 La-BCUO
  → RVB 预测 d-wave（与实验一致），但定量 Tc 计算困难 [猜想层级]

B14（Mottness）：
  铜氧化物母态（x=0）是教科书级 Mott 绝缘体的完美案例
  → ZSA 相图（Zaanen-Sawatzky-Allen）：铜氧化物是 "charge-transfer insulator"
  → 空穴掺杂 → 空穴主要在 O 2p 轨道上（不在 Cu d 上）[观察，ZSA 理论支持]
  → 正确理解铜氧化物 → 必须从 charge-transfer Mott 物理出发 [定理，B14 框架]

B3（AFM 涨落）：
  铜氧化物 Q=(π,π) AFM 涨落 + d_x²-y² 几何一致性是 B3 框架的强动机
  但定量计算在强关联区失效 → B3 给出 D3 的 d-wave 解释，但 Tc 定量差距仍在 [猜想]

H3（赝能隙），H4（Fermi arc），H5（Strange metal）：
  铜氧化物是 H3/H4/H5 的 **核心现象来源**
  → 这三个反常正常态特征都首先（也最清晰地）在铜氧化物中发现和确立
  → D3 与 H3/H4/H5 是双向定义关系：铜氧化物定义这些现象，这些现象定义铜氧化物

A8（Linear-T 电阻率）：
  铜氧化物最优掺杂 ρ ∝ T 是 strange metal 的旗帜性特征
  → Paper A（Eliashberg 框架）给出的 ρ ∝ T 预测在铜氧化物失效 → 定义 Paper A 边界

E2（化学掺杂）：
  铜氧化物相图 = 化学掺杂（E2）的最复杂案例
  → p 型（La/Sr 替换，YBCO 氧含量控制）vs n 型（Ce 替换 Nd）
  → 掌握铜氧化物相图 = 掌握 E2 中最难、最富物理的实例

E1（压力）：
  Hg-1223 在压力下 Tc 从 134K→164K，压力淬火 151K
  → 铜氧化物是 E1 中压力效应最显著、最有应用价值的材料类
  → 压力淬火（Deng 2026）是 E1 的新前沿路线

D1 vs D2 vs D3 的层级比较：
  D1（元素金属）：BCS 声子，Tc ≤ 10K（常压），理论完整
  D2（A15 化合物）：BCS 声子强耦合，Tc ≤ 23K（常压薄膜），Eliashberg 框架有效
  D3（铜氧化物）：非 BCS，强关联，Tc ≤ 134K（常压），配对机制未解
  → 三者形成 Tc 阶梯：10K → 23K → 134K，物理图像从简单到复杂
  → D3 是 Eliashberg 框架（B1-B2）失效的清晰边界材料
```
