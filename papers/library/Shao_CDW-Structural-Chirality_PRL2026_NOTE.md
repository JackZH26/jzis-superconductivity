## 核心内容（基于 Crossref 元数据）

在 kagome 超导体 AV₃Sb₅（A = Cs/K/Rb）体系中，多层 CDW（电荷密度波）之间的**层间相位关系诱导了结构手征性**。

## 关键概念

- **CDW 矢量相（vector phase）：** 不同层之间的 CDW 相位偏移量
- **结构手征性（Structural Chirality）：** CDW 相位形成特定模式 → 打破镜面对称 → 出现手征结构
- 这与 Frieze 条纹（我们之前看的 CsCr₃Sb₅ paper）形成有趣对比——那里是实空间 STM 观测，这里可能是 X 射线/中子衍射确认

## 与我们研究的关联

- AV₃Sb₅ kagome 超导体系：Kagome 的 CDW 竞争超导是我们数据库研究中的重要反例（Night 16 R framework 的 Kagome 灾难）
- 多层 CDW 相位 → 手征性 → 可能与 Josephson 效应或非常规配对对称性有关
- 与 Frieze CsCr₃Sb₅ 论文形成系列：CDW 的空间对称性在 kagome 家族中越来越复杂
