## 1. 物理定义与核心思想

### 1.1 Mermin-Wagner 定理：2D 超导为何不能有真长程序（[定理，1966]）

```
=== Mermin-Wagner 定理与 2D 超导的基本困境 [定理，Mermin & Wagner 1966] ===

定理陈述（[定理，严格证明]）：
  在有限温度 T > 0 下，具有连续对称性（如 U(1)）的二维系统
  不能发生自发对称性破缺（SSB）。

对超导的含义：
  2D 超导序参量：ψ(r) = |ψ| * exp(iφ(r))
  → 真长程序（LRO）要求：⟨ψ⟩ ≠ 0（全局 U(1) 对称性自发破缺）
  → Mermin-Wagner 定理：在 T > 0 时，⟨ψ⟩ = 0（禁止 LRO！）
  → 关联函数：⟨ψ*(r) ψ(0)⟩ 不能在 |r|→∞ 趋近非零常数

物理机制（[定理，Goldstone 相位涨落]）：
  → 连续对称群有 Goldstone 模（相位涨落 δφ）
  → 2D 中 Goldstone 模的热涨落积分对数发散：⟨(δφ)²⟩ ~ ln(L/ξ)（红外发散）
  → δφ 涨落在热力学极限中破坏 ⟨ψ⟩ ≠ 0 的可能性

BKT 的突破（[定理，Kosterlitz & Thouless 1973]）：
  → 2D 超导仍可以有"准长程序"（QLRO, quasi-long-range order）
  → QLRO：关联函数幂律衰减（不是指数衰减）
    ⟨ψ*(r) ψ(0)⟩ ~ |r|^{-η(T)}（幂律，η > 0）
  → 幂律衰减 = 超流序仍存在，但缺少严格的 LRO
  → 关键物理：QLRO 由漩涡-反漩涡对的束缚状态维持

关键区分（反谄媚！）：
  ⚠️ T < T_BKT：QLRO（准长程序，幂律衰减）→ 超流序 ✓
  ⚠️ T > T_BKT：指数衰减（真正无序态）→ 超流序消失 ✗
  ⚠️ 不存在 T_c（真正的对称性破缺相变），只有 T_BKT（拓扑相变）！
```

### 1.2 漩涡（Vortex）的拓扑性质（[定理]）

```
=== 超导漩涡作为拓扑缺陷 [定理] ===

超导序参量的相位表示：
  ψ(r) = |ψ(r)| * exp(iφ(r))
  φ(r) = 序参量相位场（实数标量场）

漩涡（涡旋）的定义（[定理，拓扑缺陷]）：
  漩涡 = 相位 φ 绕某点 r₀ 绕行一圈后转过 2πn 的拓扑缺陷
    ∮_C ∇φ · dl = 2πn（n = 漩涡绕转数，拓扑荷）
  n = +1：正漩涡（顺时针相位旋转）
  n = -1：反漩涡（逆时针相位旋转）
  n = 0：无漩涡（相位连续）

漩涡的能量（[定理，弹性介质类比]）：
  梯度能量密度：f_grad = (ρ_s/2) * (∇φ)²
  漩涡核心外的相位场：|∇φ| ~ 1/r（n=1 时，r = 距漩涡中心距离）
  漩涡总能量（[定理，对数散度]）：

    LaTeX: E_v \approx \pi \rho_s \ln\!\frac{L}{\xi}

    ASCII: E_v ≈ π * ρ_s * ln(L/ξ)

    其中：
      ρ_s = 超流刚度（能量单位，J 或 eV）
      L   = 系统线度（UV 截断 → IR 截断）
      ξ   = 超导相干长度（漩涡核心半径，UV 截断）
    ⚠️ 关键：能量随 ln(L) 发散 → 无限大系统中单个漩涡能量无穷大！

漩涡的熵（[定理，组合统计]）：
  漩涡核心可以出现在 (L/ξ)² 个独立格点位置
  漩涡熵：

    LaTeX: S_v = 2 k_B \ln\!\frac{L}{\xi}

    ASCII: S_v ≈ 2 * k_B * ln(L/ξ)

    物理：一个漩涡的"放置方式数" = (L/ξ)²，取对数乘 k_B 得熵
    注意：系数 2 来自二维面积的对数（ln[(L/ξ)²] = 2 ln[L/ξ]）

漩涡对（Vortex-Antivortex Pair）的结构：
  → 孤立漩涡能量无穷大，但漩涡对总能量有限！
    E_pair ~ 2π * ρ_s * ln(r₁₂/ξ)（r₁₂ = 漩涡对间距）
  → 漩涡对在 T = 0 时自然形成（零点涨落），相互束缚（对数势）
  → 相位涨落仅由束缚漩涡对维持 → QLRO 依然成立
```

### 1.3 Kosterlitz-Thouless 自由能论证：相变的来源（[定理]）

```
=== KT 论证：漩涡的自由能分析 [定理，Kosterlitz & Thouless 1973] ===

单个漩涡的热力学自由能：

  F = E_v - T * S_v
    = π * ρ_s * ln(L/ξ) - T * 2k_B * ln(L/ξ)
    = (π * ρ_s - 2k_B * T) * ln(L/ξ)

相变判据（两个区间）：

  T < T_BKT（漩涡束缚）：
    π * ρ_s > 2k_B * T → F > 0（漩涡自由能为正）
    → 自由漩涡在热力学上不利 → 漩涡成对束缚 → QLRO 维持
    → 超流序 ✓（幂律衰减关联函数）

  T > T_BKT（漩涡解束缚）：
    π * ρ_s < 2k_B * T → F < 0（漩涡自由能为负！）
    → 自由漩涡在热力学上有利 → 漩涡对解束缚 → 自由漩涡增殖
    → 自由漩涡破坏相位相干 → 超流序 ✗（指数衰减关联函数）

T_BKT 的自洽方程（[定理]）：

  相变点：π * ρ_s(T_BKT) = 2k_B * T_BKT

  整理得（Nelson-Kosterlitz 关系，见第 4 节）：

    LaTeX: k_B T_{\mathrm{BKT}} = \frac{\pi}{2}\, \rho_s(T_{\mathrm{BKT}})

    ASCII: k_B * T_BKT = (π/2) * ρ_s(T_BKT)

  ⚠️ 注意：这是自洽方程，因为 ρ_s 本身也依赖于 T！
     正确理解：T_BKT 是满足上述等式的温度，不是独立确定的数值。

KT 相变的拓扑本质（[定理]）：
  → BKT 不是普通的对称性破缺相变（Landau-Ginzburg 范式）
  → 是拓扑相变：以漩涡-反漩涡对的束缚↔解束缚为序参量
  → 序参量 = 漩涡对的"解束缚密度"（或等效地，超流刚度的跳变）
  → BKT 是无穷阶相变：所有热力学量在 T_BKT 处的导数均连续！
     → 比热无发散（区别于 BCS 二阶相变的比热跳变）
```

## 3. BKT 的精确数学

### 3.1 超流刚度的 Nelson-Kosterlitz 普适跳变（[定理]）

```
=== Nelson-Kosterlitz 跳变：超流刚度的普适不连续性 [定理] ===

T < T_BKT（漩涡束缚相）：
  ρ_s(T) = 有限值（随温度升高缓慢下降）
  QLRO 超流序存在（幂律关联）
  漩涡对相互束缚，贡献相位涨落但不破坏超流序

T → T_BKT^-（从下方趋近，漩涡对刚好束缚）：
  ρ_s 趋近普适极限值（Nelson-Kosterlitz 跳变值）：

    LaTeX: \rho_s(T_{\mathrm{BKT}}^-) = \frac{2}{\pi}\, k_B\, T_{\mathrm{BKT}}

    ASCII: ρ_s(T_BKT^-) = (2/π) * k_B * T_BKT

T → T_BKT^+（从上方趋近，漩涡刚解束缚）：
  ρ_s → 0（突然跳变到零！）
  → 自由漩涡大量出现，完全破坏超流序

跳变的普适性（[定理，核心结论]）：

  Δρ_s = ρ_s(T_BKT^-) - ρ_s(T_BKT^+)
        = (2/π) * k_B * T_BKT - 0
        = (2/π) * k_B * T_BKT

  ⚠️ 普适性含义：
    ① 跳变量 (2/π) * k_B * T_BKT 不依赖具体材料（铜氧化物、NbSe₂、He-4 均相同）
    ② 跳变量只由 T_BKT 决定 → 实验上，测 T_BKT 就能预测跳变量！
    ③ 但：实验测量的跳变可能因有限尺寸效应、无序、样品质量而偏离（见第 9 节反谄媚）

BKT 相变的重正化群流（[定理，Kosterlitz 1974，来源待验证]）：
  两方程描述漩涡密度 y 和耦合常数 K = πρ_s/(2k_BT) 的 RG 流：

    dK/dl = -4π³ * y² * K²（漩涡屏蔽 ρ_s）
    dy/dl = (2 - πK) * y（漩涡密度 RG 方程）

  不动点分析：
    K > 2/π（T < T_BKT）：y → 0（漩涡密度流向零，束缚相）
    K < 2/π（T > T_BKT）：y → ∞（漩涡密度发散，无序相）
    K = 2/π（T = T_BKT）：BKT 临界不动点（y = 0）
  
  → 验证：T_BKT 条件 K = 2/π ↔ πρ_s/(2k_BT) = 2/π ↔ k_B*T = (π/2)*ρ_s（一致）
```

### 3.2 I-V 特征的幂律行为（[定理，Halperin-Nelson]）

```
=== I-V 特征：BKT 相变的电学指纹 [定理，Halperin & Nelson 1979] ===

T < T_BKT（超导幂律，束缚漩涡贡献）：

  LaTeX: V \propto I^{\alpha(T)},\quad \alpha(T) > 3

  ASCII: V ∝ I^α, α(T) > 3（超导幂律）

  物理：束缚漩涡对在电流驱动下有复杂的非线性响应（不解束缚时 α >> 1）
  T → 0 时：α → ∞（完美超导，V→0）

T = T_BKT（精确立方律，判决性！）：

  LaTeX: V \propto I^3\quad (\alpha = 3)

  ASCII: V ∝ I^3（精确立方，BKT 判据）

  物理：T_BKT 处漩涡对刚开始解束缚，非线性响应恰好为立方
  ⚠️ 这是 BKT 最有力的实验判据——α = 3 是拓扑相变的精确预测！

T > T_BKT（线性趋向正常态）：

  LaTeX: V \propto I^1\quad (\alpha \to 1)

  ASCII: V ∝ I^α, α → 1（线性，欧姆响应）

  物理：自由漩涡产生线性 I-V（欧姆电阻）

实验提取 T_BKT 的流程：
  ① 测量不同温度下的 I-V 曲线
  ② 在 log V vs log I 坐标中作图 → 提取斜率 α(T)
  ③ 画 α(T) vs T 曲线 → 寻找 α = 3 的温度
  ④ 该温度即 T_BKT

⚠️ 实际困难（反谄媚）：
  → I-V 曲线受焦耳加热（Joule heating）影响：大电流加热样品 → 假 α 值
  → 无序材料：漩涡被钉扎（pinning）→ 幂律偏离，α(T) 弥散
  → 有限系统：边界效应改变幂律
  → α = 3 的精确性受多种非 BKT 效应干扰，须谨慎解释
```

### 3.3 电阻 R(T) 的 BKT 函数形式（[定理]）

```
=== R(T) 的 BKT 特征函数 [定理，Halperin-Nelson 1979，来源待验证精确推导] ===

在 T_BKT 上方（T → T_BKT^+），电阻 R(T) 具有特征函数形式：

  LaTeX: R(T) \propto \exp\!\left(-b\,\sqrt{\frac{T_{\mathrm{BKT}}}{T - T_{\mathrm{BKT}}}}\right)

  ASCII: R(T) ∝ exp(-b * sqrt(T_BKT / (T - T_BKT)))

  参数：
    b   = 无量纲常数（依赖材料，通常 b ≈ 1-2，须从拟合获得）
    T_BKT = BKT 转变温度（唯一物理参数，从拟合提取）

  物理起源：
    → 漩涡密度 y(T) 在 T_BKT 附近以 exp(-const/√(T-T_BKT)) 方式增长（RG 流预测）
    → 电阻 R ∝ 自由漩涡密度 ∝ y(T) → 得到上述函数形式

与 BCS 函数形式的对比（[定理，区分性！]）：

  BCS（3D 激活能隙）：R(T) ∝ exp(-Δ/(k_B * T))（Arrhenius 形式，线性 log R vs 1/T）
  BKT（漩涡解束缚）：R(T) ∝ exp(-b/√(T-T_BKT))（非 Arrhenius，不是线性！）

  区分方法：
    → 在 ln R vs 1/(T - T_BKT)^{1/2} 坐标系中，BKT 给出直线
    → 在 ln R vs 1/T 坐标系中，BKT 给出弯曲曲线
    → 实验上：通过数据拟合区分哪种函数形式更好

⚠️ 注意：BKT R(T) 函数形式仅在 T_BKT 附近（有限温度区间）成立
   → 远离 T_BKT：其他效应（正常态电阻、配对涨落）主导
   → 需要选择合适拟合窗口，结果依赖窗口选择
```

## 4. BKT 转变的实验识别（实操指南）

```
=== 三个判据：如何从实验数据确认 BKT ===

========================================
判据 1：I-V 特征（最广泛使用，较易实现）
========================================

实验步骤：
  ① 制备高质量 2D 超导薄膜或界面样品
  ② 四端法测量不同温度 T 下的 I-V 曲线（宽电流范围，多个数量级）
  ③ 在 log-log 坐标中分析 I-V 曲线
  ④ 提取幂律指数 α(T)（V ∝ I^α）
  ⑤ 作 α vs T 图，寻找 α = 3 处的温度 → T_BKT

BKT 确认信号：
  ✓ α(T) 从低温（α >> 3）随温度升高连续降低
  ✓ 在某温度 T_BKT 附近，α = 3 精确成立
  ✓ 高于 T_BKT，α → 1（欧姆响应）

⚠️ 困难与陷阱：
  - 加热效应：大电流下样品温度升高 → 假幂律
    缓解：使用短脉冲电流、低占空比测量
  - 无序钉扎：α(T) 曲线展宽，α = 3 处不精确
  - 有限样品尺寸：边界效应修正幂律
  - 多值拟合：同一 I-V 数据有时可用不同 α 拟合

========================================
判据 2：R(T) 的 BKT 函数拟合（与 BCS 区分）
========================================

实验步骤：
  ① 精确测量四端电阻 R vs T（10 mK 精度，宽温度范围）
  ② 在 T_BKT 附近，拟合 BKT 函数：R = A * exp(-b/√(T-T_BKT))
  ③ 提取 T_BKT 和 b 参数
  ④ 比较：BCS（exp(-Δ/kT)）vs BKT 函数形式哪个拟合质量更好

BKT 确认信号：
  ✓ R(T) 在 T_BKT 附近的 BKT 函数拟合明显优于 Arrhenius 拟合
  ✓ 提取的 T_BKT 与 I-V 判据得到的 T_BKT 自洽

⚠️ 困难与陷阱：
  - 拟合窗口依赖性：不同温度窗口给出不同 T_BKT
  - 配对涨落（Aslamazov-Larkin 修正）也可产生类似函数形式
  - b 参数（约 1-2）的物理含义依赖具体理论模型
  - 仅 R(T) 拟合不足以证明 BKT，需与 I-V 判据结合

========================================
判据 3：超流刚度 ρ_s(T) 的普适跳变（最精确，实验最难）
========================================

实验步骤：
  ① 用微波谐振器（coplanar waveguide resonator）测量复导纳 σ(ω)
  ② 穿透深度 λ_L(T) → ρ_s = ħ²/(μ₀ * e² * λ_L²)
     或等效：ρ_s = (ħ²/e²) * D_s^{2D}（连接 F7 超流权重）
  ③ 寻找 ρ_s 在 T_BKT 处的跳变
  ④ 测量跳变量 Δρ_s，与普适预测比较：Δρ_s^{pred} = (2/π) * k_B * T_BKT

BKT 确认信号：
  ✓ ρ_s(T) 在 T_BKT 处显示不连续跳变（而非连续趋零）
  ✓ 跳变量 Δρ_s ≈ (2/π) * k_B * T_BKT（与普适值吻合）

⚠️ 困难与陷阱：
  - 需要精密微波测量设备（稀释制冷机 + 微波谐振器）
  - 有限频率效应：ω → 0 外推误差
  - 样品均匀性：不均匀样品 → ρ_s 跳变弥散
  - 测量的跳变量可能与普适值偏离（样品质量问题），见第 9 节

三判据综合使用建议：
  ⭐ 最强 BKT 证据 = 三个判据同时满足，且 T_BKT 自洽
  ✓ 至少两个判据同时满足可以合理支持 BKT 解释
  ✗ 仅一个判据（尤其仅 R(T) 拟合）证据不足
```

## 5. BKT 与 BCS 相变的深层比较

```
=== BCS（3D）vs BKT（2D）相变特征全面对比 ===

特征            BCS（3D）                        BKT（2D）
─────────────────────────────────────────────────────────────────────────────
相变类型        二阶热力学相变（SSB）             拓扑相变（漩涡束缚→解束缚）
序参量          ⟨ψ⟩ ≠ 0（真长程序，LRO）         代数衰减（准长程序，QLRO）
序参量特征      全局 U(1) 对称性自发破缺           U(1) 未严格破缺（MW 定理）
低温相关联      指数趋向常数：⟨ψ*(r)ψ(0)⟩→c≠0    幂律衰减：⟨ψ*(r)ψ(0)⟩~r^{-η}
高温相关联      指数衰减（T>Tc）                   指数衰减（T>T_BKT）
漩涡作用        热激活漩涡（在 Hc₁→Hc₂ 间出现）   热激活漩涡主导 BKT 相变本身

R(T) 形式       Arrhenius：exp(-Δ/kT)             BKT：exp(-b/√(T-T_BKT))
主导物理        配对（Δ ≠ 0，Cooper 对形成）       相位相干（漩涡束缚，非配对！）
发生温度        Tc = BCS 配对温度                  T_BKT < T_MF（相位相干温度）
适用维度        3D（及 2D 平均场）                 严格 2D（或准 2D，厚度 < λ_L）
Nobel 奖        Bardeen, Cooper, Schrieffer 1972   Kosterlitz, Thouless 2016

[定理] 关键区分：
  BCS = 配对相变（Cooper 对振幅 |ψ| 从 0 变非零）
  BKT = 相位相干相变（|ψ| 在 T_BKT 以上可能已非零，但相位不相干！）
  → 两者在 2D 超导中分别对应两个不同温度：T_MF（配对）和 T_BKT（相干）
  → T_BKT < T_MF（永远！）
```

## 8. 关键反谄媚

```
⚠️ 反谄媚 8.1：T_BKT ≠ T_MF（平均场配对温度），永远！

  T_BKT 是相位相干温度（漩涡束缚→解束缚）
  T_MF 是 Cooper 对配对温度（对振幅 |ψ| 从 0 变非零）

  在真正的 2D 超导中：T_BKT < T_MF（严格不等式）
    T_MF：配对在此以下形成，|ψ| ≠ 0，但相位不相干（漩涡自由）
    T_BKT：漩涡对束缚，相位相干建立，超流零电阻出现

  混淆后果：
    → 错误地把 T_MF（比热跳变）当 T_BKT 用于 Nelson-Kosterlitz 关系 → 错误的 ρ_s
    → 错误地把 T_BKT 当 Tc（Cooper 对配对温度）→ 低估配对温度

  正确实验分析：
    → Tc（比热跳变）≈ T_MF
    → T_BKT（I-V 幂律）< T_MF
    → 两者之差 T_MF - T_BKT：受 2D 涨落影响，越薄差越大

⚠️ 反谄媚 8.2：BKT ≠ 超导 = 零电阻（关系比你想的复杂）

  T < T_BKT：理论上零电阻（QLRO，束缚漩涡 → 超流序）
  T > T_BKT：有限电阻（自由漩涡 → 相位退相干）

  但实验上：
    → 有限频率测量：任何有限频率都给出有限电阻（零频极限才是真零电阻）
    → 零磁场下：漩涡密度有限，但有限样品中仍有小电阻
    → 杂质和钉扎：即使 T < T_BKT，漩涡运动也可能产生小电阻
    → A1（零电阻 MECE）定义的零电阻比 BKT 理想情况更复杂

⚠️ 反谄媚 8.3：I-V 幂律 α = 3 并不总是精确

  理论预测：α(T_BKT) = 3（精确）
  实验现实：
    → α = 3 的精确性受多种因素干扰（加热、无序、有限尺寸）
    → 不应期待实验上得到精确的 α = 3，应期待 α(T) 连续函数在某温度附近经过 3
    → "I-V 幂律特征"的正确解读是 α(T) 的温度演化，而非在某一温度的精确值

⚠️ 反谄媚 8.4：Nelson-Kosterlitz 普适跳变有实验误差

  理论：Δρ_s = (2/π) * k_B * T_BKT（普适，精确！）
  实验：
    → 有限频率效应（ω → 0 极限难以实现）
    → 样品无序 → ρ_s 跳变弥散（不是锐跳）
    → 测量到的"跳变量"可能偏离普适值 10-30%
    → 仅测量到大致 (2/π) * k_B * T_BKT 量级的跳变，不足以精确验证普适性
    → 精确验证需要极高样品质量和极低噪声测量

⚠️ 反谄媚 8.5：量子几何下界 T_BKT 推导有很长条件链

  推导链（F7 Peotta-Törmä + F8 Nelson-Kosterlitz）：
    ① 孤立平坦带（gap_band >> |Δ|）
    ② 均匀配对振幅 Δ
    ③ D_s^BCS ≈ 0（近平坦带）
    ④ BdG 量子度规 ≈ 正常态量子度规（小 Δ 极限）
    ⑤ T_BKT 确实是真正的 BKT 相变（非 3D 效应）
    ⑥ T_BKT 与 ρ_s 通过 Nelson-Kosterlitz 关系连接（而非配对温度）

  每个条件在 MATBG 中都只是近似满足 → 最终下界也只是近似 [猜想]
  → 不应将此下界作为精确预测，应作为量级估算

⚠️ 反谄媚 8.6：BKT 证据 ≠ 量子几何起源证明

  即使清晰观测到 BKT 特征（α = 3，ρ_s 普适跳变），也只能说明：
    → 该超导体是（准）二维的 ✓
    → 相变是拓扑 BKT 类型 ✓
    → 超流刚度非零（有量子几何或 BCS 贡献）✓

  但不能推断：
    → 超流刚度 = 量子几何贡献（BCS 贡献可能更大！）
    → 量子度规预测的 T_BKT 下界是紧的（其他机制可以给更大的 T_BKT）
    → 量子几何是主导机制（强关联等效可以产生相同 BKT 特征）
```
