# Fermi-Liquid Transport Beyond the Upper Critical Field in Superconducting La₂PrNi₂O₇ Thin Films

- **Authors:** Yu-Te Hsu, Yidi Liu, Yoshimitsu Kohama, Tommy Kotte, Vikash Sharma, Yaoju Tarn + 4 more (Hwang group, Stanford)
- **Journal:** Nature Communications (2026-03-10)
- **DOI:** 10.1038/s41467-026-70250-4
- **arXiv:** 2505.19011
- **PDF:** Hsu_Fermi-Liquid-Transport-LaPrNiO_NatComm2026.pdf (20p)

## 核心发现

La₂PrNi₂O₇（La 2/3 被 Pr 替换）薄膜超过 Hc2 后：**Fermi 液体输运**，T² 依赖从 1.5K 到 300K 全程观测到

## 关键数值

| 参数 | 数值 |
|:---|:---:|
| Tc | **41K** |
| Hc2⊥(0) | **42.8 T** |
| Hc2∥(0) | **106 T** |
| 各向异性 γ | **2.48**（接近 YBCO 的 2.2）|
| ξ_ab(0) | **2.76 nm** |
| A₂（T² 系数） | 8.1 nΩ·cm/K² |
| ρ₀ | 102 μΩ·cm |
| Δ₀（能隙估计） | ~6.3 meV |
| T_F（有效费米温度） | **2330 K** |

## 重要发现

1. **无 T-linear 电阻**（到 300K）← 与体相晶体不同（体相有 T-linear）
   - 说明薄膜可能处于**过掺杂区**
   - 不是奇异金属（strange metal）
   
2. **Fermi 液体行为：** T²、cot(θ_H) ∝ T²、Kohler 标度
   - 纵向和 Hall 输运共享**单一准粒子寿命**
   
3. **Tc/T_F ≈ 0.01 标度（Uemura 图）**
   - 和高温铜氧化物在同一标度线上
   - 暗示相同的普适原理，与材料细节无关

4. **有效质量增强：** m*/m_e ≈ 10（强关联）

## 与 Paper D 的关系

**重要参考数据：**
- 薄膜 Tc = 41K（确认现有数据）
- **各向异性 γ = 2.48 ≈ YBCO(2.2)**：但这不是因为 dx²-y² 主导，而是因为双轨道结构中两者混合
- **Tc/T_F ≈ 0.01**：镍酸盐和铜氧化物在同一标度，暗示两者配对能量尺度相似

**对 C 因子问题的启示（W1）：**
- 薄膜处于**过掺杂区**（T² 而非 T-linear）→ 这解释了为什么薄膜 Tc 低于体相
- 体相 80K 可能是**最优掺杂**，而不是 Tc 天花板
