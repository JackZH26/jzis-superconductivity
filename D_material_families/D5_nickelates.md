# D5. 镍酸盐超导体（Nickelate Superconductors）— 2019 年新兴，"离铜氧化物最近"却并非第二个铜氧化物

**MECE 编号：** D5
**关联 MECE：**
- D3（铜氧化物 — 最近结构类比，Ni¹⁺ d⁹ ≈ Cu²⁺ d⁹，d_x²-y² 轨道主导，但差异大）
- D4（铁基 — 多轨道超导类比，s± 多轨道框架可能推广）
- C2（s± wave — 可能配对态，多轨道 Fermi 口袋嵌套）
- C3（d_x²-y² wave — 铜氧化物类比候选，无限层镍酸盐可能）
- B10（Hund 耦合 — Ni 多轨道 d_z²+d_x²-y² 体系，J_H 不可忽视）
- B14（Mottness — 无限层镍酸盐母态是金属 ≠ Mott 绝缘体，核心区别于铜氧化物）
- E1（压力 — La₃Ni₂O₇ 的主要超导条件，~14GPa 才超导 Tc≈80K）
- B3（AFM 自旋涨落 — La₃Ni₂O₇ 的层内超交换 J + 层间超交换 J_⊥）

**层级关系：** D5 是 2019 年以后最受关注的高温超导新体系。两大子族（无限层 RNiO₂ 和双层 La₃Ni₂O₇）物理机制不同。Ni 的"类 Cu"电子构型激发了巨大期待，但实验和理论均显示镍酸盐≠第二铜氧化物——多轨道性质、不同母态、高压要求是镍酸盐特有的挑战。

⚠️ **核心诚实声明：镍酸盐配对机制在 2026 年完全未解。无限层镍酸盐 Tc 仅 15K，La₃Ni₂O₇ 的 80K 仅在高压（~14GPa）下实现。大量"进展"是竞争复现，机制争议远大于共识。本文档不试图掩盖实验局限和理论争议，尤其强调：镍酸盐 ≠ 第二个铜氧化物。**

## 1. 物理定义与核心思想

### 1.1 为什么镍酸盐被期待？

**[猜想 — 镍酸盐被期待的理论动机]**

```
核心类比（[猜想，有理论动机，但动机并不等于结论]）：
  Cu²⁺（d⁹）在 CuO₂ 平面中 → 铜氧化物高温超导 (Tc 最高 134K)
  Ni¹⁺（d⁹）在 NiO₂ 平面中 → 类比预期可能有高温超导

  期待的逻辑链：
  (1) 无限层镍酸盐 RNiO₂（R = Nd, La, Pr）：
      Ni¹⁺ = d⁹ 构型，单 d_x²-y² 轨道主导（与 Cu²⁺ d⁹ 类似）
      NiO₂ 平面 ≈ CuO₂ 平面（结构极相似）
  (2) Sr 掺杂（空穴掺杂）→ 类比铜氧化物 La_{2-x}Sr_xCuO₄ 掺杂方案
  (3) 理论预言（2004 Anisimov/2017 Li、2019 等）：
      [来源待验证：Anisimov et al. 1999 PRB 60, 7901；预言 NiO₂ 体系可超导]

  理论期待为何合理（[猜想，有一定依据]）：
  - d_x²-y² 主导 → 类铜氧化物的强超交换 J
  - 强关联但不同（Ni d⁹ 本地矩更弱于 Cu d⁹？）
  - AFM 自旋涨落介导 d-wave 配对（或 s± via 多轨道）

⚠️ 反谄媚：上述类比 **不是** 定理，是猜想。Ni 和 Cu 的根本差异
（多轨道、不同 CT 能、不同 Hund 耦合）使"第二个铜氧化物"
的期待在物理上并不严格。2019 年后实验证实了这些差异确实重要。
```

### 1.2 实际差异：镍酸盐 ≠ 铜氧化物（最重要的反谄媚）

**[观察/定理 — 镍酸盐与铜氧化物的根本物理差异]**

```
差异 1：Ni 的多轨道性质（[观察，DFT + DMFT 计算确认]）：
  铜氧化物：Cu d_x²-y² 是唯一费米面附近的活性轨道
    → 其他 Cu 3d 轨道（d_z²，d_xz，d_yz）远高于费米面（铜氧化物四方晶体场强）
    → 单轨道 Hubbard 模型 t-J 近似合理

  无限层镍酸盐（RNiO₂）：Ni d_z² 也接近费米面！
    → Ni 的 3d 晶体场分裂比 Cu 小（Ni 配位稍不同，apical O 缺失 → 弱晶体场）
    → d_z² 轨道不能忽略（有非零 DOS at E_F）
    → 同时还有稀土 R 的 5d/4f 轨道贡献（R = Nd, La, Pr）→ 极性金属态
    结论：无限层镍酸盐 ≠ 单轨道体系 [观察，多组 DFT+DMFT 一致]

  La₃Ni₂O₇（双层）：d_z² 和 d_x²-y² **均主导**（Ni 氧化态约 Ni²·⁵⁺）
    → d_z² 电子通过 apical O 形成层间耦合（J_⊥ 机制的来源）
    → 完全不同于铜氧化物（无 apical O 的 CuO₂ 平面）

差异 2：电荷转移能 ΔCT = E(d) - E(p)（[观察，XAS + 理论]）：
  铜氧化物：ΔCT ≈ 1–2 eV（电荷转移绝缘体，Cu d⁹ → d¹⁰ L̃ 低能激发）
    → 铜氧化物母态是"电荷转移绝缘体"（ZSA 分类，B14 关联）
  无限层镍酸盐：ΔCT 估计更大（Ni d 能级更深，O 2p 与 Ni 3d 杂化更弱）
    → 镍酸盐母态更接近 Mott 侧（Hubbard 型）而非电荷转移侧 [来源待验证]
    → 但实验（输运）显示无限层镍酸盐母态是**金属**（非绝缘体！）
    → 矛盾：理论预期 Mott 绝缘体，实验见金属态 → 关联弱于铜氧化物 [观察]

差异 3：Hund 耦合 J_H（[猜想，B10 关联]）：
  无限层镍酸盐（Ni¹⁺，d⁹）：单电子 d⁹，Hund 规则平庸（全轨道几乎满）
    → J_H 效应相对较小（d⁹ 构型的 Hund 不如 d⁶ 铁基显著）[来源待验证]
  La₃Ni₂O₇（Ni²·⁵⁺，d⁷·⁵）：d_z²¹ + d_x²-y²：半满轨道 → Hund 耦合更活跃
    → J_H 对 La₃Ni₂O₇ 的双轨道物理有重要影响 [猜想，来源待验证]

差异 4：母态的性质（最关键，B14 关联）：
  铜氧化物母态（x=0）：Mott 绝缘体（零掺杂下绝缘，AFM 有序）
    → 电阻率极大，磁有序温度 T_N ≈ 300–500K
    → 超导从 Mott 态出发（强关联）[定理，实验确认]

  无限层镍酸盐（x=0，未掺杂）：**金属态！**
    → NdNiO₂ 等母化合物具有有限 DOS @ E_F（金属电导）
    → 没有明显的 Mott 能隙 [观察，输运实验]
    → 关联强度比铜氧化物弱得多
    ⚠️ 这是最关键的区别：铜氧化物 = 从 Mott 绝缘体出发；镍酸盐 = 从金属出发

差异总结（ASCII 对比）：
  ──────────────────────────────────────────────────────────────
  特征              铜氧化物（D3）    无限层镍酸盐     双层 La₃Ni₂O₇
  ──────────────────────────────────────────────────────────────
  活性 d 轨道       d_x²-y²（单）    d_x²-y²+d_z²   d_z²+d_x²-y²（双主导）
  母态（x=0）       Mott 绝缘体      金属（弱关联）  金属（常压）
  超导条件          常压掺杂         薄膜，Sr 掺杂   高压 ~14GPa（块材）
  Tc_max（常压）    134K（Hg-1223）  ~15K（薄膜）    常压不超导（块材）
  实验形态          体材+薄膜        **仅薄膜**       高压块材
  ──────────────────────────────────────────────────────────────
```

### 1.3 La₃Ni₂O₇ 的特殊物理

**[观察 — La₃Ni₂O₇ 是镍酸盐中最重要的高压超导体]**

```
La₃Ni₂O₇ 结构：
  双层 NiO₂ 平面（Ruddlesden-Popper n=2 型）
  Ni 氧化态：约 Ni²·⁵⁺（d⁷·⁵ 平均）
  双层之间通过 apical O 连接（层间超交换 J_⊥ 的来源）
  常压结构：斜方相（Amam），约 13GPa 以上转变为四方相（I4/mmm）

核心物理思路（J_⊥ 机制，[猜想，Ji-Yang 2023 等]）：
  (1) d_z² 轨道通过 apical O 在层间形成强超交换 J_⊥
  (2) J_⊥ 产生层间单态（interlayer singlet）
  (3) 高压压缩 → 层间 Ni-O-Ni 键角改变 → J_⊥ 增强 → SC 出现
  → 这是 La₃Ni₂O₇ 在高压下超导的最主流机制假说 [猜想，来源待验证]

  [参考：Ji, Yang et al. 2023（理论）；Zhang et al. 2023 等，来源待精确核查]

为什么必须高压？[观察 + 猜想]
  常压下 La₃Ni₂O₇ 处于斜方相（Amam），某种电荷/轨道/自旋有序竞争超导
  高压 ~13GPa：结构相变 Amam → I4/mmm（四方，更高对称性）
  四方相下 d_z²-d_x²-y² 轨道混合增强 → J_⊥ 增大 → 超导出现（Tc ≈ 80K）
  ⚠️ [Zhang+2025, arXiv:2511.15265（Raman 光谱确认 Amam→I4/mmm @ 14.5 GPa，超导同时出现）（来源：daily 笔记 2026-04-11）]：具体结构相变压力点和 Tc 的关系，不同实验组数据有差异
```

## 2. 历史关键节点（2019–2026 镍酸盐超导发展史）

**[历史 — 镍酸盐超导的诞生与快速发展]**

```
2019 Li, Lu, Zeng, Kirby, Zhong, Zhang, Jin, Ramesh, Birgeneau, Hwang（Stanford）：
  发现 Nd₀.₈Sr₀.₂NiO₂ 薄膜超导，Tc ≈ 9–15K
  [来源：Li et al. 2019 Nature 572, 624]（确定）
  ★ 历史地位：第一个镍酸盐超导体，引爆全球研究热潮
  实验细节：
  - 材料：SrTiO₃ 衬底上生长的 NdNiO₃ 薄膜（PLD），再用 CaH₂ 氢还原得 NdNiO₂
  - 掺杂：20% Sr 替换 Nd（空穴掺杂，等效于铜氧化物最优掺杂）
  - Tc ≈ 9–15K（报告值取决于定义方式，零电阻约 9K，能隙闭合约 15K）
  ⚠️ 注：复现困难！氢还原过程对薄膜质量极敏感，不同组报告 Tc = 3K–15K [观察]

2020–2022 多个组（MIT、SIMIT 中科院微电子所、香港大学、南京大学等）：
  陆续复现 Nd₁₋ₓSrₓNiO₂，Tc 在 3–15K 范围变化
  → Tc 对衬底类型（STO vs LAO vs LSAT）极其敏感 [观察]
  → Tc 对氧含量（还原充分度）敏感 [观察]
  ⚠️ 一些组报告未能复现（Tc 未见零电阻）
  [各组来源分散，见综述 Wang & Zhang 2021 NPJ Quantum Mater. 6, 95；来源待验证]

2021 Osada, Wang, Goodge, Harvey, Kim, Kim, Akey, Kourkoutis, Bhaskaran, Hwang（Google + Stanford）：
  La₁₋ₓSrₓNiO₂ 薄膜超导，Tc ≈ 13–15K（La 代替 Nd）
  PrNiO₂ 薄膜超导，Tc ≈ 15K（Pr 无限层镍酸盐）
  [来源：Osada et al. 2021 Nano Lett. 21, 4, 1613]（来源待精确核查）
  意义：
  - 证实不同稀土（R = Nd, La, Pr）的无限层镍酸盐均可超导
  - Tc 普遍在 9–15K 范围，不因稀土而大幅改变
  - 最大 Tc ≈ 15K 似乎是无限层镍酸盐的天花板 [观察]

2022 Nomura, Arita（东大，DFT+DMFT）：
  无限层镍酸盐的多轨道电子结构计算：d_z² 贡献显著，弱 Mott 关联
  [来源待验证：Nomura & Arita 2022 PRB 105, 094503]
  意义：确立"多轨道弱关联"框架，挑战"镍酸盐 ≈ 铜氧化物"的单轨道假设 [猜想]

★★★ 2023 Sun, Zhang, Zhu, Li, Shan, Zhang, Duan（中国科学院物理所、北大 Zhu Meng 组等）：
  La₃Ni₂O₇ 在 ~14GPa 压力下 Tc ≈ 80K
  [来源：Sun et al. 2023 Nature 621, 493]（确定）
  ★ 镍酸盐最大突破，双层镍酸盐进入高温超导级别
  实验细节：
  - 材料：多晶 La₃Ni₂O₇ 块材（不是薄膜！）
  - 压力：金刚石对顶砧（DAC）装置，~14GPa
```

## 3. 两大镍酸盐子族

### 3.1 无限层镍酸盐 RNiO₂（R = Nd, La, Pr, ...）

**[观察 — 无限层镍酸盐的结构、特征与局限]**

```
结构定义：
  化学式：RNiO₂（R = Nd, La, Pr, Sm, ...）
  晶体结构：无限层（infinite-layer）结构 — 仅 NiO₂ 平面堆叠，无 apical O
  Ni 氧化态：Ni¹⁺（d⁹）—— 未掺杂时等效于 Cu²⁺ d⁹
  晶格：四方，a ≈ 3.9 Å，与 CuO₂ 平面参数接近 [来源待验证]

  ASCII 结构示意：
    ─────── LaO 层 ─────── （稀土氧化物层，类比铜氧化物 LaO 层）
    ─── NiO₂ 平面 ───── （Ni¹⁺ d⁹ 正方形网格，**无** apical O）
    ─────── LaO 层 ───────
    ─── NiO₂ 平面 ─────
    （对比铜氧化物：CuO₂ 平面 + apical O → 铜氧化物有 apical O，镍酸盐无！）

制备方法（关键限制）：
  步骤 1：在 SrTiO₃（或 LaAlO₃）衬底上用 PLD 生长 RNiO₃ 薄膜（Ni³⁺，钙钛矿型）
  步骤 2：用 CaH₂ 在 250–280°C 氢还原 → RNiO₂（Ni³⁺ → Ni¹⁺，去除 apical O）
  [来源：Li 2019 Nature 572, 624 — 制备方法部分]（确定）

  ⚠️ 关键限制：
  (1) **无法制备块材（体材）**：无限层 RNiO₂ 在常压下热力学不稳定（块材难以合成）
  (2) 仅薄膜可用 → 无法进行比热（热容）、中子散射、μSR 等体材实验
  (3) 薄膜超导可能受衬底应变影响（很难区分本征与界面效应）
  → 无限层镍酸盐是实验测量受限最严重的高温超导体系之一 [观察]

超导性质：
  Tc 范围：3–15K（取决于稀土 R、掺杂 x、衬底、还原条件）
  最优掺杂：x ≈ 0.2 Sr 替换（空穴掺杂）
  掺杂相图：只有有限的 x 范围有超导，过掺/欠掺均无超导 [观察，Li 2019 + 多组]
  Tc 对衬底高度敏感：STO（最常用）vs LAO vs LSAT → Tc 差异达 5K [观察]

配对对称性（高度不确定）：
  d_x²-y² wave？（单轨道铜氧化物类比）[猜想]
  s± wave？（如果 d_z² 口袋足够重要）[猜想]
  2026 年：无明确实验判断，因为**仅薄膜，无法做相位灵敏实验** [观察]

电子结构特点（[观察，DFT+DMFT 计算]）：
  费米面：不简单！
  - d_x²-y² 口袋（主要，类 Cu）
  - d_z² 口袋（弱但存在，R 5d/4f 影响下）
  - 稀土 R 的 5d 轨道对费米面有贡献（"self-doping"效应）
  → 镍酸盐费米面比单带铜氧化物复杂得多 [观察，来源待验证]
```

### 3.2 双层镍酸盐 La₃Ni₂O₇ 及其他 R₃Ni₂O₇

**[观察/猜想 — 双层镍酸盐是镍酸盐高温超导的核心体系]**

```
结构定义：
  化学式：La₃Ni₂O₇（Ruddlesden-Popper n=2 型）
```

## 4. La₃Ni₂O₇ 详细物理（重点）

### 4.1 电子结构

**[观察/猜想 — La₃Ni₂O₇ 的多轨道费米面]**

```

  费米面由两个主要轨道构成：
```

```
    H_J_perp = J_⊥ Σ S_{i,top} · S_{i,bottom}（层间超交换）

机制 B：多轨道 s± 配对（类铁基推广）
  核心思路：
  - d_x²-y²（β 口袋，类铁基空穴口袋）和 d_z²（α 口袋，类铁基电子口袋）

  - 如果 d_x²-y² 主导配对（d_z² 次要）→ 类铜氧化物的 d_x²-y² wave

  [猜想，最简单类比，但 La₃Ni₂O₇ 明显有 d_z² → 单轨道 d-wave 假设可疑]
  局限：
  - La₃Ni₂O₇ 双轨道性质使 d-wave 描述不完整
  - 无实验判据

2026 年现状（诚实评估）：
  三种机制均有理论支持，无判决性实验
  ⚠️ 高压限制：DAC 环境下无法做 ARPES、中子散射、μSR → 无法直接测配对态
  ⚠️ 机制争议可能在未来几年随常压薄膜 La₃Ni₂O₇ 的质量提升而有所进展
```

## 7. 关键反谄媚（最重要）

### 7.1 无限层镍酸盐 Tc 仅 15K，差铜氧化物近一量级

```
[观察/定理 — 最直接的反谄媚：期待高，实际低]

实验事实：[定理，实验确认]
  最高 Tc（无限层镍酸盐，薄膜）≈ 15K（Pr-系，Osada 2021）[来源待精确核查]
  铜氧化物 Tc_max（常压块材）= 134K（Hg-1223）[Schilling 1993 Nature 363, 56]
  比值：134K / 15K ≈ 9

"为什么这么低"的诚实评估：
  ✗ 失败的期待：2019 年许多人期待无限层镍酸盐会有 ~50K 超导（铜氧化物的 1/3）
  ✓ 实际结果：最优薄膜仅 15K，且 Tc 对制备条件极度敏感，复现困难
  → 从实验角度，无限层镍酸盐是迄今为止最令人"失望"的高温超导新体系之一

潜在原因（[猜想，后验]）：
  (1) 金属母态 → 超交换涨落弱 → 小 Tc
  (2) d_z² 轨道干扰：d_x²-y² 配对被多轨道竞争削弱
  (3) 薄膜特异性：衬底应变、界面效应可能对超导既有增强也有抑制
  → 任何一个解释都未被验证为主因 [猜想，诚实标注]
```

### 7.2 La₃Ni₂O₇ 的 80K 仅在高压下 — 常压不超导

```
[观察 — La₃Ni₂O₇ 高压依赖是最大局限]

实验事实：[定理，Sun 2023 + 多组复现]
  La₃Ni₂O₇ 块材在常压：金属，**无超导至 2K 以下** [Zhang+2025, arXiv:2511.15265（SC 出现与 I4/mmm 结构相变在 14.5 GPa 同时发生，常压 Amam 相无超导）（来源：daily 笔记 2026-04-11）；与 Sun+2023, Nature 621, 493 一致]
  La₃Ni₂O₇ 在 ~14GPa：Tc ≈ 80K（超导）[Sun 2023 Nature 621, 493]（确定）

为什么高压要求是致命弱点：
  (1) 实用性：高压超导无法用于实际超导线材或器件
      → 目前无法制备"常压 80K 镍酸盐超导体"
  (2) 实验测量严重受限：
      ❌ 无法做 ARPES（高压 ARPES 技术不成熟）
      ❌ 无法做中子散射（DAC 散射体积太小，中子通量不足）
      ❌ 无法做 μSR（探针无法穿透高压装置）
      → 配对对称性的直接测量几乎不可能
  (3) 数据可靠性：高压 DAC 实验的系统误差大
      → 不同组报告的 Tc 有差异（75K–85K）[来源待验证]
      → 超导体积分数不确定（是否全体超导？）

常压薄膜 La₃Ni₂O₇ 超导（20–40K）的现状（2026）：
  [来源待验证 — 高度存疑]
  - 一些组报告常压薄膜超导，但 Tc 和超导体积分数差异很大
  - 主要疑虑：薄膜中可能存在相分解（混入 RP n=1 或 n=3 相）
  - 独立重复验证不充分
  → 2026 年常压薄膜 La₃Ni₂O₇ 超导应视为"尚未充分确认"的初步发现 [猜想，低置信度]
```

### 7.3 镍酸盐 ≠ 第二个铜氧化物（最核心反谄媚）

```
[观察 — 反对"镍酸盐是新铜氧化物"的简单类比]

多轨道性质（定量差异）：
  ✗ 单轨道假设（铜氧化物类比）：仅 d_x²-y² 主导
  ✓ 实际（DFT+DMFT）：d_z² 在费米面有不可忽略贡献 [观察，来源待验证]
  → t-J 单带模型对镍酸盐可能不适用

母态根本不同：
  铜氧化物：Mott 绝缘体（TN ≈ 300–500K，磁矩 ≈ 0.6 μB）
  无限层镍酸盐：金属（有限 DOS @ EF，无明确 AFM 序或极弱）[观察]
  → 铜氧化物的 t-J/RVB 框架出发点不适用于镍酸盐 [观察]
  （B14 关联：Mottness 综述明确区分了两者）

Hund 耦合差异：
  Cu d⁹（d 壳基本满）：Hund 耦合对单电子 d⁹ 影响有限
  Ni（d⁷·⁵，La₃Ni₂O₇）：d 壳非满，Hund 耦合更活跃
  → La₃Ni₂O₇ 的物理更接近铁基（多轨道 Hund 体系）而非铜氧化物 [猜想，B10 关联]

配对机制根本不同（猜想层级，但方向明确）：
  铜氧化物：d_x²-y² wave（判决性确认，Tsuei 1994）→ 单轨道 AFM 超交换
  镍酸盐：s±？d-wave？J_⊥ 超交换？— 三者竞争，无判决实验 [猜想]
  → 如果 J_⊥ 机制（层间超交换）是主因，则 La₃Ni₂O₇ 的配对物理与铜氧化物完全不同

结论：
  镍酸盐被期待的理由（Ni¹⁺ d⁹ ≈ Cu²⁺ d⁹）是类比，不是等价
  2019 年以来的实验（低 Tc 无限层，高压 La₃Ni₂O₇）表明类比有局限
  "第二个铜氧化物"的标签是误导性的 — 镍酸盐是独特的新体系，有自己的物理
```

### 7.4 配对机制争议未解，高压限制了实验

```
[观察 — 2026 年状态的诚实评估]

当前配对机制状态：
  s±（多轨道 Fermi 口袋嵌套）：[猜想，低置信度，缺乏直接实验]
  d_x²-y² wave（类铜氧化物）：[猜想，低置信度，单轨道假设可疑]
  J_⊥ 超交换（层间 singlet）：[猜想，有一定理论支持，但定量不完整]
  → 没有一个机制有判决性实验支持 [观察，2026]

高压实验的根本局限：
  La₃Ni₂O₇ 的超导在 ~14GPa 以上
  高压限制了所有关键实验：
  ❌ ARPES → 无费米面数据
  ❌ 中子散射 → 无磁激发（自旋波）数据
  ❌ μSR → 无超流密度数据
  ❌ Josephson 结实验 → 无配对相位信息
  → 判决性配对对称性测量在 La₃Ni₂O₇ 上 **技术上不可行（2026 年）** [观察]

发展路径（[猜想]）：
  最有希望的突破：常压薄膜 La₃Ni₂O₇ 超导若被可靠复现（Tc ≈ 30–40K）
  → 薄膜可做 ARPES、STM/QPI 等 → 配对机制可能在 2–3 年内有突破
  → 但常压薄膜超导本身的可靠性还需确认
```

## 9. 与 MECE 体系的关联分析

```
D3（铜氧化物）：
  镍酸盐是 D3 最直接的"邻居"材料 — 结构最相似，但物理本质不同
  → D5 是 D3 框架的"压力测试"：类铜结构却未复现铜氧化物物理
  → 关联方向：D3（对照基准）← D5（偏差揭示关键因素）

D4（铁基）：
  La₃Ni₂O₇ 与铁基的类比（双轨道，多 Fermi 口袋）
  → D4 的 s± 框架有可能推广到 La₃Ni₂O₇（d_z² + d_x²-y² 多口袋 s±）[猜想]
  → D5 的双轨道物理（Ni d_z² + d_x²-y²）是 D4（Fe 5 轨道）的简化版

C2（s± wave）：
  La₃Ni₂O₇ 的多轨道 Fermi 面嵌套 → s± 候选
  → C2 的理论框架（两带 BCS + 嵌套）可能适用于 La₃Ni₂O₇ [猜想]
  → 但无直接实验验证（高压限制）

C3（d_x²-y² wave）：
  无限层镍酸盐的单轨道类比 → d-wave 候选
  → 若 d_z² 贡献小，无限层镍酸盐配对可能接近 d-wave [猜想]
  → 完全无实验判据

B3（AFM 自旋涨落）：
  层内超交换 J（~50–80 meV）是 d-wave 或 s± 的可能驱动力
  层间超交换 J_⊥（~10–30 meV）是 La₃Ni₂O₇ J_⊥ 机制的驱动力
  → Chan 2026 + Zhou 2026 INS 实验 → 精确测量 J 和 J_⊥ → B3 的实验输入

B10（Hund 耦合）：
  La₃Ni₂O₇（Ni²·⁵⁺，d⁷·⁵）多轨道 → Hund 耦合有一定作用
  → B10 框架部分适用于理解 La₃Ni₂O₇ 的轨道物理
  → 无限层镍酸盐（Ni¹⁺，d⁹）Hund 效应相对弱

B14（Mottness）：
  无限层镍酸盐母态是金属（非 Mott 绝缘体）→ B14 框架（t-J/RVB）不适用 [观察]
  → B14 综述明确标注了"镍酸盐不是第二铜氧化物"这个反谄媚教训
  → D5 是 B14 的核心反例

E1（压力）：
  La₃Ni₂O₇ 超导完全依赖压力（~14GPa）→ E1 是 D5 的核心控制参数
  → 理解压力诱导超导的机制（结构相变 + J_⊥ 增强）是 D5 与 E1 的交叉前沿

F8（拓扑超导）：
  Nie 2026 的 1212/2222 堆叠镍酸盐 → 可能有拓扑表面态 [猜想，来源待验证]
  → D5 的多层堆叠与 F8 的拓扑物理有潜在联系

B12（多通道叠加）：
  Paper A 框架对 La₃Ni₂O₇ 的适用性：
  → J_⊥ 层间超交换（非声子通道）超出标准 Eliashberg → Paper A 边界测试
  → 若 La₃Ni₂O₇ 是多通道（J_⊥ + 层内 AFM 涨落），Paper A 或可部分描述 [猜想]
```
