# Emergence of the Logarithmic Average Phonon Frequency in SC Tc Formula
**arXiv:2504.17435** | April 2025
**Authors:** Nattawut Natkunlaphat, Pakin Tasee, Udomsilp Pinsook (Chulalongkorn)

## 核心方法论
- 解析推导 ω_log 在 Tc 公式中的基本角色
- 假设 Eliashberg 谱函数 α²F(ω) 在低频遵循 Debye 模型，高频由光学声子主导
- 从预测函数直接推导出包含 ω_log 的 Tc 公式

## 关键结果
1. **解析证明 ω_log 为什么是 Tc 公式中的"正确"频率标度**——而非 ω_D 或 ⟨ω⟩
2. 在特定条件下（Debye+光学模型），推导出 Tc 公式与 Allen-Dynes 及其修正版吻合
3. 与精确 Eliashberg 解有很好的相关性
4. 关键条件：需要声子谱的双组分性质（声学 Debye + 光学峰）

## 与研究的关联
- **直接关联我们的 ω_log vs ω̄₂ 分析！** 该论文从解析角度确认 ω_log 是 AD 公式的基本量
- 但 ω̄₂ 是 Eliashberg 的关键量——两者的差异正是我们研究的 AD→Eliashberg gap 的根源
- 双组分谱（声学+光学）的假设直接适用于氢化物：金属格 = 低频声学，H 振动 = 高频光学

## 启发的新想法
**假设：** 如果 ω̄₂/ω_log 比值可以从 α²F 的谱形状参数（如 Debye 截断频率与光学峰频率之比）解析预测，那么就不需要完整的 Eliashberg 计算来估计 AD→Eliashberg 校正。**可验证：** 用 Kimura 数据中的 α²F 拟合 Debye+光学模型参数，看能否预测 ω̄₂/ω_log。

## 数值/公式
- 推导公式与 Allen-Dynes 和修正 AD 公式对比
- 需要条件：α²F(ω) = Debye 部分(低频) + 光学部分(高频)
