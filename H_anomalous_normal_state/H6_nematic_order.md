## 1. 物理定义与核心思想

### 1.1 向列序的定义 [定理，对称性分析]

```
=== 向列序（Nematic Order）的定义 ===

【名称来源】
  "向列（Nematic）"来自液晶物理：
  → 液晶中分子方向有序但位置无序（区别于固体：方向 + 位置均有序）
  → 电子向列序类比：电子的方向分布（Fermi 面形状）各向异性，但晶格位置不改变

【铁基超导中的向列序（[定理，对称性分析]）：】

  母化合物结构：四方晶格（D4h 点群，即 C4 旋转对称性）
  向列序：C4 → C2（正交）的自发旋转对称性破缺
    → 系统区分了 a 轴和 b 轴（原本是等价的四重轴）
    → 但：平移对称性保持不变（区别于 CDW/SDW，后者引入新的周期性）

  向列序参量：φ_nem（可观测的各向异性量）
    典型测量：φ_nem = ρ_xx - ρ_yy（a 轴和 b 轴电阻率之差）
    φ_nem = 0：高温四方相（C4 对称，a≡b）
    φ_nem ≠ 0：低温向列相（C2 对称，a≠b）
    相变温度：T_nem（向列相变温度，有时也写 T_s 或 T_structural）

【向列序 vs CDW/SDW 的关键对比 [定理/观察]：】

  性质           向列序（H6）            CDW（H4）              SDW（H5）
  ---            ----                    ----                   ----
  破缺对称性     旋转（C4→C2）           平移（电荷周期调制）    时间反演 + 平移（磁序）
  序参量         φ_nem（各向异性）        ρ_Q（电荷密度振幅）    m_Q（自旋密度振幅）
  晶格畸变       轻微斜方化（结果，非原因）强烈周期性晶格畸变     通常无（纯电子序）
  平移对称性     保持                    破坏                   破坏
  实验标志       电阻率各向异性 + 弹性软模  X 射线超晶格衍射峰    中子散射磁性峰
  与 SC 关系     涨落驱动（B5），长程序各向异性  竞争压制（H4）    压制（长程）+ 涨落驱动（H5/B3）
```

### 1.2 向列序的物理图像——三种机制 [猜想，来源争议]

```
=== 向列序起源的三大候选机制 ===

【图像 A：轨道序（Orbital Order）[猜想，轨道机制阵营]】

  物理图像：
  → 铁基超导中 Fe 原子有 5 个 3d 轨道（多轨道体系）
  → d_xz 和 d_yz 轨道在四方对称下严格简并（按 C4 对称要求）
  → 向列序 = d_xz 和 d_yz 轨道的简并性自发打破（轨道占据不等）
  → d_xz 填充更多 → 沿 x 方向电子更多 → C4→C2 对称破缺
  → 这是轨道"Jahn-Teller"效应的电子版本

  直接证据 [观察，来源待验证]：
  → ARPES 在 T < T_nem 时直接测到 d_xz / d_yz 轨道的能带劈裂
    （原本简并的两条带在向列相中分开，劈裂量 ~ 50-100 meV）
  → 轨道极化有明确的温度依赖：T > T_nem 时消失

  问题/质疑：
  → 轨道简并打破的驱动力是什么？（自旋-轨道耦合？Hund 耦合？外部电声耦合？）
  → ARPES 测到的轨道劈裂是否可能是磁矩引起的次级效应？[猜想，争议]

【图像 B：自旋向列涨落（Spin Nematic）[猜想，磁性驱动阵营]】

  物理图像：
  → 从 SDW 出发：铁基中的 SDW 有两种等价的波矢 Q₁=(π,0) 和 Q₂=(0,π)
  → 在 T > T_SDW 时，长程 SDW 序尚未形成，但 Q₁ 和 Q₂ 方向的短程 SDW 涨落共存
  → 这两套涨落如果形成特定的二次组合：< S_Q₁ · S_Q₁ > ≠ < S_Q₂ · S_Q₂ >
    → 该复合算符本身破坏 C4 对称！（即使没有长程磁序）
  → 即：向列序是 SDW 有限温度涨落的"预熔"体现（向列相是 SDW 的前驱）[猜想]
  → 机制：SDW 涨落产生各向异性应力 → 耦合晶格 → 诱发 C4→C2 结构畸变

  支持证据 [观察，来源待验证]：
  → 向列序消失时（过掺杂区），SDW 涨落也消失（相关性）
  → 弹性软模在 T_nem 处出现，与 SDW 涨落增强同步

  问题/质疑：
  → T_s > T_SDW 在欠掺杂铁基中有据可查：向列序先于 SDW 出现！
    → 若向列序是 SDW 的"前驱"，为何比 SDW 先出现？[矛盾，已知]
  → 回应 [猜想]：SDW 涨落（短程）可在 T_SDW 以上存在，驱动向列序，
    而长程 SDW 序在更低温才凝结 → 不矛盾，但需要量化 [猜想]

【图像 C：独立量子临界现象 [猜想]】

  物理图像：
  → 向列序参量有自己独立的量子临界点（QCP_nem）
  → QCP_nem 与 SDW 的 QCP 可以在相图上是不同的点
  → 向列涨落从 QCP_nem 出发，独立于磁涨落，单独增强 SC [猜想，B5]
  → 这是铁基 SC 中向列涨落机制最"干净"的理论图像

  地位：
  → 目前是 [猜想]，尚无决定性实验区分 A/B/C 三种图像
  → A 和 B 在很多体系中难以区分（轨道和自旋自由度深度耦合）
```

### 1.3 向列性（Nematicity）的实验测量方法 [观察]

```
=== 探测向列性的主要实验手段 ===

【① 电阻率各向异性（主要方法，需去孪晶）】
  测量量：ρ_xx - ρ_yy（或 ρ_a - ρ_b）
  技术难点：
  → 铁基晶体在冷却时形成孪晶畴（a/b 轴定向随机）
  → 孪晶会把各向异性信号互相抵消 → 测到的是 0
  → 解决：用单轴应变（detwinning pressure）固定晶体孪晶方向
    → Chu et al. 2010 的核心实验创新 [来源待验证，Science 2010]
  特点：灵敏，可测量整体样品，但需要机械去孪晶

【② 弹性软模（C₆₆ 弹性常数）】
  测量量：B₂g 对称性的剪切弹性常数 C₆₆
  原理：
  → 向列序参量与 B₂g 晶格畸变线性耦合（向列-声子耦合）
  → 在 T_nem 附近，向列涨落增强 → C₆₆ 软化（下降）[来源待验证，Bohmer 2014]
  → 软化幅度：~20-30%（估计值，[来源待验证]）
  特点：无需去孪晶，测量晶格弹性的宏观量，直接反映向列涨落强度

【③ 弹阻抗（Elastoresistance）/ 压阻系数】
  测量量：在施加单轴应变 ε 下，电阻率的各向异性响应 dρ/dε
  原理：
  → 小的单轴应变"帮助"向列序参量取向 → 诱导出各向异性
  → 测到的系数反映向列磁化率（nematic susceptibility）χ_nem
  → χ_nem 在向列 QCP 附近发散（类 Curie-Weiss 形式）[猜想，观测趋势]
  特点：不需要真正去孪晶，可以精确定量 χ_nem 的温度依赖 [来源待验证，Hristov 2018 类实验]

【④ ARPES（角分辨光电子谱）】
  测量量：d_xz / d_yz 轨道的能带结构
  直接证据：
  → 去孪晶样品中，d_xz 和 d_yz 轨道在 T < T_nem 时出现能带劈裂
  → 劈裂量 ~ 50-100 meV（Fe-As 系，[来源待验证]）
  → 劈裂在 T > T_nem 时消失（对应轨道简并恢复）
  特点：直接看轨道分辨的电子结构，是最有力的"轨道序"证据

【⑤ Raman 散射（拉曼散射）】
  测量量：B₂g 声子软化和电子连续谱各向异性
  原理：
  → B₂g 对称性拉曼响应在 T_nem 附近增强（类 Curie-Weiss）
  → 可以区分轨道贡献（光学声子）和自旋贡献（磁 Raman 信号）
  特点：可以部分区分轨道和自旋向列涨落的相对贡献 [猜想，Gallais 等，来源待验证]
```

## 2. 历史关键节点

```
=== 向列序研究的历史里程碑 ===

【2008 铁基超导发现 + 母化合物结构相变观察】
  → Kamihara 等发现 LaFeAsO₁₋ₓFₓ 中 Tc=26K 超导 [Kamihara 2008, JACS 130, 3296]
  → Rotter 等报道 BaFe₂As₂ 的磁性和结构同时发生相变（T ≈ 138K）
    [确定，Rotter 2008, PRL 101, 107006]
  → 早期解读：结构相变和 SDW 相变同时发生，认为两者是同一件事

【2010 Chu et al.（Stanford）— 向列性是电子起源的直接证明】
  → 实验：用单轴应力去孪晶 BaFe₂As₂，直接测量电阻率各向异性
  → 发现：ρ_a - ρ_b 在 T < T_s 时显著非零，远超结构畸变（a-b 晶格差）预期
  → 结论：电阻率各向异性是"电子向列序"，不是简单晶格畸变效应
  → 意义：第一次把向列性和结构相变分开，确立"电子向列序"概念 [来源待验证，Science 2010]

【2012 前后 Fernandes-Chubukov 等 — 向列量子临界理论建立】
  → 理论：自旋向列涨落（Spin Nematic）作为铁基向列序的驱动机制
  → 预言：向列磁化率 χ_nem 应服从 Curie-Weiss 形式，在 QCP 处发散
  → 建立了向列序和 SDW 涨落关系的理论框架 [来源待验证，Fernandes-Chubukov 2012 前后]

【2014 Bohmer et al. — 弹性常数 C₆₆ 软化，确认向列相变】
  → 弹性测量：BaFe₂As₂ 系列中 C₆₆ 在 T_s 附近显著软化（下降 ~20-30%）
  → 直接证明：向列相变伴随弹性不稳定（B₂g 软模）[来源待验证，Bohmer 2014]
  → 意义：提供了与去孪晶电阻率实验独立的向列序证据

【2015-2020 铜氧化物向列行为争论兴起】
  → Ando 等早期工作（2002 前后）报告 LSCO 欠掺杂区电阻率各向异性 [来源待验证]
  → Lawler et al.：STM 观测到 Bi₂Sr₂CaCu₂O₈ 中的"向列态"（Fermi arc 各向异性）
    [来源待验证，Nature 2010 类论文]
  → 争议：是否是真正的 C4→C2 破缺，还是 CDW/条纹序/样品不均匀性导致？

【2020 前后 FeSe 成为向列序研究焦点】
  → FeSe 体材：Tc = 8K，T_s（向列相变）≈ 90K，零场下无 SDW 长程序！
  → 向列序和 SDW 的分离最为清晰（向列序 >> SDW，向列独立性最强）[来源待验证，多组]
  → FeSe 成为区分轨道序和自旋向列机制的最佳实验平台
```

## 3. 铁基超导的向列序（最详细案例）

### 3.1 BaFe₂As₂ 母化合物：向列序与 SDW 的并发

```
=== BaFe₂As₂（Ba122）母化合物：向列序 ===

基本参数 [Rotter 2008, PRL 101, 107006]（确定）：
  T_s = T_SDW ≈ 138K（结构相变 = SDW 相变，两者几乎同时发生）
  结构相变类型：四方（I4/mmm）→ 斜方（Fmmm）
  晶格各向异性：a - b ≈ 0.3%（斜方化程度，很小）
  → 晶格畸变远小于电阻率各向异性 → 向列序是电子起源而非晶格起源！

向列序参量的直接测量（去孪晶后）[观察，来源待验证]：
  → 电阻率各向异性：ρ_a / ρ_b 在 T < T_s 时偏离 1（约 10-20%，依掺杂）
  → 轨道劈裂（ARPES）：d_xz / d_yz 能带在 T < T_s 时劈裂 ~50-100 meV [来源待验证]
  → 弹性常数 C₆₆：在 T_s 附近软化 ~20-30% [来源待验证]

与 SDW 同时发生的解读：
  → 在 BaFe₂As₂ 母化合物中，两者不分离（T_s = T_SDW = 138K）
  → 无法判断哪个是"因"，哪个是"果"
  → 需要掺杂体系（T_s > T_SDW 情形）才能区分
```

### 3.2 掺杂演化：向列序先于 SDW 出现的关键证据

```
=== 掺杂体系中 T_s > T_SDW 的关键证据 ===

【Co 掺杂 BaFe₂₋ₓCoₓAs₂ 相图 [观察，来源待验证]：】

  x = 0（母化合物）：T_s ≈ T_SDW = 138K（同时相变）[确定，Rotter 2008]
  x ≈ 0.04（欠掺杂）：T_s ≈ 110K > T_SDW ≈ 80K（向列序先！）[来源待验证]
  x ≈ 0.06-0.08（最优掺杂）：T_s → 0（向列 QCP？）+ Tc 最高 [来源待验证]
  x > 0.12（过掺杂）：T_s = 0，无结构相变 [来源待验证]

  T_s > T_SDW 的重要性：
  → 在 T_SDW < T < T_s 的温度区间：
    • 向列序存在（C2 斜方相）
    • 长程 SDW 序不存在（中子散射无磁性超晶格峰）
  → 这个区间证明：向列序可以独立于 SDW 长程序而存在！
  → 但：SDW 涨落（短程磁序）在 T > T_SDW 时仍然存在
    → 不能完全排除"自旋向列涨落驱动"图像 [猜想]

【FeSe 体材——向列序最纯净的实验平台 [观察，来源待验证]：】

  FeSe：
    T_s（向列相变）≈ 90K
    T_SDW（SDW 长程序）= 0（零场下无磁序！）
    Tc ≈ 8K

  意义：
  → FeSe 是唯一已知的铁基超导体，向列序 >> SDW 完全分离（SDW 不存在）
  → 若自旋驱动图像正确：FeSe 的向列序应由 SDW 涨落驱动（但无长程磁序） [猜想]
  → 若轨道驱动图像正确：FeSe 的向列序纯粹是轨道起源（d_xz/d_yz 劈裂）[猜想]
  → ARPES 在 FeSe 中确认了轨道劈裂 [来源待验证]，但 SDW 涨落也在 FeSe 中有测量 [来源待验证]
  → FeSe 是当前争议最集中的体系，两种机制均有支持证据，尚无定论 [猜想]
```

## 4. 向列序的起源：轨道 vs 自旋？（铁基核心争议）

```
=== 向列序起源的两大阵营 ===

【自旋驱动阵营（Spin Nematic / Spin-Driven Nematic）[猜想]】

  核心论点：
  → 铁基 SDW 的两个等价波矢 Q₁=(π,0) 和 Q₂=(0,π) 在 T_SDW 以上共存
  → SDW 涨落形成"Ising-nematic"复合算符（涨落强度的 Z₂ 对称破缺）
  → 这个复合算符的凝聚（< S_Q₁ · S_Q₁ > ≠ < S_Q₂ · S_Q₂ >）即为向列序
  → 向列序是"轨道无序的磁序"，不需要独立的轨道序参量 [猜想]

  支持证据：
  → 掺杂演化中，向列序消失与 SDW 涨落消失同步（定性相关）[来源待验证]
  → 中子散射在 T_nem > T > T_SDW 区间仍测到 SDW 短程涨落 [来源待验证]
  → Raman 散射 B₂g 响应与 SDW 涨落强度定性相关 [来源待验证，Gallais 2016 类]

  困难：
  → FeSe 的向列序（T_s ≈ 90K）不伴随 SDW（T_SDW = 0）：SDW 涨落弱时向列序却很强
    → 自旋驱动图像在 FeSe 中存在张力 [猜想]
  → T_s > T_SDW 本身与"向列是 SDW 前驱"图像冲突（尽管有回应解释）[猜想]

【轨道驱动阵营（Orbital-Driven Nematic）[猜想]】

  核心论点：
  → Fe 3d 轨道的 d_xz/d_yz 简并性在 C4 对称下受保护
  → 若轨道相互作用（Hund 耦合 + 晶体场 + 轨道-晶格耦合）倾向于打破简并
  → d_xz/d_yz 轨道占据差（orbital polarization）直接产生 C4→C2 对称破缺
  → 轨道序是"主序参量"，SDW 是轨道序的次级效应（轨道各向异性 → 更好的单向嵌套 → SDW）[猜想]

  支持证据：
  → ARPES 直接测量轨道劈裂（d_xz/d_yz 能带分裂），发生在 T_nem 以下 [来源待验证]
  → FeSe 的轨道劈裂存在但无 SDW，与轨道序独立驱动一致 [来源待验证]
  → Hund 耦合（B10）在铁基多轨道体系中很重要，提供轨道简并破缺的驱动力 [猜想]

  困难：
  → ARPES 测到的轨道劈裂是否是磁序（SDW）的结果，而不是原因？
    （鸡蛋和鸡的问题：无法从 ARPES 快照判断因果 [猜想]）
  → 纯轨道序的能量尺度（什么驱动轨道简并打破？）尚无清晰第一性原理图像 [猜想]

【当前状态：两种机制可能都对（双驱动）[猜想，当前主流倾向]】

  较可能的图像（2024-2026 年共识方向）[猜想]：

  → 向列序的"普遍机制"可能不存在，需要逐体系分析 [猜想]

  → Raman 散射：分离 B₂g 响应中轨道分量（光学声子 + 轨道连续谱）
```
