## 1. 物理定义与核心思想

### 1.1 Planckian 极限的推导 [定理，量子力学起源]

```
=== Planckian 散射率的量子力学下限（散射时间上限）===

量子力学出发点：
  时间-能量不确定原理：ΔE × Δτ ≥ ℏ（标准形式：ΔE × Δt ≥ ℏ/2）
  → 物理含义：在时间 Δτ 内，能量不确定度不能小于 ℏ/Δτ

热平衡系统中的应用：
  → 热能 k_B T 是系统的特征能量不确定度（热涨落幅度 ~ k_B T）
  → 代入不确定关系：ΔE ~ k_B T → Δτ ≥ ℏ / (k_B T)

结论（Planckian 时间）[定理，量纲分析+不确定原理]：
  最短可能的散射时间：τ_P = ℏ / (k_B T)
  最大可能的散射率：1/τ_P = k_B T / ℏ

数值估算（[定理，计算值]）：
  T = 100K → τ_P = ℏ / (k_B × 100K) ≈ 76 fs（飞秒）
  T = 300K → τ_P ≈ 25 fs
  T = 10K  → τ_P ≈ 760 fs

⚠️ 重要说明（反谄媚 - 见第 9 节）：
  → ℏ/(k_BT) 是量纲分析给出的"自然"时间尺度
  → 其作为"量子力学允许的最短散射时间"的论证基于热力学不确定关系
  → 这个论证不如 KSS 极限（η/s ≥ ℏ/(4πk_B)，从全息精确推导）严格
  → "Planckian 极限"更多是经验分类标准，而非严格定理
```

### 1.2 Planckian 散射系数 α [定理，定义]

```
=== α 的定义：无量纲化的散射率 ===

定义（中心公式）：
  LaTeX: \alpha \equiv \frac{\hbar}{\tau k_B T}
  ASCII: alpha ≡ hbar / (tau * k_B * T)

等价形式（对应 Planckian 频率）：
  LaTeX: \frac{\hbar}{\tau} = \alpha k_B T
  ASCII: hbar/tau = alpha * k_B * T

物理解读：
  → α 是实际散射率（ℏ/τ）与热激发频率（k_B T）的比值
  → α << 1：准粒子寿命远超热激发周期（Fermi 液体区域，准粒子良好定义）
  → α ~ 1：散射率达到热激发频率量级（Planckian 极限，准粒子概念可疑）
  → α >> 1：超 Planckian（若真实出现，意味着理论框架有问题，
              或实验提取 m* 和 n 有系统误差）

Fermi 液体中的 α（对比）[定理，Fermi 液体理论]：
  → Fermi 液体：ℏ/τ ∝ (k_B T)² / E_F（相空间限制）
  → α_FL = ℏ/(τ k_B T) ∝ k_B T / E_F << 1（低温 T << T_F）
  → 典型金属（T = 100K，E_F = 5eV ~ 58000K）：α ~ 100/58000 ~ 0.002 << 1
  → Strange Metal：α ~ 1（与 Fermi 液体差 3 个数量级！）[观察]
```

### 1.3 从电阻率到 α 的转换 [定理，Drude 框架]

```
=== 实验测量 α 的核心步骤 ===

Drude 公式（[定理]，电导率半经典推导）：
  σ = n e² τ / m*
  → 电阻率：rho = m* / (n e² tau)

代入 α 的定义（tau = hbar / (alpha * k_B * T)）：
  LaTeX: \rho = \frac{m^*}{n e^2 \tau} = \frac{m^* k_B T}{n e^2 \hbar} \cdot \alpha
  ASCII: rho = (m* * k_B * T) / (n * e^2 * hbar) * alpha

整理：
  alpha = (rho * n * e^2 * hbar) / (m* * k_B * T)
        = (d rho/dT) * (n * e^2 * hbar) / (m* * k_B)

物理含义（对 Strange Metal）：
  → ρ ∝ T 意味着 dρ/dT = const，即 α 不依赖 T（[观察，物理推导]）
  → 从 Linear-T 斜率 dρ/dT 提取 α，需要额外知道 m* 和 n

实验提取 α 的方法（铜氧化物）[来源待验证]：
  → m*：通过 ARPES（角分辨光电子谱）测量 Fermi 速度 v_F，再用 m* = ℏ k_F / v_F
  → n（载流子浓度）：通过霍尔效应测量，n_H = 1/(R_H e)
  → 注意：n_H 是霍尔浓度（单带 Drude 假设），多带体系 n_H 的物理意义更复杂

⚠️ 关键不确定性（反谄媚 - 见第 9 节）：
  → m* 和 n 的测量本身有不确定性（ARPES 样品依赖，霍尔效应多带问题）
  → 不同实验组对同一材料提取的 α 可以相差 2-3 倍
  → "α ≈ 1"的精度是量级估算（order-of-magnitude），非精确数值
```

## 5. Planckian 极限与超导 Tc（重要猜想）

  → ⚠️ Homes 定律是现象学关系，其物理意义仍有争议 [来源待验证]

猜想的层级总结：
  [定理]：BCS 声子上限 Tc < 40K（McMillan 公式适用范围内）
  [猜想]：Planckian 散射率限制 Tc（有定性论点，无严格推导）
  [观察]：最优掺杂（Tc 最高）↔ α 最小（最接近 Planckian）的相关性

## 6. Planckian 极限的理论框架（四大路径，均为 [猜想]）

```
=== 为什么 α ≈ 1？四种理论解读 ===

【路径 A：全息 AdS/CFT 框架 [猜想，数学框架]】

  物理图像：
    → 没有准粒子的量子液体 ↔ AdS 时空中的黑洞
    → 黑洞视界的热化时间 = Planckian 时间（ℏ/(k_B T)）
    → 视界附近的 Hawking 辐射 → 自然给出 Planckian 热化率

  关键结果（全息）：
    → 粘滞率下限：eta/s = hbar/(4*pi*k_B)（KSS 极限，[定理，全息框架内严格]）
    → 散射率：1/τ = α k_B T/ℏ，全息模型给 α = O(1) [来源待验证，模型依赖]

  优势：
    → 没有准粒子的体系的自然描述语言
    → 自动给出 Planckian 量级（不需要精细调参）

  弱点：
    → 全息是数学对应（AdS/CFT），不是微观物理图像
    → 真实材料（铜氧化物晶格 + 电子）如何映射到 AdS₅ 几何？完全不清楚
    → 全息给 α = O(1) 但给不出精确数值（为什么不是 2 或 0.5？）

【路径 B：量子临界金属 / Hertz-Millis 理论 [猜想]】

  物理图像：
    → 量子临界点（QCP）附近，量子涨落作为散射浴
    → 涨落的关联时间 τ_c ~ ℏ/(k_B T)（量子临界区的动力学标度）
    → → 电子-涨落散射率：1/τ ∝ k_B T/ℏ（给 Planckian 量级）

  对铜氧化物的适用性：
    → QCP 是候选机制，但铜氧化物 QCP 存在性有争议（见 H1）
    → Hertz-Millis（3D AFM QCP）严格给 1/τ ∝ T^(3/2)，不是 T [来源待验证，理论]
    → 2D 版本可能给线性 T，但具体指数依赖 QCP 类型 [猜想，模型依赖]

  关键矛盾 [猜想层级]：
    → 标准量子临界理论 ≠ 线性 T 散射率（需要精细的 QCP 类型和维度）

【路径 C：RVB / Mottness / 预形成 Cooper 对 [猜想]】

  物理图像（Anderson, Lee, Nagaosa 等 [来源待验证]）：
    → 铜氧化物的 Mott 绝缘体背景：Mottness 残余
    → 预形成 Cooper 对存在于 Tc 以上（赝能隙区见 H3），但相位不相干

  优势：
    → 对铜氧化物相图的直觉（赝能隙 → 相位涨落）
    → 与 H3（赝能隙）节点有自然关联

  弱点：
    → 预形成 Cooper 对的实验证据仍有争议 [来源待验证]
    → 对铁基、MATBG 等体系（Mottness 较弱）的适用性不清楚
    → 相位涨落如何精确给出 α ≈ 1 未严格推导 [猜想]

【路径 D：SYK / 随机矩阵 / 最大混沌 [猜想，玩具模型]】

  物理图像（Kitaev 2015，Sachdev 等 [来源待验证]）：
    → N 个费米子的全关联随机耦合 → SYK 模型，精确可解

    → Lyapunov 指数：λ_L = 2πk_BT/ℏ（最大混沌，Maldacena-Shenker-Stanop 界 [来源待验证]）

  → 层级：[猜想] — 竞争框架，无公认答案

```
=== Planckian 散射是否违反 WF 定律？===

Wiedemann-Franz 定律（[定理，Fermi 液体]）：
  Lorenz 数：L₀ = π²k_B²/(3e²) ≈ 2.44 × 10⁻⁸ WΩK⁻²
  ASCII: L_0 = (pi^2 * k_B^2) / (3 * e^2)
  成立条件：弱关联 Fermi 液体，弹性散射主导，热载体 = 电载体（准粒子）

  WF 定律（[定理]）：
  κ / (σ T) = L₀（Lorenz 关系）

  物理起源（[定理]）：
  → 同一准粒子既传热又传电，二者比值由统计力学决定（不依赖散射细节）
  → 只要散射是弹性的且导热体 = 导电体，WF 成立 [定理，Fermi 液体框架]

Planckian 耗散下的期望 [猜想]：
  → 若散射达到 Planckian 极限（1/τ ~ k_BT/ℏ），准粒子概念已经模糊
  → 热导率 κ 和电导率 σ 的载体可能部分分离（声子携带热但不导电）
  → 预期：WF 偏离（κ/(σT) ≠ L₀）
  → 方向：L > L₀ 还是 L < L₀ 取决于具体机制 [猜想，不同理论预言不同]

实验状态（铜氧化物，最优掺杂）[来源待验证，争议中]：
  → Proust et al.（YBCO 等）：WF 定律在某些条件下接近成立 [来源待验证]
  → Hill et al. / 其他组：LSCO 某些条件下报道 L ≠ L₀ [来源待验证]
  → 争议来源：
    (1) 声子热导率的分离困难（热导率测量 = 电子 + 声子混合贡献）
    (2) 不同测量温区和磁场条件下结论不同
    (3) 最优掺杂区域的 WF 违反幅度较小，可能被误差掩盖

  → 铁基：多带体系，WF 定律本身在多带 Drude 框架下已有修正
  → Romero 精读未涉及 WF 测量（基于 ρ + R_H 提取 α）[来源：已精读]

层级总结：
  [定理]：WF 定律在 Fermi 液体（弹性散射，准粒子良好定义）中成立
  [猜想]：Planckian 散射下 WF 违反（机制依赖，方向不确定）
  [观察，争议]：铜氧化物最优掺杂的 WF 违反程度 [来源待验证]

⚠️ 实验上重要但结论尚未确定，2026 年仍是活跃前沿

## 9. 关键反谄媚（Anti-Sycophancy）

```
⚠️ 反谄媚 H2.1：Planckian 极限的理论论证不如 KSS 严格

  常见表述："ℏ/(k_BT) 是量子力学禁止突破的散射下限"
  实际层级：[量纲分析 + 热力学不确定关系 + 经验观察]，非严格定理

  对比 KSS 极限（[定理，全息框架内严格]）：
  → η/s = ℏ/(4πk_B)：从 AdS/CFT 严格推导，粘滞率的精确下界
  → Planckian τ_P = ℏ/(k_BT)：从不确定原理+热力学量纲分析得到，论证更弱
  → "最短散射时间 = Planckian 时间"是合理推测，不是数学证明

  正确表述：
  → "在强关联 Strange Metal 中，实验观测到散射率接近 k_BT/ℏ 量级" [观察]
  → "量子力学不确定原理给出量纲上的 k_BT/ℏ 作为自然散射单位" [物理推导]
  → 二者结合支持 α ≈ O(1) 是有意义的极限，但非严格定理 [猜想]

⚠️ 反谄媚 H2.2：α ≈ 1 的"普适性"可能是数量级估算的假象

  常见表述："从铜氧化物到 MATBG，所有 Strange Metal 都有 α = 1"
  实际层级：[量级估算（order-of-magnitude），系统误差大]

  具体问题：
  → m* 的来源：ARPES（铜氧化物）vs 比热（重费米子）vs 带隙估算（MATBG）
                 → 不同方法给出 m* 可以相差 2-5 倍
  → n 的来源：Hall 测量（多带修正？）vs 填充推算（Moiré）
                 → 多带体系 n_H ≠ 真实载流子浓度
  → α ∝ 1/(m* × n)：m* 和 n 的双重误差 → α 误差可达 3-10 倍
  → "0.5 到 3 都算 α ≈ 1"：这是数量级一致（量级相同），不是精确测量

  应有结论：
  → "不同强关联体系的 α 在 0.5-3 范围内集中" [观察，可信]
  → "α 精确等于 1（误差 < 10%）" [不成立]
  → "普适的 α = 1 来自共同机制" [猜想，需要理论，实验精度不足以确认]

⚠️ 反谄媚 H2.3：从 ρ 提取 α 的模型依赖性

  常见操作："用 Drude 公式 rho = m*/(n e² tau) 直接提取 τ，然后算 α"
  问题：Drude 公式假设
  → 单弛豫时间 τ（实际：非 Drude 型光导率 σ(ω)，有频率依赖）
  → 单带（实际：铁基、重费米子都是多带）
  → m* 不随 T 变化（实际：Mott 绝缘体附近 m* 可以有 T 依赖）
  → n 不随 T 变化（实际：铜氧化物赝能隙区 n(T) 可能变化）

  具体后果：
  → 如果 σ(ω) 不是 Drude 型（实际上 Strange Metal 的 σ(ω) 往往是 1/ω 型）
    → 用 DC 电阻率提取的 τ 是"有效 Drude 弛豫时间"，不是真实散射时间
    → α 的物理意义被 Drude 假设污染
  → 不同实验组选择不同的 m*（来自 ARPES / 比热 / 等离子体频率）可能给 α 相差 2-3 倍

⚠️ 反谄媚 H2.4：Homes 定律等"关联"不是因果

  常见推理："σ_n(Tc) × Tc = const → Planckian 散射决定超导 Tc"
  逻辑问题：观察到相关性 ≠ 因果关系

  正确解读：
  → Homes 定律（[观察]）：ρ_s ∝ σ_n × Tc，来自数据拟合
  → 这个关系来源可能是：
    (a) Planckian 散射决定 Cooper 对相干 → Tc [猜想]
    (b) 最优掺杂同时优化散射率和配对强度，但二者来自独立机制 [猜想]
    (c) Drude 框架的数学关系，不含特定物理 [猜想]
  → "Planckian 散射驱动超导" 和 "Planckian 散射是超导的症状" 和 "二者来自共同原因"
     在实验上难以区分

⚠️ 反谄媚 H2.5：α > 1 不一定是理论框架错误

  常见结论："α > 1 的材料违反了 Planckian 极限，说明测量有问题"
  实际层级：α > 1 可能来自：
  → Drude 模型的多带近似错误（铁基、重费米子常见）
  → m* 被低估（有效质量用 ARPES 电子带宽，忽略关联修正）
  → n 被高估（Hall 测量给多带的"等效"浓度，不是真实导电载流子）
  → 量子干涉效应（弱局域化等）影响 DC 电阻率但不影响散射率
  → 不能反过来用"α > 1 不合理"来推断"Planckian 是绝对极限"
```
