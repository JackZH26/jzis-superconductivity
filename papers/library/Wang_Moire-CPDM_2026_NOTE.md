## 核心内容

**体系:** 1QL Sb₂Te₃ / 6UC FeTe 外延异质结（拓扑绝缘体 + 反铁磁体）

**关键发现:**
1. 六方 Te 晶格（Sb₂Te₃）× 正方 Te 晶格（FeTe）→ 自然形成 moiré 超晶格
2. 两个超导能隙随 moiré 周期空间调制 → Cooper 对密度调制（CPDM）态
3. Josephson STM 直接实空间成像 CPDM，波长 = moiré 周期
4. Bi₂Te₃ 替换 Sb₂Te₃ → 可调 CPDM 周期和幅度
5. CPDM 不破坏全局平移对称性（区别于 PDW）

**Tc:** ~3-12K（依赖 Sb₂Te₃ 厚度）
**机制:** TI 拓扑表面态提供载流子 + FeTe AFM 涨落提供配对胶水
**技术:** STM/S + Josephson STM/S（实空间成像）

## 与我们研究的关联

- 不同晶系外延 moiré（六方 vs 正方）→ 全新材料空间
- quasi-2D + 双间隙 + 适度相互作用 → 符合 P1 甜点方向
- CPDM 可调性 = moiré 作为 Cooper 对空间分布的旋钮
- Tc 低但可控性强 → 基础物理平台价值 > Tc 追求
- glide symmetry breaking + nematic SC → 新的对称性分析目标
