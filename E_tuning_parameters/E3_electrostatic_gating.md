# E3. 静电门控 — 可逆载流子密度调控与 2D 超导感应

**MECE 编号：** E3
**关联 MECE：**
- D11（2D/界面超导 — 最主要的门控 SC 材料族：FeSe/STO、LAO/STO、TMD 等）
- D13（2D 半导体 — MoS₂/NbSe₂/TMD 系列，离子液体门控 SC 的主战场）
- E2（化学掺杂 — 对比：门控 vs 化学掺杂；无序的独立贡献）
- F8（BKT 相变 — 门控 2D SC 的相变机制；Berezinskii-Kosterlitz-Thouless 涡旋解束）

**层级关系：** E3 是**专为 2D 材料和超薄薄膜设计的可逆调控手段**。与 E2（化学掺杂）最根本的区别：门控不引入化学无序，且完全可逆——撤去栅压即回到初始态。E3 与 E2 的协同对比实验是分离"载流子效应"和"无序效应"的关键手段（见第 6、9 节）。E3 对体材料无效（门控深度 < 2 nm），几乎专属于 2D/超薄薄膜体系（D11/D13）。
**精读日期：** 2026-04-28
**精读模型：** Claude Sonnet 4.6（high thinking）
**层级：** 综述（FET 原理/固体栅/离子液体栅/EDLT；反幻觉规则严格执行；[来源待验证] 大量标注）

## 1. 物理定义与核心思想

### 1.1 FET 原理与静电门控

**[[定理]：FET 物理（平行板电容），→ [猜想]（对超导的具体机制）]**

```
场效应晶体管（FET）原理 [定理，基础静电学 + 半导体物理]：

结构：
  金属栅极（Gate）
       |
  介质层（Dielectric）: HfO₂、SiO₂、离子液体等
       |
  超导导电沟道（超薄薄膜 / 2D 材料）
       |
  基底（Substrate）

工作原理：
  外加栅压 V_g → 介质两侧产生电场 E = V_g / d（d = 介质厚度）
  → 在导电沟道表面感应出载流子（电子或空穴）
  → 表面载流子密度 n₂D 变化 → Fermi 能级 EF 移动
  → 超导临界温度 Tc 和其他超导参数随之变化

感应载流子密度公式 [定理，平行板电容]：

  n₂D = C_g * V_g / e

  其中：
    n₂D [cm⁻²]   = 二维载流子密度（面密度）
    C_g  [F/cm²]  = 单位面积栅电容
    V_g  [V]      = 栅压（相对源极）
    e    [C]      = 元电荷（1.6 × 10⁻¹⁹ C）

  平行板电容公式 [定理]：
    C_g = ε₀ * κ / d
    其中：κ = 介质相对介电常数，d = 介质厚度

关键参数对比：
  固体栅（SiO₂, d = 300 nm, κ ≈ 3.9）：
    C_g ≈ 10 nF/cm² → n₂D_max ≈ 10¹³ cm⁻²（受击穿电压 ~10V 限制）

  固体栅（HfO₂, d = 10 nm, κ ≈ 20）：
    C_g ≈ 1.8 μF/cm² → n₂D_max ≈ 10¹⁴ cm⁻²（受击穿场强限制）[来源待验证]

  离子液体栅（EDL，等效 d ≈ 1 nm, κ_eff 大）：
    C_g ≈ 5–10 μF/cm²（实测）[来源待验证]
    → n₂D_max ≈ 10¹⁴–10¹⁵ cm⁻²（比传统固体栅高 1–2 量级）
```

### 1.2 门控 vs 化学掺杂的核心对比

**[这是 E3 区别于 E2 的核心价值所在]**

```
=== 可逆性（Reversibility）===

化学掺杂（E2）：
  ❌ 不可逆：Sr 进入 La 格位，无法撤回
  → 实验上：每个掺杂浓度需要一个独立样品
  → 代价：无法排除样品间差异（批次效应）

静电门控（E3）：
  ✅ 完全可逆：V_g = 0 → n₂D 回到初始值 → Tc 回到初始态（或零）
  → 实验上：同一样品扫描全相图
  → 优势：消除样品间差异；同一个器件做完整掺杂相图

=== 化学无序（Chemical Disorder）===

化学掺杂（E2）：
  ❌ 每个杂质离子 ≈ 一个散射中心（电荷 + 质量差异）
  → 平均自由程 l ∝ 1/x（x = 杂质浓度）
  → d-wave SC 对此高度敏感（见 E2 Anderson 定理）

静电门控（E3）：
  ✅ 纯静电感应：载流子积累在界面/表面 ~ 1–2 nm 的薄层中
  → 理想情况：无杂质引入，无化学无序
  → 相同载流子密度下，门控 SC 的平均自由程 >> 化学掺杂
  ⚠️ 例外：离子液体门控可能引入电化学掺杂（见第 5 节反谄媚）

=== 调控深度限制 ===

门控的物理局限 [定理，Thomas-Fermi 屏蔽]：
  金属/半导体的 Thomas-Fermi 屏蔽长度：
    λ_TF = sqrt(ε₀ * ε_r * EF / (3 * n * e²))（自由电子近似）
  
  典型值：
    良导体（n ~ 10²² cm⁻³）：λ_TF ~ 0.1–0.5 nm → 门控仅影响极表面
    2D 材料 / 半导体（n ~ 10¹² cm⁻²）：λ_TF ~ 几 nm → 门控有效

  ⚠️ 对体材料超导体（如 YBCO 体材料，n ~ 10²¹ cm⁻³）：
    门控深度 < 1 nm → Tc 调制幅度极小（~1K）
    → 门控对体超导几乎无效 → E3 专属于 2D/超薄体系 [定理（屏蔽论证）]
```

## 2. 历史关键节点

**[[来源标注可靠性分类：确定/较确定/来源待验证]]**

```
★ 1960s–2000s：常规 FET 超导感应失败

  早期尝试：Si、GaAs、Ge 等半导体的 FET 超导感应实验
  失败原因：
    固体栅 C_g ~ 10 nF/cm² → n₂D_max ~ 10¹² cm⁻²
    大多数超导体需要 n₂D >> 10¹³ cm⁻² 才能形成 Cooper 对
    → 载流子密度不够，Tc 无法感应到可测量值
  意义：固体栅 FET 对超导的感应是否可行，在 2000 年代前是未解问题

★★★ 2008 Ueno et al.（东京大学）— SrTiO₃ 离子液体门控，首次氧化物 SC 感应

  来源：K. Ueno et al., Nature Materials 7, 855 (2008)（来源确定）
  材料：SrTiO₃（STO）— 钙钛矿氧化物绝缘体/半导体
  技术：离子液体（DEME-TFSI）栅
  结果：Tc ≈ 0.4K（载流子密度 ~ 10¹³–10¹⁴ cm⁻² 量级）
  意义：第一次通过离子液体门控在氧化物上实现超导感应
         证明离子液体栅可以突破固体栅的载流子密度限制
  ⚠️ Tc = 0.4K 较低，且 STO 本身有低 Tc 超导（Tc ~ 0.35K at optimal carrier density）
     门控是否真正"感应"了全新的 SC？还是调控了 STO 内在的 SC？[观察]

★★★ 2012 Ye et al.（东京大学）— MoS₂ 离子液体门控 SC

  来源：J.T. Ye et al., Science 338, 1193 (2012)（来源确定）
  材料：MoS₂（二硫化钼）— 原本是半导体（禁带宽度 ~1.8 eV for monolayer）
  技术：离子凝胶（ionic gel）栅
  结果：Tc = 10.8K（n₂D ~ 1.2 × 10¹⁴ cm⁻²）
  意义：第一次在 TMD 材料上实现离子液体门控 SC
         证明 2D 半导体可以通过门控进入超导态
         开启门控 TMD 超导的研究热潮
  ⚠️ 后续工作对 Tc 值和机制有进一步讨论（多体效应？VHS？）[来源待验证]

★★ 2012 Bollinger et al. — La₂CuO₄ 超薄膜门控

  来源：A.T. Bollinger et al., Nature 472, 458 (2011)（来源待精确核查；年份可能 2011）
  材料：La₂CuO₄（LCO）超薄膜（铜氧化物）
  技术：固体栅（LSAT 基底）
  结果：通过静电门控调控铜氧化物的超导
  意义：证明固体栅对超薄铜氧化物有效（虽然调控幅度有限）
  ⚠️ 来源年份和期刊需精确核查 [来源待验证]

★★ 2014 Yu et al. — ZrNCl 离子液体门控，Tc = 15K

  来源：[来源待验证：Yu et al. 2014?，期刊需核查]
  材料：ZrNCl（层状氮化物）
  技术：离子液体栅
  结果：Tc 最高 ≈ 15K（离子液体门控）
  意义：ZrNCl 是研究离子液体门控和离子插层效应混淆问题的重要案例
  ⚠️ ZrNCl 的超导可能部分来自离子插层（非纯静电效应）[猜想，来源待验证]

★★ 2016 前后 — TMD 系列门控 SC 系统研究

  MoSe₂、WS₂、WSe₂、NbSe₂ 等 TMD 材料的离子液体门控研究
  [来源待验证：Xi 2016？多个组，多篇论文]
  集体意义：建立 TMD 门控 SC 的系统知识；发现 Ising SC（强 SOC 限制 Cooper 对方向）[来源待验证]

★★ 2009 Caviglia et al. — LAO/STO 界面固体栅

  来源：A.D. Caviglia et al., Nature 456, 624 (2008)（来源待精确核查；可能 2008）
  材料：LaAlO₃/SrTiO₃（LAO/STO）氧化物界面
  技术：固体栅（背栅）
  结果：通过栅压连续调控 2DEG 密度 → Tc 从 ~0 到 ~0.3K
  意义：完美展示固体栅在氧化物界面的精确调控能力

★★ 2020+ — vdW 异质结 + 门控（MATBG 等）
```

## 3. 固体栅技术

**[传统 FET 结构，低载流子密度，适合低载流子密度体系]**

```
=== 结构与工作参数 ===

结构示意：
  金属栅（Au / Pt / Al）
       |
  高 κ 介质（HfO₂ / Al₂O₃ / SiO₂ / 铁电层）
       |
  超导薄膜（< 5 nm）或 2D 材料（单层/少层）
       |
  基底（SrTiO₃ / LSAT / Si）

关键参数（[定理，平行板电容]）：
  SiO₂（d = 300 nm, κ ≈ 3.9）：
    C_g ≈ 10 nF/cm² → n₂D_max ≈ 10¹³ cm⁻²

  HfO₂（d = 10 nm, κ ≈ 20）：
    C_g ≈ 1.8 μF/cm² → n₂D_max ≈ 10¹⁴ cm⁻²（受击穿场强 ~ 4 MV/cm 限制）
    [来源待验证：具体 κ 和击穿场强依赖沉积方法]

  SrTiO₃ 背栅（低温下 κ 极大！）：
    低温下 STO κ 可达 10⁴（在 ~10K 以下，量子顺电效应）[来源待验证]
    → C_g 极大 → n₂D_max 可超过 10¹³ cm⁻²（即使 d 较大）
    特例：STO 基底本身作为背栅介质（特殊用途）

=== 优势 ===

  ✅ 高速响应（ns 量级），适合动态调控
  ✅ 低温可用（无冻结问题，见第 5 节离子液体的冻结问题）
  ✅ 无电化学污染（无离子迁移，无插层风险）
  ✅ 清洁静电效应（真正的 E3，无 E2 混淆）
  ✅ 技术成熟，CMOS 兼容，可批量制备

=== 缺点 ===

  ❌ 载流子密度上限 ~ 10¹³ cm⁻²（SiO₂ 栅）
  ❌ 高 κ 介质（HfO₂ 等）可提高到 ~10¹⁴ cm⁻²，但击穿风险增加
  ❌ 对大多数非 2D 半导体材料的 SC 无效（需要更高 n）
  ❌ 均匀性要求高（介质厚度不均匀 → n₂D 空间涨落）

=== 典型应用 ===

LAO/STO 界面固体栅（最成功案例）[来源待验证]：
  背栅调控 2DEG 密度：n₂D 从 ~ 0 → 最优 → 过调
  → Tc 从 0 调到 ~0.3K
  → 可在单样品上扫描完整 Tc-n 相图（优于化学掺杂系列样品方法）
  → Rashba SOC 随 V_g 变化 → 可研究 SOC 调控的拓扑相变 [猜想，来源待验证]

超薄 YBa₂Cu₃O₇ 薄膜固体栅 [来源待验证]：
  调制幅度 ~ 1K（很小！）
  原因：YBCO 是高 Tc 铜氧化物，体载流子密度 n ~ 10²¹ cm⁻³
        门控仅影响表面 < 1 nm → 相对于总载流子数变化极微

MATBG 固体栅 [来源待验证]：
  扭转双层石墨烯的本征载流子密度极低（~ 10¹² cm⁻²/平带填充）
  → 固体栅（hBN 介质）完全够用
  → 精确调控从 Mott 绝缘体到 SC 到磁性态的全相图
  ⚠️ MATBG 是特殊极限，其超低载流子密度不代表通常超导体 [来源待验证]
```

## 4. 离子液体/离子凝胶栅技术

**[最重要的门控技术，突破载流子密度限制；但有电化学混淆风险！]**

```
=== 工作原理 ===

离子液体（Ionic Liquid, IL）的特性：
  定义：完全由离子组成的液体（室温下），无溶剂分子
  常用离子液体：
    DEME-TFSI（[N,N-diethyl-N-methyl-N-(2-methoxyethyl)ammonium bis(trifluoromethylsulfonyl)imide]）
    [EMIM][TFSI]（1-ethyl-3-methylimidazolium bis(trifluoromethylsulfonyl)imide）
    [EMI][BF₄]（1-ethyl-3-methylimidazolium tetrafluoroborate）

电化学双层（Electric Double Layer, EDL）形成过程：
  1. 在超导材料/离子液体界面施加 V_g
  2. 离子液体中的阴/阳离子向界面迁移（阳离子 → 负极，阴离子 → 正极）
  3. 在界面形成 ~ 1 nm 厚的致密离子层（Helmholtz 层）
  4. 等效于极薄（~ 1 nm）绝缘层的平行板电容器

EDL 电容参数 [来源待验证]：
  Helmholtz 层厚度 d_H ~ 1 nm（量级）
  有效 κ_eff ~ 10（量级，取决于离子液体种类）
  C_EDL ~ ε₀ * κ_eff / d_H ~ 10 μF/cm²（比固体栅高约 1000 倍！）
  → n₂D_max = C_EDL * V_g_max / e
    取 V_g_max ≈ 3V（保守，避免电化学反应）：
    n₂D_max ≈ 3 × 10 × 10⁻⁶ / 1.6 × 10⁻¹⁹ ≈ 2 × 10¹⁴ cm⁻²
  [数值估算，来源待核查精确实验值]

=== 离子凝胶（Ionic Gel）===

  定义：聚合物基质（如 PEO）中包裹离子液体 → 固态/半固态
  关键优势：
    固态，机械稳定（不流动）
    可以在低温下长期使用（到液氮温区 ~80K）[来源待验证]
    可通过旋涂直接沉积在器件上（工艺兼容性好）
  C_g 与离子液体类似：~ 5–10 μF/cm² [来源待验证]
  典型应用：Ye 2012 MoS₂ 实验使用的正是离子凝胶 [Ye 2012, Science 338, 1193]

=== 典型离子液体门控工艺 ===

  步骤：
  1. 制备超薄薄膜/2D 材料器件（光刻图案化，金属电极）
  2. 在器件上方滴加或旋涂离子液体/离子凝胶
  3. 覆盖栅极（Pt 丝 / Pt 片 / 碳电极）
  4. 在高温（~ 220K 以上，离子液体未冻结）下施加 V_g
  5. 载流子积累到目标密度后，降温到超导测量温度（< 10K）
  6. 离子液体在低温下冻结 → n₂D 被"锁定"（不再响应 V_g 变化）
  7. 进行电输运测量（R-T 曲线，磁场依赖等）

  ⚠️ 低温冻结的后果（见第 9 节反谄媚）：
    ~ 200K 以下离子液体冻结 → 无法在低温下动态调控
    → 必须在高温（220–300K）设定 V_g 后再降温测量
    → 高温阶段的电化学反应风险更高！
```

## 5. 离子液体门控的核心风险：静电 vs 电化学

**[⚠️ 最重要的反谄媚核心——很多"门控 SC"可能是化学掺杂！]**

```
=== 问题的本质 ===

理想静电门控（E3 的纯粹形式）：
  离子聚集在界面形成 EDL → 仅通过电场感应载流子
  离子不进入超导材料内部
  → 可逆（撤去 V_g → 离子分散 → n₂D 回零 → Tc 回零）
  → 无化学无序（超导材料内部无外来离子）

电化学反应（E2 的污染）：
  高 V_g（> 3–4 V）或高温（> 220–300K）→ 离子液体中的离子获得足够能量
  → 离子穿越界面，进入超导材料晶格（插层，intercalation）
  → 离子插层 = 化学掺杂（见 E2 第 1 节，类型 4）
  → 后果：
    (a) 载流子密度的增加来自化学掺杂，而非纯静电效应
    (b) 可能引入化学无序（外来离子 = 散射中心）
    (c) 不可逆：撤去 V_g 后，插层离子无法自动离开

=== 如何区分静电 vs 电化学掺杂 ===

判断标准（[定理层级的判据]）：

  判据 1：可逆性检验（最直接）
    V_g 先上升 → Tc 升高（感应 SC）
    V_g 归零 → 测量 Tc
    ✅ 静电：Tc 回到初始值（或零）
    ❌ 化学掺杂：Tc 保持不变（插层离子留在晶格内）
    ⚠️ 很多早期离子液体门控论文没有做 V_g → 0 后的 Tc 测量！

  判据 2：时间依赖性
    ✅ 静电：施加 V_g 后 n₂D 快速响应（取决于离子扩散速率，~ 分钟量级）
    ❌ 电化学：插层过程更慢，且与温度强相关（高温更快）
    → 如果 n₂D 随时间持续增加（对恒定 V_g），可能有插层 [观察]

  判据 3：离子液体冻结后的稳定性
    对静电门控：冻结后 n₂D 稳定（EDL 冻结，电荷不变）
    对电化学掺杂：冻结后 n₂D 稳定是预期的，但冻结前的插层已完成
    → 此判据无法独立区分，需结合判据 1

  判据 4：原位 X 射线/中子衍射
    电化学插层 → 晶格参数变化（层间距增大）[来源待验证]
    纯静电门控 → 晶格参数不变（仅表面载流子密度变化）
    → 最直接的结构证据，但实验难度极高（超薄样品 + 低温）

=== 实际体系的风险评估 ===

高风险（可能有电化学混淆）：
  ZrNCl 离子液体门控：
    V_g 高（> 4V），高 Tc（15K）
    Li⁺ 插层 ZrNCl 已知 → Lix ZrNCl 化学掺杂也能导致 SC
    → 很多 ZrNCl"门控 SC"可能是插层 SC [猜想，来源待验证]
  
  SrTiO₃ 离子液体门控（Ueno 2008）：
    后续研究显示 STO 对质子（H⁺）有敏感性
    → 微量水分 + 高 V_g → H⁺ 插层可能 [猜想，来源待验证]

低风险（较清洁的静电效应）：
  MoS₂ 离子液体门控（Ye 2012）：
    V_g 相对适中，可逆性已检验（文中有 V_g 关断后的测量）[来源待验证]
    → 较可信的静电门控 SC
  
  LAO/STO 固体栅（Caviglia）：
    固体栅无离子，无插层风险
    → 最干净的 E3 实例 [来源待验证]

=== 早期文献的系统性偏差 ===

  ⚠️ 估计相当比例（可能 30–50%？无文献统计，[来源待验证]）的早期离子液体
     门控 SC 论文没有做充分的可逆性检验
  → 这些实验结果可能混入了不同程度的电化学掺杂
  → 在引用这些工作时必须注意：声称的"门控 Tc"可能是掺杂 Tc
```

## 6. 门控 2D 材料超导：主要材料体系

**[E3 最主要的应用领域；与 D11/D13 深度交叉]**

```
=== MoS₂ 门控超导（经典案例）===

  基础材料：2H-MoS₂（半导体，禁带宽度 ~1.8 eV，单层）
  本征超导：无（需要门控或化学掺杂）
  
  离子液体门控 SC（Ye 2012）[Science 338, 1193，来源确定]：
    离子凝胶栅 → n₂D ~ 1.2 × 10¹⁴ cm⁻²
    Tc = 10.8K
  
  门控机制（[猜想]）：
    高 n₂D → EF 移动到 Mo 4d 带的更高能量
    → EF 接近 van Hove 奇点（vHs）→ N(EF) 增强 → λ 增大 → Tc 升高
    → 继续增加 n₂D → EF 越过 vHs → N(EF) 降低 → Tc dome（先升后降）
    → Tc-V_g 的 dome 形状类似化学掺杂相图 [来源待验证]
  
  Ising 超导（2D MoS₂ 的特殊物理）[来源待验证]：
    MoS₂ 缺乏空间反演对称（非中心对称 2H 结构）
    → 面内 Rashba + Ising SOC：K 和 K' 谷的自旋方向固定（面外方向）
    → Cooper 对自旋被 Ising SOC 锁定在面外
    → 面内磁场（H ∥ 面内）无法打破配对 → 上临界场 Hc₂ >> Pauli limit
    → 这是 MoS₂ 门控 SC 的独特 2D 物理 [猜想，来源待验证]

=== LAO/STO 界面门控超导 ===

  基础体系：LaAlO₃/SrTiO₃（LAO/STO）氧化物界面
  界面 2DEG：在 LAO 厚度 > 4 u.c. 时自发形成（原因争议：重构 vs 离子转移 [来源待验证]）
  本征 Tc（最优）：~ 0.2–0.3K（极低，但可测量）[来源待验证]
  
  固体栅调控（Caviglia 2008）[来源待验证]：
    背栅 V_g（通过 STO 基底）→ 精确控制 2DEG 密度
    Tc-n₂D 完整相图：
      V_g 过负 → n₂D 耗尽 → Tc = 0（绝缘态）
      V_g 最优 → Tc_max ~ 0.3K
      V_g 过正 → 过掺杂 → Tc 降低
    
  Rashba SOC + 门控（拓扑物理 [猜想]）：
    LAO/STO 界面有强 Rashba SOC（来自界面破缺对称性）
    V_g 改变 n₂D → 同时改变 Rashba SOC 强度（EF 移动改变局域势）
    → 可能在 SC + SOC 区域实现类拓扑 SC 态 [猜想，来源待验证]
    → 目前没有确凿实验证据 [观察]

=== 超薄铜氧化物薄膜门控 ===

  材料：超薄 YBCO 薄膜（~1–5 u.c.）、La₂CuO₄ 超薄膜
  
  固体栅调控 Tc（Bollinger 2011 等）[来源待验证]：
    La₂CuO₄ 超薄膜：固体栅可调控 Tc
    调制幅度：受限（几 K），原因：铜氧化物 n 大，门控深度 ~ 1 u.c.
    意义：证明门控可以调控铜氧化物的 SC，但效果远不如 2D 半导体
  
  ⚠️ 对 YBCO 体材料：门控完全无效
     体材料 n ~ 10²¹ cm⁻³，Thomas-Fermi 屏蔽长度 < 0.5 nm
     → 门控能引入的载流子 << 体材料总载流子

=== 有机超导体门控 ===

  部分有机 SC 通过离子液体门控感应超导（目的：避免化学掺杂的无序）
  [来源待验证：具体体系和文献需核查]
  
  概念意义：
    化学掺杂 BEDT-TTF 盐 → 引入阴离子无序
    门控方法 → 理论上无序更小 → 可研究无序对有机 SC 的独立贡献
  
  实际挑战：有机分子晶体通常不稳定，薄膜质量难以控制 [来源待验证]
```

## 7. EDLT 极端情况与超导相图的精确扫描

**[Electric Double Layer Transistor — 门控的终极形态，也是风险最高的形态]**

```
=== EDLT 的定义与特点 ===

  EDLT = Electric Double Layer Transistor（电化学双层晶体管）
  技术：离子液体/离子凝胶栅，高 V_g（2–5V 范围）
  特点：
    C_g ~ 10 μF/cm² → n₂D 可到 10¹⁴–10¹⁵ cm⁻²
    V_g 可以在单个器件上扫描完整 SC 相图
    → 效果等同于"一键完成化学掺杂系列实验"

=== ZrNCl 的 EDLT 相图 ===

  体系：ZrNCl（层状氮化物，本征绝缘体）
  离子液体门控结果：
    V_g 增加 → Tc 从 0 升到 ~ 15K（全相图） [来源待验证]
  
  重要问题（反谄媚）：
    ZrNCl 的化学掺杂路径：Li₀.ₓZrNCl（Li 插层）也能实现 SC
    → 问题：离子液体"门控"的 Tc 到底是静电效应还是 Li⁺/Na⁺ 插层？
    → 可逆性检验：撤去 V_g 后 Tc 是否回零？
    [来源待验证：是否有充分的可逆性数据？]

=== MATBG 的门控相图（固体栅特例）===

  扭转双层石墨烯（MATBG, θ ≈ 1.1°）：
    平带使得 n 极低（每个平带 ~ 4 个载流子，对应 n₂D ~ 2.4 × 10¹² cm⁻²）
    → 固体栅（hBN 介质）完全足够：C_g ~ 10 nF/cm² 就足以扫描全相图
    → 不需要离子液体！
  
  门控相图（[来源待验证]）：
    n/n₀ = -2（半填充，类 Mott 绝缘体）
    n/n₀ = -2 ± δ（Mott 绝缘体旁的 SC）
    n/n₀ = 0（Dirac 点）
    ... 多个对称位置出现 Mott 态和 SC 态
  
  ⚠️ MATBG 是超低载流子密度的特例！
    对其他超导体（YBCO：n ~ 10²¹ cm⁻³；NbSe₂：n ~ 10¹⁴ cm⁻²），
    MATBG 的固体栅方案直接失效 → 不能照搬 [来源待验证]

=== 完整相图扫描的科学价值 ===

  门控 vs 化学掺杂（关键科学问题）：
    相同载流子密度时：
      化学掺杂 → 有无序（散射）→ Tc_doped
      门控 → 无无序（纯 E3）→ Tc_gated
    
    如果 Tc_gated > Tc_doped（相同 n₂D）：
      说明化学无序抑制了 Tc → 无序对 SC 有害 [猜想]
    
    如果 Tc_gated ≈ Tc_doped：
      说明化学无序对此体系的 SC 影响很小（可能是 s-wave，Anderson 定理保护）[猜想]
    
    现有实验（FeSe/STO 等）[来源待验证]：系统性对比数据不足
```

## 9. 关键反谄媚

**[静电门控研究中最常见的过度解读和误导性结论]**

```
反谄媚 1：离子液体门控 ≠ 纯静电效应

  ✅ 理想情况：离子液体在界面形成 EDL → 纯静电感应载流子 → 可逆
  
  ⚠️ 实际风险：
    高 V_g（> 3–4 V）→ 电化学反应 → 离子插层（= 化学掺杂，E2！）
    高温（> 200K）+ 高 V_g → 风险更高
    早期文献（2008–2014）大量缺乏可逆性检验

  → 正确表述：
    "离子液体门控 SC"必须附带可逆性检验数据才有效
    无可逆性检验的结果只能定性为"疑似门控诱导 SC"

反谄媚 2：固体栅的载流子密度对大多数 SC 不够

  ✅ 固体栅对 2D 半导体（MoS₂，低 n）、MATBG（超低 n）、LAO/STO（低 n 2DEG）有效
  
  ⚠️ 对常规超导体（YBCO, MgB₂, NbSe₂ 等）：
    需要 n₂D >> 10¹³ cm⁻²
    固体栅（SiO₂）能提供的 n₂D ~ 10¹² cm⁻² 严重不足
    即使高 κ 介质（HfO₂），也仅能达到 ~ 10¹³–10¹⁴ cm⁻²
    → 固体栅对大多数超导体的调控效果极微（Tc 变化 < 1K）

  → 正确表述：
    固体栅是"精密微调"工具，离子液体才是"全相图扫描"工具

反谄媚 3：门控 Tc 幅度对体材料极小（~ 1K），不代表 E3 的极限

  ✅ 例子：YBCO 超薄膜固体栅调控 Tc 幅度 ~ 1K [来源待验证]
  
  解释：
    YBCO 体材料 n ~ 10²¹ cm⁻³（每单元格 ~ 1 个空穴）
    门控引入的载流子 ~ 10¹³ cm⁻²，约等于 0.01 u.c. 厚度的体材料载流子
    → 相对效应极微 → Tc 变化 ~ 1K
    
  → 正确表述：
    E3 对体材料的 Tc 幅度调控有根本性的物理限制
    门控 ~ 1K 的效果对 Tc = 92K 的 YBCO 是噪声量级
    E3 的最大价值是 2D/超薄体系（本身就是 E3 专属领域）

反谄媚 4：低温离子液体冻结问题严重限制实验设计

  ✅ 离子液体（DEME-TFSI 等）在 ~ 200K 以下冻结 → 无法在低温动态调控
  
  实验影响：
    (a) 必须在 220–300K 下设定目标 V_g，然后降温到测量温度（< 10K）
    (b) 降温过程中 V_g 被"冻结"，无法再精细调节
    (c) 高温设定阶段：电化学反应风险最高（温度高 + 离子可动）
    
  离子凝胶的改善 [来源待验证]：
    可以使用到 ~ 80K（液氮温区）
    但仍然无法覆盖 < 80K 的动态调控
    
  → 正确表述：
    "低温在位门控调控 SC"目前对离子液体技术几乎不可行
    改进方向：低冻点离子液体（新材料），或铁电栅（固体栅 + 高 κ）

反谄媚 5：MATBG 门控是极特殊的情况，不能推广

  ✅ MATBG 的固体栅门控之所以成功：
    平带 → 超低载流子密度（n ~ 10¹² cm⁻²）
    固体栅 C_g ~ 10 nF/cm² 刚好足够扫描全相图
  
  ⚠️ 对其他 SC：
    NbSe₂：n ~ 10¹⁴ cm⁻² → 固体栅不够，需要离子液体
    YBCO：n ~ 10²¹ cm⁻³ → 门控无意义（体材料）
    铁基 122（BaK）：n ~ 10²² cm⁻³ → 门控无意义（体材料）
    
  → 正确表述：
    MATBG 展示了 E3 技术在超低 n 体系的极致能力
    不能将此经验推广到常规超导体的门控调控
```
