## 1. 物理定义与核心思想

### 1.1 拓扑电荷（卷绕数）的数学定义（[定理]）

```
=== 超导序参量的相位结构与拓扑缺陷 [定理] ===

超导序参量（GL 理论框架）：
  ψ(r) = |ψ(r)| * exp(iφ(r))
  φ(r) = 序参量相位场（实数标量场，定义在 U(1) 圆上）

漩涡（Vortex）的拓扑电荷 n（卷绕数 / Winding Number）：

  定义（[定理，拓扑缺陷的精确定义]）：

    LaTeX: n = \frac{1}{2\pi} \oint_C \nabla\phi \cdot d\mathbf{l} \in \mathbb{Z}

    ASCII: n = (1/2π) * ∮_C ∇φ · dl ∈ Z（整数）

  其中：
    C = 围绕拓扑缺陷的任意闭合回路（不穿过缺陷核心）
    n = 拓扑电荷（卷绕数）= 相位绕回路一周所绕转的 2π 整数倍
    ∇φ = 相位梯度场（实空间）

拓扑电荷的物理分类：
  n = +1：正漩涡（顺时针绕一圈，相位增加 2π）
  n = -1：反漩涡（逆时针绕一圈，相位减少 2π）
  n = 0：无拓扑缺陷（相位场连续可微）
  |n| = 2, 3, ...：高阶漩涡（多重绕转，通常不稳定，见第 7 节）

拓扑电荷守恒（[定理，拓扑稳定性]）：
  → 拓扑电荷不能被连续形变（homotopy）消除
  → 守恒条件：无漩涡-反漩涡对的创生或湮灭
  → 总拓扑电荷：∑_i n_i = 常数（在无创生/湮灭过程中）
  → 正/反漩涡只能成对湮灭：n=+1 碰到 n=-1 → 总电荷 = 0，能量释放（F8 关联）

为什么 n 必须是整数？（[定理，U(1) 基本群]）：
  → 序参量 ψ(r) 必须单值：绕一圈回到原点 → φ 改变必须是 2π 的整数倍
  → 数学上：映射 S¹ → U(1) 的同伦类 = π₁(U(1)) = ℤ（整数群）
  → ℤ 的生成元 = 单漩涡（n=±1）
```

### 1.2 磁通量子化（[定理，London 1950；实验，Deaver-Fairbank & Doll-Näbauer 1961]）

```
=== 磁通量子化：拓扑电荷的物理后果 [定理] ===

每个漩涡携带量子化磁通 Φ₀（[定理，GL 方程推导]）：

  LaTeX: \Phi_0 = \frac{h}{2e} = 2.067\,833\,848 \times 10^{-15}\,\text{Wb}

  ASCII: Phi_0 = h/(2e) = 2.067833848e-15 Wb（磁通量子，精确定义）

  其中：
    h = 普朗克常数 = 6.626e-34 J·s
    2e = Cooper 对的电荷（两个电子！）
    Phi_0 = 磁通量子（fluxon）

物理推导（[定理，London 方程 + GL 方程]）：
  1. London 方程：超流电流 J_s = -(n_s e² / mc) * A（Cooper 对贡献）
  2. 磁场方程：∇ × B = μ₀ J_s
  3. 围绕漩涡积分：∮_C (J_s/n_s e + ħ/(2e)*∇φ) · dl = 0（London 方程的相位形式）
  4. 远离漩涡核心：J_s → 0（超流可以绕过核心流动）
  5. 因此：∮ ħ/(2e) * ∇φ · dl = Φ（穿过回路的总磁通）
  6. 由拓扑电荷定义：∮ ∇φ · dl = 2πn（整数倍）
  7. 故：Φ = n * (h/2e) = n * Φ₀（磁通量子化！）

关键：2e 来自 Cooper 对（[定理，严格推导]）：
  → 超导体是 Cooper 对的凝聚体，载流子是 2e（非 1e）
  → 如果载流子是单电子（e），则 Φ₀ = h/e（正常金属量子干涉，如 AB 效应）
  → Φ₀ = h/2e 是超导特有的量子，与普通量子化（h/e）不同！
  ⚠️ 反谄媚：Φ₀ 不是"普朗克常数 h 的量子化"，是 Cooper 对这个物理事实的直接体现

实验验证（[确定，1961]）：
  Deaver & Fairbank [来源待验证：Phys. Rev. Lett. 7, 43 (1961)]：
    锡薄圆柱体，测量磁通 → 量子化台阶 = Φ₀ = h/2e ✓
  Doll & Näbauer [来源待验证：Phys. Rev. Lett. 7, 51 (1961)]：
    独立实验，铅圆柱，同样确认 Φ₀ = h/2e ✓
  ← 两组独立实验同年发表，互相印证 → 磁通量子化被公认为超导的根本特征
  ⚠️ [来源待验证]：引用细节（页码、卷数）待原始论文独立核实

伦敦预言（[来源待验证：F. London, "Superfluids", 1950]）：
  → F. London 1950 年书中预言了磁通量子化，但预测值为 h/e（未考虑 Cooper 对）
  → 1957 年 BCS 理论澄清载流子为 2e → 正确值 Φ₀ = h/2e
  ⚠️ [来源待验证]：London 原始预言的精确表述待原始文献核实
```

## 3. Abrikosov 漩涡格子（[定理，II 型 SC，Abrikosov 1957]）

```
=== Abrikosov 漩涡格子：磁通量子的宏观有序排列 [定理] ===

II 型超导体的相图（[定理]）：
  H < Hc₁：Meissner 态（磁通全排斥，B = 0）
  Hc₁ < H < Hc₂：混合态（漩涡格子，漩涡进入超导体）← F9 核心领域
  H > Hc₂：正常态（超导消失）

漩涡格子的几何结构（[定理，Abrikosov 1957]）：
  → 每根漩涡线携带一个磁通量子 Φ₀（拓扑电荷 n=1）
  → 漩涡间存在排斥相互作用（同方向漩涡）→ 形成规则格子
  → 最低能量结构：三角格子（体心六方 = 六角密排）
  → 方形格子也稳定，但能量略高（部分材料可见）

漩涡格子间距 a₀（[定理，三角格子公式]）：

  LaTeX: a_0 = \left(\frac{2\Phi_0}{\sqrt{3}\,B}\right)^{1/2}

  ASCII: a_0 = sqrt(2*Phi_0 / (sqrt(3)*B))（三角格子，最低能量）

  数值估算（B = 0.1 T）：
    a_0 = sqrt(2 * 2.07e-15 / (1.732 * 0.1))
        ≈ sqrt(2.39e-14)
        ≈ 155 nm（典型量级：100-1000 nm）

漩涡密度（[定理，面密度]）：

  LaTeX: n_v = \frac{B}{\Phi_0}

  ASCII: n_v = B / Phi_0（单位：m^{-2}）

  → B 越大 → 漩涡越密 → 格子间距 a₀ 越小
  → 当 a₀ ~ ξ（相干长度）时 → 漩涡芯重叠 → Hc₂

漩涡的空间结构（[定理]）：
  → 漩涡芯半径 ~ ξ（相干长度，超导序参量 |ψ| 趋零的范围）
  → 漩涡间磁场范围 ~ λ_L（London 穿透深度，磁通线宽度）
  → Ginzburg-Landau 参数：κ = λ_L / ξ
    κ < 1/√2：I 型 SC（无漩涡，Meissner 态）
    κ > 1/√2：II 型 SC（存在 Abrikosov 漩涡格子）

Abrikosov 格子的拓扑意义（[定理]）：
  → 每根漩涡线携带拓扑电荷 n = +1
  → N 根漩涡线的总拓扑电荷 = N = B * 样品面积 / Φ₀
  → 总磁通量 = N * Φ₀（严格量子化！）
  → 格子的有序性 = 等价于拓扑电荷的均匀分布（最小排斥能态）

Abrikosov 格子的实验观测（[确定，多方法]）：
  → 磁粉法（Bitter decoration）：撒铁磁颗粒，在磁通管入口聚集 [来源待验证，1967 Essmann]
  → 中子小角散射（SANS）：探测漩涡格子的周期性 [来源待验证]
  → 扫描隧道显微镜（STM）：直接成像漩涡芯（|ψ| → 0 的暗点）[来源待验证，1989 Hess et al., NbSe₂]
  → 中子星类比 [猜想]：中子星磁通可能也以类 Abrikosov 格子分布（夸克超流中的类比，但属于极端物理，不在 SC 研究范围内）
```

## 4. 漩涡运动与电阻（[定理，Josephson 关系推导]）

```
=== 漩涡运动产生电阻：超导失效的微观机制 [定理] ===

核心命题（[定理]）：
  漩涡在实空间运动 → 切割超流量子线 → 产生电动势 → 有限电阻！
  ⚠️ 反谄媚：不是"有漩涡就有电阻"，是"漩涡流动才有电阻"！

电场产生机制（[定理，Josephson 关系]）：
  漩涡以速度 v_L（x 方向）运动：

    E_y = B * v_x（Faraday 定律，单个磁通量子切割超流线）

    更精确：E = Φ₀ * n_v * v_L（n_v = 漩涡密度，v_L = 漩涡速度）

  → 电场 E ∝ 漩涡流动速度 v_L
  → 电阻 R = E/J ∝ v_L（与电流密度关系通过 Magnus 力确定）

漩涡的洛伦兹力（Magnus 力）（[定理]）：

  LaTeX: \mathbf{F}_L = \Phi_0\, n\, (\mathbf{J} \times \hat{z})

  ASCII: F_L = Phi_0 * n * (J × z_hat)（单根漩涡受到的洛伦兹力）

  其中：
    J = 外加电流密度（A/m²）
    z_hat = 漩涡线方向（垂直于超导平面）
    n = 漩涡拓扑电荷（+1 或 -1）
    F_L = 洛伦兹力（垂直于 J，使漩涡横向漂移）

  物理图像：
    → 电流 J（y 方向）+ 漩涡线（z 方向）→ 洛伦兹力（x 方向）
    → 漩涡横向运动 → E 场（y 方向）→ 电阻！
    → 正漩涡（n=+1）和反漩涡（n=-1）受力方向相反（分离！）

钉扎与临界电流（[定理，A7 关联]）：
  ─────────────────────────────────
  无钉扎（理想 II 型 SC）：
    漩涡自由流动 → E ≠ 0 for all J → Jc = 0！（超导失效）
    纯 II 型 SC 在 Hc₁ < H < Hc₂ 时电阻 ≠ 0（Flux-flow 电阻）
  ─────────────────────────────────
  有钉扎（实用超导材料）：
    缺陷/杂质/边界 → 钉扎中心（漩涡能量局部极小点）
    钉扎力 F_pin > 洛伦兹力 F_L：漩涡被钉住 → v_L = 0 → R = 0 → Jc > 0
    F_L > F_pin（J > Jc）：漩涡脱钉，开始流动 → R > 0 → 超导失效

    临界电流密度（[定理，估算]）：
      Jc ≈ F_pin / (Φ₀ * d)（d = 样品厚度或漩涡长度）
      → 钉扎越强 → Jc 越大 → 超导越"实用"

Flux-flow 电阻（[定理，Bardeen-Stephen 模型，来源待验证]）：
  自由漩涡流动时的电阻率：

    ρ_ff = ρ_n * (B / Hc₂)

    其中 ρ_n = 正常态电阻率，B = 外加磁场
    → B = Hc₁ 时 ρ_ff → 0；B → Hc₂ 时 ρ_ff → ρ_n
    ⚠️ [来源待验证]：Bardeen-Stephen 1965 是经典引用，精确数值关系待核实

热激活漩涡（Thermally Activated Flux Flow, TAFF，[定理]）：
  → 即使 J < Jc，热涨落也可以帮助漩涡越过钉扎势垒（"漩涡蠕变"）
  → 电阻 R ~ exp(-U_0 / (k_B * T))（U_0 = 钉扎势垒高度）[来源待验证]
  → 这导致实际超导材料在低电流时仍有残余电阻（YBCO 等高温 SC 尤为显著）
```

## 5. 特殊拓扑电荷：半整数磁通（π 磁通）

```

  ─────────────────────────────────────────

  ─────────────────────────────────────────

  → 实验：[来源待验证，Tsuei et al. 实验组，约 1994-2000]
  → 困难：需要特殊多晶样品几何（三角形或五边形角 Josephson 结）
  → 状态：[猜想]，部分实验证据存在，但精确验证极困难

  ─────────────────────────────────────────
  体系 2：手性 SC（Chiral SC）漩涡的 MZM（F5 关联）
  ─────────────────────────────────────────
  → Class D 拓扑 SC（Chern = 1，如预期的 p+ip SC）
  → 漩涡核心束缚 Majorana 零模（MZM）
  → 含 MZM 的漩涡：有效拓扑电荷涉及 Z₂ 量子数（不是纯整数！）
  → 交换两个 MZM 漩涡：非阿贝尔统计（Ising 任意子）
  → 状态：[猜想/前沿]，理论预言（Kitaev 2001）清晰，但实验验证中（F5 详述）

  ─────────────────────────────────────────
  体系 3：Sr₂RuO₄ 的 µSR 信号（[观察，争议中]）
  ─────────────────────────────────────────

  → µSR（正电子自旋共振）测量：发现进入混合态时内场增大
  → 一种解释：漩涡携带 Φ₀/2（半量子）
  → 现状：Sr₂RuO₄ 的配对对称性有重大争议（2019 年后 p+ip 解释受到严重质疑）
  → [来源待验证]：具体 µSR 数据引用待原始论文核实
  → 状态：[观察 + 猜想]，解释高度争议

半量子化磁通的实验困难（[定理级反谄媚]）：
  → µSR 信号极弱（需要精确背景扣除，统计误差大）
  → d-wave π 结需要极特殊的多晶样品几何，制备难度极高
  → 半量子与整数量子的区分：信号量只有 Φ₀/2 ~ 1e-15 Wb，分辨率要求极苛刻
  → 大多数"半量子"实验报告在多年后被重新解释
  → 结论：[猜想]，半量子化磁通是拓扑超导的重要理论特征，但实验确认极难

## 6. 拓扑电荷的代数结构（[定理]）

```
=== 漩涡的基本群结构：π₁(U(1)) = ℤ [定理] ===

超导序参量的目标空间（[定理]）：
  → 序参量 ψ = |ψ| * exp(iφ) 的相位 φ 取值于 U(1) 圆（S¹）
  → 漩涡 = 空间路径到 U(1) 的映射（绕一圈的同伦类）
  → 漩涡的分类 = π₁(U(1)) = ℤ（整数群）

代数法则（[定理，严格]）：

  合并法则（加法）：
    两个漩涡靠近 → 合并为一个 → n_total = n₁ + n₂

  正/反漩涡湮灭（[定理]）：
    n = +1（正漩涡）+ n = -1（反漩涡） → 湮灭 → n_total = 0
    能量释放：E ~ 2π * ρ_s * ln(r₁₂/ξ)（r₁₂ → 0 时释放全部）

  高阶漩涡的不稳定性（[定理]）：
    n = ±2 漩涡：能量 E ~ n² * π * ρ_s * ln(L/ξ) = 4π * ρ_s * ln(L/ξ)
    vs 两个 n = ±1 漩涡分离：E ~ 2 * π * ρ_s * ln(L/ξ)（因子 2 < 4）
    → |n| = 2 漩涡能量 > 两个 |n| = 1 漩涡 → 不稳定，倾向分裂
    → 超导体中只有 |n| = 1 漩涡是稳定基态漩涡

  能量与拓扑电荷的关系（[定理]）：

    LaTeX: E_v \propto n^2 \cdot \pi \rho_s \ln\frac{L}{\xi}

    ASCII: E_v ∝ n^2 * π * ρ_s * ln(L/ξ)

    → 能量 ∝ n²：n = ±2 能量是 n = ±1 的 4 倍！
    → 分裂定理：n = ±2 → 2 个 n = ±1 的漩涡，总能量减半
    → 实验后果：实验中几乎不见高阶漩涡（n=±2），偶尔在特殊拓扑 SC 中见到

拓扑稳定性（[定理]）：
  → 漩涡不能被连续形变消除（n ≠ 0 的漩涡是拓扑稳定的）
  → 只有漩涡-反漩涡对湮灭（需要二者碰触）才能消灭拓扑电荷
  → 拓扑保护 ≠ 能量保护（漩涡可以被外力驱动运动，只是无法凭空消失）

## 8. 关键反谄媚

```
⚠️ 反谄媚 F9.1：Φ₀ = h/2e 的"2"来自 Cooper 对，不是数字"2"的巧合

  Φ₀ = h/(2e)：分母 2 来自 Cooper 对的电荷 = 2e（两个电子！）
  正常金属中：单电子 AB 效应 → Φ₀^normal = h/e（不同！）
  超导是 Cooper 对凝聚体，这个"2"是 BCS 理论的根本预测
  → 实验测量到 h/2e 而非 h/e 是验证 Cooper 对存在的直接证据之一

  ⚠️ 错误说法："磁通量子是 h 的量子化" → 正确说法："磁通量子来自 Cooper 对（2e）的相位量子化"

⚠️ 反谄媚 F9.2：有漩涡 ≠ 有电阻；漩涡流动才有电阻

  II 型 SC 在 Hc₁ < H < Hc₂：有漩涡（Abrikosov 格子）
  但：如果漩涡被完全钉扎（F_pin >> F_L）→ v_L = 0 → E = 0 → R = 0（超导！）
  只有漩涡克服钉扎开始流动 → E ≠ 0 → R > 0 → 超导失效

  实用超导材料（YBCO, NbTi, Nb₃Sn）的工程核心 = 钉扎工程（pinning engineering）
  → 目标：增大 Jc = 增强钉扎力，对抗更大洛伦兹力
  → 不是"消灭漩涡"，而是"钉住漩涡"

⚠️ 反谄媚 F9.3：高阶漩涡（n=±2）几乎不存在

  理论上 n=±2 漩涡是合法的拓扑对象（π₁(U(1))=ℤ 允许所有整数）
  实验现实：n=±2 漩涡能量是两个 n=±1 漩涡的 2 倍（E ∝ n²）
  → 热力学不稳定：n=±2 自发分裂为两个 n=±1 漩涡
  → 实验中极罕见（特殊几何约束下偶见）

  ⚠️ 错误推论："既然 n∈ℤ，高阶漩涡很常见" → 能量 n² 约束导致实际上只见 n=±1

⚠️ 反谄媚 F9.4：半磁通量子（Φ₀/2）的实验确认极度困难

  理论上在 d-wave π 结和手性 SC 中可能存在
  实验现实：
    → 信号极弱（Φ₀/2 ~ 1e-15 Wb），需要极精密的 SQUID 或 µSR 测量
    → µSR 信号受背景影响大，统计困难
    → "半量子"报告历史上多次被后来的实验重新解释
    → Sr₂RuO₄ 的 p+ip 解释（支持半量子）2019 年后受到严重质疑
  → 结论：[猜想]，半量子化磁通是重要理论预言，但迄今没有无争议的实验确认

⚠️ 反谄媚 F9.5：Abrikosov 格子 ≠ 固定不动的晶体

  Abrikosov 格子在真实超导体中是"软固体"：
    → 漩涡间相互排斥（弹性能）
    → 热涨落可以融化漩涡格子（Vortex Lattice Melting Line）
    → 在 T 足够高（尤其 YBCO 这类高温 SC）：漩涡格子熔化为漩涡液体（vortex liquid）
    → 漩涡液体中漩涡流动 → 电阻非零（即使 J < 名义 Jc）
    → 漩涡格子熔化线是 HTSc 在 H-T 相图中的重要界线 [来源待验证]
```
