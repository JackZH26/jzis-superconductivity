## 核心发现

在扭角 WSe₂（tWSe₂）均质双层中，通过**连续调节 U/W（关联强度/带宽比）**，首次完整展示了从 Mott 绝缘体到超导体的相图：

1. **twist angle ~4.6°**：U 和 W 可比（中等关联区），最优超导条件
2. **双门控器件**：同一器件可调出反铁磁绝缘体、超导穹顶、奇异金属
3. **电子/空穴双掺杂相图**：AFM 绝缘体两侧均有超导穹顶（类铜氧化物相图！）
4. **高 Tc 现象学**：vHS 移动、带结构调控的 Mott 转变、T-linear 电阻（奇异金属）
5. 单一器件连续展示 Mott 转变的两侧 → 弥合了铜氧化物和 Hubbard 模型的关键缺失

## 物理意义

- 这是**首个在实验上连续调控经过 Mott 转变的超导相图** — 对铜氧化物机制研究有重大意义
- tWSe₂ 作为**三角格子 Hubbard 模型的量子模拟器**，可以直接检验理论预测
- AFM 绝缘体两侧的超导穹顶 → 支持"Mott 绝缘体掺杂 → 超导"的图像

## 与我们研究的关联

- 与 Guo+ Nature 2026（WSe₂ Tc-R 反关联）是**同一体系**但不同侧重
- Night 16 的 AFM-ρ_s 竞争机制在这里有直接体现
- 量子度规研究：tWSe₂ 在 4.6° 附近 U≈W，量子度规 G_metric 应该中等各向异性 → s±-wave 配对预测
- 支持 Paper C（140-190K 间隙）中 moiré 超导的机制描述
