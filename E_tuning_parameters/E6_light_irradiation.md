## 1. 物理定义与核心思想

### 1.1 两大调控类型

```
=== 类型一：稳态光子掺杂（Persistent Photodoping）===

物理图像：
  UV/可见光（hν > E_g）照射样品
  → 光致激发电子从价带 → 导带（或从杂质能级释放）
  → 产生额外空穴（铜氧化物）或电子（取决于带结构）
  → 载流子浓度增加 → 等效于化学掺杂（但不引入化学无序！）
  → 低温下载流子无法通过声子复合消散 → 持久光电导（PPC）

与 E2 化学掺杂的关键区别：
  ✅ 可逆（升温至足够高温 → 光生载流子热激发复合消失）
  ✅ 不引入化学无序（不改变晶格结构）
  ❌ 掺杂量不精确（光强不均匀 → 局域密度不均匀）
  ❌ 调制范围有限（相比化学掺杂）
  ❌ 样品加热问题（激光照射 → 局部升温）

=== 类型二：非平衡光致 SC（Pump-Probe / Ultrafast Photoexcitation）===

物理图像：
  飞秒激光脉冲（~100 fs - 1 ps，中红外/近红外/可见光）泵浦材料
  → 激发特定声子模式（非共振：电子激发；共振：直接声子泵浦）
  → 创造非平衡电子/声子分布（远离热平衡态）
  → ps 级时间尺度内 → 探测光学响应（THz / 光学电导率）

核心实验观测：
  泵浦后数 ps 内，某些材料出现"类超导"光学电导率特征：
  (a) THz 光学电导率在低频出现类 Drude 峰消失（σ₁ → 0）+ 1/ω 型 σ₂ 发散
  (b) 类 Josephson 等离激元（Josephson plasma resonance, JPR）信号恢复
  → 这是层间超流权重的光学信号，可以在 ps 时间尺度内测量

⚠️ 核心问题（全文最重要的反谄媚）：
  "类超导"光学响应 ≠ 真正超导
  判据对比：
    零电阻 → 无法在 ps 时间尺度测量 [技术不可能]
    Meissner 效应 → 无法在 ps 时间尺度测量 [技术不可能]
    Josephson 等离激元 → 可测，但 JPR 信号也可能来自光致金属化或其他机制
  → 光致 SC 实验的所有"证据"均为光学电导率代理，
     而非超导三大判据的直接证明

=== 光照可能增强 SC 的机制 ===

  机制 A：声子模式选择性激发 → 等效增大 λ [猜想]
    → 中红外激光 → 共振激发特定软模（JT 声子、铜-氧弯曲模）
    → 该声子参与 Cooper 配对 → 非平衡占据使 λ_eff 增大 → SC 增强
    → 代表：K₃C₆₀（D9），LNSCO/YBCO（Cavalleri 组）

  机制 B：消融竞争序（CDW/条纹/磁序）→ N(EF) 恢复 → SC 增强 [猜想]
    → 激光脉冲 → 破坏 CDW/PDW 序（无序化）→ 更多谱重恢复到费米面 → SC 增强
    → 代表：LNSCO 条纹相，铜氧化物欠掺杂区

  机制 C：Floquet 调制有效哈密顿量 → 增强超导配对 [猜想，理论]
    → 周期性激光驱动 → 修改有效跳跃参数 t_eff 和 U_eff
    → 预言某些驱动频率可增强 d-wave 超导 [来源待验证]
    → 实验实现极难（热化问题，见第 7 节）
```

```
★★★ 2011 Fausti et al. — LNSCO 中红外泵浦 → 瞬态超导信号 [确定，但解释争议大]

  来源：D. Fausti et al., Science 331, 189 (2011)（来源确定）
  体系：La₁.₈₋ₓEu₀.₂SrₓCuO₄（LNSCO，条纹相铜氧化物，Tc ≈ 0）

  争议：
    ❌ 光学信号可能来自光致瞬态金属化（metallic transient），非真 SC
    ❌ 无零电阻、无 Meissner 测量（时间尺度无法实现）
    ❌ 加热效应贡献难以定量排除

★★★ 2014 Kaiser et al. / Hu et al. — 欠掺杂 YBCO 泵浦 → 瞬态超导 [来源待验证]

  来源：
    S. W. Kaiser et al., Phys. Rev. B 89, 184516 (2014)（来源待验证，具体数据需核查）
    W. Hu et al., Nat. Mater. 13, 705 (2014)（来源待验证，具体数据需核查）
  体系：YBa₂Cu₃O₆.₅（欠掺杂 YBCO，Tc ≈ 51K）
  实验：中红外脉冲泵浦铜-氧 B₁u 弯曲声子 → THz 探测
  结果：T = 60-80K（Tc 以上 ~10-30K）→ 泵浦后瞬态 Josephson 等离激元恢复
  解释：声子泵浦 → 层间 Josephson 相干恢复 → T > Tc 的"瞬态超导" [来源待验证，猜想]
  争议：
    ❌ 与 Fausti 2011 相同：光学代理信号，非直接超导证据
    ❌ 加热效应（铜-氧弯曲声子 → 晶格升温）贡献未充分排除

★★★ 2016 Mitrano et al. — K₃C₆₀ @ 100K 光致超导（最强光致 SC 案例）[确定，争议较少]

  结果：T = 100K（远高于 Tc = 18K）→ ps 级"类超导"光学电导率（σ₁ → 0, σ₂ ~ 1/ω）

    ✅ 效应显著（100K >> Tc = 18K，完全正常态泵浦后出现 SC 信号）
```

## 3. 铜氧化物的光致超导（Cavalleri 组主战场）

```
=== 体系一：LNSCO 条纹相（Fausti 2011 Science）===

材料：La₁.₈₋ₓEu₀.₂SrₓCuO₄（LNSCO，典型 x ~ 0.12）
常态特征：
  → PDW/条纹序 绝缘体（CuO₂ 面内自旋/电荷有序，抑制超导相位相干）
  → Tc ≈ 0（无宏观超导，尽管可能存在局域 Cooper 对）[观察，实验共识]

泵浦实验 [Fausti 2011, Science 331, 189（来源确定）]：
  → 15 μm 中红外脉冲（共振铜-氧面外弯曲声子 Ag 模）
  → T = 20K 探测（高于 Tc ≈ 0）
  → 泵浦后 ~ 数 ps 内：THz 频段出现类 Josephson 等离激元

物理解释（[猜想，争议大]）：
  声子模式激发 → 无序化条纹序 → CDW/PDW 序参量迅速降低
  → 本已存在的局域 Cooper 对（赝能隙区）重新获得相位相干
  → 瞬态超流密度恢复 → THz 信号类似超导

关键反谄媚：
  ❌ 条纹序消融 = 瞬态金属化（普通金属也有类似 σ₂ 上升趋势）
  ❌ 无法区分"光致超导"与"光致非超导金属"的 THz 信号
  ❌ 信号寿命 ~ ps（热平衡后消失）—— 与样品加热时间尺度重叠

=== 体系二：欠掺杂 YBCO（Kaiser/Hu 2014）===

材料：YBa₂Cu₃O₆.₅（欠掺杂，T_c ≈ 51K，赝能隙区的典型样品）
常态特征（T > Tc = 51K）：
  → 赝能隙态（Cooper 对预形成，但无相位相干）
  → 面外 Josephson 等离激元在正常态不存在

泵浦实验 [Kaiser 2014 / Hu 2014，来源待验证]：
  → 中红外泵浦铜-氧 B₁u 弯曲声子（~16–20 μm）
  → T = 70-80K（高于 Tc = 51K）
  → 泵浦后瞬态 Josephson 等离激元恢复（THz 光导率中 1/ω 型 σ₂ 出现）

物理解释（[猜想，同样争议大]）：
  声子泵浦 → 层间超流权重瞬态恢复 → T > Tc 区域的"瞬态超导"
  机制候选：
    ① 激发声子 → 弛豫时改变层间耦合 → Josephson 相干恢复 [猜想]
    ② 声子泵浦 → 局域加热 CDW/nematic 区域 → 更大超流密度恢复 [猜想]
    ③ 非平衡 Cooper 对涨落（Berezinskii-Kosterlitz-Thouless 驱动）[猜想]

关键反谄媚：
  ❌ 最强替代解释：赝能隙区本已有预形成 Cooper 对
     → 声子泵浦仅是打破 CDW 竞争序 → 已有 Cooper 对重新获得相干
     → 这不是"凭空"产生超导，而是释放预存的 Cooper 对
     ❌ 加热铜-氧声子 → 局部有效温度可能短暂"冷却"特定自由度，但整体升温
     ❌ 独立于 Cavalleri 组的完全重现工作有限 [来源待验证，截至 2026]
```

## 4. K₃C₆₀ 光致超导（Mitrano 2016，详见 D9）

```
简要总结（详细内容 → D9 富勒烯节）：

体系：K₃C₆₀（Tc = 18K，Jahn-Teller 声子配对超导体）

实验参数 [Mitrano et al., Nature 530, 461 (2016)（来源确定）]：
  → 激发波长：9.6 μm（中红外，共振 H_u(4) Jahn-Teller 声子模式）
  → 实验温度：T = 100K（>> Tc = 18K，完全正常态）
  → 信号持续时间：~ 1 ps（ps 量级，热平衡后消失）
  → 信号特征：σ₁(ω→0) → 0，σ₂(ω) ~ 1/ω（类超导 THz 光导率）

配对增强机制 [猜想]：
  非平衡 JT 声子激发 → λ_eff（有效电声耦合常数）瞬态增大
  → BCS Tc ∝ exp(-1/λ_eff) 急剧升高 → 100K 下仍有超导配对 [猜想，数量级未确认]
  
  正式表达（ASCII）：
    Allen-Dynes（BCS 适用）：Tc ~ omega_log * exp(-1.04*(1+lambda)/(lambda-mu*(1+0.62*lambda)))
    非平衡 lambda_eff > lambda_eq → 非平衡 Tc_eff >> Tc_eq [猜想，框架定性]

关键限制：
  ❌ 无零电阻测量（ps 时间尺度内无法实现）
  ❌ 无 Meissner 效应（无法测量）
  ❌ 其他实验室完全独立重现有限 [观察，截至 2026]
  ✅ 光学信号的"类超导"特征相对清晰（σ₂ ~ 1/ω 型符合超流密度框架）

光致 SC 的"最干净"案例（相比铜氧化物）：
  K₃C₆₀ 正常态（100K）没有预形成 Cooper 对 → 信号不来自预存配对释放
  → 机制更可能是真正的非平衡 λ 增大 [猜想]
  → 但仍是光学代理，非直接超导判据
```

## 5. 稳态光子掺杂

```
=== 物理原理 ===

UV 光照射（hν > 带隙/激发能）：
  → 光致载流子产生（电子-空穴对）
  → 低温下某些材料中载流子"冻住"不消散（陷阱态捕获）
  → 持久光电导（Persistent Photoconductivity, PPC）效应
  → 有效空穴（或电子）浓度增加 → Fermi 面移动 → Tc 改变

=== 铜氧化物稳态光子掺杂 ===

体系：La₂CuO₄（LCO，母态 Mott 绝缘体，Tc = 0）

实验（[来源待验证，具体数值需核查]）：
  → UV 光照射（hν > E_CuO 转移能，约 2 eV）
  → 产生光生空穴 → 增加 CuO₂ 面空穴浓度
  → PPC：低温下光生空穴不消散（陷阱在 LaO 层）
  → 表现：电导率升高 → 甚至可能出现 Tc > 0 的 SC 迹象 [来源待验证]

与 E3 门控的比较：
  相似：均改变载流子浓度，不引入化学无序
  不同：
    光子掺杂：方向固定（UV 仅产生空穴，不能产生电子掺杂）
    静电门控：双向可调（正负 gate voltage 双向掺杂）
    光子掺杂：空间均匀性差（光强分布不均）
    静电门控：均匀，精确

=== PPC 效应的局限 ===

  [观察，共识]：
  → PPC 的空穴来自表面/界面陷阱态，非均匀掺杂
  → PPC 不等于均匀的化学掺杂 → Tc 调制不可靠
  → "PPC 诱导 SC" = 局域超导块 → 渗流导电，非真均匀 SC [猜想]
  → 升温（>100K 量级）→ PPC 消失，Tc 回到零 → 可逆

=== 优点与缺点总结 ===

  优点：
    ✅ 不引入化学无序（对比 E2 化学掺杂）
    ✅ 可逆（升温消除）
    ✅ 不需要复杂电极（对比 E3 门控）
    ✅ 能够在宏观薄膜上操作

  缺点：
    ❌ 不精确（光强不均匀 → 局域空穴密度不均匀）
    ❌ 方向固定（只能给空穴，对某些体系无效）
    ❌ 样品加热（UV 光功率 → 局部升温）
    ❌ 掺杂量小（不能大幅移动相图）
    ❌ 持久性差（低温冻结，升温消除）
```

## 6. Floquet 工程 / 激光驱动超导（理论前沿）

```
=== Floquet 理论基础 ===

Floquet 定理 [定理，数学定理]：
  周期性驱动的量子哈密顿量 H(t) = H(t + T)：
  → 解为 Floquet 态（时间上类似 Bloch 态）
  → 等效静态哈密顿量 H_eff（Floquet 有效哈密顿量）

高频展开（Magnus 展开，驱动频率 Ω >> 带宽）[定理，级数展开]：
  H_eff = H_0 + (1/Omega) * [H_{+1}, H_{-1}] / (2i) + O(Omega^{-2})
  其中 H_{+n} = n-th Fourier 分量

物理直觉：
  → 高频激光驱动 → 修改有效跳跃参数 t_eff（Bessel 函数修正）
  → t_eff / t_0 = J_0(A/Omega)（J_0: 零阶 Bessel 函数，A: 驱动幅度）[定理，Floquet-BH 模型]
  → 改变有效 U/t 比值 → 改变 Hubbard 模型相图位置 → 理论上可调控 SC

=== 激光驱动增强超导（理论预言）===

Sentef 等 Floquet-Hubbard 模型 [来源待验证]：
  1D/2D Hubbard 模型 + 周期激光驱动
  → 特定驱动频率 Ω/t ~ O(1)：有效 U_eff 减小 → SC 配对增强 [猜想，数值模拟]
  → 预言：d-wave 超导 Tc 可能通过 Floquet 调控升高 [猜想，未实验验证]

Floquet 配对增强机制（ASCII）：
  H_eff ~ -t_eff * sum_{<ij>} c†_i c_j + U_eff * sum_i n_{i↑} * n_{i↓}
  其中 t_eff = t_0 * J_0(A/Omega)，U_eff 由更高阶 Magnus 展开决定
  → 适当 A/Omega 可使 U_eff/t_eff 落入 Mott-BCS 相图的 SC 区域 [猜想]

=== 实验实现的关键障碍 ===

  [观察，公认的技术困难，2026]

障碍一：热化（Thermalization）
  → 激光驱动 → 光子能量被晶格吸收 → 升温（~K 量级 / 脉冲）
  → 热化时间（电子-声子弛豫）~ ps
  → Floquet 稳态建立需要 ~ 1/Γ（耗散率逆）>> ps → 几乎无法在固体中实现
  → 现实：Floquet "态"在热平衡破坏之前就消失了

障碍二：区分 Floquet 效应 vs 热效应
  → 激光 → 局部加热 → 超导被破坏/恢复 → 光学响应变化
  → 这些变化可以看起来和 Floquet 增强一样
  → 没有可靠的实验判据区分真正的 Floquet 调控 vs 热效应 [观察]

障碍三：驱动频率约束
  → 有效 Floquet 理论需要 Ω >> 带宽
  → 强关联材料（铜氧化物、铁基）带宽 ~ eV
  → 对应频率 >> 可见光（UV 量级）→ 材料直接损伤

结论：[猜想，理论有趣，实验挑战目前无法克服]
  Floquet 工程增强超导是严肃的理论预言，
  但在固体材料中的实际实现（截至 2026）尚无无争议的实验验证。

=== 2D 材料：Floquet 工程的更好平台？===

  [猜想，活跃理论前沿，截至 2026]
  
  TMD（MoS₂/WSe₂/WS₂）+ 圆偏光照射：
    → Floquet-Bloch 态 → 打开拓扑能隙 → 可能诱导拓扑 SC [猜想]
    → 带宽更小（~0.1-0.5 eV）→ 驱动频率要求更宽松
    → 时间分辨 ARPES：已有 Floquet 带结构实验观测 [来源待验证，Gedik 组等]
  
  石墨烯 Floquet：
    → 类似 TMD，带宽小 → Floquet 理论适用范围更宽
    → MATBG + 光照：理论预言平坦带 SC 增强 [猜想，来源待验证]
    → 实验：尚无确认 [观察，2026]
```

## 8. 关键反谄媚

```
⚠️ 反谄媚 1（最重要）：光学类超导信号 ≠ 真正超导

  核心问题：所有光致 SC 实验测量的是 THz 光学电导率
  σ₁(ω→0) → 0 和 σ₂ ~ 1/ω 是超导的"典型特征"
  但这两个特征也可以来自：

    (a) 瞬态金属化：非超导金属在超快时间尺度内的 Drude 响应
        → 光致金属化 → σ₁ Drude 峰在低频可以接近零（宽 Drude + 非平衡加宽）
    (b) 光致载流子的等离激元共振
        → 等离激元在低频 → σ₂ ~ 1/ω 型响应
        → 不是超流密度，而是自由载流子的等离子体效应
    (c) 赝能隙区预形成 Cooper 对的相位相干恢复
        → 不是"产生新的超导"，而是"释放预存的配对"
  
  → 光致 SC 的光学证据在没有独立零电阻或 Meissner 确认的情况下，
     只是"类超导光学响应"，不能称为"超导"

⚠️ 反谄媚 2：样品加热是最顽固的竞争解释

  激光泵浦能量 → 必然产生热量（除非 100% 被特定模式吸收）
  加热路径：
    光子 → 电子激发 → 电子-声子弛豫（~ ps）→ 声子升温
  加热效应的光学后果：
    → 局部有效温度 T_local > T_sample（公称温度）
    → T_local 可能在 Tc 下方（某些样品区域）→ 恢复局域 SC 迹象 → 光学信号
    → 不是光照诱导了超导，而是局部冷却到 SC 区域
  
  反驳方案（[来源待验证，不同实验组的尝试]）：
    → 测量非共振（失谐）频率泵浦下是否有相同效应
    → 若共振频率特异性 → 加热解释被削弱（非选择性加热不应有频率特异性）
    → 实际：铜氧化物实验的频率特异性数据存在，但解读有争议 [来源待验证]

⚠️ 反谄媚 3：Mitrano 2016 的独立验证不充分

  [观察，截至 2026]
  → Nature 530, 461 (2016) 主要来自 Cavalleri 组（MPSD Hamburg）
  → 同一组的后续文章继续扩展，但其他独立实验室完全重现的报告有限
  → K₃C₆₀ 样品制备困难（空气敏感，高纯度 C₆₀ 需要）→ 重现实验数量少
  → 结论：[猜想]Mitrano 2016 是该领域最可信的光致 SC 实验之一，
           但独立验证不充分，不应视为确立事实

⚠️ 反谄媚 4：Fausti 2011 和 YBCO 实验的替代解释更简洁

  铜氧化物实验的替代解释链：
    赝能隙区 → 预形成 Cooper 对（已有）
    + CDW/条纹序竞争（抑制超导）
    + 声子泵浦 → CDW 无序化（短暂）
    → Cooper 对重新获得相位相干（瞬态）
    → 光学信号类超导
  
  这与"真正光致超导"的区别：
    替代解释：释放预存配对（不增加 Tc，不需要新配对机制）
    光致超导声称：在 Tc 以上几十 K 产生全新的超导态
  
  → Occam's 剃刀：替代解释（预形成 Cooper 对释放）更简洁，不需要新物理
  → 需要额外证据才能排除替代解释

⚠️ 反谄媚 5：持久光电导（PPC）≠ 均匀掺杂，≠ 超导掺杂

  PPC 效应的实际物理：
    → 光生载流子被局域缺陷/晶界捕获（浅陷阱 → 低温冻结）
    → 这是局域的（空间不均匀）
    → "光诱导 Tc"可能来自超导渗流（局域超导块的连通性）
    → 而非均匀的 SC 相
  → 光子掺杂的 Tc 值不能与化学掺杂的 Tc 值直接对比 [观察]

⚠️ 反谄媚 6：Floquet 工程在固体中的实用性极低

  物理现实：
    Floquet 稳态寿命受热化时间限制 ~ ps（固体中电子-声子耦合）
    Floquet 有效哈密顿量成立的条件：
      (1) 驱动频率 Ω >> 耗散率 Γ（不满足）
      (2) 系统在 Floquet 态稳定之前不热化（不满足，固体中 ~ ps 热化）
    → 固体中的 Floquet 超导增强，在理论预言的 Floquet 稳态实现之前
       就已经被热化破坏
  
  TMD/石墨烯的改善：
    带宽更小 → 要求的 Ω 更低 → 理论适用范围稍宽
    但热化时间（电子-声子弛豫）仍然是 ps 量级 → 问题本质不变
  → 结论：[猜想，理论严肃，实验挑战截至 2026 尚未克服]
```
