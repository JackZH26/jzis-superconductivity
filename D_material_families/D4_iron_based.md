## 1. 物理定义与核心思想

### 1.1 铁基超导体的结构定义

**[定义 — 铁基超导体]**

```
铁基超导体（Iron-Based Superconductors, IBS）：
  定义：含 Fe₂As₂ 或 Fe₂Se₂ 层的层状化合物，铁（磁性 3d 过渡金属）为配对主角
  化学特征：
    - Fe²⁺（d⁶，半满 3d 壳层，Hund 耦合活跃）
    - Fe 原子形成正方格子（square lattice）
    - As/Se 原子交替排列在 Fe 平面上下（四面体协调）
    - Fe₂As₂/Fe₂Se₂ 层被"spacer"层（LaO、Ba 等）隔开
    → spacer 层通过化学掺杂调控电子浓度

  结构示意（层状堆叠，ASCII）：
    ─────────────── spacer 层（例：LaO、BaF、K、SrF）
    ─ Fe₂As₂ 层 ─  ← 超导在此发生（类比铜氧化物 CuO₂ 平面）
    ─────────────── spacer 层
    ─ Fe₂As₂ 层 ─
    ─────────────── spacer 层
    （层间耦合较弱，但比铜氧化物稍强 → 各向异性较小）

  Tc 范围（[定理，实验确认]）：
    常压块材：8K（FeSe bulk）~ 55K（SmFeAsO）
    界面/薄膜：FeSe/STO 单层 Tc ≈ 65K（ARPES），输运 Tc 仍争议中 [来源待验证]
    高压：FeSe 高压 ~37K [来源待验证]
```

### 1.2 多轨道电子结构：为什么铁基与铜氧化物根本不同

**[定理/观察 — 铁基的核心物理特征]**

```
铜氧化物：单轨道（Cu d_x²-y² 主导），d-wave 配对，Mott 绝缘体母态
铁基：多轨道（5 个 Fe 3d 轨道同时接近费米面），s± 配对，金属母态

5 个 Fe 3d 轨道（[定理，晶体场 + DFT 计算]）：
  铁基的 Fe²⁺ d⁶ 构型，在 As/Se 四面体晶体场中：
  - d_xz, d_yz（二重简并，主要形成 Γ 点空穴口袋）
  - d_xy（主要形成 M 点电子口袋的内层）
  - d_z²（形成 Γ 点部分空穴口袋）
  - d_x²-y²（能量较高，远费米面，对超导贡献较小）
  → 所有 5 个轨道都在费米面附近 ≠ 铜氧化物的单轨道主导

多费米面拓扑（[观察，ARPES + DFT 一致]）：
  典型铁基（如 Ba₁₋ₓKₓFe₂As₂）的费米面：
  - Γ 点（布里渊区中心）：2-3 个圆柱形空穴口袋（hole pockets）
    → d_xz + d_yz + d_z² 轨道成分
  - M 点（布里渊区角落，等价于 (π,0) 在 1-Fe 单元格）：2 个椭圆形电子口袋
    → d_xz + d_xy 轨道成分（视具体材料而定）

  关键几何关系（s± 配对的几何基础）：
  Γ → M 的嵌套向量 Q ≈ (π, 0) 或 (π, π)（取决于布里渊区折叠）
  → AFM 自旋涨落在 Q 处最强（连接空穴和电子口袋）
  → 有效相互作用 V(Q) < 0（在 Γ 和 M 之间引起符号翻转）
  → s± 配对：Δ_h > 0（空穴口袋），Δ_e < 0（电子口袋）[见 C2 综述]
```

### 1.3 为什么铁磁性元素却有高温超导？

**[观察/猜想 — 铁基核心悖论及其解决]**

```
旧观念（BCS 框架内，Abrikosov-Gor'kov 定理）：[定理，1961]
  磁性杂质对 s-wave 超导有"对破坏"（pair-breaking）效应
  → 磁性元素 = 超导的敌人（尤其是强磁性的 Fe）
  → 1990 年代，"Fe 化合物不可能超导"是半隐含的共识

铁基的解决（[观察/猜想，2008 后建立]）：
  (1) Fe 3d 轨道有局部磁矩，但 **长程磁序被抑制**（掺杂或施压后）
      → 没有长程磁序 → Abrikosov-Gor'kov 定理不适用（它针对稀磁杂质）
  (2) **自旋涨落（spin fluctuations）** 替代磁序本身起配对"胶水"作用
      → 这是 B3（AFM 涨落）框架的核心思想（铜氧化物也相同逻辑）
  (3) 多轨道结构 + Hund 耦合：J_H > 0 导致铁基是"Hund's metal"（不是 Mott 绝缘体）
      → 局部磁矩被 J_H 稳定，但 Kondo 屏蔽弱 → 磁矩涨落而非冻结
      → 涨落的磁矩 = 配对的"胶水"，而不是对破坏者 [猜想，B10 关联]

  ⚠️ 反谄媚：这个解释是 **合理的机制猜想**，不是严格推导。
  "自旋涨落驱动 s± 配对"的能量学仍有争议（定量 Tc 计算困难）。
  铁基配对机制比铜氧化物"好一些"，但依然未完全解决。
```

### 1.4 铁基 vs 铜氧化物：系统对比

**[观察 — 两大高温超导体系的核心区别]**

```
对比维度        | 铁基（IBS）              | 铜氧化物（Cuprate）
────────────────|──────────────────────────|────────────────────────────
活跃轨道        | 5 个 Fe 3d 轨道          | 1 个 Cu d_x²-y² 轨道
母态            | 金属（SDW + 向列相）      | Mott 绝缘体（AFM）
超交换 J        | ~50 meV（铁基）          | ~130 meV（铜氧化物）
关联强度        | 中等（Hund's metal）      | 强关联（接近 Mott）
配对对称性      | s±（主流，有反例）        | d_x²-y²（判决性确认）
配对机制        | 多轨道 AFM 涨落（猜想）  | 未解（40 年争议）
Tc_max（常压）  | 55K（SmFeAsO）           | 134K（Hg-1223）
理论成熟度      | 较铜氧化物更好，仍不完整  | 机制完全未解
Paper A 适用性  | 部分适用（Eliashberg+涨落）| 不适用（强关联）
实用现状        | 高场磁体应用（REBCO 竞争）| 实用带材（Bi-2223，YBCO）
```

## 2. 历史关键节点（2008-2026 铁基超导发现史）

**[历史 — 铁基超导从发现到 2026 年最新进展]**

```
2008 Kamihara, Watanabe, Hirano, Hosono（东京工业大学）：
  发现 LaFeAsO₁₋ₓFₓ（1111 族，LaO 系），Tc = 26K
  [来源：Kamihara et al. 2008 JACS 130, 3296]
  ★ 铁基超导的诞生，引发全球研究热潮
  重要性：
  - 首次证明含铁（强磁性元素）化合物可以有 26K 的高温超导
  - 推翻"磁性元素阻碍超导"的旧框架
  - 仅 2008 年春，全球超过 30 个实验室开始竞赛推高铁基 Tc
  ⚠️ 注：Hosono 组 2006 年在 LaFePO 中已发现 4K 超导（前驱工作），
  但未引起广泛关注 [来源待验证]

2008 Ren, Yang, Yi, Shen, Zhou 等（中国科学院、中国科技大学等）：
  SmFeAsO（F 掺杂），Tc = 55K — 一个月内将 1111 族 Tc 推至 55K
  [来源：Ren et al. 2008 Chin. Phys. Lett. 25, 2215 和相关论文；来源待精确核查]
  ⚠️ 注：2008 年中国多个组在短短数周内将 Tc 从 26K 推到 43K（Ce）→
  52K（Pr, Nd）→ 55K（Sm）；多个组独立发现，引用时需查对应原始论文
  重要性：
  - 55K 确立铁基作为高温超导（第二大体系）的地位
  - 55K 至今（2026）仍是 **铁基超导常压块材 Tc 的最高记录** [观察]

2008 Mazin, Singh, Johannes, Du（Naval Research Lab）：
  提出 s± 配对对称性的理论框架
  [来源：Mazin et al. 2008 PRL 101, 057003]
  核心预言：多带 BCS 框架 → 空穴口袋（Γ）和电子口袋（M）能隙符号相反
  → 中子散射共振峰（resonance peak）= s± 的判决性信号
  重要性：
  - s± 框架迅速成为铁基理论的主流框架
  - 该框架对多个实验给出定性正确的预测 [猜想，有实验支持]

2008 Kuroki, Onari, Arita, Usui, Tanaka, Kontani, Aoki（东大等）：
  RPA（随机相近似）计算 1111 族的配对：独立预言 s±
  [来源：Kuroki et al. 2008 PRL 101, 087004]
  → 与 Mazin 2008 同期，两篇背景独立但结论一致，增加 s± 的理论可信度

2008 Rotter, Tegel, Johrendt（慕尼黑大学）：
  发现 BaFe₂As₂（122 族），K 掺杂 Ba₁₋ₓKₓFe₂As₂，Tc = 38K
  [来源：Rotter et al. 2008 PRL 101, 107006]
  重要性：
  - 122 族是 **铁基研究最多的体系**（最易制备大单晶）
  - Ba-122 单晶促成了后来大量精密测量（中子、ARPES、STM、μSR）
  - 38K < 55K（1111），但实验条件好得多

2008 Hsu, Luo, Yeh, Chen, Huang, Wu, Lee, Huang, Chu（台湾成功大学/清华大学）：
  发现 FeSe（11 族，最简单铁基结构），Tc = 8K
  [来源：Hsu et al. 2008 PNAS 105, 14262]
  重要性：
  - FeSe 结构最简单（无 spacer 层，仅 Fe₂Se₂ 层堆叠）
  - 体材 Tc = 8K 最低，但后来成为铁基研究的核心体系（FeSe/STO 突破）
  - FeSe 无 Γ 点空穴口袋（与标准 s± 矛盾）→ 最大理论挑战之一

2010 Hanaguri, Niitaka, Kuroki, Takagi（理化学研究所 RIKEN）：
  FeSe 上的 QPI（准粒子干涉）测量，支持 s±（无磁杂质散射格局）
  [来源：Hanaguri et al. 2010 Science 328, 474]
  重要性：
  - 这是 s± 的最重要实验证据之一（来自当时 Tc 最低的铁基！）
  - 但仍是 [猜想层级]：QPI 格局排除 s++，支持 s±，但不是决定性判决

2012 Wang, Zhang, Zhang, Li, Chen, Xue（北京清华大学，Xue Qikun 薛其坤 组）：
  单层 FeSe 薄膜在 SrTiO₃（STO）衬底上，ARPES 发现能隙开放温度远高于体材
  [来源：Wang et al. 2012 Chin. Phys. Lett. 29, 037402；来源待精确核查]
  ★ FeSe/STO：Tc ≈ 65K（ARPES），突破体材铁基 55K 记录
  重要性：
  - 界面超导的突破性案例（D11 关联）
  - 单层 FeSe 的电子结构与体材完全不同（无空穴口袋，仅电子口袋）
  - 机制至今争议（声子 + 电荷转移 + 界面效应）[猜想]

2013-2015 多个组（北京清华、Princeton、MIT 等）：
  FeSe/STO 输运测量和 ARPES 补充研究
  部分输运实验报告 Tc > 100K（疑似）[来源待验证]
  ⚠️ 注：输运 Tc 在不同样品间变化大（60-80K 到疑似 100K+）
  → 数据差异来源：样品质量、厚度控制、测量方法 [来源待验证]
  → 2026 年"FeSe/STO 输运 Tc" 仍未有统一公认值

2026 Romero et al.（Nature Physics）：
  多通道涨落框架（Paper A）分析铁基 α 参数（涨落通道权重）
  [来源：Romero 2026 NatPhys，本知识库已精读]
  → 铁基多个体系的 α_{B3}（AFM 涨落通道）占主导（但 α_{B5} 也非零）
  → 向列涨落（B5）和 AFM 涨落（B3）有协同效应（协同增强 Tc）
  → 证实 Paper A（多通道 Eliashberg）框架可以定量描述铁基 Tc 的材料趋势 [猜想]

2025-2026 La₃Ni₂O₇ 镍酸盐高压超导：
  部分机制分析（DFT + 多轨道）类比铁基框架（多轨道 s± 思路）[见 D5 综述]
```

## 4. 铁基相图（掺杂与温度）

### 4.1 标准铁基相图（以 Ba-122 为例）

**[观察/定理 — 铁基相图基本结构]**

```
Ba(Fe₁₋ₓCo_x)₂As₂ 相图（[观察，多组实验一致]）：

  温度 T
  ▲
  |   TN（SDW，条纹 AFM）
  | ╲
  |  ╲
TS |  ╲╲ 向列相（结构畸变 T4 → Ortho）
  |   ╲╲╲
Tc |    ╲╲╲╲ SC 穹（超导穹）
  |     ╲╲╲╲╲___
  ────────────────────────→ x（Co 掺杂浓度）
     x=0  0.04  0.07  0.15

  各区域物理特征：
  ─────────────────────────────────────────────────────────
  x = 0         | 四方相（PM）→ 斜方相（AFM SDW）T_S = T_N ≈ 138K
  0 < x < 0.04  | 向列相（T_S > T_N，两相分离）+ SDW 逐渐弱化
  x ≈ 0.04–0.07 | SC 出现，与 SDW 共存（争议：真共存 vs 相分离）
  x ≈ 0.07–0.12 | SC 穹顶端，最高 Tc ≈ 25K（Co 掺杂 Ba-122）
  x > 0.12      | SC 消退，趋向正常金属
  ─────────────────────────────────────────────────────────

  向列相（nematic phase，B5 关联）：
  - 铁基特有：在 T_N < T < T_S 区间，结构已从四方变斜方，但 AFM 序未建立
  - 物理实质：Fe₂As₂ 层的 C₄ 旋转对称性自发破缺为 C₂（向列序）
  - 向列转变温度 T_S 在 SC 穹边界附近 → 向列涨落增强 SC？[猜想，B5 关联]
  - 铜氧化物 **没有** 类似的向列相 → 这是铁基特有的物理 [观察]

  SDW（自旋密度波）的性质：[定理，中子散射确认]
  - 铁基母化合物（x=0）：条纹型反铁磁（stripe AFM）
  - 磁波矢 Q = (π, 0) 或等价的 (0, π)（斜方相对称性破缺选一个）
  - 每个 Fe 磁矩 m ≈ 0.36–0.9 μ_B（视材料而定）[来源待验证]

**[观察 — FeSe 是铁基中最反常的体系]**

### 5.1 中子散射共振峰（最重要的间接证据）

```
中子散射共振峰（spin resonance peak）[猜想，高置信度]：
  机制：s± → Δ_h 和 Δ_e 符号相反 → 在能量 ω_res = Δ_h + Δ_e 处，

  [来源：Christianson 2008 来源待精确核查；Nature 456 引用较常见，需确认]

  重要性与局限：
  ✅ 共振峰 T < Tc 才出现 → 与 SC 能隙打开直接关联

  ⚠️ FeSe 无空穴口袋 → 在 FeSe 中该共振峰解释不适用
```

### 5.2 QPI（准粒子干涉）实验

```
Hanaguri 2010 FeSe QPI 实验 [猜想，重要]：
  [来源：Hanaguri et al. 2010 Science 328, 474]
  方法：STM 扫描 FeSe 单晶，对比磁性杂质（磁铁）和非磁性杂质（Zn）的 QPI 格局
  关键对比（[观察]）：
  - 非磁性杂质（Zn）：不破坏 time-reversal → 能散射任何 q 向量
  - 磁性杂质（Fe, Mn）：破坏 time-reversal → 对于 s± 配对，特定 q 方向的散射被禁阻

  理论预期（[猜想，Hanaguri 2010 分析]）：
  - s++（同号）：磁性和非磁性杂质的 QPI 格局相同（因为 Δ_kf/Δ_ki > 0）

  结果：实验中磁性杂质的 QPI 格局与 s± 预期一致，与 s++ 不一致

  ⚠️ 局限：FeSe 无空穴口袋，实际嵌套向量与 Ba-122 不同，解释仍有细节争议

## 6. Hund 耦合与多轨道效应（B10 关联）

**[猜想/观察 — 铁基"Hund's metal"框架]**

```
铁基是"Hund's metal"（[猜想，有 DMFT 计算支持]）：
  定义：
  - 不是 Mott 绝缘体（Coulomb U 对多轨道体系被 Hund J_H 部分补偿）
  - 但 Hund 耦合 J_H 使得各 3d 轨道的准粒子质量显著增重（m* >> m_e）
  - 典型铁基：m*/m_e ≈ 2–8（视轨道而定）[来源待验证，DMFT 计算结果]
  - 对比 BCS 金属（Al, Nb）：m*/m_e ≈ 1–2

Hund's metal vs Mott insulator vs BCS metal（[观察/猜想，de'Medici 框架]）：
  [来源：de'Medici et al. 2011 PRL 107, 267401]
  核心结论（[猜想，DMFT 支持]）：
  - 多轨道体系中，增大 J_H（固定 U/t）：
    → 轨道间关联增强（轨道间 Kondo 屏蔽减弱）
    → 各轨道准粒子权重 Z_α 分化（轨道选择性）→ 轨道选择 Mott（OSMP）
  - 铁基在"Hund's metal"区域：每个轨道都有准粒子（非 Mott），但有效质量大

Hund 耦合的"Janus 特性"（[猜想，B10 关联]）：
  J_H 既促进磁性（铁基有大局域磁矩），又促进多轨道增强 DOS
  → 多轨道高 DOS（N(E_F)↑）+ 有效相互作用 → 配对增强 [猜想]
  反谄媚：J_H 不是单纯"好事"——它也使准粒子质量增重，降低 Fermi 速度
  → Tc ∝ ω_B × e^{-1/λ}，J_H 通过增大 λ 提升 Tc，但通过降低 ω_B 可能削弱 Tc

轨道选择性超导（[猜想，前沿研究]）：
  不同轨道的准粒子质量不同 → 不同轨道对超导贡献不同？
  → FeSe：d_xz/d_yz 轨道主导能隙，d_xy 轨道能隙小得多（各向异性 s±？）
  [来源待验证，Fe-3d 轨道分辨 ARPES 研究]
  → 这可能解释 FeSe 能隙高度各向异性的来源 [猜想]

## 7. FeSe/STO 接口超导（D11 关联）

```

  - STO 衬底通过界面向 FeSe 转移电子（电子掺杂）

  Romero 2026 框架：α_{B2}（界面声子）+ α_{B3}（残留 AFM 涨落）协同 [猜想]

### 9.2 Tc 55K 仍远低于铜氧化物 134K

```
[观察 — Tc 天花板的原因（猜想）和意义]

实验事实：[定理，实验确认]
  铁基最高体块 Tc = 55K（SmFeAsO，常压）[Ren 2008，来源待精确核查]
  铜氧化物最高 Tc = 134K（Hg-1223，常压）[Schilling 1993 Nature 363, 56]
  比值：134K / 55K ≈ 2.4

可能的 Tc 天花板原因（[猜想，非定理]）：
  (1) 超交换/涨落能量尺度：铁基 J ≈ 50 meV vs 铜氧化物 J ≈ 130 meV
      → Tc 上限 ∝ J（配对"胶水"强度）→ 比值 130/50 ≈ 2.6 ≈ 实验比值
      → 这个"解释"可能是后验（RETRODICTION），不是真正的理论预测 [猜想]
  (2) 多轨道竞争：铁基 s± 中空穴和电子口袋能隙相互部分抵消（相位相反）
      → 净配对强度降低 [猜想，与单轨道 d-wave 相比]
  (3) Hund metal 中的强相关效应：准粒子质量增重 → Fermi 速度下降 → 影响 Tc

  ⚠️ 实际上以上三点都是事后解释，能否严格推导 Tc 天花板：不能 [诚实声明]

FeSe/STO ≈ 65K 是否超过 55K？
  ARPES 能隙打开温度 ≈ 65K（但这是能隙，不一定等于 Tc）
  输运 Tc：报告值 60–80K（不同样品、不同组，来源待验证）
  → FeSe/STO 薄膜 Tc ≈ 65K 基本上被接受为可靠的 Tc（高于体块 55K）
  但 65K 仍低于铜氧化物 92K（YBCO）→ 铁基即使算界面增强也达不到液氮温区以上
```

### 9.3 铁基理论"比铜氧化物好一些"但未解决

```
[观察 — 铁基理论的诚实评估（2026）]

相对成功之处（[观察]）：
  ✅ s± 配对理论（Mazin 2008）在发现后很快提出，且多个实验支持 [猜想，高置信度]
  ✅ 多带 BCS（B12 多通道）框架可以定量描述 Ba-122 等体系的 Tc 趋势 [猜想]
  ✅ Hund's metal（B10）框架给出电子结构（质量增重）的合理解释 [猜想]
  ✅ DFT + DMFT + 自旋涨落计算对 Ba-122 的 Tc 定量在因子 2 内 [猜想，来源待验证]
  → 对比铜氧化物：没有任何理论能定量计算 Tc

绝对失败之处（[观察]）：
  ❌ FeSe 的机制完全不清楚（无空穴口袋，无 SDW，但超导 8K）
  ❌ FeSe/STO 机制争议（三种假说，无判决实验）
  ❌ LiFeAs 是 s± 还是 s++ 未定
  ❌ 铁基超导 Tc 预测：没有理论预测到 55K 之前（1111 族被发现是实验突破，非理论指导）
  ❌ 没有铁基版的"三角结实验"来判决性确认 s± 相位（铁基缺乏铜氧化物那样的判决实验）

结论：铁基是"理论友好型"高温超导材料
  但"相对理解得好一些"≠"已解决"
  2026 年铁基配对机制的正确性：
  - s±（Ba-122、1111 族）：[猜想，高置信度，~70–80%]
  - FeSe 机制：[猜想，低置信度，完全开放]
  - LiFeAs：[猜想，争议中]
```

### 9.4 FeSe/STO 65K 能否工程化？

```
[观察 — 实用性的现实]

技术挑战：
  (1) 仅单层有效（多层 Tc 下降）→ 超薄膜，界面质量关键
  (2) 输运 Tc 随样品制备条件变化大（真空度、退火条件、STO 表面终止）
  (3) STO 是绝缘体衬底 → 导电测量需要特殊设计
  (4) FeSe 在空气中不稳定 → 需要封装保护 [来源待验证]
  (5) 规模化制备单层 FeSe/STO：MBE（分子束外延），工艺复杂，成本高

实用前景（[观察，2026]）：
  暂无明确的 FeSe/STO 超导线材或器件实用化路线
  → 目前主要是科学研究平台（机制研究 + 界面增强物理探索）
  → 即使 Tc = 65K 确认，仍低于液氮（77K）→ 无直接应用价值
  ⚠️ 如果实际 Tc 证实超过 77K（部分实验声称），情况会改变，但仍待确认
```

## 11. 与 MECE 体系的关联分析

```
C2（s± wave）：
  铁基是 s± 配对对称性的 **核心实验平台**（唯一高温超导材料类别）
  → Mazin 2008 PRL 预言 s± → 中子共振峰、QPI、μSR 给出支持 [猜想，有实验支持]
  → D4 是 C2 的核心材料基础；C2 综述了 s± 的理论推导，D4 给出实验证据
  → 但 FeSe 等反例提醒：s± ≠ 铁基配对的终极答案

B3（AFM 自旋涨落）：
  铁基是 B3 框架的 **最重要实验验证体系**（相较于铜氧化物的不确定性）
  → Q=(π,0) 处的 AFM 涨落 → 连接 Γ 和 M 口袋 → s± 配对（定量描述较铜氧化物成功）
  → D4 是 B3 的核心材料案例；B3 综述 AFM 涨落 → s± 的理论机制

B5（向列涨落）：
  铁基是 **向列涨落在超导中作用的核心研究平台**
  → Ba-122 等材料在掺杂不足区有明确的向列相 → B5 框架的最直接实验对象
  → D4 是 B5 的核心材料；B5 综述 form factor 选择性和向列涨落增强机制

B10（Hund 耦合）：
  铁基是 Hund's metal 的 **代表材料**（de'Medici 2011 框架）
  → D4 是 B10 的核心材料案例；B10 综述 J_H 的 Janus 特性和多轨道物理

B12（多通道叠加）：
  Romero 2026 用 Paper A 框架分析铁基 → 铁基是 **多通道 Eliashberg 框架的验证材料**
  → B12 的成功（对 Ba-122 等）是 D4 的核心量化引用；D4 是 B12 的材料平台

D3（铜氧化物）：
  D4 与 D3 是 **高温超导双峰**：铁基 + 铜氧化物 = 两大高温超导体系
  → 两者对比是理解"高温超导共同物理"或"各自独特物理"的关键
  → D3: 单轨道/Mott/d-wave；D4: 多轨道/Hund metal/s±（通常情况）

D11（2D/界面）：
  FeSe/STO 是 **界面增强超导的最重要案例**
  → D4 → D11 方向：铁基母体（FeSe）在界面上 Tc 增强 8 倍的现象和机制
  → D11 综述界面 SC 的系统性物理；D4 提供铁基界面的具体背景

E1/E2（压力/掺杂）：
  铁基相图（掺杂 → SDW 消失 → SC 出现）是 E2 的核心案例
  FeSe 压力相图（无磁序 → 高压 SDW 出现 + Tc 上升）是 E1 最反直觉的案例

D6（氢化物）vs D4（铁基）：
  两者 Tc_max（常压）= 55K vs 铜氧化物 134K → 同为高温超导，但机制完全不同
  铁基：BCS+涨落（准粒子有效），Tc 受限于涨落能量尺度
  氢化物：BCS 声子（Eliashberg），Tc 受限于 ω_log 和 λ

B1-B2（BCS/Eliashberg）与 D4 的边界：
  铁基比铜氧化物"更接近 BCS 框架"——多带 BCS 有效 [猜想]
  但铁基比 A15/MgB₂ 更复杂：需要多轨道 + 涨落 + Hund 修正
  → 铁基是"BCS 框架有效但需要重要修正"的材料区间
  → Paper A（多通道 Eliashberg）对铁基有一定适用性（Romero 2026 验证）
  → 对比铜氧化物（D3）：Paper A 对铜氧化物完全不适用（Mott 物理主导）
```
