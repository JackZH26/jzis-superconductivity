## 1. 物理定义与核心思想

### 1.1 CDW 的基本定义 [观察，实验确认]

```
=== 电荷密度波（CDW）的定义 ===

核心现象 [观察]：
  电子系统中出现空间周期性的电荷密度调制，同时伴随晶格畸变。

CDW 序参量（空间调制形式）：
  ρ(r) = ρ₀ + ρ_Q cos(Q·r + φ)
  其中：
    ρ₀ = 均匀背景电荷密度
    ρ_Q = CDW 调制振幅（序参量大小）
    Q = CDW 波矢（与嵌套波矢 2k_F 相关）
    φ = CDW 相位（滑动 CDW 时相位连续变化）

实验标志 [观察]：
  → 电子衍射 / X 射线：出现超晶格衍射峰（波矢 Q 的 Bragg 峰的伴随峰）
  → STM：实空间周期性电子密度调制直接成像
  → ARPES：在 k 和 k+Q 处能谱出现 CDW 能隙（折叠带）
  → 比热：CDW 相变处出现比热跳变（二级或弱一级相变）

CDW vs 超导的根本区别 [观察]：
  维度       CDW                    超导（SC）
  ---        ---                    ------
  序参量     ρ_Q（实标量/复数相位） Δ（复数配对振幅）
  时间反演   保持                   保持（singlet）
  平移对称性 破坏（周期调制）       保持
  U(1) 规范  保持（CDW 不涉及规范场）破坏（SC 是规范对称破缺）
  输运响应   绝缘/半金属（CDW 能隙）零电阻（Meissner）
```

### 1.2 CDW 的微观起源：Peierls 不稳定性 [定理，1D；推广为猜想，2D/3D]

```
=== Peierls 不稳定性：CDW 的理论根基 ===

【1D 严格结论（[定理]，T=0 时绝对不稳定）：】

  1D 电子链（半填充，Fermi 波矢 k_F）：
  完美 Fermi 面嵌套：k_F → -k_F 的波矢 Q = 2k_F

  Lindhard 响应函数 χ(Q = 2k_F) 在 1D 发散（对数发散）：
    χ(2k_F) ~ -N(EF) × ln(Λ/T)（T→0 时发散）
  → 即使无穷小的电子-声子耦合 g 也能触发 CDW 不稳定性
  → CDW 能隙 Δ_CDW ~ Λ × exp(-1/N(EF)g²)（类 BCS 指数形式）

  CDW 形成后：
  → 能带折叠：在 k = ±k_F 处打开 CDW 能隙 2Δ_CDW
  → 能谱：ε_k → ±√(ξ_k² + |Δ_CDW|²)（完全类似 BCS 配对能隙）
  → 晶格畸变：声子在 Q=2k_F 处软化 → 静态畸变（Kohn 异常）
  → 系统总能量降低（Peierls 能量增益 ~ Δ_CDW²/2 × N(EF)）

【2D/3D 推广（[猜想，近似，依赖嵌套质量]）：】

  2D/3D 中 χ(2k_F) 不再对数发散（Fermi 面曲率抑制嵌套）
  → CDW 只在特定嵌套波矢 Q 处出现（Fermi 面嵌套"好"的方向）
  → CDW 的稳定性依赖嵌套质量（完美嵌套 → CDW 强）
  → 铜氧化物、NbSe₂ 等 2D 体系的 CDW 起源争议：
    - 完全是 Fermi 面嵌套？[猜想，不完美]
    - 还是有 Mott / 强关联的额外贡献？[猜想，铜氧化物]
    - 或者 van Hove 奇点（态密度发散）驱动？[猜想，Kagome]

【CDW 对 Fermi 面的影响（[定理，CDW 形成后的单粒子谱]）：】

  CDW 打开部分能隙（而非全部）：
  → Q = 嵌套波矢时，k 和 k+Q 处的电子混合
  → 能谱分裂成两个分支：±√(ξ_k² + Δ_CDW²)
  → N(EF) 降低（被 CDW 能隙"吃掉"的 k 区域失去态密度）
  → SC 受影响：BCS Tc ∝ exp(-1/(N(EF)×V))
                N(EF) 降低 → Tc 降低（CDW 压制 SC）[定理，在 BCS 框架内]
```

### 1.3 CDW 与超导的能量竞争 [猜想，现象学]

```
=== CDW 与 SC 的竞争图像（现象学）===

简单竞争视图 [猜想，现象学]：
  → CDW 和 SC 都利用 Fermi 面附近的电子配对能量
  → CDW 消耗 N(EF) → SC 无法获得足够的态密度配对
  → 两者争夺相同的电子自由度
  → 经验规律：压力/掺杂 → CDW 消失 → SC 出现/增强 [来源待验证，多体系]

压制 CDW → 增强 SC 的机制（两种可能，均[猜想]）：
  1. 直接竞争：CDW 消失 → N(EF) 恢复 → BCS Tc 升高
  2. 涨落驱动：CDW 量子临界涨落（接近 QCP）→ 增强配对 [猜想，B6 相关]

协同视图（争议更大，[猜想]）：
  → CDW 涨落本身可以作为配对玻色子（软声子 = 交换粒子）
  → CDW 相邻区域 CDW 量子临界点 → SC 穹顶
  → 支持证据：多种体系 SC 穹顶在 CDW 消失处出现 [来源待验证]
  → 反对：这只是相图的巧合，不一定是因果 [猜想，因果链未证明]
```

## 2. 历史关键节点

```
=== CDW 研究的历史里程碑 ===

【1954-1955 Peierls/Fröhlich】[来源待验证，理论奠基]
  → Peierls：1D 金属的晶格不稳定性（"Peierls 不稳定性"）[来源待验证]
  → Fröhlich：无摩擦滑动 CDW 的超导类比（CDW ≠ 超导，后来被否定）
  → 奠定了 CDW 理论基础，但 1D CDW ≠ 超导

【1970-1980s NbSe₂、NbS₂、TaS₂ 早期研究】[来源待验证]
  → 层状过渡金属二硫化物（TMDs）是经典 CDW 体系
  → NbSe₂ 体材：CDW（33K）+ SC（7.2K）共存 [来源待验证]
  → TaS₂：多个 CDW 相（不公度/公度/近公度），无超导 → CDW 不必然伴随 SC

【1988-1992 铜氧化物中 CDW 早期信号】[来源待验证]
  → LSCO 欠掺杂区中子散射发现异常散射信号（早期条纹序迹象）
  → 尚不清晰，解读有争议

【1995 Tranquada et al. — 条纹序（CDW+SDW 复合序）】[来源待验证，经典实验]
  → La₁.₈₇₅Ba₀.₁₂₅CuO₄（LBCO，x=1/8）中子散射
  → 发现自旋（SDW）+ 电荷（CDW）条纹序在 x=1/8 完美稳定
  → Tc 严重抑制至约 4K（正常 La₂₋ₓSrₓCuO₄ 的 Tc 在 x=1/8 约 30K）
  → 这是 CDW 与 SC 竞争的最清晰实验案例之一 [来源待验证]
  → ⚠️ "Tranquada 1995" 的精确期刊引用：Nature 375, 561 (1995) [来源待验证，需核对]

【2012-2015 铜氧化物 CDW 系统性发现（X 射线时代）】[来源待验证，多组]
  → Chang et al. (2012) [来源待验证]：YBCO 的 X 射线衍射发现 CDW 波矢 Q ≈ (0.315, 0) a*
  → Blackburn et al. (2013) [来源待验证]：中子散射确认
  → Comin et al. (2014) [来源待验证]：BSCCO 中 X 射线 CDW
  → 多组在不同铜氧化物体系（YBCO/BSCCO/Hg-1223）发现 CDW 普遍存在 [来源待验证]
  → 标志：铜氧化物 CDW 从"偶然现象"升格为"普遍特征"

【2016 Xi et al. — NbSe₂ 单层的 CDW 增强 + SC 降低】[确定，NatPhys]
  → Xi et al. (2016) Nature Physics 12, 139
  → 单层 NbSe₂：CDW Tc（33K → 145K），SC Tc（7.2K → 3K）
  → 最干净的维度效应 + CDW/SC 竞争实验证据

【2021 AV₃Sb₅ Kagome CDW 发现】[来源待验证，Ortiz 2020/2021]
  → CsV₃Sb₅（Kagome 超导体）：CDW（94K）+ SC（2.5K）共存 [来源待验证]
  → Ortiz et al. (2020/2021)：发现 AV₃Sb₅ 系列材料 [来源待验证，精确引用待核]
  → CDW 的起源（Fermi 面嵌套/van Hove 奇点/平坦带）争议持续至今
```

## 3. 铜氧化物的 CDW（最重要案例，D3 关联）

### 3.1 LBCO x=1/8 效应

```
=== La₁.₈₇₅Ba₀.₁₂₅CuO₄（LBCO）的条纹序 ===

x=1/8 的特殊性 [观察，Tranquada 1995，来源待验证]：
  → La₂₋ₓBaₓCuO₄ 在 x=1/8（空穴浓度 p=1/8）处：
    - 条纹序（stripe order）完美稳定：SDW + CDW 联合序
    - SDW 波矢 Q_SDW ≈ (π/a, π/a ± π/4a)（自旋条纹）
    - CDW 波矢 Q_CDW ≈ (π/2a, 0) 或 (0, π/2a)（电荷条纹，周期 4a）
    - 条纹是"空穴聚集线 + 反铁磁绝缘域"交替出现的空间调制
  → Tc 严重抑制：约 4K [来源待验证]
    对比：La₂₋ₓSrₓCuO₄ 在 x=0.125 处 Tc ≈ 30K [来源待验证]
    → x=1/8 使 Tc 降低约 26K → CDW+SDW 复合序强烈压制 SC

x=1/8 效应的物理图像 [猜想，强关联视角]：
  → 1/8 掺杂 = 每 8 个铜原子填入 1 个空穴 → 约分成 4a 周期最稳定
  → La₁.₈₇₅Ba₀.₁₂₅CuO₄ 中 Ba²⁺ 替代 La³⁺ 导致 LTT（低温四方）晶格畸变
  → LTT 相钉扎住条纹序（条纹与晶格畸变协同），使其特别稳定 [猜想，LTT 钉扎机制]
  → 稳定的条纹序 → SC 配对通道被完全压制 → Tc 极低

La₂₋ₓSrₓCuO₄（LSCO）的 x=1/8 弱化版 [观察]：
  → LSCO 在 x ≈ 1/8 附近也有 Tc 的弱压制（"1/8 anomaly"），但远没有 LBCO 严重
  → 因为 LSCO 的晶格没有 LTT 畸变，条纹序不被钉扎，只是涨落而非长程序
  → 对比：长程 CDW 序压制 SC >> CDW 涨落对 SC 的影响
```

### 3.2 YBCO、BSCCO 等体系的 CDW

```
=== YBCO / BSCCO 体系的 CDW（X 射线时代，2012-2015 之后）===

CDW 的基本特征 [观察，多组 X 射线实验，来源待验证]：
  波矢：Q ≈ (0.25-0.35) a*（不同铜氧化物体系略有不同）
    YBCO：Q_CDW ≈ (0.315, 0) a* [来源待验证，Chang 2012]
    BSCCO：Q_CDW ≈ (0.25, 0) a* [来源待验证]
  出现温度：T_CDW ≈ 100-200K（在欠掺杂区，低于赝能隙温度 T*）[来源待验证]
  强度：欠掺杂越深，CDW 越强（与赝能隙强度趋势一致）

SC 和 CDW 的磁场竞争（关键实验证据）[观察，来源待验证]：
  → 零场条件：SC + CDW 在铜氧化物欠掺杂区共存（低场）
  → 高磁场（H >> Hc₂，超导被压制）：
    CDW 衍射峰强度显著增强！
    → 磁场压制 SC → CDW 取而代之，填充"空出来"的 Fermi 面
    → 这是 CDW 和 SC 竞争同一 Fermi 面最直接的实验证据 [来源待验证，Chang 2012 等]
  → 量子振荡（高场 de Haas-van Alphen）：
    发现小 Fermi 口袋（F ≈ 530 T，Doiron-Leyraud 2007 [来源待验证]）
    → 与 CDW 重构 Fermi 面产生的口袋一致 [猜想，Fermi 面重构解读]

CDW 与 T* 的时序关系（关键！）[观察]：
  T* >> T_CDW：在铜氧化物欠掺杂区，CDW 在赝能隙内出现（CDW 晚于赝能隙）
  → 这暗示 CDW 可能是赝能隙区的次生序（不是赝能隙的原因）
  → 但也可能：T* 以下出现的 CDW 涨落先出现，长程 CDW 序在更低温稳定
  → 因果关系未定（H3 的核心争议之一，此处不重复展开）
```

## 4. NbSe₂ 的 CDW（维度效应最干净案例，D13 关联）

```
=== NbSe₂：CDW + SC 竞争的教科书案例 ===

体材 NbSe₂（3D 极限）[观察，来源待验证]：
  → CDW 转变温度：T_CDW = 33K（三倍周期超结构，Q ≈ (1/3, 0) a*）
  → SC 转变温度：Tc = 7.2K
  → 两者共存（T_CDW > Tc，先形成 CDW 再进入超导）
  → CDW 为非公度 CDW（incommensurate CDW），来自 Fermi 面嵌套 [猜想，嵌套机制]

单层 NbSe₂（2D 极限）[确定，Xi 2016, Nat Phys 12, 139]：
  → CDW 转变温度：T_CDW = 145K（大幅升高！相比体材 33K）
  → SC 转变温度：Tc = 3K（大幅降低！相比体材 7.2K）
  → 维度效应极为清晰：2D → CDW 增强 + SC 降低

维度效应的物理图像 [猜想，多种解读竞争]：
  → CDW 增强（体材 33K → 单层 145K）：
    1. 2D 中 Fermi 面嵌套更"平"，χ(Q) 发散更强 [猜想，嵌套增强]
    2. 层间耦合 J_perp 在 3D 体材中减弱 CDW（相位涨落被抑制），
       2D 中无层间耦合 → CDW 更稳定 [猜想]
    3. 表面效应（单层≠真2D，可能有基底影响）[猜想，实验注意]
  → SC 降低（体材 7.2K → 单层 3K）：
    1. 2D 中 Mermin-Wagner 定理：长程 SC 序受限（→ BKT 转变），
       有效 Tc 降低 [定理适用框架]
    2. CDW 增强消耗更多 N(EF) → SC 配对资源减少 [猜想]
    3. 相位涨落在 2D 更强烈 → Tc（超流相干）进一步降低

NbSe₂ 的关键意义：
  → 相同材料，只改变厚度（层数），CDW 和 SC 反向演化
  → 这是 CDW/SC 竞争关系最干净的实验论据之一
  → 铜氧化物体系 CDW 与 SC 的竞争证据来自不同掺杂样品（混杂了多个变量），
    NbSe₂ 提供了变量最少的比较
```

## 5. Kagome AV₃Sb₅ 的 CDW（最新热点，D14 关联）

```
=== AV₃Sb₅（A=Cs, K, Rb）Kagome 超导体中的 CDW ===

基本参数 [来源待验证，Ortiz 2020/2021，精确引用待核]：
  CsV₃Sb₅：CDW T_CDW = 94K，SC Tc = 2.5K
  KV₃Sb₅：CDW T_CDW = 80K，SC Tc = 0.93K [来源待验证]
  RbV₃Sb₅：CDW T_CDW = 103K，SC Tc = 0.92K [来源待验证]
  → 三个兄弟材料均呈现 CDW + SC 共存，T_CDW >> Tc

CDW 的复杂性（Kagome CDW 的独特之处）[来源待验证，多组 STM/X 射线]：
  → 超结构：2×2×4（面内 2×2 + 层间 4 倍周期）
  → CDW 对称性争议：
    - 是否有手性（chirality，TRS 破缺）？[猜想，部分 STM 实验报告]
    - 可能是 Star-of-David 型 vs Tri-Hexagonal 型 [来源待验证]
  → AHE（反常霍尔效应）：
    - 部分实验报道 CDW 相中有 AHE → 暗示 TRS 破缺？[猜想，争议大]
    - ⚠️ AHE ≠ 确认 TRS 破缺（量子几何，Berry 曲率，等多种来源）

CDW 与 SC 的关系（压力调控）[观察，来源待验证]：
  → 施加压力 → T_CDW 降低 → Tc 升高（在 CsV₃Sb₅ 中）[来源待验证]
  → 这与 CDW 和 SC 竞争的图像一致（CDW 被抑制 → SC 获得更多资源）
  → ⚠️ 压力同时改变带宽/嵌套/电声耦合，不能确认 CDW 是唯一原因 [猜想]

Kagome CDW 可能的非常规起源 [猜想，多竞争机制]：
  1. Fermi 面嵌套：Kagome 格子在 M 点附近有嵌套向量 Q ≈ (1/4, 0) a* [猜想]
  2. van Hove 奇点（VHS）驱动：Kagome 在 M 点有 VHS（态密度发散）[来源待验证]
  3. 平坦带关联：Kagome 平坦带虽不在 EF（关键反谄媚！D14 已述），
     但平坦带的残余 + VHS 协同可能驱动 CDW [猜想]
  4. Bond-order（键序）：CDW 可能更好地理解为键序（bond-order wave），
     而非纯电荷调制 [猜想，争议]

意义（Kagome CDW 为什么特别值得研究）：
  → Kagome 的几何挫败（frustration）可能引入新的 CDW 物理（超越传统 Peierls）
```

## 6. CDW 竞争 vs CDW 涨落驱动 SC 的争论

```
=== 核心争论：CDW 压制 SC（传统）还是 CDW 涨落增强 SC（现代）？===

【传统竞争视图（占主导）[观察支持]：】

  论据：
  → N(EF) 机制：CDW 打开 Fermi 面部分能隙 → N(EF) 降低 → SC 减弱 [定理，BCS 框架内]
  → 实验：NbSe₂ 单层 CDW 增强+SC 降低 [Xi 2016，确定]
  → 实验：LBCO x=1/8 条纹序稳定+Tc 几乎归零 [Tranquada 1995，来源待验证]
  → 实验：高磁场压制 SC → CDW 增强（铜氧化物，[来源待验证]）
  → 实验：压力抑制 CDW → Tc 升高（多体系，[来源待验证]）
  结论：长程 CDW 序压制 SC —— 这个方向有较强实验支持

【现代涨落视图（B6 的核心，争议中）[猜想]：】

  论据：
  → CDW 量子临界点（QCP）附近：CDW 涨落（软声子）增强 → 配对增强
  → 相图观察：SC 穹顶经常出现在 CDW 消失点附近（量子临界涨落驱动？）[来源待验证]
  → 理论：CDW 软声子作为配对媒介（B6 的机制），类似 AFM 涨落驱动 SC（B3）
  反对：
  → SC 穹顶在 CDW 消失处 → 这是巧合？还是 CDW 消失 + N(EF) 恢复驱动 SC？[猜想]
  → "涨落增强配对"需要 CDW 涨落谱函数的定量论证，目前仅是定性猜想

【与铁基超导 SDW/SC 的平行 [猜想，类比]：】

  铁基：SDW（自旋密度波）与 SC 也有类似的竞争/协同讨论（G1 关联）
  → AFM/SDW 压制 SC（竞争）：实验支持
  → AFM 涨落驱动 SC（协同）：理论/实验部分支持（B3）
  → CDW/SC 的关系与 SDW/SC 高度平行，但 CDW 是电荷序、SDW 是自旋序

【当前共识（2026-04）：】

  [观察，较强支持]：长程 CDW 序压制 SC（N(EF) 机制 + 多体系实验）
  [猜想，弱支持]：CDW 涨落可能增强特定配对通道（定量论证缺乏）
  [猜想，因果未定]：SC 穹顶在 CDW QCP 附近的机制（CDW 涨落 vs N(EF) 恢复）
```

## 7. CDW 的实验测量方法

```
=== 测量 CDW 的主要实验手段 ===

【1】X 射线 / 中子散射（最直接的长程序判据）[观察]
  → 互补信息：
    X 射线：电荷衍射峰（超晶格反射，波矢 Q 处）
    中子散射：磁性衍射（SDW）+ 晶格（声子）
  → 优点：直接给出 CDW 波矢 Q 和温度演化（T_CDW 处强度突然增加）
  → 缺点：需要大体积单晶，空间平均（无实空间分辨）

【2】STM（扫描隧道显微镜）（实空间最直接）[观察]
  → 直接成像 CDW 的空间调制（周期性条纹）
  → 可以观察 CDW 畴结构、缺陷钉扎、CDW 相位滑移
  → 局域谱测量（dI/dV）：CDW 能隙在 ±Δ_CDW 处的相干峰
  → ⚠️ 表面敏感：测量的是表面态，体态信息间接

【3】ARPES（角分辨光电子谱）（k 空间能隙直接观测）[观察]
  → CDW 折叠带：在 k 和 k+Q 处的带交叉处打开能隙（"背折叠"）
  → 直接观测 CDW 在 k 空间的能隙位置和大小
  → 可以比较 CDW 能隙 vs 超导能隙的 k 空间分布

【4】高磁场 + SC 压制（动态竞争最直接）[观察]
  → 施加 H >> Hc₂：SC 被场压制
  → 同时测量 X 射线 CDW 强度：CDW 是否在 H 增大后增强？
  → 若 CDW 增强 → 直接证明 CDW 和 SC 竞争同一 Fermi 面资源 [来源待验证，铜氧化物]
  → 量子振荡（de Haas-van Alphen）：高场下 Fermi 面口袋信息

【5】比热（热力学相变判据）[观察]
  → T_CDW 处比热 C(T) 的跳变（二级相变的热力学信号）
  → CDW 比热跳变通常比 SC 相变的跳变小（CDW 只涉及部分 Fermi 面）
  → 与 Tc 处的比热跳变区分（温度分离时容易区分）

【6】输运（间接但简便）[观察]
  → CDW 出现时电阻率 ρ(T) 在 T_CDW 处可能有斜率变化（部分 Fermi 面能隙化）
  → 霍尔系数 R_H 变化（Fermi 面重构后载流子数改变）
  → ⚠️ 输运对 CDW 和其他散射机制难以区分，需结合其他方法
```

## 9. 关键反谄媚（Anti-Sycophancy）

```
⚠️ 反谄媚 H4.1：CDW 压制 SC ≠ CDW 是赝能隙的原因

  流行表述："CDW 打开了赝能隙"或"赝能隙就是 CDW 能隙"
  实际层级：[猜想，因果关系严重不确定]

  核心问题：
  → T_CDW（CDW 出现温度）< T*（赝能隙温度）（铜氧化物欠掺杂区）
  → 如果 CDW 是赝能隙的原因，CDW 应该先于赝能隙出现（T_CDW ≥ T*）
  → 实际上：赝能隙先出现（T*），CDW 后出现（T_CDW）→ CDW 更可能是次生序
  → 此外：CDW 波矢 Q ≈ (0.3, 0) a* 对应 Fermi 面某特定嵌套方向，
    赝能隙集中在反节点 (π,0)（更大 k 空间区域）→ k 空间位置不完全重合

  正确表述：
  → "CDW 和赝能隙在铜氧化物欠掺杂区共存，但因果关系未定" [猜想]
  → "CDW 本身压制 SC（N(EF) 机制），但与赝能隙的关系是独立的争议"

⚠️ 反谄媚 H4.2：CDW 竞争 SC ≠ 长程 CDW 序必然出现

  流行表述："只要有 CDW 竞争，就会看到 CDW 衍射峰"
  实际层级：[错误！长程 CDW 序和 CDW 涨落（短程序）效应完全不同]

  具体区别：
  → 长程 CDW 序（T < T_CDW）：X 射线出现清晰超晶格衍射峰，Fermi 面完全重构
  → CDW 涨落（T > T_CDW 或无长程序）：谱函数弥散，X 射线只有漫散射
  → 铁基超导某些体系（e.g., FeSe）：有类似 CDW 的涨落信号，但无长程 CDW 序 [来源待验证]
  → 铜氧化物某些欠掌杂样品：CDW 涨落在 T_CDW 以上已存在，但不影响 SC 同等效果

  正确表述：
  → "区分长程 CDW 序（压制 SC 效果强）和 CDW 涨落（效果弱，可能驱动 SC）[观察 vs 猜想]"

⚠️ 反谄媚 H4.3：不同材料的 CDW 起源可能完全不同

  流行表述："CDW 都是由 Fermi 面嵌套（Peierls 不稳定性）驱动的"
  实际层级：[猜想，NbSe₂ 有证据，铜氧化物和 Kagome 争议大]

  体系分析：
  → NbSe₂：Fermi 面嵌套机制相对有说服力（准 2D + 特定嵌套 Q [来源待验证]）
  → 铜氧化物：Mott 背景 + 强关联 + 可能根本不是简单 Fermi 嵌套
    → RVB/Mott 效应可能驱动 CDW（B14/B9 关联）[猜想]
  → Kagome（AV₃Sb₅）：van Hove 奇点（M 点 VHS）vs Fermi 嵌套 vs 平坦带关联 [猜想，争议中]
  → 把三个体系的 CDW 都归因为"Fermi 面嵌套"是严重过度简化

  正确表述：
  → "NbSe₂ CDW 的嵌套机制有一定支持；铜氧化物 CDW 起源争议大（可能强关联主导）；
    Kagome CDW 涉及 VHS/平坦带等 Kagome 特有机制" [各为猜想]

⚠️ 反谄媚 H4.4：CDW + SC 空间共存 ≠ 两者不竞争

  流行表述："NbSe₂（或铜氧化物）中 CDW 和 SC 共存，所以两者不竞争"
  实际层级：[逻辑错误，共存不等于不竞争]

  物理分析：
  → 空间共存的两种情形：
    1. 相分离：空间中不同区域分别是 CDW 相和 SC 相（样品不均匀）
    2. 微观共存：同一晶胞内，两种序参量共存（各自占据不同 k 空间区域）
  → 即使微观共存，SC 仍然因 N(EF) 减少而比没有 CDW 时弱！
    → 共存意味着"两者都存在"，不意味着"两者不相互影响"
  → NbSe₂：CDW（33K）和 SC（7.2K）共存，但 SC 的 Tc 明显低于 CDW 消失后的预期 [猜想]
  → 单层 NbSe₂：CDW 增强时 SC 降低 → 最直接的竞争证据，即使在"共存"体材中也竞争

  正确表述：
  → "共存不等于不竞争：长程 CDW 序即使与 SC 共存，也通过 N(EF) 压制 SC Tc" [猜想→观察]
```
