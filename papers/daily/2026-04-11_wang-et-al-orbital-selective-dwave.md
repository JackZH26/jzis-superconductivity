# Orbital-Selective d-wave Superconductivity in the Two-Band t-J Model: Applications to La3Ni2O7
- **作者:** Zhan Wang, Kun Jiang, Fu-Chun Zhang, Hui-Ke Jin
- **来源:** arXiv:2604.08319
- **年份:** 2026 (April)
- **研究线:** SC
- **精读日期:** 2026-04-11

## 核心贡献（一句话）
用 VMC 证明双带 t-J 模型中超导完全来自流动轨道（dx²-y²），而局域化轨道（dz²）通过形成轨道间束缚态抑制超导——指出提升 Tc 的路径是抑制 dz² 轨道参与。

## 关键方法论
- **双带 t-J 模型:** 轨道-0（dx²-y² like, 流动, t⁰⁰ 大）+ 轨道-1（dz² like, 准局域, t¹¹ 小）
- **VMC (Variational Monte Carlo):** Gutzwiller 投影波函数 |ψ⟩ = P_G |ψ_MF⟩
- **超交换能量层级分析:** 
  - 最低能：轨道-0 内 AFM 单态 (-4(t⁰⁰)²/U per pair)
  - 次低能：轨道间密度-密度吸引 (-n_{i,0}n_{j,1})，与自旋无关
  - 最高能：轨道-1 内交换 (~(t¹¹)²/U)
- **SC 序参量:** Δ^{αβ} 从 pair-pair 关联函数 Φ^{αβ}(r) 的饱和值提取
- **参数空间扫描:** t⁰¹ ∈ [0.1, 0.6], t¹¹ ∈ [0.1, 0.6], U=8, 20×20 格子

## 主要结果
1. **[VMC 定理]** Δ⁰⁰（轨道-0 内配对）是唯一显著的 SC 序参量，Δ¹¹ ≈ Δ⁰¹ ≈ 0.005（接近误差棒）
2. **[关键机制]** 轨道-1 占据 ⟨n₁⟩ 增加 → Δ⁰⁰ 单调下降（能量缺陷机制）
3. **[dome]** Δ⁰⁰ 在 δ ≈ 0.16-0.2 有 dome 最优掺杂（与单带 t-J 一致）
4. **[La3Ni2O7 应用]** t⁰¹ ≈ 0.5t⁰⁰, t¹¹ ≈ 0.2t⁰⁰ 对应 La3Ni2O7 参数范围
5. **[μ_z 效应]** 增加轨道-1 的 onsite 能量分裂 μ_z → n₁ 增加 → Δ⁰⁰ 降低
6. **[处方]** 提升 Tc 的路径：减小轨道能量分裂，抑制 |z,-⟩ 轨道参与

## 与我们研究的关联
- **直接挑战/补充 Ji+2026 框架：**
  - Ji+2026 说 J⊥ 控制 Tc magnitude → 本文说 J⊥ 来自 dz² 层间耦合
  - 但本文同时说 dz² 占据抑制 SC → **存在最优 dz² 参与度**
- **对 PSPD 模型的启示：** ΔJT（Jahn-Teller 畸变）控制 eg 轨道分裂 → 控制 n₁ → 控制 Tc
  - ΔJT 大 → 轨道分裂大 → n₁ 大 → Tc 低（与 PSPD onset 条件一致！）
  - ΔJT ≈ 0.33 eV 是 onset 阈值 → 可能对应 n₁ 从 ~0 到 ~5% 的跨越
- **STO/20GPa 解释：** STO 的大 a_p 给出大 ΔJT → 大 n₁ → 弱 SC（Tc=10K）

## 潜在问题/局限
- VMC 精度有限（trial wavefunction bias）
- 参数范围 t¹¹ ≈ 0.2t⁰⁰ 下轨道-1 占据仅 ~10% → 量化预测困难
- 没有考虑 Hund's coupling（J_H = 0）
- 单层 2D 模型 vs La3Ni2O7 的双层 3D 结构

## 关键数值（如有）
- 最优掺杂: δ ≈ 0.16-0.20 [VMC, 与实验一致]
- 现实参数: t⁰¹ ≈ 0.5t⁰⁰, t¹¹ ≈ 0.2t⁰⁰ [来自 Wang+2025d DFT]
- n₁/n ≲ 10% 对 t⁰¹, t¹¹ ~ 0.5t⁰⁰ [VMC]
- U = 8t⁰⁰ [标准 Hubbard 参数]

## 启发的新想法
- **[可验证假设]** 存在最优 ΔJT*，使得 dz² 参与刚好足以提供 J⊥ 但不过多抑制 SC。预测：SLAO 基板 (a_p=3.756) 的 ΔJT ≈ 0.39 eV 接近这个最优值 → Tc = 40-48K。更大压缩（bulk 14 GPa）→ ΔJT 更大但 I4/mmm 结构 → Tc = 80K
- **[理论方向]** 将 VMC 结果与 Ji+2026 SBMF+DMRG 结合：定量确定最优 n₁ 值
