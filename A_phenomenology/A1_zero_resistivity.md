# A1. 零电阻（Zero Resistivity）

**MECE 编号：** A1
**关联 MECE：** A2（Meissner，配对的两个独立证据）、A6（临界场）、E7（无序）、H1（strange metal 正常态）
**精读日期：** 2026-04-27
**精读模型：** Claude Opus 4.7（high thinking）
**层级：** 综述（基于经典文献）

## 1. 物理定义

**零电阻：** 当温度 T < T_c（临界温度）时，超导体的直流电阻率
$$\rho(T < T_c) = 0$$
在仪器精度内。这是超导体最早被识别的特征，也是定义性观察之一。

### 1.1 严格定义需要区分

| 概念 | 含义 |
|------|------|
| ρ = 0（理想超导）| 无限长持续电流 |
| ρ = 完美导体 | 电场 E = 0，但 dB/dt = 0（不要求 B = 0）|
| **超导 ≠ 完美导体** | 超导还要求 B = 0（即 Meissner 效应，A2）|

> **关键澄清：** 1911-1933 间，物理学界曾把超导误认为"完美导体"。1933 Meissner 实验证明超导是更复杂的现象——它**主动排出磁场**（不只是阻止变化）。这是 A1 必须与 A2 联动的物理原因。

## 2. 历史关键节点

| 年份 | 事件 | 来源 |
|------|------|------|
| **1908** | Kamerlingh Onnes 在 Leiden 实现氦液化（4.2 K）| Onnes, Comm. Phys. Lab. Leiden 108 |
| **1911** | Onnes 在固态水银中发现 4.2 K 时电阻骤降至 0 | Onnes, Comm. Phys. Lab. Leiden 120b, 122b, 124c |
| **1913** | Onnes 获诺贝尔物理学奖（液氦 + 超导发现）| Nobel Lecture |
| **1933** | Meissner-Ochsenfeld 发现完全抗磁性（A2）| Naturwissenschaften 21, 787 |

| **1962** | File-Mills 持续电流测量将 ρ 上限定为 < 10⁻²⁵ Ω·cm | PRL 10, 93 |

[定理] Onnes 的实验是历史第一次观测，用经典 4 端法测电阻在 Hg 凝固后突然 < 10⁻⁵ Ω。

## 3. 关键测量量

### 3.1 临界温度 T_c

T_c 有多种定义，必须明确：

| 定义 | 缩写 | 说明 |
|------|------|------|
| Onset Tc | T_c^onset | dρ/dT 急升点（最高温度）|
| Midpoint | T_c^mid | ρ = 0.5·ρ_n 处 |
| 90%-10% width | ΔT | 转变宽度，反映样品均匀性 |
| Zero-resistance | T_c^0 | ρ 首次低于仪器分辨率 |

不同论文用不同定义，对比时必须统一。

### 3.2 持续电流寿命

最严格的 ρ = 0 验证：在闭合超导环中诱导持续电流，监测衰减。
- File-Mills 1962 [PRL 10, 93]：持续电流寿命 > 10⁵ 年（实验下限）
- 由此推论 ρ < 10⁻²⁵ Ω·cm（远小于任何正常金属，铜约 1.7×10⁻⁶ Ω·cm @ 室温）

### 3.3 上下文中的电阻率

| 状态 | 典型 ρ（Ω·cm）|
|------|---------------|
| 室温铜 | 1.7 × 10⁻⁶ |
| 4 K 高纯铜 | ~10⁻¹⁰（受杂质散射限制）|
| 超导体（File 1962 上限）| < 10⁻²⁵ |
| 严格 ρ = 0 | 0 |

## 4. 测量方法

### 4.1 四探针法（最常用）
两端通电流 I，另两端测电压 V。消除接触电阻效应。
- 优点：标准、简单
- 局限：仪器分辨率（典型 ~10⁻⁹ Ω·cm）

### 4.2 持续电流衰减
- 闭合超导环中诱导电流
- 用 SQUID 或外部磁通计长期监测
- 优点：最高精度
- 局限：实验时间长

### 4.3 微波/光学探测
- 复电导率 σ(ω) 在低频极限给出 σ → ∞ 的标志
- 例如 TDTS（A8 / Romero 2026 已用）

### 4.4 远红外反射率
- 反射率在低 ω 上升至 1
- 间接验证

## 5. 已知例外 / 复杂情形

### 5.1 超导涨落（Tc 附近）
- 临界温度上方 ε = (T - T_c)/T_c 时间已有涨落性 SC
- ρ(T) 不是阶跃，而是平滑下降
- 在低维或强关联体系尤其显著

### 5.2 低维（2D）BKT 转变
- 严格 2D 中没有真正长程序（Mermin-Wagner）
- BKT 转变（涡旋-反涡旋）使电阻在某温度下降但不是 BCS 阶跃
- 例：FeSe/STO 单层、MATBG

### 5.3 Anderson 局域化
- 强无序使正常态导带局域化，低 T 时 ρ 可能也增大
- 这与超导的零电阻**性质不同**：Anderson 局域化是非超导态的金属-绝缘体转变

### 5.4 非平衡瞬态超导
- 光致 SC（K₃C₆₀ Mitrano 2016, A8/E6）：~ps 时间尺度的零电阻状响应
- 不是稳定 SC 态，但具有部分 SC 光学特征

### 5.5 拓扑保护的"准零电阻"
- 量子霍尔边态：1D 边态电阻 = h/e² 量子化（非零）
- 不属于超导，但容易混淆

### 5.6 高温部分零电阻
- LK-99 类伪声明：可能是杂质相局部 SC + 体材料正常
- 必须用 Meissner 效应（A2）+ 比热跃变（A3）联合验证

## 8. 数据汇总

[定理] BCS 框架下，T < T_c 时序参量 Ψ ≠ 0 自动给出 ρ = 0。

[观察] 持续电流寿命下限 > 10⁵ 年 [File-Mills 1962, PRL 10, 93]

[定理] Anderson 1959 原始论证：弱杂质散射不破坏 s-wave SC（Anderson's theorem）。但磁性杂质破坏（Abrikosov-Gor'kov 1961）。
