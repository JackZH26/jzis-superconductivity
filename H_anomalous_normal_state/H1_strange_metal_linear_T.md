## 1. 物理定义与核心思想

### 1.1 Fermi 液体 vs Strange Metal [定理 vs 观察]

```
=== 两种正常态的对比框架 ===

【正常金属（Fermi 液体）— [定理，Landau Fermi 液体理论]】

  Landau 准粒子图像（严格成立条件：弱相互作用，低温，短程屏蔽）：
    → 电子-电子散射截面 ∝ (k_B T)²（相空间限制）
    → 弛豫时间：τ ∝ 1/T²（准粒子寿命 ℏ/τ << k_B T）
    → 电阻率（Drude 模型 + Matthiessen 规则）：
        ρ = ρ₀ + A T²   （A 由 e-e 相互作用强度决定）

  Fermi 液体的 T² 电阻率是严格推导结果：
    → 前提：准粒子概念成立（有定义良好的 Fermi 面）
    → 前提：e-e 散射主导（低温，其他散射可忽略）
    → 在低温（T << T_F）干净金属中，T² 定律是量子多体理论的定理级结论

【Strange Metal — [观察]】

  铜氧化物最优掺杂附近（p ≈ 0.16）等强关联体系的实验观测：
    → 电阻率：ρ = ρ₀ + B T   （线性，无 T² 项，或 T² 项极小）
    → 线性 T 区间极宽（从 Tc 以上一直到数百 K 甚至更高）
    → 关键：这个线性不是"近似"，而是在 2-3 个数量级温度范围内成立 [观察]

  物理意义（[观察，物理推导]）：
    ρ ∝ T  →  1/τ ∝ T  →  τ ∝ 1/T
    → Drude 公式：ρ = m*/(n e² τ)，若 ρ ∝ T → 1/τ ∝ T
    → 弛豫率 ℏ/τ ∝ k_B T（散射率与热能成正比）
    → 对比 Fermi 液体：ℏ/τ ∝ (k_B T)² / E_F << k_B T（低温时准粒子寿命长）
    → Strange Metal 中：ℏ/τ 不小于 k_B T，准粒子描述失效？

  ⚠️ 核心谜题 [观察层级]：
    → Fermi 液体中，散射相空间 ∝ T²（e-e 散射的 Pauli 阻塞）
    → Linear-T 中，散射率 ∝ T：需要什么机制提供额外的散射相空间？
    → 没有公认的微观理论解释这个问题 [这是当前领域的核心未解问题]
```

### 1.2 散射率的量子极限与 Planckian 极限 [观察 → H2 详述]

```
=== 从 Linear-T 电阻率到 Planckian 散射率 ===

  Planckian 极限（H2 完整推导）：
    → 定义：1/τ ≡ α × k_B T / ℏ，其中 α = O(1)（量子力学的天然散射率单位）
    → Strange Metal 中实测（铜氧化物最优掺杂）：α ≈ 1 [来源待验证，Legros 2019 等]
    → 含义：散射率接近量子力学允许的最大值（~ 热能频率 k_B T/ℏ）
    → 这不是某个特定机制的预言，而是对 Strange Metal 散射率的实验总结

  Planckian 极限与 Linear-T 的连接 [观察，物理推导]：
    → 若 1/τ = α k_B T/ℏ（Planckian），Drude 公式给出：
        ρ = (m*/n e²) × (1/τ) = (m*/n e²) × α k_B T/ℏ ∝ T
    → 即：Planckian 散射 → Linear-T 电阻率（前提：m*、n 随 T 变化可忽略）
    → 反之：若观察到 Linear-T，不一定是 Planckian（需要额外测量 m*, n 等）

  注意（反谄媚）：
    → Planckian 散射率是对实验结果的描述，不是理论解释
    → "为什么 α ≈ 1？"仍是开放问题
    → H2 专节处理 Planckian 极限的理论框架和实验验证
```

### 1.3 Strange Metal 的实验签名汇总 [观察]

```
=== Strange Metal 的多维实验特征（注意 A8 已详细覆盖，此处仅概览）===

1. ρ(T) = ρ₀ + AT（线性，宽温度范围）[观察，最核心特征]
   → 最优掺杂铜氧化物、铁基最优掺杂附近、重费米子 QCP 附近等

2. 霍尔系数 R_H 温度依赖（H9 关联）[观察]
   → 正常 Fermi 液体：R_H ≈ const（与 T 无关）
   → Strange Metal：R_H(T) 随 T 变化（暗示有效载流子密度或移动率随温度演化）

3. 光导率 σ(ω)：1/ω 型（非 Drude 峰）[观察，来源待验证]
   → 正常 Drude 型：σ(ω) = σ₀/(1 + (ωτ)²)（洛伦兹峰形）
   → Strange Metal：σ(ω) ∝ 1/ω（更宽的谱，无明显 Drude 峰）
   → 暗示：缺乏单一弛豫时间的准粒子描述

4. 热导率 / Wiedemann-Franz 定律偏离 [观察，来源待验证，部分体系]
   → 标准金属：κ/(σT) = L₀（Lorenz 数，独立于材料）
   → 某些 Strange Metal（铜氧化物，来源待验证）：L ≠ L₀
   → 偏离暗示：热导载体与电导载体可能不同（准粒子不完整描述）

⚠️ 反谄媚：ρ ∝ T 不等于所有输运量都线性！
   → ρ(T) 线性是 Strange Metal 最清晰的实验特征
   → R_H(T)、σ(ω)、κ/T 等其他量的异常有额外的系统依赖和争议
   → 不能将单一测量（仅 ρ ∝ T）等同于全面验证了 Strange Metal 状态
```

## 3. Strange Metal 的理论框架（三大竞争，各为 [猜想]）

### 3.1 量子临界点（QCP）涨落机制 [猜想]

```
=== QCP 框架：量子相变附近的临界涨落驱动散射 ===

核心思路：
  → 在量子相变点（T=0 的相变，由非热参数调控），量子涨落发散
  → 临界涨落作为散射"浴"（bath）→ 准粒子散射率增强
  → 不同 QCP 类型给出不同的散射率温度依赖

QCP 类型与散射率（[猜想，取决于 QCP 细节]）：
  → AFM QCP（三维，O(3) 对称性）：1/τ ∝ T^(3/2)（非线性，Hertz-Millis 理论）[猜想]
  → AFM QCP（二维，准二维铜氧化物？）：1/τ ∝ T^α，α 接近 1？[猜想，争议大]
  → 轨道/向列/电荷 QCP 类型：具体指数取决于序参量维度和动力学指数 z
  → 某些 2D 费米子-玻色子耦合模型可能给 α = 1 [猜想，模型依赖]

铜氧化物的 QCP 场景：
  → 假设：最优掺杂 p ≈ 0.16 附近存在一个隐藏的 QCP（赝能隙 → 0 的端点）[猜想]
  → 该 QCP 的临界涨落驱动宽温区的 Linear-T
  → 问题 1：铜氧化物的 QCP 是否真实存在？（隐藏于 SC 穹下，难以直接实验确认）
  → 问题 2：即使 QCP 存在，QCP 通常只影响 QCP 附近有限温区，
             但铜氧化物 Linear-T 在极宽温度范围成立（从 4K 到 ~1000K）
             → QCP 涨落的"扩散"如何影响如此宽的温区？[猜想，机制不清楚]
  → 问题 3：不同类型 QCP 涨落（AFM vs 电荷 vs 向列）给出 T^α，α 不一定恰好 = 1
             需要精细调参才能给 α ≈ 1，缺乏普适预言

重费米子体系的 QCP 支持证据（相对较强）：
  → CeRhIn₅（压力调控 AFM QCP）：QCP 附近线性 T 电阻率 [来源待验证]
  → YbRh₂Si₂（磁场调控 AFM QCP）：线性 T + 非 Fermi 液体行为 [来源待验证]
  → 重费米子的 QCP 可以通过调控参数系统研究（铜氧化物 QCP 难以单独孤立）
  → 但重费米子体系能量尺度（f 电子 Kondo 尺度 ~ meV）与铜氧化物（eV）不同

层级总结：QCP 机制对部分体系（重费米子）有较好支持，
           对铜氧化物的解释仍有重大困难（QCP 存在性争议 + 宽温区问题）
```

### 3.2 Planckian 耗散 / Mottness / 全息理论 [猜想]

```
=== 全息 / 强关联框架：准粒子不存在时的散射 ===

核心思路：
  → 强关联体系（如 Mott 绝缘体附近的铜氧化物）中，准粒子概念可能根本失效
  → 用全息（AdS/CFT）或 "Mottness" 框架描述没有准粒子的金属
  → 预言：散射率自然趋向 Planckian 极限 1/τ ∝ k_B T/ℏ

Marginal Fermi Liquid（MFL）[猜想，唯象模型]：
  → Varma, Littlewood, Schmitt-Rink, Abrahams, Ruckenstein（1989）[来源待验证]
  → 假设：某个临界谱（量子临界形式）提供的散射：Σ''(ω, T) ∝ max(ω, T)
  → 导致：准粒子残余量 Z → 0（log 修正），ρ ∝ T
  → 问题：MFL 是唯象描述，没有给出谱的微观起源（"批出"结果不等于解释）

全息 / AdS-CFT 方法 [猜想，数学框架]：
  → Hartnoll, McGreevy, Horowitz 等（2007-2012）[来源待验证]
  → 思路：将强关联金属对应到 AdS₂ × R² 时空中的 Reissner-Nordström 黑洞
  → 预言：电阻率 ρ ∝ T²/μ（其中 μ 是化学势）[来源待验证]；
           或修正版本给 ρ ∝ T（随散射势选择不同）
  → Planckian 散射 1/τ ≈ k_B T/ℏ 从黑洞热力学自然出现
  → 优点：可以处理没有准粒子的体系（黑洞本身就是非准粒子描述）
  → 问题 1：全息理论是数学框架（AdS/CFT 对应），不是铜氧化物的物理图像
             → "铜氧化物等价于 5 维黑洞"是数学类比，不是物理机制
  → 问题 2：全息模型的微观参数（AdS 几何细节）如何对应到铜氧化物的晶格/轨道/相互作用？
             → 目前无法系统连接
  → 层级：全息预言 ρ ∝ T 是一致性支持，不是微观解释 [猜想]

SYK（Sachdev-Ye-Kitaev）模型 [猜想，玩具模型]：
  → Kitaev（2015）、Sachdev（2015）[来源待验证]
  → N 个费米子的全关联随机相互作用 → 精确可解，无准粒子
  → 给出 ρ ∝ T（某些条件下）[来源待验证]
  → SYK 是模型系统，铜氧化物是否真正实现 SYK 物理？未知 [猜想]

层级总结：Planckian / 全息 / SYK 框架在数学上自洽，
           给出 ρ ∝ T 作为"没有准粒子的金属"的自然预言，
           但这些框架到铜氧化物实际物理的映射尚未建立
```

### 3.3 多临界点 / 相分离 / 非均匀性机制 [猜想]

```
=== 非均匀性机制：Local-T 散射来自相界面 ===

核心思路：
  → 部分体系（有机 SC、某些氧化物）中，Linear-T 可能来自相分离
  → 在 CDW/SC/AFM 相竞争区域，相界面散射给出与 T 成比例的贡献 [猜想]

有机超导体（κ-BEDT-TTF）的讨论 [来源待验证]：
  → Mott 绝缘体附近：第一序相变 + 相分离区域
  → Linear-T 可能部分来自相界面散射（与 QCP 涨落不同）[猜想]

对铜氧化物的适用性（弱）：
  → 铜氧化物最优掺杂区域：相分离和 CDW 竞争较弱（欠掺杂区更明显）
  → 最优掺杂的极宽 Linear-T 温区难以用局域相分离解释
  → 这个机制对铜氧化物主流 Strange Metal 的解释力最弱

层级总结：非均匀性机制可能对部分体系（欠掺杂、有机 SC）有贡献，
           但不是最优掺杂铜氧化物 Strange Metal 的主要解释
```

## 4. 铜氧化物 Strange Metal 详解（最经典案例）

### 4.1 掺杂相图与 Strange Metal 区域

```
=== 铜氧化物掺杂相图中的 Strange Metal 区域 [观察] ===

掺杂依赖（LSCO、YBCO、BSCCO 等均有类似规律）：

  欠掺杂（p < 0.12）：
    → 高温：Linear-T（Strange Metal）
    → 低温：赝能隙（H3），Fermi arc（局部 Fermi 面），CDW 竞争
    → 磁场压制 SC 后：绝缘体（某些 LSCO）或线性 ρ ∝ T [来源待验证]

  最优掺杂（p ≈ 0.16）：
    → 全温范围：Linear-T（从 Tc 以上到 1000K+ 某些材料）[观察，来源待验证]
    → 赝能隙不明显（甚至消失）
    → Fermi 面接近完整（ARPES 可见）
    → Tc 最高

  过掺杂（p > 0.20）：
    → 低温：向 T² 过渡（Fermi 液体逐渐恢复）
    → 高温：仍可能有 Linear-T（但温区收窄）
    → p → 0.27+：接近标准 Fermi 液体（T² 项主导）

关键特征（最优掺杂）：
  → 磁场压制 SC 后（Hc₂ ~ 30-70T），正常态 ρ ∝ T 仍成立 [来源待验证]
  → 这排除了"Linear-T 是 SC 涨落效应"的解释
  → Linear-T 是独立的正常态特征，与 SC 配对分开

主要材料（最优掺杂）[来源待验证]：
  → LSCO (La₂₋ₓSrₓCuO₄)：x ≈ 0.15，Tc ≈ 38K，Linear-T 从 ~10K 到 ~800K
  → YBCO (YBa₂Cu₃O₇₋δ)：δ ≈ 0，Tc ≈ 93K，Linear-T 宽温区
  → BSCCO (Bi₂Sr₂CaCu₂O₈₊δ)：最优掺杂，Tc ≈ 91K
  → Tl₂Ba₂CuO₆₊δ（Tl-2201）：最优掺杂，常用于 Planckian 测量 [来源待验证]
```

## 8. 关键反谄媚

```
⚠️ 反谄媚 H1.1：Linear-T 电阻率不是铜氧化物独有

  错误认知："Strange Metal 是铜氧化物特有的现象，来自 CuO₂ 面的特殊电子结构"
  现实 [观察]：
    → 铁基超导（多带，s± 配对，Fe d 轨道）：最优掺杂附近 Linear-T [Romero 2026]
    → 重费米子（QCP 附近，f 电子，极重质量）：Linear-T [来源待验证]
    → 有机超导（κ-BEDT-TTF 等）：部分报道 Linear-T [来源待验证]
    → 魔角石墨烯（MATBG）：特定填充下 Linear-T [来源待验证]
    → Moiré 系统（TMD 等）：Linear-T 报道 [来源待验证]
  结论：Strange Metal 是超越特定材料化学的普遍现象，
         任何成功的理论必须解释这种普遍性

⚠️ 反谄媚 H1.2：Linear-T 不等于一定是 QCP

  错误推理："ρ ∝ T → 体系处于 QCP 附近"
  现实 [猜想层级]：
    → QCP 是 Linear-T 的一种解释，不是唯一解释
    → 铜氧化物的极宽温区 Linear-T（Tc → 1000K）远超传统 QCP 影响范围
    → QCP 涨落通常只影响 T << J（相互作用尺度）的温区
    → 从 Tc ~ 100K 到 1000K 的 Linear-T 需要额外的解释，QCP 标准理论不够
    → 全息/Mottness 框架同样可以给 Linear-T，且对宽温区更自洽 [猜想]

⚠️ 反谄媚 H1.3：没有公认的微观理论

  当前状态（2026 年）[观察]：
    → QCP 涨落 [猜想]：对重费米子较好，对铜氧化物有困难
    → 全息 / AdS-CFT [猜想]：数学自洽，但物理映射未建立
    → Mottness / RVB [猜想]：铜氧化物专用，其他体系难推广
    → MFL [猜想]：唯象，无微观基础
    → SYK [猜想]：玩具模型，现实材料的适用性未知
  → 40 年研究（1986-2026），仍无公认的 Strange Metal 微观理论
  → 这是强关联凝聚态物理的最核心开放问题之一

⚠️ 反谄媚 H1.4：电阻率线性 ≠ 所有输运量线性

  常见误解：
    "铜氧化物所有输运性质都是线性 T"
  现实 [观察]：
    → ρ(T) ∝ T（线性）：最清晰，多种材料和方向证实
    → R_H(T)：温度依赖，但函数形式比 ρ 更复杂（非简单线性）[来源待验证]
    → σ(ω)：1/ω 型，不是简单线性频率依赖 [来源待验证]
    → κ/T（Lorenz 数）：偏离 L₀，但偏离幅度和方向有争议 [来源待验证]
    → 不同输运量的 Strange Metal 特征不同，需要理论解释所有量，不只是 ρ
```
