# C4. d_xy wave 配对对称性 — B₂g 表示，D₄h 第二 d-wave

**MECE 编号：** C4
**关联 MECE：**
- C3（d_x²-y² — 同族 B₁g/B₂g 对比；d_xy 是 d_x²-y² 旋转 45° 的对应物，在正方晶格中物理不同）
- C1（s-wave 基础 — A₁g vs B₂g 对称性区分）
- C9（Singlet — d_xy 是 singlet 配对，自旋 S=0，与 d_x²-y² 相同自旋结构）
- C10（节点结构 — d_xy 节线在轴方向，vs C3 的对角线方向；节点位置不同但数量相同）
- B3（AFM 涨落 — d_xy 在特定费米面嵌套几何下可从自旋涨落涌现）
- B10（Hund — 铁基多轨道竞争；d_xy 作为 s± 的次要竞争态）
- D4（铁基 — 铁基超导中 d_xy 是理论竞争候选，但实验未确认）
- D8（重费米子 — CeCu₂Si₂ 等早期理论曾提议 d_xy，现已多倾向 d_x²-y²）
- D10（有机 SC — κ-BEDT 三角晶格上 d_xy 和 d_x²-y² 近简并的讨论）
- C7（Chiral — d_xy + d_x²-y² 混合可形成 d+id 手性态，见未解问题）

**层级关系：** C4 是 D₄h 第二个 d-wave 通道（B₂g），与 C3（B₁g）是同族"兄弟"，但在正方晶格中两者物理不同；C4 目前仅有理论支撑，**无材料实验确认**
**精读日期：** 2026-04-28
**精读模型：** Claude Sonnet 4.6（high thinking）
**层级：** 综述（基于 Bickers 1989、Monthoux-Pines 1991、Kuroki 2008、Graser 2009 等；C4 领域原始文献相对稀少）

## 1. 物理定义与核心思想

### 1.1 d_xy 在 D₄h 点群中的位置

**[定理 — 群论事实]**

CuO₂ 平面或铁基 FeAs 层的对称群为 D₄h（四方点群）。D₄h 的 d-wave 通道恰有两个：

| 表示 | 名称 | 基函数（动量空间）| 节线方向 |
|------|------|--------------|---------|
| B₁g | d_x²-y² | cos k_x - cos k_y | 对角线（k_y = ±k_x）|
| B₂g | **d_xy** | **sin k_x sin k_y** | **轴线（k_x = 0, k_y = 0）**|

d_xy（B₂g）是 d-wave 家族的第二成员，与 d_x²-y²（B₁g）正交，节线方向旋转 45°。

**[定理 — 45° 旋转关系]**

在**连续极限**（圆形费米面，忽略晶格）中：

```
B₁g 极坐标形式：Δ(φ) = Δ₀ cos(2φ)
B₂g 极坐标形式：Δ(φ) = Δ₀ sin(2φ) = Δ₀ cos(2(φ - π/4))

即：d_xy 等价于 d_x²-y² 旋转 45°（π/4）
```

**但**在实际正方晶格中，旋转 45° 会改变物理（不同晶轴对应不同轨道和格点），因此 B₁g ≠ B₂g，两者是不等价的不可约表示。

### 1.2 d_xy 的 k 空间结构

**[定义]** d_xy 配对振幅（动量空间，晶格形式）：

$$\Delta(\mathbf{k}) = \Delta_0 \sin k_x \sin k_y$$

LaTeX: $\Delta(\mathbf{k}) = \Delta_0 \sin k_x \sin k_y$

ASCII:
```
Δ(k) = Δ₀ sin k_x sin k_y

参数：
  Δ₀ = 最大配对振幅
  k_x, k_y = 约化布里渊区单位（k × a）
  范围：k_x, k_y ∈ [-π, π]
```

**k 空间极大值位置：**
```
最大点（|Δ| = Δ₀）：(π/2, π/2)、(π/2, -π/2)、(-π/2, π/2)、(-π/2, -π/2)
                    即布里渊区对角线（与 d_x²-y² 的节点位置相同！）
节线（Δ = 0）：      k_x = 0（Γ-Y 线）和 k_y = 0（Γ-X 线）
```

**k 空间分布（ASCII 示意）：**
```
布里渊区 Δ(k) 的正负号分布（B₂g）：

       k_y
        ↑
  (-) (+) │ (-) (+)
    ──────┼──────→ k_x
  (+) (-)  │ (+) (-)

节线在 k_x = 0 和 k_y = 0 轴上！

对比 d_x²-y²（B₁g）节线在对角线 k_y = ±k_x：
  B₁g 节线：对角线方向 → 节点在 (±π/2, ±π/2)
  B₂g 节线：轴方向   → 节点在 (π/2, 0)、(0, π/2) 等费米面截止处
```

### 1.3 节线位置与节点

**节线方程**（Δ(k) = 0 的轨迹）：

$$\sin k_x = 0 \text{ 或 } \sin k_y = 0$$
$$\Rightarrow k_x = n\pi \text{ 或 } k_y = n\pi \quad (n \in \mathbb{Z})$$

ASCII: 节线：k_x = 0, ±π 或 k_y = 0, ±π（沿 BZ 边界的轴线）

**与 d_x²-y² 节点的关键区别：**
```
d_x²-y² 节点（B₁g）：费米面与对角线 k_y = ±k_x 的交点 → (±π/2, ±π/2)
d_xy 节点（B₂g）：  费米面与轴线 k_x = 0, k_y = 0 的交点 → 位于 BZ 轴上

在铜氧化物的近似正方费米面上：
  B₁g 节点：(π/2, π/2) 区域（BZ 对角线，"反节点"远离）
  B₂g 节点：(π, 0) 和 (0, π) 附近的轴线上（即铜氧化物的"反节点"区域！）

→ 铜氧化物费米面的反节点区域正好是 d_xy 的节点位置
→ 而铜氧化物 ARPES 在反节点区测到最大能隙 → 铜氧化物不能是 d_xy
```

**[定理 — 对称性强制节点]** 与 B₁g 相同，B₂g 配对的节点也是对称性强制的，在晶格 D₄h 对称性保持的条件下不可移动。

### 1.4 为什么 d_xy 在铜氧化物中不优先

**[观察 — 强支持]** 三重原因使铜氧化物选 B₁g 而非 B₂g：

1. **Cu 轨道特征**：Cu 导电轨道是 d_x²-y²（B₁g），d_xy 轨道（B₂g）在 CuO₂ 面上能量更高（能带不穿过费米面），天然不产生 B₂g 配对
2. **费米面嵌套**：铜氧化物的 Q=(π,π) AFM 涨落在 B₁g 通道吸引，在 B₂g 通道排斥或中性，d_xy 不优先
3. **ARPES 反节点观测**：铜氧化物在 (π,0) 方向有最大能隙，这正是 d_xy 的节点位置 → 直接排除 d_xy 为铜氧化物主态

**[猜想]** 在铁基超导中，多带结构（d_xz/d_yz/d_xy/d_z²/d_x²-y² 五个轨道）可能在特定掺杂下提供 d_xy（B₂g）配对的嵌套条件，但 s± 通常更优先（更好的带间嵌套）。

## 3. 数学框架

### 3.1 d_xy 能隙函数的等价形式

**[定理 — 代数等价]**

$$\Delta(\mathbf{k}) = \Delta_0 \sin k_x \sin k_y$$

等价表达式：

**(1) 积化和差展开：**
```
Δ(k) = Δ₀ sin k_x sin k_y
     = (Δ₀/2)[cos(k_x - k_y) - cos(k_x + k_y)]

在近圆形费米面（k → k_F）：
  Δ(k_F, φ) = Δ₀ sin(2φ)    ← 极坐标形式（φ = arctan(k_y/k_x)）
```

**(2) 极坐标形式（近圆形费米面）：**

$$\Delta(k_F, \phi) \approx \Delta_0 \sin(2\phi)$$

LaTeX: $\Delta(k_F, \phi) = \Delta_0 \sin(2\phi)$

ASCII: Δ(k_F, φ) = Δ₀ sin(2φ)

```
与 d_x²-y² 对比：
  d_x²-y²（B₁g）：Δ = Δ₀ cos(2φ)  → 节点在 φ = π/4, 3π/4
  d_xy（B₂g）：   Δ = Δ₀ sin(2φ)  → 节点在 φ = 0, π/2（即轴方向）

相差正好 π/4（45°）的相位偏移。
```

**(3) 连续极限：**
```
Δ(k) ≈ Δ₀ k_x k_y / k_F²      ← k 小时的展开
      = Δ₀ k² sin φ cos φ / k_F²
      = (Δ₀/2) k² sin(2φ) / k_F²
```

**(4) 与 d_x²-y² 的 45° 旋转关系（圆形费米面中的等价性）：**
```
若作坐标旋转 45°：k_x → (k_x + k_y)/√2，k_y → (-k_x + k_y)/√2
则：cos k_x - cos k_y  →  sin k_x sin k_y（近似）
即 B₁g 和 B₂g 在圆形费米面极限下互相转换，简并！
在正方晶格中，晶格破坏旋转对称 → 打破简并 → B₁g ≠ B₂g
```

### 3.2 BCS 自洽方程的 B₂g 投影

**[定理]**

B₂g 通道的配对势：

$$V(\mathbf{k}, \mathbf{k}') = V_{xy} \cdot \sin k_x \sin k_y \cdot \sin k'_x \sin k'_y$$

ASCII:
```
V(k,k') = V_xy × sin k_x sin k_y × sin k'_x sin k'_y

代入 BCS gap equation：
Δ(k) = Δ₀ sin k_x sin k_y  （若 V_xy < 0，B₂g 通道吸引）
→ 自洽解的形状因子 = sin k_x sin k_y，确认 B₂g 形式
```

**[猜想]** 在铁基超导中，特定掺杂下某些铁 d 轨道（如 d_xy 轨道本身）的带间散射矢量 Q 可能在 B₂g 通道给出 V_xy < 0，从而稳定 d_xy 配对。但这依赖于细节的轨道参数，不同计算组给出不同结论 [来源待验证]。

### 3.3 节点处的低能行为

**[定理 — 类比 C3，节点位置不同]**

d_xy 也有线性节点（非全能隙），节点处形成各向异性 Dirac 锥：

$$H_{\rm eff} = v_F \delta k_F \sigma_z + v_\Delta \delta k_\Delta \sigma_x$$

低能态密度（线性 DOS）：

$$N(\omega) \propto \frac{|\omega|}{v_F v_\Delta}$$

ASCII: N(ω) ∝ |ω|（线性，与 d_x²-y² 相同形式，但节点位置不同）

**重要区别：**
```
d_x²-y²（B₁g）：节点在 (±π/2, ±π/2)，费米速度方向不同
d_xy（B₂g）：   节点在轴方向上（与费米面的交点）
→ v_F 和 v_Δ 的值不同，热导率各向异性方向不同
→ 热流方向选择性实验可以区分
```

**穿透深度和比热（低温行为，[定理]）：**
```
d_xy 也有节点 → 低温行为同样是：
  δλ(T) ∝ T（穿透深度线性，干净极限）
  C_s ∝ T²（比热，线性 DOS 积分）
  1/T₁ ∝ T³（NMR，Hebel-Slichter 峰消失）

⚠️ 注意：上述低温行为与 d_x²-y² 定性相同！
仅靠穿透深度或比热无法区分 B₁g 和 B₂g。
```

## 4. d_xy 的实验识别方法

⚠️ **核心困难：d_xy 和 d_x²-y² 的低温热力学行为（穿透深度、比热、NMR）定性相同，因为两者都有线性节点。必须使用位置敏感或相位敏感方法才能区分。**

### 4.1 穿透深度 λ(T)：有节点但无法区分 B₁g/B₂g

**[定理]** 两种 d-wave 都给出线性穿透深度：

```
d_xy：   δλ(T) ∝ T（节点线性 DOS，与 d_x²-y² 相同）
d_x²-y²：δλ(T) ∝ T（相同）

→ 穿透深度证明有节点，但不能区分 B₁g 和 B₂g！
```

**[观察]** 实验上，穿透深度只能排除全能隙 s-wave，不能区分两种 d-wave。需要方向分辨或相位实验。

### 4.2 ARPES：节点位置区分（理论上最直接）

**[猜想 — 原则上可行，实践上困难]**

```
d_x²-y²（B₁g）节点：(±π/2, ±π/2) 方向（BZ 对角线）
d_xy（B₂g）节点：   (π/2, 0) 或 (0, π/2) 附近（BZ 轴方向）

ARPES 原则上可以区分：
  若 ARPES 在 (π/2, π/2) 测到 Δ = 0（节点）→ d_x²-y²
  若 ARPES 在 (π, 0) 附近测到 Δ = 0（节点）→ d_xy
```

**实际困难：**
- 铜氧化物中 ARPES 在 (π,0) 方向明确测到**最大能隙**（不是节点）→ 铜氧化物直接排除 d_xy
- 铁基 ARPES 分辨率和费米面复杂性使节点位置判断困难
- 无材料有 ARPES 确认的 d_xy 节点（在 BZ 轴方向）

### 4.3 热导率各向异性

**[猜想 — 原则上有区分能力]**

```
d_x²-y²：节点在对角线 → 热流沿对角线方向最强
d_xy：   节点在轴方向 → 热流沿轴方向最强

旋转磁场下的热导率实验（Matsuda 组方法）原则上可以区分：
  C₄ 对称性下的热导率方向依赖在 B₁g 和 B₂g 中方向相差 45°
```

**现实：** 没有体系被用此方法确认为 d_xy [来源待验证]。

### 4.4 Josephson 相位敏感实验（最判决性）

**[定理 — 原则上判决性]**

类比 Tsuei-Kirtley 三角结实验（C3 Section 2.4），可设计判别 B₁g vs B₂g 的实验：

```
Tsuei-Kirtley 原始实验区分 B₁g 和 A₁g（d-wave vs s-wave）。
若要区分 B₁g 和 B₂g（d_x²-y² vs d_xy）：
  → 需要与晶轴旋转 45° 设计约瑟夫森结
  → B₁g 和 B₂g 在这个角度下给出不同相位行为

原则上可行，但实际要求：
  (1) 需要 d_xy 候选材料的高质量单晶
  (2) 需要精确 45° 旋转界面制备
  (3) 目前没有候选材料足够可靠
```

**[观察]** 由于缺乏实验候选材料，这个实验从未被执行。

### 4.5 QPI（准粒子干涉）

**[猜想]**

```
QPI 散射强度正比于 |Δ(k)|² × |Δ(k+q)|²（相干因子）
  d_x²-y²：节点在对角线 → 强散射 q 连接 (π/2,π/2) 到 (π/2,π/2)
  d_xy：   节点在轴线 → 强散射 q 连接 (π,0) 附近的节点

QPI 格局方向性不同，STM 原则上可以区分。
但没有材料做过 d_xy 的 QPI 实验确认。
```

## 5. 关键定量结果与典型材料

⚠️ **整节以理论计算为主，实验数值极少。所有无 DOI 标注的数值须视为 [来源待验证]。**

### 5.1 铁基超导中的 d_xy 竞争态

**背景：** 铁基超导（LaFeAsO、BaFe₂As₂ 系列）有五个铁 d 轨道（d_xz、d_yz、d_xy、d_z²、d_x²-y²），多带费米面导致配对通道竞争。

**Kuroki 等 2008 RPA 计算（LaFeAsO）[猜想，理论]：**
- s± 通道（A₁g，带间 Q=(π,0)/(0,π) 嵌套）为主导配对通道
- d_xy（B₂g）通道存在但强度低于 s±
- 具体领先磁化率 χ(q) 的比例 [来源待验证，参考 PRL 101, 087004 (2008)]

**Graser 等 2009 五轨道 RPA（NJP 11, 025016）[猜想，理论]：**
- 最优掺杂（x=0.1 附近）：s± 主导，d_xy 次要
- 过掺杂区（空穴口袋消失）：d_xy 竞争相对增强（s± 弱化因嵌套变差）
- [定量值来源待验证，DOI: 10.1088/1367-2630/11/2/025016]

**Fernandes 等 2010 铁基对称性分析 [猜想，理论]：**
- 在特定嵌套矢量 Q 满足 d_xy 形状因子的条件下，d_xy 可以超过 s± 成为主态
- 强烈依赖于轨道权重和掺杂，不是普遍结论
- [来源待验证，参考 PRB 82, 014521 (2010)]

**⚠️ 关键反谄媚点：** 没有任何铁基超导体被实验确认为 d_xy 主态。现有实验证据（中子散射、穿透深度、ARPES、NMR 的整合）对主要铁基超导体（BaFe₂As₂、FeSe 等）最一致的描述仍是 s± 或其变体。

### 5.2 重费米子系统

**CeCu₂Si₂ [猜想，历史理论]：**
- 早期（1980-1990 年代）理论：部分计算预言 d_xy 或混合 d-wave 配对
- 现代实验（穿透深度、比热、NMR）：倾向节点 d-wave，但具体 B₁g vs B₂g 争议未完全解决
- [来源待验证，参考 Miyake-Schmitt-Rink-Varma 1986, PRB 34, 6554]

**CeIn₃、CePd₂Si₂（压力下 SC）[猜想]：**
- 部分理论预言 d_xy，尤其在三角或正交晶格的重费米子体系
- 实验区分 d_x²-y² vs d_xy 在重费米子体系极困难（需要单晶相位实验，而重费米子单晶极小）
- [来源待验证]

### 5.3 有机超导体 κ-(BEDT-TTF)₂X

**[猜想 — 近简并，不确定]：**
- κ-BEDT 系列有三角晶格，晶格对称性低于 D₄h
- 部分计算（FLEX、RPA）显示三角晶格上 d_x²-y² 和 d_xy 能量近简并
- 实验倾向 d_x²-y²（类比正方铜氧化物），但三角晶格的对称性分析更复杂
- [来源待验证，参考 Schmalian 1998, PRL 81, 4232; Kino-Fukuyama 1996]

**⚠️ 有机 SC 的对称性分析更复杂：** κ-BEDT 晶格本身偏离正方（单斜或正交），D₄h 的 B₁g/B₂g 分类不完全适用，需要实际晶体场群论分析。

## 6. d_xy vs d_x²-y² 核心对比

| 特征 | d_x²-y²（C3，B₁g）| d_xy（C4，B₂g）|
|------|-----------------|----------------|
| **能隙函数（晶格）** | cos k_x - cos k_y | sin k_x sin k_y |
| **极坐标** | Δ₀ cos(2φ) | Δ₀ sin(2φ) |
| **节线方向** | k_y = ±k_x（对角线）| k_x = 0, k_y = 0（轴线）|
| **节点位置** | (±π/2, ±π/2) 附近 | BZ 轴上的费米面交点 |
| **对称表示** | B₁g（D₄h）| B₂g（D₄h）|
| **旋转关系** | 基准 | 45° 旋转 ≈ d_x²-y²（圆形 FS）|
| **穿透深度低温** | δλ ∝ T（线性）| δλ ∝ T（线性，**相同！**）|
| **比热低温** | C_s ∝ T²（相同）| C_s ∝ T²（**相同！**）|
| **热导率方向** | 极大沿对角线 | 极大沿轴方向（**不同**）|
| **ARPES 节点位置** | (π/2, π/2) 对角线 | (π/2, 0) 轴方向（**不同**）|
| **相位实验（Josephson）** | B₁g → π 结（旋转 90°）| B₂g → 不同相位行为（旋转 45° 判别）|
| **实验确认材料** | YBCO/BSCCO/LSCO（确认）| **无（0 个材料确认）**|
| **铜氧化物** | ✅ 主态（判决性实验确认）| ❌ 非主态（ARPES 直接排除）|
| **铁基** | 竞争候选（理论）| 竞争候选（理论，更弱）|
| **重费米子** | 多数更倾向 B₁g | 部分理论候选，实验未区分 |
| **有机 SC** | 多数计算倾向 B₁g | 少数计算近简并，不确定 |

## 7. 关键反谄媚发现

### 7.1 核心结论：d_xy 是理论构造，无实验确认材料

**[观察 — 诚实评估]**

截至 2026 年，没有任何超导材料被判决性实验确认为 d_xy（B₂g）主态。
- 所有 d_xy 证据来自 RPA/FLEX 理论计算
- 实验候选（铁基过掺杂端、部分重费米子）均无判决性相位实验
- 与 d_x²-y² 的区分在所有候选体系中均未完成

这与 d_x²-y² 的情况形成鲜明对比：d_x²-y² 在 1994 年即有 Tsuei-Kirtley 三角结实验的判决性确认，而 d_xy 至今连候选材料都不确定。

### 7.2 与 d_x²-y² 的区分极度困难

**[观察]**

- 穿透深度、比热、NMR 三项经典判据对 d_xy 和 d_x²-y² 给出定性相同结果（都是有节点），无法区分
- ARPES 原则上可以区分节点位置，但在铁基等多轨道体系中 k 空间分辨困难
- Josephson 相位实验原则上是判决性的，但需要 d_xy 候选材料的单晶，目前无合适材料

### 7.3 铁基中 d_xy 的地位：理论敏感，实验沉默

**[观察]**

铁基 d_xy 的理论预言强烈依赖于：
- 具体的紧束缚模型参数（铁轨道 hopping 矩阵元）
- 掺杂浓度（影响费米面拓扑）
- 是否包含轨道序、向列序等次级效应

不同理论组使用不同参数可以得到 d_xy 主导或次要的不同结论 [来源待验证]，说明这是参数敏感的理论结果而非普遍预言。

### 7.4 圆形费米面中近简并

**[定理 — 在自由电子极限]**

当费米面接近圆形（k_F 各向同性）时，B₁g 和 B₂g 通道的配对势几乎相等：

```
V_{B₁g} = <(cos k_x - cos k_y)(cos k'_x - cos k'_y)> V
V_{B₂g} = <sin k_x sin k_y sin k'_x sin k'_y> V

对于圆形费米面，这两个矩阵元相等（由旋转对称性保证）
→ d_x²-y² 和 d_xy 简并！

打破简并需要：
  (1) 正方晶格效应（费米面方形化）
  (2) 轨道特异性（哪个轨道主导费米面）
  (3) 自旋涨落的方向选择性（Q=(π,π) vs Q=(π/2,π/2)）
```

**[猜想]** 在费米面接近圆形的体系中（某些有机 SC、重费米子），d_xy 和 d_x²-y² 能量差可能很小，实验区分更难。

## 8. 与 Paper E 的关联

**Paper E（轨道-能隙对应原则）的框架：** 配对能隙的对称性 = 配对轨道的对称性。

**d_xy 在 Paper E 框架中的角色（[猜想]）：**

```
Cu d_x²-y² 轨道（B₁g）→ 对应 d_x²-y² 能隙（B₁g）  ← C3 的完美案例
Cu d_xy 轨道（B₂g）     → 原则上对应 d_xy 能隙（B₂g）← C4 的逻辑

但：在 CuO₂ 平面，Cu 的 d_xy 轨道能量比 d_x²-y² 高约 1-2 eV（晶体场劈裂），
不穿过费米面 → d_xy 轨道在铜氧化物中对配对贡献极小
→ Paper E 的轨道选择原则自动给出 d_x²-y²（B₁g）而非 d_xy（B₂g）

d_xy 是 Paper E 框架的"备选通道"：
  若导电轨道主要是 d_xy（如某些铁基中的 d_xy 轨道带），
  Paper E 预言对应 B₂g 能隙 → 提供一个可验证的预言框架
```

**[观察]** 铁基超导体中 Fe d_xy 轨道确实参与费米面，Paper E 框架在这里的预言值得关注，但需要结合多轨道混合效应，不是简单的单轨道对应关系。

## 9. 未解问题 / 开放前沿

| 问题 | 当前状态 | 挑战 |
|------|---------|------|
| **铁基过掺杂端是否有 d_xy 相？** | 理论预言，实验空白 | 过掺杂端单晶制备困难；穿透深度/热导率各向异性测量需要高精度旋转场实验 |
| **有机 SC κ-BEDT 的真实配对对称性** | 多数证据指向 d_x²-y²，但三角晶格使 B₁g/B₂g 分类不清晰 | 晶体对称性低，群论分类需要重新分析；单晶质量限制 |
| **d_xy + d_x²-y² 混合态（d+id）** | 理论预言为手性 SC（C7 关联），破时间反演 | 手性 SC 的实验签名（Kerr 效应、自发霍尔电导）在候选体系中争议极大 |
| **重费米子 B₁g vs B₂g 判别** | 理论未统一，实验未判决 | 重费米子单晶微小，Josephson 相位实验困难；低 Tc（<2K）的相位噪声大 |
| **Paper E 轨道对应在铁基的适用性** | 尚未系统验证 | 五轨道混合下轨道-能隙对应的量化需要新的形式框架 |
