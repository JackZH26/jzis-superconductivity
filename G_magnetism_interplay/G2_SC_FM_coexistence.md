# G2. 超导与铁磁共存（SC + FM Coexistence）

**MECE 编号：** G2
**关联 MECE：**
- B4（FM 涨落驱动 triplet SC — FM 涨落是 G2 材料中 triplet 配对的理论核心；B4 侧重配对机制，G2 侧重序的本征共存与实验体系）
- C5（p-wave / triplet 配对 — triplet（奇宇称）配对是 FM+SC 本征共存的必要条件；C5 侧重配对对称性，G2 侧重为什么 FM 迫使 triplet）
- D8（重费米子 — URhGe/UCoGe/UGe₂ 均是铀基重费米子体系；D8 侧重 Kondo 物理，G2 侧重 FM+SC 的共存条件）

## 1. 物理定义与核心思想

### 1.1 铁磁为什么摧毁 singlet SC（[定理]）

```
=== FM 破坏 singlet SC 的两条物理机制 [定理] ===

─────────────────────────────────────────────────────
机制 1：轨道（Meissner）效应
─────────────────────────────────────────────────────
铁磁内部磁感应强度 B_int（由自发磁化 M 产生）：
  → B_int = μ₀(H + M) ≈ μ₀ M（退磁场近似下）
  → 对于强铁磁（如 Fe），M ~ 10⁶ A/m，B_int ~ 1-2 T
  → B_int 超过 Hc1（第一类 SC 或第 II 类 SC 的下临界场）
  → SC 无法完全排开内磁场 → Meissner 效应失败
  → 物理图像：Cooper 对的 Lorentz 力折断了对
  ← 但：对于弱铁磁（如 URhGe，M ~ 0.42 μ_B/U），
    B_int 极小 → 轨道效应不是主要破坏者

─────────────────────────────────────────────────────
机制 2：Pauli（Zeeman）拆配效应 ← 主要凶手！
─────────────────────────────────────────────────────
铁磁自发磁化 → 内部等效交换场（exchange field）B_ex：
  → singlet Cooper 对（↑↓）的能量被 Zeeman 劈裂：
    ΔE = g μ_B B_ex
  → 当 ΔE ~ Δ（SC 能隙）时，singlet 配对被拆散

Chandrasekhar-Clogston 极限（Pauli 极限）[定理, E5 关联]：
  → H_P = Δ / (g μ_B √2) = 1.84 Tc（对 BCS 超导）
  → 单位：T，Tc 单位 K（当 g=2 时）
  → 即：H_P(T) ≈ 1.84 × Tc(K)

对铁磁内部交换场的估计：
  → 铀基弱铁磁（如 URhGe）：J_ex ~ 数百 meV ~ 等效 B_ex ~ 10³ T 量级
  → 这比任何 Chandrasekhar-Clogston 极限都大得多！
  → 结论：铁磁内部交换场 >> H_P → singlet SC 完全被压制 [定理]

─────────────────────────────────────────────────────
物理层级总结：
─────────────────────────────────────────────────────
  [定理] FM + singlet SC 在同一电子系统中本征共存是不可能的。
          → 这是 Pauli 拆配的必然结果。
          → singlet 配对（S=0，↑↓ 态）在自发交换场中能量代价太大。

  [猜想] 但 FM + triplet SC 可能本征共存：
          → triplet Cooper 对（S=1，↑↑ 或 ↓↓ 态）不受 Zeeman 磁场拆散
          → FM 背景中 ↑↑ 或 ↓↓ 配对天然受益于交换极化 [B4 关联]
          → URhGe 等铀基体系提供了强有力的[观察]支持，但配对对称性仍是[猜想]
```

### 1.2 Triplet SC 为何能与 FM 共存（[猜想，物理论证强]）

```
=== Triplet Cooper 对与 FM 的相容性 ===

Triplet 配对的自旋结构（[定理，量子力学]）：
  → S=1 的三个 m_s 量子态：
    |↑↑⟩（m_s = +1）
    (|↑↓⟩ + |↓↑⟩)/√2（m_s = 0）
    |↓↓⟩（m_s = -1）
  → d-vector（C5 详述）：描述 triplet 配对的矢量序参量
    Δ_αβ = (d·σ iσ_y)_αβ
    d ∥ z 轴：等权 ↑↑ 和 ↓↓ 配对（"等自旋配对"，Equal Spin Pairing）

与 FM 交换场的关系 [猜想，但物理论证强]：
  → FM 磁化方向 M ∥ ẑ（选为 z 轴）
  → Zeeman 能量：H_Z = -g μ_B B_ex ∑ σ_z
    → 对 ↑↑ 对（m_s=+1）：Zeeman 能量 = -2g μ_B B_ex（降低能量！）
    → 对 ↓↓ 对（m_s=-1）：Zeeman 能量 = +2g μ_B B_ex（升高能量）
    → 对 singlet 对（↑↓，m_s=0）：Zeeman 能量 = 0，但被 Pauli 拆配破坏

  → 结果：
    - d ⊥ M（d-vector ⊥ FM 磁化）：↑↑ 和 ↓↓ 态简并被打破，但非致命
    - d ∥ M（d-vector ∥ FM 磁化）：等自旋配对（↑↑ 或 ↓↓ 之一），最稳定
    
  ← FM 背景 → 选择 d ∥ M 的 triplet 配对 → 最小化能量代价
  ← FM 涨落（B4）→ 提供 triplet 配对的媒介玻色子（横向自旋涨落）

两种本征共存机制（[物理分类]）：
  A. 本征共存（Intrinsic Coexistence）：
     → 同一电子系统，同一晶胞内既有 FM 序参量（M≠0）又有 SC 序参量（Δ≠0）
     → 代表体系：URhGe、UCoGe、UGe₂（高压）
     → [观察]：实验确认两个相变（Tc 和 T_Curie），但微观共存仍需精确判定

  B. 近邻共存（Proximity Coexistence，S/F/S 异质结）：
     → FM 层夹在 SC 层之间的人工结构
     → 近邻效应：Cooper 对泄漏进 FM 层 → odd-frequency triplet 诱导
     → 不是本征共存！G2 专注 A 类，B 类在 D11 中讨论
```

## 2. 历史关键节点

```
=== SC + FM 研究的历史发展 ===

[~1960s-1970s] 理论预言：FM 与 singlet SC 不相容
  → Matthias 等早期讨论：铁磁金属不可能是 BCS 超导体 [来源待验证，历史综述引用]
  → 铁磁交换场 >> Chandrasekhar-Clogston 极限 → 理论上 singlet SC 被完全压制
  → 当时共识：FM 和 SC 是完全对立的，无共存可能

[1980] Fischer-Maple 综述（早期铁磁超导）[来源待验证，综述文献]：
  → Er₁₋ₓHoₓRh₄B₄ 系列：SC 和铁磁的竞争与共存
  → 关键发现：在某些 RE（稀土）化合物中，磁序和 SC 可以共存
  → 重要区别：这些材料中的磁序是 RE 的 4f 局域磁矩（并非传导电子铁磁！）
  → 4f 局域磁矩与传导电子耦合弱 → 不破坏 singlet SC（机制与 G2 完全不同）
  ⚠️ 这些早期"铁磁超导体"实际上是稀土局域磁矩共存，不是传导电子铁磁+SC！

[2000] Saxena et al.：UGe₂ 在压力下 FM+SC 共存（划时代！）
  → Nature 406, 587 (2000) [来源待验证，DOI 需核实]
  → UGe₂ 常压：强铁磁（T_Curie = 52K）[来源待验证]，无 SC
  → 压力 > 10 kbar：FM 减弱 + SC 突然出现（Tc ~ 0.7-1K）[来源待验证]
  → SC 只在铁磁相内出现（非铁磁相无 SC！）→ 强烈暗示 FM 与 SC 有内在联系
  → 意义：第一个铀基铁磁超导体，彻底改变"FM 和 SC 完全对立"的认知
  ⚠️ [来源待验证]：以上数值引自综述和二手资料，需查原始 Nature 论文核实

[2001] Huxley et al.：URhGe 本征 FM+SC 共存（常压！）
  → 被引频繁，来源：Nature 413, 613 (2001) [来源待验证，DOI 需核实]
  → URhGe 常压：T_Curie = 9.5K，Tc = 0.25K [来源待验证]
  → 关键：不需要压力，FM 和 SC 在常压下共存！
  → 比 UGe₂ 更干净的体系（常压共存，实验更容易）
  → Tc 极低（0.25K），需要稀释制冷机（dilution refrigerator）

[2005] Lévy et al.：URhGe 高场再入超导（reentrant SC）（震撼！）
  → Science 309, 1343 (2005) [来源待验证，DOI 需核实]
  → 在 H ∥ b 轴（硬轴）~ 12T 处，出现第二个 SC 区域！
  → 常压下 Tc = 0.25K，但 H ~ 12T 处 SC 再次出现（Tc' ~ 0.4K，更高！）
  → 这是在高磁场下 SC 被增强的极端反常现象（正常情况下磁场压制 SC）
  → 机制：12T 处 FM 矩发生重取向（c 轴 → b 轴），横向 FM 涨落极大增强
  ⚠️ [来源待验证]：Tc' 数值 ~0.4K 取自综述文献，需查原始 Science 论文核实

[2007] Hattori et al. / Huy et al.：UCoGe FM+SC 共存
  → UCoGe 发现：Huy et al. PRL 99, 067006 (2007) [来源待验证，DOI 需核实]
  → T_Curie = 2.5K，Tc = 0.6K（Tc 比 URhGe 更高）[来源待验证]
  → 弱铁磁（M ≈ 0.03 μ_B/U）[来源待验证]，比 URhGe（0.42 μ_B/U）弱得多
  → NMR：Aoki et al. 等对 UCoGe 的 Knight shift 测量 → 暗示 triplet 配对 [来源待验证]

[2011+] Aoki et al. 系列综述：铀基 FM 超导的系统研究
  → J. Phys. Soc. Japan 系列综述 [来源待验证]
  → UTe₂ 前驱：探索 FM 涨落但无长程 FM 序的超导体（UTe₂ 在 G4 中详述）
  → 形成"铀基 FM 超导家族"：UGe₂ / URhGe / UCoGe 三巨头

[近期] UTe₂（2019+）的崛起
  → Tc ~ 1.65K，无长程 FM 序，但有强 FM 涨落 [来源待验证]
  → 被认为是 FM 涨落驱动 triplet SC 的"最纯净"案例
  → 严格说 UTe₂ 不属于 G2（无 FM 序），在 B4 + D8 中详述
  → 但 UTe₂ 的研究极大推进了对 G2 材料的理解
```

## 3. 主要 FM+SC 本征共存材料（铀基体系）

### 3.1 URhGe（最重要的案例，最干净的本征共存）

```
=== URhGe：常压 FM+SC 共存体系 ===

基本参数（[来源待验证，综述值，需查 Huxley 2001 原始论文]）：
  晶体结构：斜方（orthorhombic），空间群 Pnma
  T_Curie = 9.5K（FM 铁磁有序温度）
  Tc = 0.25K（SC 超导转变温度）
  FM 磁矩方向：沿 c 轴（Ising 型铁磁，强各向异性）
  FM 磁矩大小：M = 0.42 μ_B/U [来源待验证]（弱铁磁！）

  ← 关键比例：T_Curie/Tc = 9.5/0.25 = 38（FM 比 SC 稳健 38 倍）
  ← 配对类型（[猜想]）：强烈暗示 triplet p-wave，但未直接确认

为什么 FM 弱很重要：
  → M = 0.42 μ_B/U 对应内部 B_int 极小（纳米到皮科 T 量级）
  → 等效交换场 J_ex 虽大，但 FM 矩极弱 → 轨道效应可忽略
  → Pauli 拆配依然很大（J_ex >> H_P），因此配对必须是 triplet！

─────────────────────────────────────────────────────
高场再入超导（Reentrant SC，最独特现象！）
─────────────────────────────────────────────────────
[观察，Lévy et al. 2005，来源待验证]：

外加磁场方向：B ∥ b 轴（晶体硬轴，垂直于 FM 磁化方向 c 轴）

H-T 相图（示意，ASCII）：

  温度                         再入 SC
  (K)                          区域！
        ↑
  0.4  ─╮                     ╭─╮
        │ 低场 SC              │ │ 高场 SC（再入）
  0.25 ─╰──╮                  │ │
             │                 │ │
  0.1        │                 │ │
              ╰────────────────╯ ╰─────→
              0      9T    12T 14T      H(T)
                           ↑
                        发生 FM 矩
                        重取向的磁场

关键现象（[观察，来源待验证]）：
  → H = 0：SC（Tc = 0.25K）和 FM（T_Curie = 9.5K）本征共存
  → H ~ 2T（∥ b 轴）：低场 SC 被磁场压制（Tc → 0）
  → H ~ 9-12T（∥ b 轴）：出现新的 SC 区域（Tc' ~ 0.4K，比零场更高！）
  → H > 14T：高场 SC 消失

物理解释（[猜想，FM 涨落机制，B4 关联]）：
  → 在 H ~ 12T 时，外加磁场使 FM 磁化方向从 c 轴（易轴）翻向 b 轴（硬轴）
  → 这个磁矩重取向（moment reorientation）发生在 H_R ~ 12T 处
  → 在 H_R 附近：FM 矩方向"摇摆"→ 横向（transverse）FM 涨落极大增强
  → 横向 FM 涨落 → 增强 triplet 配对（B4 机制）→ Tc 反常升高！
  ⚠️ 这是 B4 FM 涨落机制在 G2 中最直接的实验体现
  ⚠️ 但 FM 横向涨落是否是唯一机制？其他贡献（轨道、向列）未被排除 [猜想]
```

### 3.2 UCoGe（FM+SC 共存，Tc 较高）

```
=== UCoGe：弱铁磁超导体 ===

基本参数（[来源待验证，综述值，需查 Huy 2007 原始论文]）：
  晶体结构：斜方（orthorhombic），空间群 Pnma（与 URhGe 同构型）
  T_Curie = 2.5K（FM 有序温度）
  Tc = 0.6K（SC 转变温度）[来源待验证]
  弱铁磁：M ≈ 0.03 μ_B/U [来源待验证]（比 URhGe 的 0.42 μ_B/U 更弱！）

  ← 关键比例：T_Curie/Tc = 2.5/0.6 ≈ 4（FM 和 SC 尺度更接近，相互竞争更强）
  ← Tc 比 URhGe 更高，更容易在实验上研究

为什么 UCoGe 重要：
  → 极弱铁磁（M ≈ 0.03 μ_B/U）→ 轨道效应几乎可以忽略
  → 但 SC 依然存在 → 说明即使极弱的 FM，也有强到足以影响配对的机制
  → T_Curie = 2.5K 和 Tc = 0.6K 都在液氦温区，比 URhGe（Tc=0.25K）更容易研究

NMR/Knight shift 证据（[观察，来源待验证]）：
  → Ohta et al., PRL 2010（[来源待验证，DOI 需核实]）：
    ⁵⁹Co NMR 测量 SC 态的 Knight shift
  → 特定方向（c 轴？）的 Knight shift 在 Tc 以下不降低 → 暗示 triplet 配对
  → 但：Sr₂RuO₄ 教训（D8 教训）：Knight shift 不降也可以有其他解释（自旋-轨道耦合）
  ← [猜想]：UCoGe 的 triplet 标签基于 Knight shift，但要小心 Sr₂RuO₄ 的前车之鉴

磁场效应（[观察，来源待验证]）：
  → UCoGe 在 B ∥ c 轴时：Tc 随磁场线性增加（反常！），后再降低
  → 与 URhGe 的再入超导类比，但细节不同
  → 解释：c 轴磁场增强 FM 涨落，进而增强 triplet 配对
```

### 3.3 UGe₂（高压才 FM+SC 共存）

```
=== UGe₂：压力诱导 FM+SC 共存 ===

基本参数（[来源待验证，综述值，需查 Saxena 2000 原始论文]）：
  晶体结构：斜方，空间群 Cmmm
  常压 T_Curie = 52K（强铁磁！）[来源待验证]
  常压：无 SC

压力相图（示意）[来源待验证]：

  温度(K)
   ↑
  60  ─ FM1（T_Curie ~ 52K at P=0）
       │  \
  30  ─│   \
       │    ╲
  10  ─│     ╲  FM2
       │      ╲──────── SC+FM2 共存区（Tc~0.7-1K）
   1  ─│              ╲────╮
       │                   ╰── SC 消失
   0  ─└──────────────────────→ 压力（GPa）
         0       1       1.5

关键特征（[来源待验证，来自 Saxena 2000 和后续研究]）：
  → P > 10 kbar（~ 1.0 GPa）：FM（FM1）减弱 → FM2 相出现
  → P ~ 10-15 kbar：SC 在 FM2 相内出现！Tc ~ 0.7-1K [来源待验证]
  → SC 只在铁磁相（FM2）内存在 → SC 完全嵌套在 FM 相内（强力证据！）
  → P > 16 kbar（约）：FM 消失 → SC 也消失！
  ← 这一相关性（FM 消失 = SC 消失）是 FM 驱动 SC 的最强间接证据

两个铁磁相（FM1 和 FM2）的区别（[来源待验证，猜想]）：
  → FM1（低压）：强铁磁，磁矩大
  → FM2（高压）：弱铁磁，磁矩略小
  → FM1 → FM2 的转变是一阶相变（T < T_Curie 时的压力诱导）
  → SC 在 FM1-FM2 边界附近最强 → 暗示 FM1-FM2 相变的量子临界涨落驱动 SC？[猜想]

与 URhGe/UCoGe 的比较：
  → UGe₂：压力才共存，体系更复杂（两个 FM 相）
  → URhGe/UCoGe：常压共存，更"干净"
  → 共同点：SC 总是嵌套在 FM 相内，FM 减弱则 SC 也减弱
```

## 4. FM 涨落驱动 Triplet SC 的物理（[猜想，B4 详述]）

```
=== FM 涨落（q→0 软模）驱动 triplet 配对的理论图像 ===

FM 涨落的特征（[物理分类，B4 详述]）：
  → 铁磁涨落：自旋涨落波矢 q → 0（与 AFM 涨落 q → Q_AFM 相反！）
  → 近铁磁临界点（T → T_Curie 或 P → P_c）：涨落软化（Stoner 软模）
  → 两种分量：
    纵向涨落（δM ∥ M₀）：密度型，不改变自旋方向
    横向涨落（δM ⊥ M₀）：使 FM 矩偏离平衡 → 自旋翻转过程

─────────────────────────────────────────────────────
横向 FM 涨落 → Triplet 配对（[猜想，核心机制]）
─────────────────────────────────────────────────────

类比 AFM 涨落驱动 d-wave（G1, B3）：
  AFM q~Q_AFM 涨落 + Fermi 面嵌套 → d_x²-y² singlet 配对（铜氧化物）

FM 横向涨落驱动 triplet（[猜想]）：
  FM 横向涨落（q~0，但 ω≠0）→ 自旋翻转：↑ ↔ ↓
  → 有效相互作用（Stoner-Wohlfarth 类型）：
    V_triplet(q→0) ~ J_ex²/（ω - ω_SW(q)）（SW=spin wave）
    → 在 q→0 处为吸引（对 triplet 配对）
    → 对 singlet 配对（需要对抗交换场）仍为排斥

  → 选出的配对对称性 [猜想]：
    p-wave（l=1，奇宇称，与 triplet S=1 组合满足 Pauli 不相容原理）
    具体 d-vector：d ∥ FM 磁化方向（c 轴，URhGe 型）

─────────────────────────────────────────────────────
URhGe 再入超导的微观解释（[猜想，B4 关联]）
─────────────────────────────────────────────────────

外加场 H ∥ b 轴（硬轴）的效果：
  → H 增大 → FM 矩从 c 轴（易轴）向 b 轴（硬轴）倾斜
  → 在 H = H_R ~ 12T：FM 矩几乎全部转向 b 轴（磁矩重取向）
  → 在 H_R 附近：FM 矩"摇摆"不定 → 横向 FM 涨落发散（临界点！）

在磁矩重取向临界点附近：
  → 横向涨落强度 χ_T(q=0) ∝ 1/(H - H_R)（类 Curie-Weiss 发散）[猜想]
  → χ_T 极大 → triplet 配对强度极大 → Tc 反常升高（再入超导）
  → 这是 B4 FM 涨落机制的直接实验验证（如果机制正确的话）

⚠️ 反谄媚：
  → FM 横向涨落是目前再入 SC 最有力解释，但不是唯一解释 [猜想]
  → 可能的其他贡献：轨道效应（磁场重组 Fermi 面）、向列涨落（Ising FM）
  → 2026 年理论界对"横向涨落是唯一机制"无共识

─────────────────────────────────────────────────────
与 AFM 涨落机制的类比与对比
─────────────────────────────────────────────────────

  特征        AFM 涨落（G1 关联, B3）        FM 涨落（G2 核心, B4）
  ────────────────────────────────────────────────────────────
  涨落波矢    q ~ Q_AFM（有限 q）             q ~ 0（长波）
  配对对称性  d-wave singlet（[猜想]）         p-wave triplet（[猜想]）
  奈尔/居里   高（铜 300K，铁 140K）           低（URhGe 9.5K）
  温度
  驱动材料    铜氧化物/铁基/部分重费米子       URhGe/UCoGe/UGe₂（仅几种）
  Tc 量级     1-130K                          < 1K（极低！）
```

## 6. 近邻效应中的 SC+FM（S/F 异质结，≠本征共存）

```
=== 区分本征共存（G2 主题）与近邻效应（D11）===

─────────────────────────────────────────────────────
S/F 近邻效应的基本物理（[定理]）：
─────────────────────────────────────────────────────

  结构：SC | FM | SC（或 SC | FM 双层）

  标准 Andreev 近邻效应：Cooper 对泄漏进 FM 层
  → FM 中的 singlet Cooper 对（↑↓）：被交换场 J_ex 调制
    ψ(x) ∝ exp(-x/ξ_F) × cos(J_ex x/ħv_F)
    → 快速振荡 + 衰减（ξ_F 极短，~ nm 量级）
    → 产生 0-π Josephson 结（π 相位跳跃）[观察，已确认]

  奇频 triplet 近邻效应（odd-frequency triplet，[猜想，实验争议]）：
  → 在非共线磁化结构（如 SC/F1/F2 三层）：
    F1 和 F2 磁化方向不同 → 产生 S=1，m_s=0 的 triplet（↑↓+↓↑）
    但这是 m_s=0 triplet，不是长程渗透的 m_s=±1 triplet！
  → 理论预言：在特殊几何（磁化不共线）下，长程 m_s=±1 triplet 可以渗透进 FM 很深 [猜想]
  → 实验证据：部分实验报告了超导电流在 FM 层中异常长程传输 [来源待验证，Khaire 2010 等]
  ← [猜想层级]：奇频 triplet 长程渗透理论完善，但实验确认仍有争议

─────────────────────────────────────────────────────
为什么 S/F 近邻效应 ≠ G2 的本征共存：
─────────────────────────────────────────────────────

  近邻效应（S/F 异质结）：
    → SC 和 FM 是两个不同材料层（宏观分离）
    → SC 层提供 Cooper 对，FM 层提供磁性
    → "共存"只在界面 nm 尺度
    → 本质上是近邻诱导（induced），不是本征的（intrinsic）

  本征共存（G2，URhGe 等）：
    → 同一晶体，同一电子系统既形成 FM 序又形成 SC
    → f 电子（或 5d 电子）同时贡献 FM 矩和 Cooper 配对
    → 这才是真正的"量子材料中 FM 和 SC 共同生活"

  实验判据（如何区分）：
    → 本征共存：单晶中两个相变（T_Curie 和 Tc）+ 磁化率和电阻同步变化
    → 近邻效应：两个材料的比热分别显示各自相变（SC 在一层，FM 在另一层）
```

## 8. 关键反谄媚

```
⚠️ 反谄媚 G2.1：FM+SC 本征共存 ≠ S/F 近邻效应（最重要！）

  最常见的混淆：
    → 文献中"铁磁超导体"大多是 S/F 薄膜异质结或 π-Josephson 结
    → 这些不是同一电子系统的本征 FM+SC 共存！
    → 真正的本征 FM+SC 共存材料（2026年）只有：
      URhGe、UCoGe、UGe₂（高压下）
      也许还有其他稀少铀基化合物

  正确表述：
    → "S/F/S 结展示了π相位跳跃"是近邻效应的 [观察]
    → "URhGe 中 FM 和 SC 本征共存"是 [观察]，但机制是 [猜想]
    → 两者物理机制完全不同，不可混淆

⚠️ 反谄媚 G2.2：Triplet 配对是[猜想]，不是已确认事实（Sr₂RuO₄ 教训！）

  常见过激说法：
    → "URhGe/UCoGe 是 p-wave 超导体" → 过度自信！
    → Knight shift 不降 → 暗示 triplet，但不是证明！

  正确层级：
    → [观察]：URhGe/UCoGe 在低温有超导转变（Tc）+ 铁磁序（T_Curie）
    → [观察]：某些方向 Knight shift 在 Tc 以下变化不明显
    → [猜想]：配对对称性是 triplet（p-wave）
    → 需要：相位测量（SQUID 干涉、角分辨比热、中子散射测 Cooper 对自旋）才能确认

  Sr₂RuO₄ 前车之鉴（D8 教训，2019 Knight shift 反转）：
    → Sr₂RuO₄ 曾被认为是"确认"的 triplet 超导体（Knight shift 不降）
    → 2019-2020 新实验推翻：Knight shift 实际会降低（早期实验有误）
    → 教训：Knight shift 不降 → 不能直接等于 triplet 配对确认！
    → URhGe/UCoGe 的 triplet 证据比 Sr₂RuO₄ 弱很多，更应谨慎

⚠️ 反谄媚 G2.3：再入超导的 FM 横向涨落机制是[猜想]，不是[定理]

  URhGe 再入超导：
    → [观察]：H ~ 12T（∥ b 轴）时出现第二个 SC 区域 [来源待验证]
    → [观察]：H_R 处 FM 矩发生重取向（从 c 轴翻向 b 轴）[来源待验证]
    → [猜想]：横向 FM 涨落在 H_R 处增强 → 驱动再入 SC

  ⚠️ 但还有其他可能的机制：
    → 轨道效应：H_R 处 Fermi 面重组 → 改变 Fermi 面形状 → 影响 Tc
    → 轨道效应和涨落效应可能共同作用
    → 2026 年无定论

⚠️ 反谄媚 G2.4：铀基 FM 超导的 Tc 极低（< 1K）→ 无实用价值

  → URhGe：Tc = 0.25K（需要稀释制冷机，~ mK 级别的实验环境）
  → UCoGe：Tc = 0.6K（仍需稀释制冷机）
  → UGe₂（高压）：Tc ~ 0.7-1K，还需要高压设备
  → 这些材料完全没有技术应用价值（Tc 远低于 4K 液氦温度）
  → 研究价值：纯基础物理（triplet 配对机制、FM+SC 的量子共存）
  → 不要夸大其"室温超导潜力"——G2 材料与室温超导没有直接关联

⚠️ 反谄媚 G2.5：UTe₂ 不是 G2 材料（无 FM 序！）

  → UTe₂（Tc ~ 1.65K）：没有长程铁磁序（T_Curie 不存在）
  → 有强铁磁涨落（Stoner 增强），但无自发磁化
  → 严格说：UTe₂ 属于 B4（FM 涨落驱动 triplet SC）+ D8（重费米子）
  → 放在 G2 是错误分类！
  → 正确说法："UTe₂ 是 FM 涨落（B4）驱动 triplet SC 的最佳候选，但不是 FM+SC 本征共存（G2）"
```
