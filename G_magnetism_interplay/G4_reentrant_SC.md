# G4. 再入超导（Re-entrant Superconductivity）

**MECE 编号：** G4
**关联 MECE：**
- G2（SC+FM 共存 — URhGe 磁场再入的材料来源，FM 横向涨落机制，G4 的核心实验案例）
- G3（场诱导 SC — 磁场再入是场诱导 SC 的一个子类型，G4 侧重"再入"概念本身）
- G1（SC+AFM 共存 — CeRhIn₅/CeCu₂Si₂ 等压力/掺杂相图中两个 SC 穹的来源）
- E5（磁场效应 — 磁场再入的外参量；Pauli 极限/Hc₂ 的基础框架）
- D8（重费米子 — 再入 SC 的主要材料体系：铀基/铈基重费米子化合物）

**层级关系：** G4 是 G 部分中的概念整合章。再入超导（Re-entrant SC）不是一种材料，而是一种现象类型——超导在外参量扫描中先出现→消失→再次出现。G4 整合了三类驱动参量（温度/磁场/压力掺杂）下的再入现象，系统梳理判断标准、物理机制，并与 G2/G3 明确区分。G4 本身不深入材料细节（G2 已述 URhGe，G3 已述场诱导，G1 已述压力相图），而是提供"再入超导"这个概念的完整拓扑图景。

## 1. 物理定义与核心思想

### 1.1 再入超导的拓扑定义（[定理，描述性]）

```
=== 再入超导：参数空间中的非连通超导区域 ===

定义（[定理，描述性，适用于所有类型]）：

  在参数空间 Ω（其中某个外参量 λ = T, H, P, x 等）中，
  若超导相 SC 在 Ω 内是非连通的——即：

    SC 区域 A（在 λ₁ < λ < λ₂）
                   ↓ 被非超导区域隔开
    SC 区域 B（在 λ₃ < λ < λ₄，λ₃ > λ₂）

  则当 λ 从 A 扫到 non-SC 区域再到 B 时，就发生"再入超导"。

  注意："再入"描述的是扫描路径，不是材料的固有分类：
  → 同一材料在温度维度是再入，在磁场维度可能是普通超导
  → 驱动参量不同 → 再入类型不同 → 物理机制可能完全不同
```

### 1.2 再入的三大物理机制（分类）

```
=== 三类再入机制 ===

─────────────────────────────────────────────────────
类型 1：竞争序导致的相图间隙（最常见，"软性"再入）
─────────────────────────────────────────────────────
驱动：AFM/CDW/SDW 在某参量范围内打败 SC
逻辑：外参量（P/x）调控竞争序强度 → 超出竞争序区域 → SC 重现

ASCII 相图示意（压力为例）：
  Tc ↑
     │   SC1(低压)    AFM主导区    SC2(高压)
     │   ╭────╮                    ╭────╮
     │   │    │    （正常态）      │    │
     │   │ SC1│       or          │ SC2│
     │   │    │    （AFM相）      │    │
     └───╯    ╰──────────────────╯    ╰───→ P
         0   P1          P2          P3

典型体系：
→ CeRhIn₅：低压 AFM，高压 SC；AFM-QCP 附近 SC 最强 [来源待验证，G1 详述]
→ CeCu₂Si₂：SC1（~0.5GPa，AFM-QCP附近）和 SC2（~2.5GPa，价键涨落）[来源待验证]
→ 铁基 BaFe₂As₂ 系列掺杂相图：SC 穹被 SDW 区域截断（是否真"再入"有争议）[猜想]

─────────────────────────────────────────────────────
类型 2：磁场驱动涨落增强（再入 SC，G2/G3 已述）
─────────────────────────────────────────────────────
驱动：外磁场使 FM 矩翻转/重取向 → 横向涨落增强 → triplet 配对复活
逻辑：高场 SC（再入 SC）具有更高 Tc' 比零场 SC 的 Tc（反常！）

典型体系：
→ URhGe（B ∥ b 轴，H_R ~ 12T）：最清晰磁场再入案例 [来源待验证，G2 G3 详述]
→ UCoGe（B ∥ ab 方向）：类似效应，但更弱 [来源待验证，G2 详述]

─────────────────────────────────────────────────────
类型 3：温度再入（极罕见，真正定义上的温度再入）
─────────────────────────────────────────────────────
驱动：磁有序与 SC 的竞争随温度的非单调演化
逻辑：升温 → 磁有序强度先增后减 → SC 在磁有序减弱的高温区"复活"
实验现状：几乎无确认案例（详见第 4 节）[争议]
```

## 2. 历史关键节点

```
=== 再入超导研究的历史发展 ===

[1972] Maple et al.（温度再入理论预言）[来源待验证]
  → 理论分析：在局域磁矩和 SC 共存体系中，温度再入超导原则上可能
  → 条件：SC 配对强度和磁序的温度依赖需精确匹配
  → 意义：首次给出温度再入 SC 的理论框架，但实验从未清晰验证
  → [Maple et al. 1972，来源待验证，原始文献未核实]

[1976-1985] 磁性超导体时代：ErRh₄B₄ 和 HoMo₆S₈
  → ErRh₄B₄（Tc≈8.6K，TM≈0.8K）：低温出现铁磁序，SC 消失 [来源待验证]
  → HoMo₆S₈（Tc≈1.8K）：低温磁序破坏 SC [来源待验证]
  → 这些被误称为"温度再入"的现象实质：
    FM 在低温取代 SC（不同序参量竞争），SC 消失后不再复活
    → 不是真正的温度再入！（详见第 4 节反谄媚）
  → [Fischer 1990 等系列综述，来源待验证]

[2001] Huxley et al.：URhGe 发现（G2 里程碑，G4 磁场再入的材料基础）
  → Nature 413, 613 (2001) [来源待验证，DOI 需核实]
  → 首次确认 FM+SC 本征共存（常压），G2 已详述

[2005] Lévy et al.：URhGe 磁场再入超导（G4 最重要实验节点）
  → Science 309, 1343 (2005) [来源待验证，DOI 需核实]
  → B ∥ b 轴，H ~ 12T 处出现再入 SC（Tc' ~ 0.4K）
  → 再入 Tc' > 零场 Tc（0.25K）——反常的磁场增强效应
  → 第一个明确的磁场驱动再入 SC 实验案例

[2000s-2010s] 重费米子压力相图中的双穹
  → CeCu₂Si₂（Steglich 系列）：发现两个 SC 区域，被 non-SC 区间隔开 [来源待验证]
  → CeRhIn₅（Park 2006 等）：压力相图 AFM+SC 区域转变 [来源待验证]
  → 这是"掺杂/压力再入"（类型1）的典型案例
  → 意义：表明两个 SC 穹在重费米子体系中并非偶然

[2019-今] UTe₂ 和磁场增强 SC
  → UTe₂（Ran 2019）：SC Tc ~ 1.6K，磁场下 Tc 不降反升（部分方向）[来源待验证]
  → 是否属于"磁场再入"？还是仅仅是 Hc₂ 增大？争议未定 [猜想]
  → 区别：无明确的"SC消失→再出现"双相变（与 URhGe 不同）
```

## 3. 温度再入（最稀奇的类型）

```
=== 温度再入：理论可能，实验极罕见 ===

【以下讨论均以标准 BCS 行为作为对照基准】

标准 BCS 行为 [定理]：
  Tc(参量) 是参量的单调函数（对于温度扫描：超导在 T=0 到 Tc 之间存在，T > Tc 消失）
  无再入 → 正常态在高温，超导在低温，单一转变

─────────────────────────────────────────────────────
温度再入的发生条件（[猜想，理论框架]）：
─────────────────────────────────────────────────────
  需要三个条件同时满足：
  1. 材料中存在局域磁有序（FM 或 AFM），磁序有自己的 TM
  2. SC 和磁序存在竞争（磁序增强 → SC 被压制）
  3. 磁序强度随温度的变化非单调（通常不满足！）

  极端情况（理论上可能）：
    T 从低到高扫描：
      T < T₁：SC 存在（Cooper 配对形成）
      T₁ < T < T₂：局域磁序增强（偶然机制），压制 SC → 正常态
      T₂ < T < T₃：磁序开始热涨落减弱 → SC 复活？（再入！）
      T > T₃：热涨落完全破坏 SC（终极 Tc）

  实际障碍：
    → 磁序强度 M(T) 几乎总是单调下降（随升温减弱）
    → M(T) 先增后减（非单调）需要非常特殊的竞争机制
    → 参数窗口极窄（对 T₁, T₂, TM 的相对大小要求严苛）

─────────────────────────────────────────────────────
最接近的历史案例：ErRh₄B₄（[观察，来源待验证]）
─────────────────────────────────────────────────────
  ErRh₄B₄（Tc ~ 8.6K，TM ~ 0.8K）[来源待验证]：
  温度扫描（降温方向）：
    T > 8.6K：正常态
    0.8K < T < 8.6K：SC 存在（LREE 稀土 SC）
    T < 0.8K：铁磁序出现 → SC 消失（Meissner 效应消失）
    T → 0K：仍无 SC（FM 主导）

  ⚠️ 这不是温度再入：
    → SC 在低温被 FM 取代，之后不再复活
    → 正确描述：FM 竞争序消灭 SC（SC 单次消失，不重现）
    → 被历史文献误称为"再入"是命名混乱导致的
    → [来源待验证，Fischer 1990 综述，来源待验证]

─────────────────────────────────────────────────────
结论（[观察]）：
─────────────────────────────────────────────────────
  截至 2026 年，没有任何固体材料被无争议地确认展示
  真正的"温度再入超导"（T₁→T₂：SC；T₂→T₃：正常；T₃→T₄：SC 再次出现）。
  温度再入至今是理论可能但实验从未实现的现象。[猜想/争议]
```

## 4. 压力/掺杂相图中的双 SC 穹（常见的"软性"再入）

```
=== 双 SC 穹：参数空间中的两个超导区域 ===

概念澄清：
  双 SC 穹（two SC domes）= 在 P-T 或 x-T 相图中，
  超导区域出现两个独立的穹形区域，中间被非超导区隔开。
  这是"参数维度的再入"，是最常见、最确实的再入超导类型。

─────────────────────────────────────────────────────
案例 1：CeCu₂Si₂（重费米子，[D8/G1 关联，来源待验证]）
─────────────────────────────────────────────────────
  背景：CeCu₂Si₂（CCS）是第一个重费米子超导体（Steglich 1979）
  压力相图（粗略描述）：
    SC1（P < 1 GPa 附近）：在 AFM-QCP 附近出现，Tc ~ 0.6K
    Non-SC 区（中间压力段）：AFM 或价键过渡区
    SC2（P > 2 GPa 附近）：在价键涨落驱动区出现，Tc ~ 0.6-0.7K
  
  关键问题：SC1 和 SC2 配对机制是否相同？[猜想]
    → SC1：d 波，由 AFM 自旋涨落介导（B3 关联）[猜想]
    → SC2：s 波，由价键涨落（Kondo 涨落）介导？[猜想]
    → 两个穹的节点结构/配对对称性可能完全不同！
    → 这意味着"双穹"≠"同一超导态的再入"，更像是两种不同的 SC
  → [Steglich et al. 系列；Holmes et al.；来源待验证]

─────────────────────────────────────────────────────
案例 2：CeRhIn₅（重费米子 + 压力，G1 已述，[来源待验证]）
─────────────────────────────────────────────────────
  压力相图特征：
    低压：AFM 为主（TN ≈ 3.8K），SC 弱（Tc 低）[来源待验证]
    中压（~1.7-2.4 GPa）：AFM+SC 微观共存区域
    高压（> 2.4 GPa）：AFM 消失，SC 更强（Tc ≈ 2.3K）[来源待验证]
  
  是否算"双穹"？[猜想]：
    → 部分文献认为低压和高压的 SC 是同一序参量，只是被 AFM 调制
    → 另一些认为存在两个独立的 SC 穹
    → G1 已详述此体系，G4 不重复材料细节

─────────────────────────────────────────────────────
案例 3：铜氧化物 LSCO（掺杂相图 x=1/8 区域，[争议]）
─────────────────────────────────────────────────────
  LSCO（La₂₋ₓSrₓCuO₄）掺杂相图特征：
    x ≈ 0.12（1/8 掺杂）处：Tc 异常压低（"1/8 效应"）
    → CDW 序在 x=1/8 处最强，竞争 SC
    → 产生 Tc(x) 曲线上的"凹陷"（dip）
  
  是否构成"双穹"？[争议]：
    → 严格意义上：Tc 未降为零，仅有 dip，不是真正的两个分离的穹
    → 部分理论（条纹相）预言 1/8 处 Tc=0，但实验未达到
    → 不宜将 LSCO 的 1/8 dip 称为"双穹再入"[反谄媚]
  → [Tranquada 1995 等，来源待验证]

─────────────────────────────────────────────────────
案例 4：铁基 BaFe₂As₂ 系列（[猜想，来源待验证]）
─────────────────────────────────────────────────────
  BaFe₂(As₁₋ₓPₓ)₂ 或 Ba(Fe₁₋ₓCoₓ)₂As₂ 掺杂相图：
    → SDW（条纹有序）在低掺杂消失 → SC 穹出现
    → 是否有第二个 SC 穹在高掺杂端？[猜想，来源待验证]
    → 实验报道有限，部分体系（如电子掺杂端）有 Tc 再次升高迹象
    → 层级：[猜想，实验证据弱，来源待验证]
```

## 5. 磁场再入的相图结构（最重要的实验类型）

```
=== 磁场再入超导的 H-T 相图 ===

URhGe（B ∥ b 轴，最清晰案例）[来源待验证，G2 G3 已详述]

  H-T 相图示意：

  T（K）↑
  0.4 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ╭──────────────────
        │                        / SC2（再入）
  0.3   │  SC1（零场 SC）       /  Tc'(H) 先升后平
        │    ╭──────╮          /
  0.2   │   /        \        /
        │  /   SC1    \      /  Normal
  0.1   │ /            \    /
        │/              \  /
  0.0 ──┴───────────────────────────────────→ H（T）
        0     2T    5T  9T   12T   15T

  说明：
    → SC1（H < 9T）：零场 FM+SC 本征共存；H↑ → Pauli 效应压制 SC → Tc 下降
    → Normal（9T < H < 12T）：SC1 消失，正常态
    → SC2/再入 SC（H > 12T）：FM 矩翻转（b 轴→c 轴），横向涨落增强
      → triplet 配对强度增大 → Tc' 反常升高（≈ 0.4K）
    → H_R（矩重取向场）≈ 12T（B ∥ b 轴）[来源待验证，Lévy 2005]

  关键特征（与普通超导的区别）：
    1. Hc₂(T) 曲线非单调（双峰/再入）
    2. 再入 SC 的 Tc' > 零场 Tc（磁场增强而非压制！）
    3. 再入 SC 仅在特定磁场方向出现（高度各向异性）
    4. 再入 SC 区域有限（H > 15T 后 Tc 再次下降）
```

## 6. 实验判断标准（区分再入超导与其他现象）

```
=== 如何判断一个材料是否存在再入超导 ===

判据 1（输运测量）：
  R(H) 或 R(T)：观察到双 dip（两次电阻骤降）
  → 第一次骤降：进入 SC1（正常→SC1 转变）
  → 骤升：回到正常态（SC1→正常态）
  → 第二次骤降：进入 SC2（正常→SC2 再入）
  → ⚠️ 必须排除：样品不均匀性导致的两步转变

判据 2（相图构建）：
  在 H-T 或 P-T 或 x-T 平面上绘制完整相图
  → 两个在参数空间中不连通的 SC 区域
  → 两个区域之间有明确的正常态区域（而非 SC 特征只是减弱）

判据 3（配对机制判别）：
  关键问题：两个 SC 区域的配对机制是否相同？
  实验方法：
    → 节点结构（NMR / 热导率 / 穿透深度）：SC1 和 SC2 是否有相同的节点？
    → 配对对称性（相敏实验 / 相干峰）：order parameter 对称性是否连续变化？
    → Knight shift（NMR）：singlet vs triplet 在两个区域的比较

判断规则：
  ┌──────────────────────────────────────────────────────────┐
  │ 同机制再入（参量连续变化但有间断）：                      │
  │   → 更可能是涨落增强（横向 FM 涨落 / QCP 涨落）           │
  │   → 典型：URhGe 的两个 SC 区域（均为 triplet，机制相同？）  │
  │                                                          │
  │ 不同机制的两个穹（两套独立 SC 态）：                       │
  │   → 更严格意义上不应称为"再入"，而是"双超导穹"             │
  │   → 典型：CeCu₂Si₂ SC1（d 波 + AFM-QCP）vs SC2（s? + 价键）│
  └──────────────────────────────────────────────────────────┘

⚠️ 重要警告：
  → "双 Tc"不等于再入超导！
    相分离（两相共存）也能产生双转变温度：
    不同比例的两相在不同温度发生转变 → 两步转变
    区分方法：单晶 vs 多晶；局域探针（µSR/NMR）vs 整体探针（输运）
```

  SC1 出现压力范围：P < 1 GPa（AFM-QCP 附近）[来源待验证]
  SC2 出现压力范围：P > 2 GPa（价键涨落区）[来源待验证]
  SC1 Tc（最高值）：~ 0.6K [来源待验证]
  SC2 Tc（最高值）：~ 0.6-0.7K [来源待验证]
  ← ⚠️ 以上压力数值为估计，需查 Holmes et al. 或 Yuan et al. 原始文献核实

ErRh₄B₄（非温度再入，历史参考）[来源待验证]：
  高温 SC 转变温度 Tc：~ 8.6K（降温进入 SC）[来源待验证]
  低温 FM 出现温度 TM：~ 0.8K（SC 消失）[来源待验证]
  → SC 消失后不再复活：这不是再入超导！

⚠️ 反幻觉说明：
  → 所有数值均为来源待验证，未直接查原始文献
  → CeCu₂Si₂ 双穹的压力边界数值来自二次引用（G4 写作时未查原文）
  → URhGe 数值已在 G2/G3 中引用，来源相同
  → 铁基 / LSCO 双穹相关数值未列入（证据弱，来源待验证）

## 8. 关键反谄媚

```
⚠️ 反谄媚 G4.1：ErRh₄B₄ 不是温度再入超导（历史命名混乱！）

  常见的错误说法：
    → "ErRh₄B₄ 是温度再入超导的经典案例"
  
  现实：
    → 降温过程：T > 8.6K（正常态）→ 8.6K（进入 SC）→ 0.8K（FM 出现，SC 消失）
    → SC 消失后不会随温度进一步降低而"复活"
    → 正确描述：局域磁矩铁磁序在低温取代超导（两种竞争序的相变）
    → 这与"再入"的定义（SC 消失后重现）完全不符！
    → 历史文献有时将此称为"re-entrant normal state"（再入正常态），
      指的是从 SC 又回到正常态，不是"再入超导"！
    → 命名混乱源头：different authors using "re-entrant" for different phenomena

⚠️ 反谄媚 G4.2：温度再入超导至今无确认案例

  截至 2026 年，没有任何材料被无争议地确认展示真正的温度再入超导。
  "真正的温度再入"= 升温过程中：正常→SC→正常→SC 四个相区依次出现。
  理论允许存在，实验从未实现，层级：[猜想]。
  不要在教科书叙述中将此当作已知事实引用。

⚠️ 反谄媚 G4.3：CeCu₂Si₂ 的双穹可能是两种完全不同的超导态

  "双穹"≠"同一超导态的再入"。
  CeCu₂Si₂ 中，SC1（低压，AFM-QCP 驱动）和 SC2（高压，价键涨落驱动）
  可能具有完全不同的：
    → 配对对称性（d 波 vs s 波？）
    → 驱动机制（自旋涨落 vs 价键涨落）
    → 节点结构
  将 CeCu₂Si₂ 的两个穹简单称为"再入超导"是对物理内容的过度简化。
  正确表述："CeCu₂Si₂ 有两个分离的超导穹，机制可能不同。"[猜想]

⚠️ 反谄媚 G4.4：磁场再入 SC 只在极少数特殊磁性超导体中出现

  常见的夸大说法：
    → "URhGe 的再入 SC 机制可以推广到其他体系"
  
  现实（2026年）：
    → 无争议的磁场再入 SC：URhGe（最清晰），UCoGe（较弱）[来源待验证]
    → 普通铁磁金属（Fe/Ni/Co）：无 SC，无再入
    → 普通 BCS 超导体：无 FM 矩翻转机制，无再入
    → 必要条件极苛刻（triplet 配对 + FM 矩重取向 + 横向涨落增强）
    → 铀基重费米子体系之外，至今无确认的磁场再入案例

⚠️ 反谄媚 G4.5："双 Tc"观察必须排除相分离

  多晶样品中，两种不同比例的相可以各自发生超导转变，
  在 R(T) 曲线上产生"双 dip"，外观与再入超导完全相似。
  区分方法（按可信度排序）：
    1. 单晶 + 局域探针（µSR NMR STM）：相分离会有局域不均匀性信号
    2. 外场（压力/磁场）扫描：再入 SC 双 dip 的相对位置随外场系统变化
    3. 样品依赖性：相分离 → 双 dip 位置随样品批次变化大；再入 → 位置稳定
    4. 比热测量：真正再入 → 两个清晰的比热跳变；相分离 → 宽化转变
```
