## 1. 物理定义与核心思想

### 1.1 节点的定义

**[定理 — 节点是 Δ(k) = 0 的 k 空间集合]**

```
超导能隙函数 Δ(k) 的零点集合称为节点（nodes）：

  节点集合 N = { k ∈ 费米面 : Δ(k) = 0 }

物理意义：
  在 N 中的 k 点，超导能隙消失
  → 准粒子在这些 k 点有零能量激发（零能模）
  → 这些零能模主导了超导体的低温物理

与全能隙对比：
  全能隙超导（无节点）：N = ∅
    → 所有 k 点均 |Δ(k)| > 0
    → 最小激发能 = min|Δ(k)| = Δ_min > 0
    → 热力学量指数激活 ~ exp(-Δ_min / kT)

  节点超导：N ≠ ∅
    → 节点附近存在任意小能量的激发
    → 热力学量呈幂律行为（T^n，n > 0）
```

LaTeX 等价：
$$N = \{\mathbf{k} \in \text{FS} : \Delta(\mathbf{k}) = 0\}$$

### 1.2 节点的类型分类

**[定理 — 节点几何分类]**

```
根据节点集合 N 的维度，节点分为三类：

[1] 线节点（Line nodes）：
    N 是费米面上的 1D 曲线（通常是直线/圆圈）
    三维体系中，节点线将费米面分成正/负能隙区域
    例：d_x²-y² 的节点线在 k_y = ±k_x（对角线方向）
    
    低能 DOS（态密度）：
    N(ω) ∝ ω    （线性，三维 d-wave 极限）[定理]
    
    物理原因：节点线附近 Δ(k) 线性 → 在 2D 情形：
    N(ω) = (2/π) × (ω/v_F) × (1/v_Δ)
    其中 v_Δ = |∇_k Δ(k)|_{节点} 是节点处的能隙斜率

[2] 点节点（Point nodes）：
    N 是费米面上的孤立点（0D）
    通常出现在高对称 k 点（如球形费米面的极点）
    例：p-wave 在 k_z 轴的极点（(0,0,±k_F)）
    
    低能 DOS（三维）：
    N(ω) ∝ ω²    （三维，二次）[定理]
    N(ω) ∝ ω     （二维，Dirac 类比）[定理]
    
    物理原因：孤立点 → k 空间中节点测度更小 → DOS 更低

[3] 意外节点（Accidental nodes）：
    满足自洽方程但不受晶格对称性保护的节点
    参数（掺杂、压力）变化可以使意外节点出现或消失
    与拓扑保护节点（第 5 节）相对
    
    例：s± 在特定参数下的意外节点（铁基超导 P 掺杂系列）

ASCII 维度总结：
  线节点：N 是 1D → N(ω) ∝ ω   → C ∝ T²，Δλ ∝ T
  点节点：N 是 0D → N(ω) ∝ ω²  → C ∝ T³，Δλ ∝ T²
  全能隙：N = ∅  → N(ω) = 0   → C ∝ exp(-Δ/T)，Δλ ∝ exp(-Δ/T)
```

### 1.3 节点→低温物理的因果链

**[定理 — 节点 DOS → 热力学幂律的推导链]**

```
核心推导链（以线节点为例）：

Step 1：节点 DOS（[定理]）
  N(ω) ∝ ω    （线性，线节点，干净极限）

Step 2：比热（[定理]）
  C(T) = ∫ N(ω) × ω × (-∂f/∂ω) dω
       ∝ T²    （线节点）
  
  其中 f(ω) = 1/(e^{ω/kT}+1) 是 Fermi-Dirac 分布
  → 维度分析：N(ω)∝ω × ω × (1/T²) × T ∝ T²（[定理]）

Step 3：穿透深度（[定理]，Gorkov 1959 推广）
  Δλ(T) = λ(T) - λ(0) ∝ ∫ N(ω) × (-∂f/∂ω) dω
         ∝ T    （线节点）

Step 4：NMR 1/T₁（[定理]）
  1/T₁ ∝ T × ∫ N²(ω) × (-∂f/∂ω) dω
        ∝ T × T² ∝ T³    （线节点）

点节点类比（替换 N(ω) → ω²）：
  C(T) ∝ T³，Δλ ∝ T²，1/T₁ ∝ T⁵（点节点）

=== 幂律汇总表（无磁场，干净极限）===

              | 线节点  | 点节点  | 全能隙
  -----------|---------|---------|-------------------
  N(ω)       | ∝ ω    | ∝ ω²   | 0（ω < Δ₀）
  C(T)/T     | ∝ T    | ∝ T²   | ∝ exp(-Δ₀/T)/T
  Δλ(T)      | ∝ T    | ∝ T²   | ∝ exp(-Δ₀/T)
  1/T₁       | ∝ T³   | ∝ T⁵   | ∝ exp(-2Δ₀/T)（相干峰后）
  κ/T|_{T→0} | 有限量  | 有限量  | → 0（指数）
```

## 3. 实验测量节点的方法

### 3.1 穿透深度 λ(T)（最常用，最常误判）

**[定理 — 以下为干净极限结果，见 Hardy et al. 1993 PRL]**

```
穿透深度变化 Δλ(T) = λ(T) - λ(0)：

无节点（全能隙）：
  Δλ(T) ∝ exp(-Δ₀ / k_B T)    T→0 时指数饱和

线节点（干净极限，T << Tc）：[定理]
  Δλ(T) ∝ T    （线性）
  Hardy et al. 1993 在 YBa₂Cu₃O₇ 首次测量到线性 Δλ(T)
  → 这是 d_x²-y² 节点的第一个热力学直接证据

点节点：
  Δλ(T) ∝ T²    （二次方）

脏极限修正（[定理]，非磁性杂质散射）：
  线节点体系在脏极限下：Δλ(T) → T²
  原因：杂质散射将线节点附近的线性 DOS 转变为类全能隙行为（AGD 理论）
  → 强调：脏极限的 T² ≠ 点节点！
  
  判别方法：
    改变样品质量（掺杂对照），观察 Δλ(T) 的温度幂次是否变化
    真正的线节点：随纯化程度升高，Δλ → T（从脏的 T² 趋向干净的 T）
    真正的点节点：Δλ ∝ T² 在所有纯度下不变
```

⚠️ **反谄媚警告：穿透深度 T 线性 ≠ 一定是线节点**
```
非节点原因的线性 Δλ(T)：
  [1] 超低温（T << Tc）下，无节点能隙 + 极低杂质水平：
      实验误差和拟合范围可以产生虚假线性
  [2] 两带效应：一个带有节点，另一个带全能隙 → 实际比热幂次可能接近线性
  [3] 局域配对不均匀性（granular SC）：可模拟节点行为
  → 必须结合比热 + 热导率 + NMR 综合判断
```

### 3.2 比热 C(T)/T

**[定理 — 低温比热行为是节点的热力学证据]**

```
无节点：C/T ∝ exp(-Δ₀/T)（指数，T→0 趋零）
线节点：C/T ∝ T          → C ∝ T²    [定理]
点节点：C/T ∝ T²         → C ∝ T³    [定理]

磁场下的 Volovik 效应（[定理]，Volovik 1993 JETP Lett.）：
  线节点超导在磁场 H 下：
  C/T ∝ √H    （H << H_c2，含节点的 Doppler shift）
  
  物理机制：涡旋的超流速场在节点附近产生 Doppler 位移
  v_s → Doppler shift ω → ω + v_s·k
  → 节点附近低能态密度被 v_s 扩散 → N(ω, H) ∝ √H
  
  → C/T ∝ √H 是线节点的磁场指纹（Volovik effect，[定理]）
  → 全能隙超导：C/T ∝ H（亚间隙 bound state，Caroli-Matricon）
  
  如何区分：
  线节点体系：C/T ∝ √H（低场下）
  全能隙体系：C/T ∝ H（所有场下，Andreev bound states）
  → 实验：log-log 图 C/T vs H，斜率 1/2（线节点）vs 1（全能隙）
```

⚠️ **反谄媚警告：比热 T² ≠ 一定是线节点**
```
T² 比热的非节点来源：
  [1] 多带 s-wave：两个 s-wave 能隙 Δ₁ >> Δ₂，T << Δ₂ 时
      小能隙 Δ₂ 带的激发主导 → 可以产生类似 T² 行为（伪节点效应）
      例：MgB₂ 的双能隙 → 中间温区类 T² 行为，并非节点
  [2] 量子涨落修正（重费米子）：T << T* 的非 Fermi-Liquid 残余
  → 多带 T² 的判据：T² 行为在极低温（T << Δ₂）消失，出现饱和
  → 真正线节点 T² 行为延伸到 T → 0 不饱和（没有下限能量截断）
```

### 3.3 热导率 κ(T, H) — 节点方向识别

**[定理 — Durst & Lee 2000 PRB; Izawa et al. 2001 PRL]**

```
通用热导率（Universal Thermal Conductivity）：
  线节点超导在 T → 0、H → 0 极限下：
  
  κ_xx / T|_{T→0} = C_universal（与杂质浓度无关的有限量）[定理]
  
  Durst-Lee 公式（[定理]，干净极限）：
  κ_xx / T = (k_B²/3) × (v_F / v_Δ) × n_nodes × (k_F / π)
  
  其中 v_F = Fermi 速度，v_Δ = 节点处能隙斜率，n_nodes = 节点对数
  
  物理意义：即使 T → 0，线节点体系也有有限热导率
  → 这是节点存在的最强热力学证据之一（不依赖杂质浓度！）
  
  全能隙超导：κ/T|_{T→0} → 0（指数），无通用量

磁场方向依赖——Izawa 方法（[定理]）：
  旋转磁场 H 方向，测量 κ(H 角度) 的振荡
  节点方向 ≡ κ极小值方向（涡旋格子旋入节点方向，准粒子被捕获）
  
  d_x²-y² 节点：κ 在 H ∥ (110)（对角线）方向出现极大值
  → 节点在 45° 对角线 → 磁场旋转实验确认 [Izawa et al. 2001 PRL]
  
  这一方法（磁场角度依赖热导率）是节点方向的直接确认工具
```

### 3.4 NMR 1/T₁

**[定理 — BCS 相干峰 + d-wave 无相干峰是经典判据]**

```
BCS s-wave（全能隙）：
  Tc 以下出现 Hebel-Slichter 相干峰：1/T₁T 在 Tc 以下先升后降
  物理：BCS 相干因子（u²+v²）在求和时同号叠加 → 增强散射
  → 相干峰是 s-wave 的经典指纹 [定理]

d-wave（线节点）：
  无 Hebel-Slichter 峰：Tc 以下 1/T₁T 单调下降
  物理：d-wave 相干因子（u²-v²）在节点方向相消 → 峰被压制
  → 无相干峰 + 1/T₁ ∝ T³（低温）是 d-wave 的双重 NMR 判据 [定理]
  
  实验要求：高纯单晶，低射频功率（Sr₂RuO₄ 教训！）

定量行为：
  无节点（s-wave）：1/T₁ ∝ exp(-2Δ₀/T)（低温指数，双能隙对因子 2）
  线节点（d-wave）：1/T₁ ∝ T³（[定理]，N²(ω) ∝ ω² 贡献两个 DOS）
  点节点：         1/T₁ ∝ T⁵（[定理]，N²(ω) ∝ ω⁴）
```

### 3.5 ARPES 直接成像

**[观察 — 最直接但技术限制严格]**

```
原理：
  ARPES（角分辨光电子能谱）直接测量 k 空间的 Δ(k) 分布
  扫描费米面 → 找 Δ(k) = 0 的 k 点（节点位置）

经典成功案例：
  d_x²-y² in BSCCO（Bi₂Sr₂CaCu₂O₈+δ）：
  节点在 k = (π/2, π/2) 方向（对角线，D₄h 中的 B₁g 节点线）[来源待验证：Ding et al. 1996 Nature]
  → 是 d_x²-y² 配对的最直接 k 空间证据

技术限制：
  [1] 能量分辨率 ~ 1-3 meV：小能隙（< 1 meV）无法分辨
  [2] 低 Tc 材料：Sr₂RuO₄（Tc = 1.5 K），UPt₃（Tc = 0.5 K）— 几乎不可能做 ARPES
  [3] 表面态污染：体态与表面态混叠
  [4] 需要极干净的劈裂面（只适合层状材料）
  
  → ARPES 仅适用于高 Tc 层状材料（铜氧化物）的节点识别
  → 低温/三维材料的节点只能靠热力学方法
```

## 4. 拓扑保护节点 vs 意外节点

**[定理 — 对称性保护是节点稳定性的关键]**

### 4.1 对称保护节点（Symmetry-Protected Nodes）

```
定义：
  由晶格点群对称性 + 时间反演对称性（TRS）保护的节点
  只要对称性不破缺，这些节点不能被扰动（掺杂、压力）移除

d_x²-y² 节点的对称保护机制（[定理]）：
  D₄h 点群中，B₁g 表示在 k_y = ±k_x 方向必须有节点
  原因：Δ(k_y = k_x) 在 C₄ 旋转下变号（B₁g 对称性要求）
       → Δ = 0 在 45° 对角线上
  
  保护算子：C₄ 旋转 + TRS（时间反演）的组合
  → 任何保持 D₄h + TRS 的扰动不能打开节点
  → 即使加入各向同性 s-wave 成分（s+d 混合），节点也保持，
    仅节点位置在 k 空间略微移动（允许的，对称保护是点，不是精确位置）

拓扑节点：Weyl 点和 Dirac 点
  Weyl 点：手性 ±1 的孤立点 → 拓扑荷 = ±1，对所有扰动稳定
            从费米面外看：如同磁单极
  Dirac 点：正负手性的 Weyl 点重合（高对称点），可被扰动劈裂
  
  → 超导配对态中的 Weyl 点（超导 Weyl 节点）：需要 TRS 破缺 + 反演破缺
  → 例：非中心对称手性 SC（NCS chiral）中可能出现
```

### 4.2 意外节点（Accidental Nodes）

**[猜想 — 铁基 s± 意外节点的理论框架，以下为观察与猜想级别]**

```
定义：
  满足 BdG 自洽方程但不受点群对称性约束的节点
  属于参数空间的偶然解，不受拓扑保护

s± 意外节点的机制（[猜想]，铁基超导）：
  s± 的序参量：Δ(k) = Δ₁（在电子口袋）和 Δ₂（在空穴口袋），Δ₁·Δ₂ < 0
  标准 s±：两个能隙值都有限，|Δ₁|, |Δ₂| > 0 → 无节点
  
  意外节点的出现条件（[猜想]，参数依赖）：
  当 intra-band interaction（带内排斥）强于 inter-band interaction（带间引力）
  → Δ(k) 在某个轨道分量改变符号 → 在费米面上出现 Δ = 0 的点/线
  → 这些节点不受 A₁g 对称保护（因为 A₁g 只要求全态的空间平均不为零）

Ba(Fe₁₋ₓPₓ)₂As₂ 的实验证据（[观察]）：
  Reid et al. 2012 PRL 在该系列测量到 κ/T → 有限（通用热导率）
  → 与标准 s±（无节点）矛盾
  → 解释：P 掺杂改变了带结构，使 s± 演化出意外节点 [来源待验证：Reid et al. 2012 PRL 超声]
  
  与真节点（对称保护）的区分：
  意外节点：随 x 变化可以连续出现或消失（连续演化）
  对称保护节点：在对称性存在的整个参数范围内不消失
  
  → Ba(Fe₁₋ₓPₓ)₂As₂ 随 x 变化：从无节点 s± → 节点 s± 的连续演化
    这正是意外节点的特征（相变但无拓扑相变）

实验判断意外节点的额外困难：
  意外节点可能只存在于费米面的一小块区域（"gap minima" 极小值）
  → 穿透深度可以被"缺口"中的极小值主导，但不会显示精确的线性 T
  → 这使节点/无节点的判别更加困难 [观察]
```

## 5. 节点在铁基超导中的争议（专题）

**[观察 — 领域内最活跃的节点争议之一]**

```
基本格局：
  铁基超导被普遍接受为 s± 配对（C2），标准预测是无节点
  但多个测量显示节点行为 → 引发 "nodeless s± vs nodal s±" 之争

材料依赖性（[观察]，不同铁基体系结果不同）：

  BaFe₂(As₁₋ₓPₓ)₂（BaFeP 系列）：
  → x = 0（As）端：无节点 s±（低 T 比热指数）
  → x ≈ 0.3（SC 最佳掺杂）：热导率通用量 → 节点信号 [Reid 2012]
  → x → 1（P）端：节点强化
  → 解读：意外节点随 x 调制（带结构随 P/As 替换变化）

  LaFePO（早期铁基）：
  → 穿透深度线性 Δλ ∝ T → 节点信号 [Fletcher et al. 2009 PRL，来源待验证]
  → 但注意：LaFePO 的费米面结构不同于 BaFe₂As₂（无空穴口袋 @ Γ）

  KFe₂As₂（全电子口袋，重掺杂）：
  → 线节点证据强烈（Δλ ∝ T, C ∝ T²）[来源待验证]
  → 可能是 d-wave 或节点 s-wave（仍争议）

理论解释框架（[猜想]）：
  [1] 轨道权重依赖 Δ(k)：
      铁基的多轨道（Fe 3d）使得 s± 的每个 k 点上有多个轨道贡献
      → 如果 3d_xy 轨道的 Δ < 0 而 3d_xz 轨道的 Δ > 0
      → 在某 k 点两个贡献正好抵消 → 意外节点
  [2] 自旋轨道耦合（SOC）效应：
      SOC 在铁基中量级 ~ 50-100 meV，不可忽略
      → SOC 重组轨道 → 修改节点位置和存在性
  [3] 层间耦合：三维效应使线节点 "puffed up" → 变成点节点

结论（目前共识，[观察]）：
  铁基超导的节点结构是材料依赖的，没有统一答案
  标准 s±（BaFe₂As₂ at optimal doping）：无节点 → 最强证据
  P 掺杂、重空穴掺杂端：节点可能出现 → 意外节点框架
  → 节点/无节点不是铁基的本质特征，而是费米面拓扑的函数
```

## 6. 节点与超导体的拓扑分类

**[猜想 — 节点的拓扑意义是近年的新认识框架，部分为理论预言]**

```
节点与拓扑边缘态的对应（[猜想]，来源：Schnyder et al. 2008 PRB，[来源待验证：卷号]）：

[1] 线节点（1D 节点线） → 平坦带边缘态（Arc states）：
    线节点将费米面分成拓扑不同的两个区域
    在两个区域的边界，表面态形成"平坦带"（arc states）
    类比：半金属中 Weyl 费米弧（Fermi arc）
    
    d_x²-y² 超导的表面：
    （100）表面：节点在表面 BZ 的 (0,±π) 处 → 零能表面束缚态
    → 在 STM 实验中表现为 ZBCP（零偏压电导峰）[观察，来源待验证：Tanuma et al.]

[2] 点节点（Weyl 型） → Fermi arc（半金属类比）：
    手性 Weyl 节点（拓扑荷 ±1）→ 表面上的 Fermi arc
    但超导中的 Weyl 节点需要 TRS + 反演对称性同时破缺才能出现
    → 目前无实验确认的 Weyl 节点超导体 [来源待验证]

[3] 与拓扑超导（TSC，B11）的重要区分：
    拓扑保护节点的超导体 ≠ 拓扑超导（TSC）！
    
    区分标准：
    有拓扑保护节点的 SC（如 d_x²-y² 铜氧化物）：
    → 有边缘零模，但不是马约拉纳模
    → 这些表面束缚态是 Bogoliubov 准粒子，有 U(1) 电荷
    
    拓扑超导（TSC，B11 类型，如 chiral p-wave）：
    → 马约拉纳零模，无电荷，属于不同拓扑类
    → 需要能隙超导体 + 奇数 Chern 数（或 Z₂ 拓扑不变量）
    
    → 能量节点 ≠ 拓扑不变量的条件（两者不是同一件事！）
    → 可以有节点但无拓扑 SC：d_x²-y²（有节点，非 TSC）
    → 可以有拓扑 SC 但无节点：chiral p+ip（无节点，是 TSC）
```

## 7. 关键反谄媚汇总

### 7.1 穿透深度单一判据不可信

**[观察 — 实验历史的重复教训]**

```
常见误判路径：
  观察到 Δλ(T) ∝ T → 宣布"线节点超导" ❌

错误根源：
  [1] 杂质/格点缺陷：即使全能隙 s-wave，极小能隙带杂质 → 亚能隙态
      → Δλ(T) 出现低温线性，来自 subgap impurity states，非节点
  [2] 表面/界面效应：表面电流对穿透深度有贡献
      → 在薄膜或粉末样品中，表面态导致虚假线性
  [3] 拟合范围：T 范围选取不当（太宽）→ 可以把指数拟合成"线性"
  
  → 判断节点必须：排除杂质效应（纯化对照）+ 比热交叉验证 + 热导率通用量
```

### 7.2 比热 T²/T³ 幂律不唯一确定节点类型

```
T² 比热的竞争解释：
  [真节点]   线节点超导：C ∝ T²（线性 DOS）
  [伪节点1] 多带 s-wave（Δ₁ >> Δ₂）：在 Δ₂ < T < Δ₁ 区间显似 T²
  [伪节点2] 非费米液体正常态残余（重费米子）：正常态 T² 项污染超导态
  [伪节点3] 磁性杂质导致的亚能隙态：类线性 DOS
  
  判别策略：
  → 真线节点：C/T ∝ T 延伸到 T → 0 不饱和 + √H 磁场依赖（Volovik）
  → 多带 s-wave：在极低温 C/T 再次指数激活（小能隙被激活）
  → 重费米子污染：磁场对 C/T 的幂次影响弱（异常行为）
```

### 7.3 意外节点对参数高度敏感

```
实验体系：Ba(Fe₁₋ₓPₓ)₂As₂ 的节点随 x 变化

这一事实的启示：
  → 同一配对对称性（s±）可以在参数空间中有节点/无节点的两相
  → "这种材料是否有节点"不是固定属性，是参数依赖的
  → 比较不同样品或不同组的测量时，掺杂量 x 的误差 ±0.01 可能改变节点存在性
  
  反谄媚立场：
  → 声称"铁基是有节点的"或"铁基是无节点的"都是过度简化
  → 正确表述："在 x = 0.3 的 Ba122P 体系，热导率通用量表明节点存在，
    这与标准无节点 s± 矛盾，可能是意外节点在此掺杂点出现"
```

## 8. 与 Paper A / E 的关联

```
Paper A（两通道 Allen-Dynes / Eliashberg）：
  节点信息的实用价值：
  → Paper A 预测 Tc 并不直接用到节点结构
  → 但节点判断影响材料选样：
    [1] 全能隙材料（无节点）的 Tc 对 s-wave 声子机制判断更干净
    [2] 有线节点（d-wave）的材料暗示 AFM 涨落主导（B3）而非 Eliashberg 声子机制
    → 铜氧化物（d_x²-y²，线节点）被 Paper A 显式排除在外
    → 铁基（s± 无节点，或弱意外节点）可以通过 Paper A 框架分析 Tc
    
  节点→机制：
  线节点（d-wave） → 几乎确定是 AFM 涨落/RVB 机制（B3/B9），不适合 Paper A
  无节点（s-wave, s±）→ 可能是 Eliashberg 声子 or 多通道，Paper A 适用范围

Paper E（轨道-能隙对应）：
  节点是 Paper E 的核心预测量：
  → Paper E 从轨道对称性出发预测节点位置和能隙结构
  → C10 节点分类是 Paper E 预测的验证框架
  
  具体关联：
  B₁g 轨道（d_x²-y²）→ d_x²-y² 配对 → 节线在 (±π/2, ±π/2) 方向
  A₁g 轨道 → s 或 s± 配对 → 无节点（标准）
  
  Paper E 的关键声明：
  [猜想] "轨道-能隙对应原则：主导费米面的轨道对称性决定了超导能隙的点群表示，
  从而决定了节点的位置和类型"
  → 如果此原则成立，节点是轨道特征的直接下游量
  → 反之，节点实验验证可以支持或质疑 Paper E 的轨道-能隙对应
```
