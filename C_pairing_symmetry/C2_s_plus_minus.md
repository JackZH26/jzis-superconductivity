## 1. 物理定义与核心思想

### 1.1 多带超导中的配对对称性问题

**背景：** 当一个超导体有多个费米面（多带结构）时，每个费米面上可以独立携带不同的配对振幅 Δ(k)。单带 BCS 中 Δ(k) = Δ₀（各向同性 s-wave），但多带体系中情况更丰富。

**铁基超导的多带结构：**
- 铁基超导（LaFeAsO、BaFe₂As₂ 等）的费米面来自 Fe 的 3d 轨道（t₂g + eg）
- 典型结构：Γ 点（布里渊区中心）附近 2-3 个**空穴口袋（hole pockets）**，M 点（区角）附近 2 个**电子口袋（electron pockets）**
- 空穴带（hole bands）：圆形口袋，包围 Γ 点，记为 h₁、h₂
- 电子带（electron bands）：椭圆形口袋，包围 M=(π,0)/(0,π)，记为 e₁、e₂

**核心问题：** 多个费米面上的 Δ 可以同号（s++）还是异号（s±）？

### 1.2 s± 的精确定义

**[定义]** s± wave（符号交替 s-wave，sign-changing s-wave）：

$$\Delta_{\rm hole}(\mathbf{k}) = +\Delta_h, \quad \Delta_{\rm electron}(\mathbf{k}) = -\Delta_e$$

ASCII: Δ_hole = +Δ_h，Δ_electron = -Δ_e（不同费米面间符号相反）

核心特征：
- **同属 A₁g 表示**：每个费米面内部，Δ(k) 对晶格旋转和反射对称（全对称），无节线
- **费米面间符号差 π**：空穴带和电子带的序参量相差一个负号（相位差 π）
- **无节点（标准版）**：在各个费米面内部，Δ(k) 处处非零（全能隙）
- **区别于 d-wave**：d-wave 节点在同一费米面内部，s± 节点（如果存在）是"意外节点"，非对称性强制

**与 s++ 的对比：**

s++（同号 s-wave）：Δ_hole = +Δ_h，Δ_electron = +Δ_e（同符号）

两者均属 A₁g，从单带实验（比热、穿透深度）无法直接区分，需要多费米面敏感实验（中子散射、QPI、Josephson）。

### 1.3 群论解释：A₁g 表示中的相位自由度

**[定理 — 对称性分析]** 在四方晶系 D₄h 点群（铁基 FeAs 面）中，s-wave 对应 A₁g 表示：
- 在所有对称操作（C₄、C₂、σ_h、i 等）下，Δ(k) 变换为 +1
- s± 中的符号差不是由晶格对称性"强制"的，而是由相互作用（AFM 涨落）动力学产生的

**关键点：** s± 是 A₁g 对称性范畴内的一个**特定动力学解**，不是不同的表示。因此：
- s± 和 s++ 在群论上属于同一类（都是 A₁g）
- 两者的区别是"偶然性"的（accidental），由 Hamiltonian 参数决定
- 这与 d-wave（B₁g）属于不同对称性类别，有本质区别

**物理直觉（ASCII 示意）：**
```
s++ (A₁g)：           s± (A₁g)：
  ┌──────────┐           ┌──────────┐
  │  +Δ      │           │  +Δ      │
  │ (hole)   │           │ (hole)   │
  └──────────┘           └──────────┘
  ┌──────────┐           ┌──────────┐
  │  +Δ      │           │  -Δ      │
  │(electron)│           │(electron)│
  └──────────┘           └──────────┘
   同符号，各向同性       符号反转，但各自各向同性
```

### 1.4 核心物理：AFM 波矢 Q 驱动符号反转

**[猜想 — 强烈支持，Mazin 2008 预言]** s± 的物理起源：

**步骤 1 — 铁基费米面拓扑：**
空穴带（Γ 点）和电子带（M 点）之间的嵌套（nesting）波矢 Q ≈ (π, π)（展开布里渊区）。

**步骤 2 — 自旋涨落在 Q 处增强（B3）：**
接近 AFM 相（如 BaFe₂As₂ 的母相是条纹 AFM），Q=(π,π) 处自旋涨落显著增强：

$$\chi_s(\mathbf{Q}) \gg \chi_s(\mathbf{0})$$

ASCII: χ_s(Q) >> χ_s(0)（Q 处自旋磁化率发散）

**步骤 3 — 带间配对相互作用为排斥：**

```
V_{12}(Q) = g² × χ_s(Q) > 0   （注意符号约定！排斥对配对方程中是负号）
```

在配对 Hamiltonian 中，带间相互作用项 V₁₂ 有效为负（排斥态，推动 Δ₁Δ₂ < 0）：

$$\Delta_1 = -|V_{12}| \sum_k \frac{\Delta_2}{2E_k^{(2)}}$$

ASCII: Δ₁ = -|V₁₂| Σ_k Δ₂/(2E_k^(2))

**步骤 4 — 自洽方程有 s± 解：**
若 V₁₂ < 0（排斥），上式要求 Δ₁ 和 Δ₂ 符号相反 → s± 是稳定解。

**总结（物理图像）：**
```
AFM 涨落在 Q=(π,π) 处增强
→ 带间"排斥"相互作用
→ 能量要求：Cooper 对在散射 Q 后能量不变
→ 条件：Δ(k) × Δ(k+Q) < 0
→ 即 Δ_hole × Δ_electron < 0  →  s±
```

## 3. 数学框架

### 3.1 两带 BCS Hamiltonian

**[定理框架 — 标准两带模型]**

$$H = \sum_{\mathbf{k}} \left[\xi_1(\mathbf{k}) c^\dagger_{1\mathbf{k}}c_{1\mathbf{k}} + \xi_2(\mathbf{k}) c^\dagger_{2\mathbf{k}}c_{2\mathbf{k}}\right] + \sum_{\mathbf{k}} \left[\Delta_1 c^\dagger_{1\mathbf{k}\uparrow}c^\dagger_{1{-\mathbf{k}}\downarrow} + \Delta_2 c^\dagger_{2\mathbf{k}\uparrow}c^\dagger_{2{-\mathbf{k}}\downarrow} + \text{h.c.}\right]$$

ASCII:
```
H = Σ_k [ξ₁(k) c†_{1k}c_{1k} + ξ₂(k) c†_{2k}c_{2k}]
    + Σ_k [Δ₁ c†_{1k↑}c†_{1,-k↓} + Δ₂ c†_{2k↑}c†_{2,-k↓} + h.c.]

带 1 = 空穴带（Γ 点附近，ξ₁ < 0 在费米面内）
带 2 = 电子带（M 点附近，ξ₂ < 0 在费米面内）
Δ₁, Δ₂：两带序参量（可以同号或异号）
```

### 3.2 两带自洽方程

**[定理 — 两带 BCS gap equation]**

$$\Delta_i = \sum_j V_{ij} \sum_{\mathbf{k}} \frac{\Delta_j}{2E_j(\mathbf{k})} \tanh\left(\frac{E_j(\mathbf{k})}{2k_BT}\right)$$

ASCII:
```
Δᵢ = Σⱼ Vᵢⱼ × Σ_k Δⱼ / (2Eⱼ_k) × tanh(Eⱼ_k / 2k_BT)

i, j ∈ {1, 2}（带指标）
V = [[V₁₁, V₁₂],     配对相互作用矩阵
     [V₂₁, V₂₂]]
Eⱼ_k = sqrt(ξⱼ_k² + Δⱼ²)   （准粒子能量）
```

### 3.3 s± 条件 vs s++ 条件

**[定理 — 配对符号由 V 矩阵符号决定]**

矩阵方程（线性化，T → Tc）：

$$\begin{pmatrix} \Delta_1 \\ \Delta_2 \end{pmatrix} = \begin{pmatrix} V_{11}\Lambda_1 & V_{12}\Lambda_2 \\ V_{21}\Lambda_1 & V_{22}\Lambda_2 \end{pmatrix} \begin{pmatrix} \Delta_1 \\ \Delta_2 \end{pmatrix}$$

ASCII:
```
[Δ₁]   [V₁₁Λ₁  V₁₂Λ₂] [Δ₁]
[Δ₂] = [V₂₁Λ₁  V₂₂Λ₂] [Δ₂]

Λᵢ = N_i(0) × ln(2γℏω_c/πk_BT_c)  （对数因子，均 > 0）
```

**本征值分析：**

| 情形 | 条件 | 解 |
|------|------|-----|
| **s++ 稳定** | V₁₂ > 0（带间吸引，声子主导）| Δ₁ 和 Δ₂ 同号 |
| **s± 稳定** | V₁₂ < 0（带间排斥，AFM 涨落）| Δ₁ 和 Δ₂ 异号 |
| 退化情形 | V₁₁ ≈ V₂₂ ≈ 0，|V₁₂| 大 | s± 临界，Tc 由 V₁₂ 决定 |

**AFM 涨落对 V₁₂ 的贡献：**

$$V_{12}^{\rm spin} \approx -\frac{3}{2}g^2 \chi_s(\mathbf{Q}) < 0 \quad (\chi_s(\mathbf{Q}) > 0)$$

ASCII: V₁₂^spin ≈ -3g²χ_s(Q)/2 < 0（自旋涨落给出带间排斥）→ 铁基 s±

**声子对 V₁₂ 的贡献：**
$$V_{12}^{\rm ph} > 0 \quad \text{（声子通常给出带间吸引）→ s++}$$

**总相互作用决定配对符号：**
$$V_{12}^{\rm total} = V_{12}^{\rm spin} + V_{12}^{\rm ph}$$

若自旋涨落主导（铁基）：V₁₂ < 0 → s±
若声子主导（MgB₂ 类）：V₁₂ > 0 → s++

### 3.4 能隙比

**[观察 — 两带模型估算]**

在各向同性近似（每带内 Δ = 常数）下，能隙比由费米面大小和相互作用强度决定：

$$\frac{\Delta_h}{\Delta_e} \sim \frac{N_e(0)}{N_h(0)} \times \frac{|V_{12}|}{|V_{11}|}$$

ASCII: Δ_h/Δ_e ～ [N_e(0)/N_h(0)] × [|V₁₂|/|V₁₁|]

**[观察]** 典型铁基：Δ_hole/Δ_electron ≈ 1-2（空穴带能隙略大于电子带）[来源待验证]

### 3.5 无节点 s± 的准粒子谱

**[定理 — 两带准粒子谱]**

每个带的准粒子能量：

$$E_{h}(\mathbf{k}) = \sqrt{\xi_h(\mathbf{k})^2 + \Delta_h^2} \geq |\Delta_h| > 0$$
$$E_{e}(\mathbf{k}) = \sqrt{\xi_e(\mathbf{k})^2 + \Delta_e^2} \geq |\Delta_e| > 0$$

ASCII:
```
E_h(k) = sqrt(ξ_h(k)² + Δ_h²) ≥ |Δ_h| > 0  ← 空穴带全能隙
E_e(k) = sqrt(ξ_e(k)² + Δ_e²) ≥ |Δ_e| > 0  ← 电子带全能隙
```

**标准无节点 s± 的物理后果：**
- 低温比热：指数激活（∝ exp(-Δ_min/k_BT)，Δ_min = min(|Δ_h|, |Δ_e|)）
- 穿透深度：指数饱和（与 s++ 相同，无法由此区分）
- London 方程：两带均有超流刚度

**DOS 示意（ASCII）：**
```
双峰 DOS（两带各自有能隙边缘 van Hove 奇异性）：

N(ω):
  |     ↑ h-band edge (|Δ_h|)   ↑ e-band edge (|Δ_e|)
  |     |                         |
  |─────┼─────────────────────────┼──────────
  0   |Δ_e|                    |Δ_h|     ω

（若 |Δ_h| > |Δ_e|：空穴带能隙较大，先出现较小的 e-band 边缘）
```

## 4. s± 的实验识别方法

### 4.1 中子散射自旋共振模（最有力证据）

**[定理 — 符号反转的中子学签名]**

**机制：** 中子非弹性散射在 Q=(π,π)（铁基的 AFM 波矢）处测量动态自旋结构因子 S(Q,ω)：

$$S(\mathbf{Q}, \omega) \propto \frac{\chi''(\mathbf{Q}, \omega)}{1 - e^{-\omega/k_BT}}$$

ASCII: S(Q,ω) ∝ χ''(Q,ω) / [1 - exp(-ω/k_BT)]

**s± 的共振峰条件：**

$$\omega_{\rm res} = |\Delta_h| + |\Delta_e| \quad \text{（两带能隙之和）}$$

ASCII: ω_res = |Δ_h| + |Δ_e|（两带加和，特征频率）

**物理来源：**
- 自旋翻转过程（Q=(π,π) 散射）将空穴带的 Cooper 对激发到电子带
- s± 中：初态 Δ_h > 0，末态 Δ_e < 0，相干因子 u_h v_e + v_h u_e ≠ 0（不相消）
- s++ 中：相干因子相消（Bogoliubov 相干因子为零）→ 无共振峰

**判断规则：**
- 在 Tc 以下出现 ω_res ≈ Δ_h + Δ_e 处的共振峰 → 支持 s±
- 无共振峰或峰强度弱 → 倾向 s++
- 共振峰强度随温度增加消失（Tc 以上）是关键验证

**典型材料（[观察]，各参考文献一致）：**
- BaFe₁.₈Co₀.₂As₂（Co 掺杂 BaFe₂As₂）：ω_res ≈ 9 meV [来源待验证]
- BaKFe₂As₂：ω_res ≈ 14 meV [来源待验证]
- LaFeAsO 系：共振峰已报道 [来源待验证，参考 Inosov 综述]

### 4.2 准粒子干涉（QPI）：非磁杂质敏感性

**[猜想 — 理论预测，Hanaguri 2010 部分验证]**

**机制（A10 关联）：** STM 在杂质附近观测到 QPI 波矢 q，由散射 q = k_f - k_i 决定：

- **s± 中，非磁杂质散射：**
  - 带内散射（同费米面，不穿越符号变化区域）：增强 → 有 QPI 强度
  - 带间散射 Q ≈ (π,π)（穿越符号反转区域）：**相干因子相消** → QPI 强度**压制**

- **s++ 中，非磁杂质散射：**
  - 带间散射：相干因子不相消 → QPI 强度**不压制**

- **对磁杂质的互补行为：**
  - 磁杂质（时间反演破缺）：s± 中带间散射增强，s++ 中带间散射压制

**判断规则：**
- 非磁杂质压制 Q=(π,π) 附近的 QPI 强度 → s±
- 非磁杂质不压制 → s++

**Hanaguri 2010（FeSe）：** 在 FeSe 中观测到与 s±（或 s++，因 FeSe 无空穴带存在争议）一致的 QPI 行为 [来源: Science 328, 474, 2010]。

### 4.3 NMR 1/T₁T：Hebel-Slichter 峰的压制

**[观察 — 一致性强]**

**s-wave（单带，C1）：** Tc 以下 1/T₁T 出现 Hebel-Slichter 相干峰（DOS 发散引起）。

**s± 中的 Hebel-Slichter 峰压制：**

相干因子在两带和中：
$$\left(\frac{1}{T_1T}\right)_{\rm s\pm} \propto (u_h v_e - v_h u_e)^2 + (u_e v_h - v_e u_h)^2 \to 0$$

ASCII: (1/T₁T)_{s±} ∝ (u_h·v_e - v_h·u_e)² → 0（相消）

两带相干因子相减（因符号相反）→ **Hebel-Slichter 峰被压制**。

**实验结果 [观察]：**
- 多种铁基 SC 的 NMR 显示 Tc 以下 1/T₁T 直接下降，无 Hebel-Slichter 峰 [来源待验证]
- 这与 d-wave 类似，但原因不同（s± 是相干因子相消，d-wave 是节点 DOS 线性）

### 4.4 Josephson 实验：π 结

**[猜想 — 理论清晰，实验困难]**

**s±-s 接触的 Josephson 效应：**
- 若将铁基（s±）与 s-wave 超导体构成约瑟夫森结
- 电流通过电子带（-Δ）一侧：π 结（相位差 π，Josephson 电流为负）
- 电流通过空穴带（+Δ）一侧：0 结（普通正结）

**实验挑战：** 如何选择性地接触特定费米面？需要高度可控的界面技术。目前实验证据有限 [来源待验证]。

**理论预言（[猜想]）：** s±-s 异质结在特定几何下应出现π结行为，这是 s± 与 s++ 的最干净区分方法。

### 4.5 穿透深度：指数饱和（与 s++ 相同，无法区分）

**[定理 — 两者相同]**

标准无节点 s± 和 s++ 的穿透深度均表现为指数饱和：

$$\frac{\Delta\lambda(T)}{\lambda(0)} \propto \sqrt{\frac{\pi\Delta_{\min}}{2k_BT}} e^{-\Delta_{\min}/k_BT}$$

ASCII: Δλ/λ(0) ∝ sqrt(πΔ_min/2k_BT) × exp(-Δ_min/k_BT)（指数饱和）

**无法区分 s± 和 s++。** 但可与 d-wave（线性 T）区分。

**例外：** 若存在"意外节点"（节点 s±），则穿透深度会出现线性 T 行为 → 这是 s± 多样性的关键判据。

## 5. 关键定量结果与典型材料

### 5.1 BaFe₂As₂ 体系（铁基原型）

**Tc：** K 掺杂 Ba₀.₆K₀.₄Fe₂As₂，Tc ≈ 38 K [来源: Rotter et al. 2008, PRL 101, 107006; DOI: 10.1103/PhysRevLett.101.107006]

**能隙（ARPES）：**
- Δ_hole ≈ 12 meV，Δ_electron ≈ 6 meV（符号相反，通过中子散射间接推断）[来源待验证，参考 Ding et al. 2008 EPL]
- 两个能隙之和 Δ_h + Δ_e ≈ 18 meV

**中子共振峰：**
- ω_res ≈ 14-15 meV（Ba₀.₆K₀.₄Fe₂As₂），与 Δ_h + Δ_e 量级一致 [来源待验证，参考 Christianson et al. 2008 Nature]
- 共振峰在 Tc 以上消失，是 s± 的强烈支持证据

**[反谄媚]** 注意：BaKFe₂As₂ 的 Δ 值在不同 ARPES 测量中有约 30% 的分散（表面态 vs 体态，光子能量依赖），上述数值 [来源待验证]。

### 5.2 LaFeAsO₁₋ₓFₓ（首个铁基）

**Tc：** = 26 K（F 掺杂，常压）[来源: Kamihara et al. 2008, Nature 453, 376; DOI: 10.1038/nature06972]

**s± 状态：** Mazin 等 2008 基于 LaFeAsO 费米面结构预言 s±，是铁基 s± 最早的理论体系 [来源: Mazin et al. 2008, PRL 101, 057003]

**[观察]** LaFeAsO 系的 SC 对杂质不敏感（比 d-wave 鲁棒），与 s± 全能隙预期一致（d-wave 的杂质敏感性更强）。

### 5.3 FeSe（最简铁基，有争议）

**Tc：** bulk = 8 K，单层 FeSe/SrTiO₃ ≈ 65 K [来源: 单层数值，He et al. 2013 等，多篇，[来源待验证]]

**费米面结构（关键反例）：**
- Bulk FeSe 的空穴带**已沉入费米面以下**（Γ 点无空穴口袋）
- 费米面只有电子口袋（M 点）和一对特殊空穴口袋（可能在 Γ 点附近但极浅）
- **这使标准"空穴-电子 s±"机制受到质疑！**

**[猜想 — 争议中]** FeSe 的配对对称性可能是：
1. 改进版 s±（利用微弱的 Γ 点附近带）
2. s++（声子或轨道涨落主导）
3. 节点 s±（带有意外节点）
4. 拓扑非平凡相（FeSe/STO 体系中的界面拓扑超导）

**QPI（Hanaguri 2010）：** 支持无节点配对，但无法区分 s± 和 s++ [来源: Science 328, 474, 2010]。

### 5.4 LiFeAs（无磁序的铁基）

**Tc：** = 18 K，无 AFM 磁有序相邻 [来源待验证，广泛引用]

**特殊性：** LiFeAs 没有 LaFeAsO 那样的邻近 AFM 序 → 自旋涨落较弱 → 理论上 s± 动机较弱。

**ARPES：** 显示多个能隙，无节点，但 s± vs s++ 解释存在争议 [来源待验证]。

**[反谄媚]** LiFeAs 是对"AFM 涨落 → s±"图像的挑战：如果 AFM 涨落弱，为何仍有 Tc=18K？可能轨道涨落（B5 向列涨落）在此体系更重要。

### 5.5 BaFe₂(As₁₋ₓPₓ)₂（P 掺杂，节点争议）

**特点：** P 替代 As，增强费米面嵌套，可以连续调控从 AFM 到 SC

**Tc：** 最高约 31 K（最优掺杂）[来源待验证]

**Reid 2012 热导率测量：** 在 P 掺杂体系中发现热导率的线性低温行为 → 暗示**节点的存在**[来源待验证，参考 Reid et al. 2012, PRL 109, 087001]

**解释争议：**
- 节点 s±（意外节点）：s± 在 As-P 置换轨道杂化改变后出现意外节点
- d-wave 混合
- 多带中的特殊节点结构

**[观察]** 这是"节点 s±"概念的关键实验动机，说明 s± 图像比最简单的两带无节点版本更复杂。

## 6. s± vs s++ vs d-wave 三角比较

| 特征 | s± | s++ | d-wave（B₁g） |
|------|----|----|--------------|
| **对称性类别** | A₁g | A₁g | B₁g（不同！）|
| **费米面内节点** | 无（标准）| 无 | 有（4 个线节点）|
| **费米面间符号** | 异号（+/-）| 同号（+/+）| 在面内变号 |
| **群论强制节点** | 否 | 否 | ✅ 是 |
| **中子共振峰** | ✅ 强（Q=(π,π)）| ❌ 弱/无 | ✅（cuprate Q=(π,π)）|
| **NMR Hebel-Slichter 峰** | ❌ 压制 | 部分保留 | ❌ 无 |
| **穿透深度 T→0** | 指数饱和（标准）| 指数饱和 | 线性 T（Hardy 1993）|
| **低温比热** | 指数（标准）| 指数 | T²（节点准粒子）|
| **对非磁杂质** | QPI 带间压制 | QPI 带间不压制 | 节点态 broadening |
| **驱动机制** | AFM 涨落（B3）| 声子或轨道涨落（B1/B5）| AFM 涨落（B3）|
| **Josephson 结** | 可出现 π 结 | 只有 0 结 | 依赖几何（0 或 π）|
| **同位素效应** | 弱（非声子主导）| 可能强（声子主导）| 弱/无（AFM 机制）|
| **主要材料** | 铁基（多数）| FeSe?，MgB₂ 类 | 铜氧化物 |
| **理论区分实验** | 中子散射共振峰（最好）| （同上，无峰）| Tsuei-Kirtley 相敏 |

**[重要提示]** s± 和 s++ 属于同一群论类别（A₁g），无法用晶格对称性区分，只能通过动力学/多费米面敏感实验区分。d-wave（B₁g）有根本性的对称性差异，可以用相敏 Josephson 实验判断。

## 7. 关键反谄媚：s± 的局限性与反例

### 7.1 FeSe 反例：无空穴费米面也有超导

**问题：** 标准 s± 需要"空穴-电子带间散射"（Q=(π,π) 嵌套），但 FeSe 的 Γ 点**无**空穴费米面（空穴带沉入费米面以下）。

**矛盾：** FeSe 仍有超导（Tc = 8 K bulk，65 K 单层），s± 的标准 Q 嵌套图像无法适用。

**[观察]** 这是对 Mazin 2008 图像的重要约束：Q=(π,π) 嵌套**不是** s± 的充分条件，s± 图像在 FeSe 中需要修改或放弃。

**可能的解决方案（均为 [猜想]）：**
1. 带间轨道涨落（B5 向列涨落）在 FeSe 中取代 AFM 涨落
2. FeSe 是 s++ 而非 s±
3. 单层 FeSe/STO 的高 Tc 来自界面声子（B1 机制），与 bulk 不同

### 7.2 节点 s±：意外节点的存在

**问题：** 标准两带模型预言无节点 s±（每带全能隙），但某些铁基（P 掺杂 BaFe₂As₂）热导率显示节点。

**解释（[猜想]）：** 铁基有 5 个 3d 轨道（t₂g: dxy, dxz, dyz；eg: dx²-y², dz²），实际上是 3-5 带模型。当带间相互作用复杂时，可以出现"意外节点"——即 A₁g 对称的能隙函数 Δ(k) 在某条费米面上碰巧为零（非对称性强制）。

**[反谄媚]** "无节点 s±"是**过度简化**。更完整的图像是：s± 配对对称性（A₁g 跨带符号变化）+ 各种节点结构（取决于材料）。

### 7.3 s++ 不可排除

**在某些铁基体系，s++ 也是可行候选：**

1. **FeSe（bulk）：** 无空穴 Fermi 面 → 标准 s± 弱，s++ 合理
2. **掺杂量较大时（远离 AFM 相）：** 自旋涨落减弱，声子/轨道涨落可能主导 → s++
3. **理论：** Kontani-Onari 轨道涨落模型预言 s++ 在某些铁基体系中稳定

**[定理]** 从 Anderson 定理推论：均匀无序不破坏 s++ 超导（配对态全对称），但对 s± 的影响取决于杂质散射是否跨越符号变化的带。

### 7.4 ARPES 能隙测量精度限制

**[观察]** ARPES 测量能隙符号**极度困难**：

- ARPES 只能测 |Δ(k)|，不能直接测符号
- 区分 Δ_hole = +12 meV 和 Δ_electron = -6 meV vs Δ_hole = +12 meV 和 Δ_electron = +6 meV：两者在 ARPES 强度上几乎相同
- 需要间接方法（中子、QPI、Josephson）来推断符号

### 7.5 多带模型的参数敏感性

**[观察]** 铁基超导有 5 个 Fe 3d 轨道贡献费米面，实际上是 4-5 带（或更多）模型。两带模型虽然抓住主要物理，但参数 V₁₂ 依赖于：

- 轨道权重（费米面上各轨道成分）
- 动量依赖的配对（不是简单的常数 V）
- 能带的轨道混合（BCS 中忽略，但铁基中 Hund 耦合（B10）导致强混合）

**结论：** 两带 s± 模型是理论简化，实际铁基的配对对称性需要 5 轨道模型 [猜想：5 轨道 RPA 可能给出更复杂的能隙结构，不是简单 ±Δ]。

## 8. 与 Paper A 的关联

- **配对态假设：** Paper A 的铁基材料部分基于 s± 配对态（空穴带 +Δ，电子带 -Δ）
- **多通道框架（B12）：** Paper A 的多通道叠加需要知道每个带的配对符号，以计算不同通道的 Josephson 耦合强度
- **s± 的 Tc 上限：** bulk 铁基 Tc 最高约 55 K（SmFeAs(O,F) 加压体系）[来源待验证]；单层 FeSe/STO 65 K 属于界面增强，机制不同
- **警告：** 如果 FeSe 类体系不是 s±，Paper A 中针对 FeSe 体系的 Josephson 耦合计算需要用 s++ 修正

=== C2 s± wave 核心公式速查 ===

[1] s± 定义（两带最简模型）：
  Δ_hole = +Δ_h（空穴带正能隙）
  Δ_electron = -Δ_e（电子带负能隙，符号相反）
  同属 A₁g 表示（晶格全对称），但 Δ_h × Δ_e < 0

[2] 两带 BCS 自洽方程：
  Δᵢ = Σⱼ Vᵢⱼ × Σ_k Δⱼ/(2Eⱼ_k) × tanh(Eⱼ_k/2k_BT)
  Eⱼ_k = sqrt(ξⱼ_k² + Δⱼ²)

[3] s± 条件（V₁₂ 符号）：
  V₁₂^spin ≈ -3g²χ_s(Q)/2 < 0  （AFM 涨落 → 带间排斥）→ s±
  V₁₂^ph > 0（声子）→ s++
  判断：若自旋涨落主导，V₁₂ < 0 → s±

[4] 中子散射共振峰（s± 最有力证据）：
  ω_res = |Δ_h| + |Δ_e|（两带能隙之和）
  s±：共振峰强（相干因子不相消）
  s++：无峰（相干因子相消）

[5] 准粒子谱（无节点 s±）：
  E_h(k) = sqrt(ξ_h² + Δ_h²) ≥ |Δ_h| > 0（全能隙）
  E_e(k) = sqrt(ξ_e² + Δ_e²) ≥ |Δ_e| > 0（全能隙）
  低温比热：C ∝ exp(-Δ_min/k_BT)（指数激活，与 s++ 相同）

[6] 穿透深度（s± 和 s++ 无法区分）：
  Δλ/λ(0) ∝ exp(-Δ_min/k_BT)（指数饱和）
  d-wave：Δλ/λ(0) ∝ T（线性 T）← 可以排除两者

=== s± 实验判据优先级 ===
  1. 中子散射共振峰 ω_res = Δ_h + Δ_e（最有力）
  2. QPI 非磁杂质压制 Q=(π,π) 散射
  3. NMR Hebel-Slichter 峰压制（与 d-wave 相似效果，但原因不同）
  4. Josephson π 结（理论清晰，实验困难）
  5. 穿透深度（只能排除 d-wave，无法区分 s± vs s++）

=== 关键材料速查 ===
  LaFeAsO₁₋ₓFₓ：Tc = 26K，s± 首发理论体系 [Kamihara 2008]
  Ba₀.₆K₀.₄Fe₂As₂：Tc ≈ 38K，中子共振峰 ω_res ≈ 14-15 meV [Rotter 2008]
  FeSe（bulk）：Tc = 8K，无空穴费米面，s± 争议 [Hanaguri 2010]
  BaFe₂(As,P)₂：节点 s±（热导率节点）[Reid 2012]
  La₃Ni₂O₇：Tc ≈ 80K @ 20GPa，s± 候选（2024-2025 争议）

=== 反谄媚核心点 ===
  ✗ 标准 s± 在 FeSe 中面临费米面拓扑困难（无空穴带）
  ✗ 节点 s± 存在（P 掺杂 BaFe₂As₂），两带无节点模型过简
  ✗ s++ 不可完全排除（轨道涨落可能在某些体系主导）
  ✗ ARPES 无法直接测 Δ 符号，只能间接推断
  ✗ 两带模型是过简化，铁基实际有 3-5 带
