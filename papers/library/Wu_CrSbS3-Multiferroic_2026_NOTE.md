# New High-Tc Charge-Transfer Multiferroicity in the Quasi-2D Antiferromagnet CrSbS₃

- **Authors:** Hung-Cheng Wu, Yu-Hsin Liao, Chih-Chung Tsai et al. (台湾国立中山大学)
- **Journal:** Advanced Science (Wiley), Open Access CC-BY 4.0
- **DOI:** 10.1002/advs.202523579
- **Published:** 2026-04-02

## 核心结论

CrSbS₃ 是罕见的准 2D 高温（~90K）Type-II 多铁材料，具有两种独立极化机制。

## 晶体与磁结构

- 正交 Pnma (No.62), a=8.7537Å, b=3.6551Å, c=13.0041Å
- Cr 子晶格：锯齿链形成 ab 面准 2D 框架
- C-type AFM，自旋沿 a 轴，磁空间群 **Pnm'a**（非中心对称）
- T_N ≈ 90K, β = 0.23(2)（准 2D AFM 范围）
- μ_eff = 3.84 μ_B/Cr（与 Cr³⁺ 自旋-only 值 3.87 吻合）

## 双极化机制

**P₁（零场自发极化）:** Cr 部分电荷转移驱动
- 证据：BVS 异常 + Cr-S 键长拐点 + XAS L₂边变化 + SHG增强
- SHG 强度与 |P₁|² 完美追踪

**P₂（磁场诱导极化）:** Pnm'a 磁对称性破缺反演
- 随场增大（1→7T），7T 下 ~0.8-1.2 μC/m²
- 线性磁电效应

## 自旋-晶格耦合

- c 轴 NTE / a 轴 PTE / b 轴 ZTE — 高度各向异性
- NTE 机制：J_AFM (直接 Cr-Cr) vs J_FM (90° Cr-S-Cr) 竞争
- Raman 四个模式均在 T_E 以下反常软化 → 协同自旋-声子耦合

## 实验手段（6种交叉验证）

同步辐射XRD (KEK PF BL-8B) | 中子衍射 (ANSTO Echidna) | Cr L-edge XAS (NSRRC) | Raman (532nm) | SHG (800nm Ti:sapphire) | 介电/热释电

## 与超导关联

- 同构体 CrSbSe₃：高压下超导，S→Se 翻转磁序(AFM→FM)和维度(2D→1D)
- MPnCh₃ 家族对晶格调控极其敏感
- 电荷转移 + AFM + 准2D + 自旋-晶格耦合 = 超导前驱态常见配方
