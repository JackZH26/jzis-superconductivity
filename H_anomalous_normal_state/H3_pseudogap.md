## 1. 物理定义与核心思想

### 1.1 赝能隙的基本定义 [观察，实验现象]

```
=== 赝能隙的定义：费米面的部分关闭 ===

核心现象 [观察]：
  在铜氧化物欠掺杂区，当温度 Tc < T < T* 时，
  电子谱（态密度）在费米面的某些方向出现部分抑制。
  这不是超导能隙（全向），而是有 k 空间方向选择性的"赝"能隙。

"赝"的含义：
  → 不是真正的（全能隙）能隙：只关闭了布里渊区（BZ）某些方向
  → 不是零温序参量：随温度升高至 T* 逐渐消失，而非突变消失
  → 不是完全关闭：仍有有限态密度，只是部分压低

两个关键温标：
  T*（赝能隙温度）：赝能隙开始出现（从高温冷却）的特征温度
                    在不同实验（NMR / ARPES / 比热 / 光导率）中的 T* 定义不同！
  Tc（超导转变温度）：相位相干建立、超导序参量出现
  关系：欠掺杂区  T* >> Tc（赝能隙区远宽于超导区）
         最优掺杂  T* ≈ Tc（赝能隙区消失或与 Tc 重合）
         过掺杂   T* < Tc 或完全消失

空间位置（k 空间）[观察，ARPES 直接测量]：
  → 反节点方向（(π,0) 和 (0,π)）：赝能隙最强，态密度被压低最显著
  → 节点方向（沿 (π,π)/√2 对角线）：保留金属性，形成 Fermi arc（→ H4 专节）
  → BZ 对称性：赝能隙保持 d-wave（B₁g）对称——与超导能隙 d_{x²-y²} 相同

赝能隙 vs 超导能隙的区别 [观察]：
  维度       赝能隙              超导能隙
  ---        ------              ------
  k 空间     部分（反节点附近）   全向（但 d-wave 有节线）
  T 演化     T* 处逐渐消失        Tc 处相变（突变）
  相位相干   无（正常态）         有（相位锁定）
  序参量     无（或未知序参量）   |Δ| > 0（配对振幅）
  磁场响应   弱（非迈斯纳）       强（Meissner 效应）
```

### 1.2 T* 相图 [观察，实验确认]

```
=== 铜氧化物相图中赝能隙线的位置 ===

标准铜氧化物相图（欠掺杂到过掺杂，以空穴浓度 p 为横轴）[观察]：

       T
       |  ×  ×  T*（赝能隙温度线）
       |     ×  ×  ×
       |        × ×  ×
T*    _|          × ×   ×_____T*消失（p* ≈ 0.19）
       |           ╲
       |    SC      ╲  Fermi 液体
Tc_max_|    phase    ╲____
       |   ╭────────╮     ╲
       |   │  Tc    │      ╲___
       |───┴─────────┴─────────────→ p（空穴浓度）
       0  0.05  0.10  0.15  0.19  0.25

关键区间：
  欠掺杂（p < 0.19）：T* >> Tc，赝能隙区很宽（Mott 绝缘体近邻）
  最优掺杂（p ≈ 0.16）：Tc 最高（Tc_max），T* ≈ Tc 或略高
  p*（≈ 0.19，量级估算 [来源待验证]）：赝能隙线的终点（T* → 0）
  过掺杂（p > 0.19）：无赝能隙，正常 Fermi 液体行为

关键数值（⚠️ 全部 [来源待验证]，实验方法依赖，见第 9 节反谄媚）：
  LSCO（La₂-xSrxCuO₄）：T*(p=0.10) ≈ 250-300 K，Tc_max ≈ 38 K
  YBCO（YBa₂Cu₃O₇-δ）：T*(p=0.10) ≈ 150-200 K，Tc_max ≈ 93 K
  Bi2212（BSCCO）：T*(p=0.10) ≈ 150-200 K，Tc_max ≈ 90 K
  ⚠️ 这些 T* 值依赖测量方法（NMR/ARPES/比热给出的 T* 可差 50-100K！）

T* 和 Tc 掺杂演化的关键对立 [观察，重要反谄媚见第 9 节]：
  → T*：随 p 增大单调递减（欠掺杂越深 → T* 越高）
  → Tc：随 p 先升后降，呈钟形（有最优掺杂 p ≈ 0.16 处的 Tc_max）
  → 两者掌杂演化方向在欠掺杂区相反 → 这是反对"赝能隙=超导前驱"最强的实验约束之一
```

### 1.3 赝能隙的实验签名 [观察]

```
=== 四大主要实验签名 ===

【1】ARPES（角分辨光电子谱）[观察，最直接]
  → 反节点方向 (π,0)：在 T < T* 下观测到能量谱的能隙（谱权重压低）
  → 节点方向 (π,π)/√2：保留金属性（有限 DOS），形成 Fermi arc（H4）
  → d-wave 对称性：能隙大小 ~ |cos k_x - cos k_y|（与超导能隙相同形式）[观察]
  → 典型能隙大小：Δ_PG ≈ 30-50 meV（欠掺杂 BSCCO）[来源待验证]
  → ⚠️ ARPES 测量表面敏感，体态信息有限

【2】NMR/Knight shift [观察，最早证据，1989-1991]
  → Knight shift K_s（自旋部分）正比于费米面处的电子态密度（Pauli 磁化率）
  → T < T* 下：K_s 降低 → 说明 DOS(E_F) 被压低 → 赝能隙存在
  → 完整的超导能隙：T < Tc 下 K_s 快速下降到 0（singlet 配对）
  → 赝能隙：T*以下 K_s 提前开始降低（正常态中态密度部分降低）
  → 这是赝能隙最早的实验证据来源 [来源待验证，Warren/Walstedt/Alloul 等]

【3】比热 [观察，弱信号]
  → T* 处有弱的比热异常（不是 Tc 处超导转变那样清晰的 jump）
  → 具体表现：C/T vs T 曲线在 T* 附近有弱的斜率变化
  → 信号较弱，在铜氧化物中不是最好的赝能隙判据
  → ⚠️ 不同测量组报道 T* 的比热信号差异大，一致性较差

【4】光导率 σ₁(ω) [观察]
  → Drude 峰变窄（低频部分谱权重减小）：准粒子部分"消失"进入赝能隙
  → T < T* 下高频谱权重（inter-band 或 mid-infrared 峰）部分转移
  → 谱权重总和（f-sum rule）守恒，但分布重排 → 赝能隙的光导率信号
  → ⚠️ 光导率无法直接给出 k 空间分辨信息，与 ARPES 互补

附加实验签名：
  → Nernst 效应（热电）：Tc 以上出现 Nernst 信号（可能是预形成对涡旋，[猜想，争议]）
  → STM：局域谱在不同位置显示赝能隙的空间不均匀性 [Wise 2008 et al.，来源待验证]
  → 输运：霍尔系数在 T* 附近的异常（与 Strange Metal 区的输运重叠，→ H1）
```

## 2. 历史关键节点

```
=== 赝能隙研究的历史里程碑 ===

【1986-1988】Bednorz-Müller 发现铜氧化物 SC + 早期相图测量
  → LSCO / YBCO 欠掺杂区正常态电阻率异常引起注意
  → 第一批 NMR 实验开始

【1989-1991】NMR Knight shift 异常 [来源待验证]
  → Warren 1989 / Alloul 1989 / Walstedt 1991 等 [来源待验证]
  → YBCO 中 Knight shift 在 T < T*（约 150-200 K）开始下降（T > Tc 时）
  → 解读：铜氧化物正常态中已有部分 DOS 被压低 → 赝能隙最早证据
  → 当时尚未命名"赝能隙"，解读有争议

【1995-1998】ARPES 直接观测 [来源待验证，Loeser 1996, Ding 1996]
  → Loeser et al. (1996) Science：BSCCO 欠掺杂样品，反节点处谱权重压低 [来源待验证]
  → Ding et al. (1996)：BSCCO 中赝能隙能谱 k 空间分布的直接测量 [来源待验证]
  → 确认：反节点能隙 + 节点处 Fermi arc 的 k 空间结构
  → 这两篇工作奠定了赝能隙的 ARPES 基础

【1995-1996】相位涨落模型 [来源待验证]
  → Emery & Kivelson (1995) Nature：预形成 Cooper 对 + 相位涨落
  → 论点：Tc 不是配对温度，而是相位相干温度（Berezinskii-Kosterlitz-Thouless 类比）
  → Emery-Kivelson 模型是假说 A（预形成对）的重要理论支柱

【1996】轨道序（Circulating current order）模型 [来源待验证]
  → Varma (1997/1999 等) [来源待验证]：时间反演对称破缺的轨道环电流序
  → 是假说 B（竞争序）的具体候选之一
  → 预言：在 T* 处时间反演对称破缺

【2000-2010】相图精细研究与赝能隙普适性
  → 系统测量 T*(p) 相线，建立相图
  → Nernst 效应测量（Wang, Ong 等）：Tc 以上 Nernst 信号 [来源待验证]
  → 解读争议：涡旋（预形成对）vs 非 SC 的 Nernst 来源

【2006】中子散射报告时间反演破缺 [来源待验证，争议]
  → Fauqué et al. (2006) PRL [来源待验证]：YBCO 中子散射在 T* 处发现磁弹峰
  → 解读为轨道磁序（Varma 模型的支持证据）
  → ⚠️ 仍有严重争议：独立重复困难，信号可能来自磁性杂质

【2008】STM 空间成像赝能隙 [来源待验证]
  → Wise et al. (2008) [来源待验证]：Ca₂-xNaxCuO₂Cl₂（CNCOC）中 STM 局域 DOS 测量
  → 发现：赝能隙幅度在空间上不均匀，与 CDW/SDW 空间调制相关
  → 意义：赝能隙不是均匀的 k 空间现象，有真实空间的不均匀性

【2012-2014】高场量子振荡 [来源待验证]
  → 强磁场下 YBCO 的量子振荡（de Haas-van Alphen）
  → 发现小 Fermi 口袋（pocket），与 CDW 重构的 Fermi 面一致
  → 赝能隙可能与 CDW 重构的 Fermi 面有关（支持假说 B 的间接证据）

【2014-2022】CDW 与赝能隙的关联研究热潮 [来源待验证]
  → X 射线衍射：铜氧化物欠掺杂区 CDW 普遍存在（多种铜氧化物）
  → CDW 波矢 Q ≈ (0.25-0.33) a* 在 YBCO / BSCCO / HBCCO 等普遍发现
  → 赝能隙与 CDW 共存 → CDW 是赝能隙原因还是结果？[争议，猜想]

【2016】PDW 的 Josephson STM 证据 [来源待验证，→ C8]
  → Hamidian et al. (2016) Nature [来源待验证]：Bi₂Sr₂CaCu₂O₈ 中 Josephson STM
  → 发现有限波矢的配对幅度调制 → PDW 的直接证据
  → PDW 是赝能隙的竞争序候选（C8 关联）

【2020s 至今】赝能隙与量子临界点（QCP）
  → p* ≈ 0.19 处的 QCP 假说获得更多实验关注 [来源待验证]
  → Badoux et al. (2016) Nature [来源待验证]：LSCO / YBCO 中在 p* 处霍尔系数突变
  → 输运测量暗示 p* 是相变点（费米面拓扑转变？QCP？）
  → 争议：QCP 在 Tc 穹顶之下（隐藏），直接验证困难
```

## 3. 三大假说的详细分析

### 3.1 假说 A：预形成 Cooper 对（Precursor Pairing）[猜想，Anderson / Kivelson / Emery]

```
=== 假说 A：赝能隙 = 超导前驱（预形成配对态）===

核心论点 [猜想]：
  → 在 T < T* 时，铜氧化物中已形成 Cooper 对（配对振幅 |Δ| > 0）
  → 但 T*  > T > Tc 时，各 Cooper 对之间的相位 θ 没有建立长程相干（相位无序）
  → Tc 是相位相干建立的温度（而非配对开始的温度）
  → 两阶段超导：T*（配对形成）→ Tc（相位相干）

理论基础 [猜想]：
  → Anderson 1987 RVB（B9）：共振价键态中自旋单态已经配对，
    离超导只缺乏相干性
  → Emery-Kivelson 1995（相位涨落）：Tc 由相位刚度（ρ_s）决定，
    而非由配对能量尺度（|Δ|~k_B T*）决定
  → 在 2D 类体系中，相位刚度可以远小于配对能量
    → T_c ~ ρ_s / (m* × 基本常数)（Kosterlitz-Thouless 式的相干温度）[猜想]

支持证据：

  [1] k 空间对称性一致 [观察]：
      → 赝能隙的 d-wave 对称（|cos k_x - cos k_y|）
         与超导能隙的 d_{x²-y²} 对称性相同（C3 节）
      → 这是最天然支持"同一配对机制"（预形成对）的实验证据
      → 假说 B（竞争序）中 CDW 也会在反节点打开能隙，
         但 CDW 的对称性应该是格子平移对称性破缺，不是 d-wave 超导的形式

  [2] Nernst 效应（有限争议） [来源待验证，猜想]：
      → T > Tc 时观测到非零 Nernst 信号（铜氧化物欠掺杂区）
      → 可能解读：涡旋（vortex）在 Tc 以上仍存在 → 预形成对的热激发涡旋
      → ⚠️ 反对意见：正常态中的准粒子也可以产生 Nernst 信号（不唯一）[来源待验证]

反对证据（关键！）：

  [1] T* 和 Tc 掌杂演化方向相反 [观察，核心反对]:
      → T*：随 p 增大，T* 单调减小（欠掺杂越深 → T* 越高）
      → Tc：随 p 先升后降（最优掌杂 p ≈ 0.16 处的 Tc_max）
      → 预形成对假说隐含：T* 越高 → 配对越强 → Tc 应该越高
      → 实际上：深度欠掺杂的 T* 最高，但 Tc 最低！
      → 这是假说 A 最难回答的实验约束。Emery-Kivelson 用相位刚度（ρ_s 随欠掌杂降低）
         解释此矛盾：欠掺杂区 ρ_s 太低 → Tc 低（即便配对强）[猜想，仍有争议]

  [2] 时间反演对称破缺 [观察，争议]：
      → 中子散射（Fauqué 2006 等）报道 T* 处时间反演对称破缺 [来源待验证，争议]
      → 预形成 Cooper 对是单态（singlet）配对 → 不应破坏 TRS
      → 如果 TRS 真的在 T* 处破缺 → 赝能隙是竞争序参量（假说 B），不是预形成对
      → ⚠️ 此实验结果独立重复存在困难，争议巨大

层级：假说 A 全部为 [猜想]，有支持证据和反对证据，无确证
```

### 3.2 假说 B：竞争序参量（Competing Order）[猜想，多种竞争序]

```
=== 假说 B：赝能隙 = 独立于超导的竞争序参量 ===

核心论点 [猜想]：
  → T < T* 时，铜氧化物中出现独立于超导的新序参量
  → 此序参量压制正常态的费米面态密度（尤其在反节点方向）→ 产生赝能隙
  → 序参量与超导竞争：欠掺杂区→竞争序主导（赝能隙，低 Tc）；
                       最优掺杂区→竞争序消退（SC 获胜，高 Tc）
  → T*是竞争序的序参量温度（二级相变或交叉）

三大竞争序候选：

【B1】CDW（电荷密度波）[猜想→部分观察]：
  来源：铜氧化物正常态费米面嵌套 → 不稳定性 → 电荷调制
  波矢：Q ≈ (0.25-0.33) a* [观察，X 射线直接测量]
  关联：高场量子振荡 → Fermi 面口袋（与 CDW 重构一致）
  问题：CDW 波矢（面内方向调制）≠ 赝能隙集中的 (π,0) 反节点
         → k 空间位置不完全吻合（CDW 是局域的，赝能隙是弥散的）

【B2】PDW（对密度波，C8 关联）[猜想，直接观测困难]：
  来源：有限波矢 q 的 Cooper 对凝聚（空间调制超导序参量）
  与赝能隙：PDW 可以在反节点打开能隙，同时保持节点附近的金属性
  关键实验：Hamidian 2016 Josephson STM（C8）[来源待验证]
  问题：PDW 是超导序参量的变体，严格说是"配对序参量"，
         属于假说 A 和 B 的中间地带

【B3】轨道序（Circulating current / Varma 序）[猜想，实验争议大]：
  来源：Varma 模型（1997 等）：铜氧面内 Cu-O 轨道环电流
  预言：T* 处时间反演对称性自发破缺（轨道磁矩），无净电荷/自旋序
  支持：中子散射信号（Fauqué 2006）[来源待验证，仍争议]
  问题：其他组独立重复中子散射结果困难；STM/µSR 没有明确看到 TRS 破缺迹象
         → 此模型实验证据基础最弱，但预言最有区分度

支持假说 B 的整体论点：
  → CDW 在铜氧化物欠掌杂区普遍存在（X 射线直接观测，[观察]）
```

### 3.3 假说 C：Mottness 起源（Strong Correlation）[猜想，B14 关联]

```
=== 假说 C：赝能隙 = 强关联/Mottness 残留效应 ===

核心论点 [猜想]：
  → 铜氧化物是 Mott 绝缘体（n=1，单占据）的掺杂产物
  → Mottness 残余（强关联效应）在 T < T* 时产生谱权重重分配

  → 在 Mott 绝缘体（半填充 Hubbard 模型）中：
    DOS 在费米能级处分裂为上 Hubbard 带（UHB）和下 Hubbard 带（LHB），
    中间形成 Mott 能隙

理论依据 [猜想]：
  → t-J 模型的精确对角化：反节点处谱权重在有限掺杂时被强烈压低 [来源待验证]
  → Cluster DMFT 计算：在欠掺杂区出现赝能隙结构（k 依赖的自能虚部）[来源待验证]

  → 不需要假设新序参量（假说 B）
  → 与 Mott 相图（B14）天然一致：欠掌杂越深 → Mottness 越强 → 赝能隙越大 → T* 越高

弱点：
  → 纯 Mottness 机制很难解释赝能隙的锐利 k 空间结构（ARPES 看到相对清晰的边界）
  → 如何从 Mottness（连续变化的强关联效应）产生 T*（相对明确的特征温度）？[猜想]

  → 假说 C 中没有新序参量，但假说 B 中的竞争序可能由 Mottness 触发
```

## 4. 赝能隙的 k 空间结构（关键实验结果）

```
=== 赝能隙的 k 空间分辨图像（ARPES 直接测量）===

布里渊区内的分布 [观察，ARPES]：

  k_y
   │          × × × ×（反节点 (π,0)：赝能隙最大）
   │         × × × × ×
   │        ×  ___  ×
   │       × /     \ ×（节点方向：Fermi arc，无能隙）
   │      ×│Fermi   │×
 ─ ─ ─ ─ ─│  arc   │─ ─ → k_x
   │      ×│        │×
   │       × \_____/×
   │        × × × × ×（反节点 (0,π)：赝能隙最大）
   │         × × × ×

关键特征：
  1. 反节点能隙 [观察]：
     → (π,0) 和 (0,π) 附近：明显的谱权重压低
     → 能隙幅度：Δ_PG ≈ 20-60 meV（依掺杂和材料，[来源待验证]）
     → 随欠掌杂增深：Δ_PG 增大（更强赝能隙）

  2. Fermi arc [观察，→ H4 专节]：
     → 节点方向（(π,π)/√2 对角线附近）：保留金属性
     → 可见"弧形"费米面片段，而非闭合费米面
     → Fermi arc 长度 vs 温度：
       T → T*（从低温升高）：Fermi arc 逐渐变长
       T = T*：Fermi arc 连接成完整的 Fermi 面（正常金属）

  3. d-wave 对称性 [观察]：
     → 赝能隙大小 Δ_PG(k) ~ |cos k_x a - cos k_y a|（d-wave 形式）
     → 在节点 ((π/2, π/2) 方向)：Δ_PG = 0（与超导能隙形式相同）
     → 在反节点 ((π, 0) 方向)：Δ_PG 最大
     → 这一对称性支持假说 A（赝能隙与超导能隙同源）
     → 但：CDW 也优先在反节点打开能隙（因为 van Hove 奇点），支持假说 B 的部分论点

  4. 节点处的弧长和 Tc 的关联 [观察，猜想]：
     → Fermi arc 长度 L_arc（角度量化）：L_arc(T→Tc) → 0（Tc 处几乎收缩到点）
     → Fermi arc 长度 L_arc(T→T*) → 完整 Fermi 面
     → 猜想：Fermi arc 长度正比于 Tc / T*（[猜想，来源待验证]）
     → ⚠️ 这是近似关系，不同材料和测量结果有偏差

Fermi arc 的两种解读 [猜想，竞争中]：
  → 解读 1（预形成对，假说 A）：
    Fermi arc 是 d-wave 超导的节点区（Δ=0 的方向），赝能隙区是配对态（Δ≠0）
  → 解读 2（竞争序，假说 B）：
    Fermi arc 是 CDW/PDW 重构后剩余的 Fermi 面片段
  → ⚠️ 两种解读在 ARPES 上的谱函数形式有细微差异，目前实验精度难以判决
```

## 5. 赝能隙与 CDW 的关系（热点争议）

```
=== 赝能隙 + CDW：共存、竞争还是因果？===

实验事实（共存，[观察]）：
  → 铜氧化物欠掌杂区（p ≲ 0.19）：
    - 赝能隙（T* 以下，ARPES/NMR 可见）
    - CDW（波矢 Q ≈ (0.25-0.33) a*，X 射线衍射可见，强场下增强）
  → 两者在相图中高度重叠，共存区域广泛 [观察]
  → CDW 通常在 T_CDW < T* 处出现（CDW 比赝能隙"晚出现"）

三种可能的关系（均为 [猜想]）：

  [关系 A]：CDW 是赝能隙的原因
    → T_CDW 以下，CDW 重构费米面 → 反节点处能隙 → 赝能隙
    → 问题：T_CDW 往往 << T*，但赝能隙从 T* 就开始了
    → CDW 充当赝能隙的机制只能解释 T_CDW < T < T* 区间的部分

  [关系 B]：赝能隙先出现，CDW 是赝能隙区的次生序
    → T* 以下，Fermi 面部分消失（赝能隙）→ 剩余 Fermi 面片段不稳定 → CDW 涨落
    → CDW 作为赝能隙区的"二次相变"，不是原因而是结果
    → 这与 T_CDW < T* 的观测一致

  [关系 C]：两者来自共同起源（Mottness / 强关联）
    → 强关联产生多种有序趋势，CDW 和赝能隙是同一微观机制的不同表现
    → 竞争：在某些掺杂区 CDW 更稳定，在另一些区域赝能隙（或 SC）更稳定
    → 这是最"无特征"的回答，但可能最接近真实

强场实验约束 [观察，来源待验证]：
  → 强磁场（H ≫ Hc₂）压制 Tc，但不能压制赝能隙本身 → 赝能隙 ≠ 超导
  → 强磁场增强 CDW（量子振荡条件）→ CDW 与 SC 竞争（场抑制 SC → CDW 增强）
  → 但压力实验：在 Hg-1223 等体系，压力可以抑制 CDW 并提升 Tc [来源待验证]
    → 支持 CDW 与 SC 竞争的论点

结论层级：
  [观察]：赝能隙和 CDW 在铜氧化物欠掌杂区共存
  [观察]：T_CDW < T*（CDW 在赝能隙区域内出现，晚于 T*）
  [猜想]：CDW 是赝能隙的原因、结果或共同起源之一 — 三者均有支持证据
  [猜想]：压力/场抑制 CDW 后 Tc 升高（CDW 与 SC 竞争） — 因果关系待确认
```

## 8. 关键反谄媚（Anti-Sycophancy）

```
⚠️ 反谄媚 H3.1：赝能隙 ≠ 超导前驱（最重要！）

  流行表述："赝能隙区是超导的孵化器，T* 标志着 Cooper 对开始形成"
  实际层级：[猜想，有重要反对证据]

  → T* 和 Tc 的掌杂演化方向相反！

    - 如果赝能隙 = 超导前驱，更强的配对（更高 T*）应该给出更高 Tc

    → 这一辩护是合理的 [猜想]，但不能证明预形成对一定是正确答案

  → "假说 A（预形成对）是三大假说之一，有合理的理论动机，但也有重要的反对证据" [猜想]

  → NMR（Knight shift 下降）：给出特定的 T*_NMR

⚠️ 反谄媚 H3.3：CDW 和赝能隙的因果关系未定

  → 相反方向：赝能隙区域可能"催生"了 CDW 不稳定性
  → 因果关系完全不确定，目前有正反两方论据

⚠️ 反谄媚 H3.4：赝能隙问题 30 年未解 — 不是缺乏数据

  常见误解："赝能隙问题只要更精确的实验就能解决"

  已有极好的实验数据：
  → 高分辨 ARPES（μeV 能量分辨）直接成像 k 空间结构
  → 单晶中子散射（磁性 / 声子 / 电荷，动量空间全谱）
```
