## 第二批的角色定位

I–N 不是核心物理课题，而是研究的**辅助/参考层**：

```
I 实验技术  → 怎么做实验
J 理论工具  → 怎么做计算
K 实用应用  → 工程价值
L 历史里程碑 → 时间脉络
M 元方法论   → 流程与工具链
N 开放问题   → 未解之谜
```

每个维度用一次性的"主题文档"维护即可，不需要每个子课题都做精读。

## 暂缓的具体含义

- I 实验技术 → 在精读论文时自然学到（每篇笔记都涉及具体技术）

- M 方法 → 已建立流程
