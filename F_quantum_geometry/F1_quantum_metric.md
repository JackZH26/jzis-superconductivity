## 1. 物理定义与核心思想

### 1.1 量子几何张量（Quantum Geometric Tensor，QGT）

量子态 |u_k⟩ 在 k 空间移动 dk → 量子态发生变化。描述这种变化的核心工具是**量子几何张量（QGT）**，由 Provost 和 Vallee 于 1980 年引入 [来源待验证，J. Phys. 系列]。

```
=== 量子几何张量（QGT）的数学定义 [定理，Provost-Vallee 1980] ===

设 |u_k⟩ 为第 n 个 Bloch 能带的周期部分波函数（Bloch 波函数）。

QGT 的定义：

  LaTeX: Q_{mn}(\mathbf{k}) = \langle \partial_m u_\mathbf{k} | (1 - |u_\mathbf{k}\rangle\langle u_\mathbf{k}|) | \partial_n u_\mathbf{k} \rangle

  ASCII: Q_mn(k) = <∂_m u_k | (1 - |u_k><u_k|) | ∂_n u_k>

其中：
  ∂_m = ∂/∂k_m（对 k 空间第 m 方向的偏导数）
  投影算符 (1 - |u_k><u_k|) = 投影到 |u_k⟩ 的正交补空间
  这确保 Q_mn 是规范不变量（gauge-covariant）

QGT 的分解：

  实部 = 量子度规（对称张量）：
    LaTeX: g_{mn}(\mathbf{k}) = \text{Re}[Q_{mn}(\mathbf{k})] = \text{Re}\langle \partial_m u_\mathbf{k} | (1-|u_\mathbf{k}\rangle\langle u_\mathbf{k}|) | \partial_n u_\mathbf{k}\rangle
    ASCII: g_mn(k) = Re[Q_mn] = Re<∂_m u_k | (1 - |u_k><u_k|) | ∂_n u_k>
    性质：g_mn = g_nm（实对称），g_mn ≥ 0（半正定）

  虚部 = Berry 曲率（反对称张量）：
    LaTeX: \Omega_{mn}(\mathbf{k}) = -2\,\text{Im}[Q_{mn}(\mathbf{k})]
    ASCII: Omega_mn(k) = -2 * Im[Q_mn(k)]
    性质：Omega_mn = -Omega_nm（实反对称）

  合并写法：
    LaTeX: Q_{mn} = g_{mn} + \frac{i}{2}\Omega_{mn}
    ASCII: Q_mn = g_mn + (i/2) * Omega_mn

关键不等式 [定理，数学严格]：

  LaTeX: g_{mn} \geq \frac{1}{2}|\Omega_{mn}|
  迹不等式：
  LaTeX: \text{tr}(g) \geq \frac{1}{2}|\Omega_{xy}|
  ASCII: tr(g) >= |Omega_xy| / 2

  这是纯数学不等式，由 QGT 的半正定性直接得到。
  不等式的意义：量子度规为 Berry 曲率提供了"度量下界"。
```

### 1.2 量子度规的物理意义

```
=== 量子度规的物理解释 ===

从量子力学 Fidelity（保真度）出发：

  考虑 k 点的波函数 |u_k⟩ 与相邻 k+dk 点的波函数 |u_{k+dk}⟩
  它们之间的保真度（重叠平方）：

    |⟨u_k | u_{k+dk}⟩|² ≈ 1 - g_mn dk_m dk_n + O(dk⁴)

  解释：
  → g_mn 衡量 k 空间中量子态重叠（Fidelity）减小的速率
  → g_mn = 0：相邻 k 点的波函数完全相同（无量子变化）
  → g_mn 大：相邻 k 点的波函数变化快（量子态距离大）

几何类比：

  g_mn 是 k 空间中量子态流形（参数空间）的黎曼度规张量
  类比微分几何：ds² = g_mn dk_m dk_n 是量子态空间的"距离元素"
  → 称为 Fubini-Study 度规（Fubini 1904, Study 1905 在射影空间中的推广）[来源待验证]

与 Berry 相/Berry 曲率的对比：

  Berry 相（几何相位）：
    φ_Berry = ∮ A_m dk_m（沿 k 空间回路积分）
    A_m = i⟨u_k|∂_m u_k⟩（Berry 联络，规范依赖）

  Berry 曲率：
    Omega_mn = ∂_m A_n - ∂_n A_m（规范不变，F2/F3 的主题）

  量子度规 g_mn：
    描述量子态的"距离"（局部几何，非拓扑）
    g_mn 积分 ≠ 拓扑不变量（非量子化，不是 Chern 数那样的整数）
    但通过 Peotta-Törmä 定理连接到超导物理（见第 4 节）

简洁总结：
  Berry 曲率（Omega）= 量子态空间的"曲率"（拓扑特征）
  量子度规（g）= 量子态空间的"度量"（局部几何特征）
  两者来自同一对象（QGT），但物理意义截然不同
```

## 2. 历史关键节点

```
★ 1904/1905 — Fubini-Study 度规（射影空间几何）[来源待验证]

  背景：纯粹数学，射影空间 CP^n 上的黎曼度规
  → 量子力学中量子态 |ψ⟩ 自然生活在射影 Hilbert 空间（模和相位无物理意义）
  → Fubini-Study 度规是射影 Hilbert 空间的自然度规
  意义：量子度规的数学背景可追溯到经典微分几何

★ 1984 Berry — Berry 相和 Berry 曲率 [来源待验证，Proc. R. Soc. Lond. A 1984]

  背景：量子绝热演化中的几何相位（与自然哈密顿量演变速率无关）
  → Berry 曲率 Omega = QGT 虚部（-2 Im[Q_mn]），成为凝聚态拓扑理论的基石
  → Berry 曲率 → Chern 数 → 拓扑绝缘体（1980s-2000s 发展）
  注意：Berry 1984 侧重曲率（虚部），量子度规（实部）在固体物理中的应用相对滞后

★★ 1980 Provost & Vallee — 量子几何张量原始定义 [来源待验证，J. Phys. 系列]

  背景：纯粹量子力学，参数化量子态的几何结构
  → 明确给出 QGT = g_mn + (i/2) Omega_mn 的完整框架
  → 这是 g_mn（量子度规）正式引入量子力学的原始文献
  说明：实际上 Provost-Vallee 早于 Berry 1984，但固体物理圈子的应用晚很多

★★ 1990s — Berry 曲率与拓扑绝缘体的联系（TKNN, Haldane 等）[来源待验证]

★★★ 2015 Peotta & Törmä — 平坦带中超流权重 = 量子度规积分 [确定]
```

## 3. Peotta-Törmä 定理（核心定理）

### 3.1 定理表述

```
=== Peotta-Törmä 定理 [定理，Peotta-Törmä 2015, Nat. Commun. 6, 8944] ===

物理背景：
  平坦带（W → 0）中，Drude 公式给出超流权重：
    D_s^BCS ∝ n / m*
  但平坦带 m* → ∞（有效质量无穷大）→ D_s^BCS → 0
  问题：平坦带超导如何维持非零超流权重？

Peotta-Törmä 定理（超流权重下界）：

  LaTeX（精确形式）:
    D_s \geq \frac{e^2}{\hbar} \cdot \frac{2}{\pi} |\Delta| \int_{\text{BZ}} \text{tr}[g(\mathbf{k})] \frac{d^2k}{(2\pi)^2}

  ASCII:
    D_s >= (e^2/hbar) * (2/pi) * |Delta| * ∫_BZ tr[g(k)] d^2k/(2pi)^2

其中：
  D_s  = 超流权重（drude 权重）[单位 e²/ℏ]
  |Δ|  = 配对势（超导能隙）[eV]
  g(k) = 量子度规张量（k 点处）[无量纲，或长度²]
  tr[g(k)] = g_xx(k) + g_yy(k)（迹，对 2D 系统）
  积分范围 = 第一布里渊区（BZ）

定理的关键含义：

  1. 下界 ≠ 等号：
     D_s 可以大于右侧（BCS 贡献叠加），不能从 tr(g) 单独算出 D_s
     → 这是下界，不是等式！

  2. 平坦带中的"救援"：
     当 W → 0，m* → ∞，D_s^BCS → 0
     但如果 ∫ tr(g) d²k ≠ 0，则 D_s 下界 > 0
     → 量子度规"拯救"了平坦带超导

  3. 量子度规决定超导相干性（非 Tc 本身）：
     Tc 由 |Δ|（配对势）决定，量子度规决定超流权重 D_s
     超流权重 ↔ London 穿透深度 λ_L ↔ 刚度
     → g_mn 影响的是超导的"机械刚度"而非配对温度本身

  4. 拓扑平坦带的推论：
     如果 Chern 数 C ≠ 0，则 ∫ tr(g) d²k ≥ |C| × (BZ面积)/(2π) > 0
     → 拓扑非平凡 → 量子度规非零 → D_s > 0（有超流权重下界）

平坦带超导的能量尺度比较（定性，[猜想，数量级估算]）：

  BCS 超流权重（W>0 常规）：
    D_s^BCS ~ n e² / (m* ω_p²) ~ |Δ|² / W
    当 W → 0：D_s^BCS → 0

  量子几何超流权重（平坦带极限）：
    D_s^geom ~ (e²/ℏ) * (2/π) * |Δ| * ∫ tr(g) d²k/(2π)²
    当 W = 0，W ≠ 限制，|Δ| 有限：D_s^geom > 0（如果 g ≠ 0）

  交叉条件（粗略，[猜想，依赖具体模型]）：
    几何效应主导 ≈ 当 |Δ| >> W（配对能量大于带宽）
    → MATBG 中：W ~ 1-10 meV，|Δ| ~ 0.5-1 meV → 边界情形
    → 完美 Landau 级（W = 0）：几何效应完全主导
```

### 3.2 原始推导思路

```
=== Peotta-Törmä 定理推导框架（[定理，概念层面]）===

出发点：Kubo 公式计算超流权重

  D_s = D_s^diamagnetic + D_s^paramagnetic

  diamagnetic 项（绝对值大，正）：
    与动能算符的期望值相关（∝ ⟨T⟩）
    在平坦带中，⟨T⟩ ~ W → 0

  paramagnetic 项（负，抵消 diamagnetic）：
    与流-流关联函数（电流响应）相关
    在平坦带中，激发谱有能隙（超导 |Δ|）
    → 虚激发（虚拟跨带跃迁）贡献量子几何项

关键步骤（概念）：

  在平坦带（W → 0）中，BCS 超流权重变为：
    D_s = [与多带矩阵元相关的量子度规项] × |Δ|

  利用 Bloch 波函数正交完备性：
    ∑_{n'≠n} |⟨u_{n',k}|∂_m u_{n,k}⟩|² = g_mm^(n)(k)（带内量子度规）

  最终得到：
    D_s ≥ (e²/ℏ) × (2/π) × |Δ| × ∫ tr[g(k)] d²k/(2π)²

  [定理，严格性]：这是严格的下界（从不等式推导，非近似）

重要前提条件 [反谄媚]：
  → 定理推导假设：孤立平坦带（与其他带有能隙分离）
  → 对于"准平坦带"（有限带宽 W ≠ 0），定理需要修正
  → 对于多带情形，需要非阿贝尔推广
  → 以上细节使定理在实际材料（MATBG、Kagome）的应用更复杂
```

## 4. 量子度规 vs Berry 曲率：同源，不同义

```
=== 量子度规（g）vs Berry 曲率（Omega）的系统比较 ===

共同来源：
  两者都来自量子几何张量 Q_mn：
    Q_mn = g_mn + (i/2) Omega_mn
  同一个数学对象，不同分量。

数学性质对比：

              量子度规 g_mn          Berry 曲率 Omega_mn
─────────────────────────────────────────────────────────────
对称性       g_mn = g_nm（对称）      Omega_mn = -Omega_nm（反对称）
数学类型     实对称矩阵              实反对称矩阵
几何角色     黎曼度规（度量）         场强（曲率 2-form）
正定性       g_mn ≥ 0（半正定）       无限制（可正可负）
规范依赖性   规范不变（由 QGT 定义）  规范不变（由 dA 定义）
积分量化     tr(g) 积分不量化         Omega 积分 = 2π × Chern 数（量化！）
拓扑信息     不含拓扑量化信息         含拓扑信息（整数 Chern 数）
─────────────────────────────────────────────────────────────

物理含义对比：

              量子度规 g_mn          Berry 曲率 Omega_mn
─────────────────────────────────────────────────────────────
测量的是什么  量子态的"几何距离"      量子态空间的"弯曲程度"
与超导的关系  超流权重下界（PT定理）  Chern 数 → 拓扑超导分类
与输运的关系  超流刚度、光谱权重      反常霍尔效应（AHE）
与磁场的关系  Landau 级度规结构       轨道磁化率（Omega 积分）
实验可测性   通过超流刚度间接测量     AHE + ARPES + QO 磁场响应
─────────────────────────────────────────────────────────────

关键不等式（再次强调）[定理]：

  对于 2D 单带系统：
    LaTeX: \text{tr}(g(\mathbf{k})) \geq \frac{1}{2}|\Omega_{xy}(\mathbf{k})|
    ASCII: tr(g(k)) >= |Omega_xy(k)| / 2

  对 BZ 积分（结合 Chern 数 C）：
    LaTeX: \int_\text{BZ} \text{tr}(g(\mathbf{k})) \frac{d^2k}{(2\pi)^2} \geq \frac{|C|}{2}
    ASCII: ∫_BZ tr(g(k)) d²k/(2π)² >= |C| / 2

    注意：这里的 |C| 是 Chern 数的绝对值（整数）
    意义：Chern 数非零 → 量子度规积分有非零下界 → 超流权重有非零下界！

平坦带中的核心关联：

  平坦带 + Chern 数 C ≠ 0（拓扑平坦带）：
    → ∫ tr(g) d²k ≥ |C|/2 > 0
    → Peotta-Törmä：D_s > 0
    → 拓扑 → 几何 → 超导（三级联）

  平坦带 + Chern 数 C = 0（拓扑平凡平坦带）：
    → ∫ tr(g) d²k ≥ 0（下界不排除 g = 0）
    → 需要单独计算 g_mn 才能判断超流权重
    → 例：理想 Kagome 平坦带（C=0，但 g ≠ 0）[来源待验证]

常见混淆 [反谄媚]：
  "量子度规大 → 拓扑非平凡" ← 错误！
  g 大只意味着量子态变化快（度量性质），不等于 Chern 数非零（拓扑性质）
  反例：理想 Kagome 平坦带 C=0，但 ∫ tr(g) > 0 [来源待验证]
```

## 5. 量子度规的计算方法

```
=== 量子度规的计算方法（理论 + 实验）===

[A] 直接定义计算 [定理，定义出发]

  展开 QGT 定义：
    g_mn(k) = Re⟨∂_m u_k|∂_n u_k⟩ - Re(⟨∂_m u_k|u_k⟩ × ⟨u_k|∂_n u_k⟩)
    = Re⟨∂_m u_k|∂_n u_k⟩ - A_m(k) × A_n(k)

  其中：
    A_m(k) = i⟨u_k|∂_m u_k⟩（Berry 联络，实数）
    ⟨∂_m u_k|∂_n u_k⟩：需要计算 Bloch 波函数的 k 导数

[B] Bloch 波函数数值方法（[观察，主流计算方案]）

  步骤：
    1. 紧束缚模型或 DFT 计算 Bloch 波函数 |u_{n,k}⟩ 在 k 网格上的值
    2. 数值差分 ∂_m u_k ≈ (|u_{k+δk_m}⟩ - |u_{k-δk_m}⟩) / (2δk_m)
    3. 代入 g_mn 定义，逐 k 点计算
    4. 对 BZ 积分（数值 k 网格求和）→ ∫ tr(g) d²k

  计算代码（概念）：
    for k in BZ_grid:
        u_k = bloch_wavefunction(k, band_index)
        du_x = (bloch_wavefunction(k+dk_x) - bloch_wavefunction(k-dk_x)) / (2*dk)
        du_y = (bloch_wavefunction(k+dk_y) - bloch_wavefunction(k-dk_y)) / (2*dk)
        g_xx[k] = Re(<du_x|du_x> - |<u_k|du_x>|²)
        g_yy[k] = Re(<du_y|du_y> - |<u_k|du_y>|²)
        tr_g[k] = g_xx[k] + g_yy[k]

  典型计算量：
    MATBG 连续模型：k 网格 100×100 → 10⁴ 点，计算时间 <1h（普通服务器）[来源待验证]
    第一性原理 DFT：计算量更大，通常需要专用软件（VASP + Wannier90 等）[来源待验证]

[C] Wannier 函数方法 [来源待验证]

  量子度规与最局域 Wannier 函数的第二矩（展宽）有关：
    ∫ tr(g) d²k / (2π)² = ∑_n ⟨n,0|r²|n,0⟩ - |⟨n,0|r|n,0⟩|²（Wannier 展宽公式，近似）

  意义：
    → 量子度规积分 ≈ Wannier 函数的"空间展宽"
    → Wannier 函数越局域 → g 积分越小（量子态在 k 空间变化越慢）
    → Chern 数 C ≠ 0 的带：Wannier 函数无法完全局域化（必有 g > 0）

  工具：Wannier90 软件可直接计算 g 相关量 [来源待验证]

[D] 实验测量（间接）[观察]

  直接途径（仅 MATBG 有进展）：
    → 测量超流刚度 ρ_s（微波谐振、电感测量、THz 透射）
    → 与 Peotta-Törmä 下界比较
    → 如果 ρ_s 远大于 BCS 估算 → 量子几何贡献

  困难（[观察，实验挑战]）：
    → ρ_s = BCS 贡献 + 量子几何贡献（两者叠加难以分离）
    → 需要精确的 BCS 贡献基准（需独立测量带宽、|Δ| 等）
    → MATBG 样品本身的不均匀性增加测量难度
    → 除 MATBG 以外，Kagome 等体系的直接 g 测量极少 [来源待验证]

  备选途径（理论指导）：
    → 量子度规可影响光学吸收（光电导虚部）→ 光学测量 [来源待验证]
    → STS/STM 测量局域态密度 → 间接约束 g [来源待验证，定性关联]
```

## 6. 典型体系中的量子度规

```
=== 典型体系量子度规（从简单到复杂）===

[体系 1] Landau 级（磁场中的准粒子，完全平坦带）[来源待验证]

  物理：2D 电子气在均匀磁场 B 中 → 完全简并的 Landau 级（WLL = 0）
  拓扑：第 n 个 Landau 级的 Chern 数 C = 1（每个完整 Landau 级）
  量子度规：

    tr(g) = l_B²（均匀分布，各向同性）
    其中 l_B = √(ℏ/eB)（磁长度）

    原因：Berry 曲率 Omega = l_B²（均匀），g = Omega/2（不等式取等号！）
    → Landau 级是"不等式饱和"的特殊例子：tr(g) = |Omega_xy|/2
    → 这类带称为"理想带"（ideal band）[来源待验证]

  超流权重（PT 定理）：
    D_s ≥ (e²/ℏ) × (2/π) × |Δ| × l_B² × N_φ（N_φ = BZ 内 Landau 级简并度）[来源待验证]

[体系 2] MATBG（魔角双层石墨烯，θ ≈ 1.1°）

  物理：连续模型（Bistritzer-MacDonald）平坦带 [来源待验证]
  拓扑：
    → 无能隙（谷-自旋简并）：每个谷的平坦带 C = ±1 [来源待验证]
    → 带有时间反演 + 粒子-空穴对称性：不同参数给出 C = 0 或 C = ±1
    → 视截断方案而定 [来源待验证]

  量子度规（定性，[来源待验证]）：
    → tr(g(k)) 随 k 变化，在 VHS（范霍夫奇点）附近最大
    → ∫ tr(g) d²k / (2π)² ~ O(a_M²)（a_M = Moiré 超晶格常数 ~ 13 nm）
    → 量子几何贡献到超流权重的比例：依赖 W 和 |Δ| 的具体比值

  2025 MIT 实验（[来源待验证，B13]）：
    → 测量超流刚度 ρ_s ~ 数 μK（极小，但非零且可测）
    → 与 BCS 预测相比：量子几何贡献不可忽略（~50%？）[来源待验证，估算]
    → 这是量子度规在超导中最直接的实验证据之一

[体系 3] Kagome 平坦带（AV₃Sb₅，理想 vs 实际）

  理想 Kagome（紧束缚模型平坦带）：
    → 平坦带 C = 0（拓扑平凡）
    → 但 tr(g) > 0！（量子度规非零）[来源待验证]
    → 超流权重下界 > 0（即使 C = 0）
    → 机理：Kagome 的特殊波函数结构（caged state）导致非零 g [来源待验证]

  实际 AV₃Sb₅（CsV₃Sb₅ 等）：
    → 平坦带不在费米面（EF ~ -70 meV 以下）[D14 已述]
    → 量子度规对 AV₃Sb₅ 超导（Tc = 2.5K）的实际贡献：有限 [猜想，D14]
    → AV₃Sb₅ 超导更可能由 VHS 处常规 BCS/CDW 涨落驱动，非量子几何 [猜想]

[体系 4] 拓扑平坦带（一般情形）[定理 + 猜想]

  设 Chern 带（C = n 整数）且带宽 W → 0：
    → 不等式：∫ tr(g) d²k / (2π)² ≥ |C| / 2
    → Peotta-Törmä：D_s ≥ (e²/ℏ) × (2/π) × |Δ| × |C|/2

  这给出了一个简洁的普遍性结论 [定理]：
    拓扑 Chern 数 C ≠ 0 的平坦带中，只要有超导配对（|Δ| > 0），
    超流权重必然非零（下界由 |C| 决定），超导相干性不会被平坦带"杀死"。
```

## 8. 关键反谄媚

```
⚠️ 反谄媚 8.1：量子度规 ≠ Berry 曲率（最常见混淆！）

  错误表述：
    "量子几何大，所以 Berry 曲率大，所以系统是拓扑的"

  正确表述：
    量子度规 g_mn（实部）和 Berry 曲率 Omega_mn（虚部）来自同一 QGT，
    但物理意义完全不同：
    - g_mn：度量（局部几何），决定量子态"距离"，与超流权重有关
    - Omega_mn：曲率（拓扑），积分给出 Chern 数（拓扑不变量），与 AHE 等有关
    两者由不等式 tr(g) >= |Omega|/2 约束，但独立存在：
    - 可以有 g > 0 但 Omega = 0（C=0，如某些平凡带）
    - 可以有 Omega > 0 的同时 g 不同（比 Omega/2 大）

⚠️ 反谄媚 8.2：Peotta-Törmä 是下界，不是等号，不能算出 D_s

  错误表述：
    "量子度规积分 = 超流权重"或"D_s = (e²/ℏ)(2/π)|Δ|∫tr(g)"

  正确表述：
    Peotta-Törmä 给出的是严格下界（≥，不是 =）
    真实 D_s = D_s^BCS + D_s^geometric + （可能的多带补正项）
    量子度规积分只是下界，真实超流权重通常更大（BCS 贡献叠加）
    实验上测到的 ρ_s > PT 下界 ≠ 验证了量子度规（也可能是 BCS 贡献）

⚠️ 反谄媚 8.3：拓扑 → 量子度规 > 0，但 → 超导 ≠ 自动成立

  完整推理链（正确）：
    [定理] Chern 数 C ≠ 0 → ∫ tr(g) d²k > 0（量子度规非零）
    [定理] ∫ tr(g) d²k > 0 + |Δ| > 0 → D_s > 0（PT 下界 > 0）
    [猜想] 但 |Δ| > 0 需要吸引相互作用实现超导配对！

  错误表述：
    "平坦带有 Chern 数 → 超导"
  
  正确：
    拓扑只保证"如果有配对，超流权重非零"
    配对本身（|Δ| > 0）需要独立的吸引机制（声子、自旋涨落等，见 B 部分）

⚠️ 反谄媚 8.4：量子度规的实验直接测量极困难

  当前状态（2026-04）：
    → 量子度规本身无法直接测量（不对应任何直接可观测量）
    → 所有"测量"都是间接的：超流刚度测量 → 通过 PT 下界反推 tr(g)
    → 实际 ρ_s = BCS + 几何，分离需要独立 BCS 基准
    → 只有 MATBG 有质量较高的间接证据 [来源待验证，2025 MIT]
    → Kagome、有机超导、氢化物中：量子度规贡献的直接实验证据极少

⚠️ 反谄媚 8.5：MATBG 量子度规贡献的比例仍有争议

  背景：MATBG 平坦带带宽 W ~ 1-10 meV，|Δ| ~ 0.5-1 meV
    → W 和 |Δ| 量级相当 → 不是纯平坦带极限（W → 0）
    → PT 定理在 W > 0 时需要修正

  当前争议 [猜想，2026 前沿]：
    → 量子几何贡献：~30-70%（不同理论模型给出不同比例）[来源待验证]
    → BCS（Fermi 液体）贡献：~30-70%（与几何贡献互补）[来源待验证]
    → 两者难以从实验上精确分离
    → 任何"量子几何主导 MATBG 超导"的说法都是 [猜想]

⚠️ 反谄媚 8.6：定理适用范围（孤立带假设）

  Peotta-Törmä 定理的隐含假设：
    → 平坦带与其他带之间有能隙分离（孤立带）
    → 在 MATBG 等实际系统中，平坦带附近有其他带（Dirac 带等）
    → 带间混合（band mixing）修正 PT 定理的精确形式
    → 多带推广（non-Abelian QGT）需要额外工作 [来源待验证，Törmä 等 2022?]
```
