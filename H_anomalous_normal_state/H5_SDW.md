# H5. 自旋密度波（SDW，Spin Density Wave）

**MECE 编号：** H5
**关联 MECE：**
- B3（AFM 涨落 — SDW 涨落是铁基 SC 的驱动机制；B3 聚焦 AFM 涨落→s± 配对增强，H5 聚焦 SDW 作为正常态竞争序）
- H4（CDW — 与 SDW 的对比：H4 是电荷密度波，H5 是自旋密度波；两者同为竞争序但物理机制不同）

## 1. 物理定义与核心思想

### 1.1 SDW 的基本定义 [观察，实验确认]

```
=== 自旋密度波（SDW）的定义 ===

核心现象 [观察]：
  电子系统中出现空间周期性的自旋密度调制，保持电荷密度均匀。

SDW 序参量（空间调制形式）：
  m(r) = m_Q cos(Q·r + φ)
  其中：
    m_Q = SDW 调制振幅（序参量大小，即磁矩大小）
    Q   = SDW 波矢（与 Fermi 面嵌套波矢相关，通常 Q = 2k_F 方向）
    φ   = SDW 相位
    注：m(r) 是自旋极化密度，不是电荷 → 电荷密度 ρ(r) 保持均匀

实验标志 [观察]：
  → 中子散射（最直接）：在波矢 Q 处出现磁性超晶格衍射峰
  → µSR（µ 子自旋弛豫）：局域磁场导致 µ 子进动频率变化 → 定量测 m_Q
  → Mössbauer 谱：铁基中 Fe 位局域磁场（超精细场）探测
  → NMR：自旋晶格弛豫时间 1/T₁ 在 T_SDW 处异常
  → ARPES：SDW 能隙（自旋极化能带劈裂）在嵌套区域打开
  → 电阻率：T_SDW 处斜率变化（但与其他散射机制混淆，需结合其他方法）

SDW vs CDW（H4）vs AFM 的关键对比 [定理/观察]：
  性质           SDW                     CDW（H4）               AFM（局域矩）
  ---            ---                     ------                  ----
  序参量         m_Q（自旋调制）         ρ_Q（电荷调制）         μ_B（局域磁矩）
  调制方式       空间周期性磁矩          空间周期性电荷 + 晶格畸变  双子晶格反平行磁矩
  晶格畸变       通常无（纯电子序）      有（声子软化）           很小
  传导性         金属性 SDW（铁基）      半金属/绝缘（CDW 能隙）   通常绝缘（Mott AFM）
  起源           Fermi 面嵌套 + 交换互作 Fermi 面嵌套 + 电声耦合  Mott 局域化 + 超交换
  实验区分       中子（磁性衍射）        X 射线（电荷超晶格）      中子（局域磁矩）
```

### 1.2 SDW 的微观起源：Overhauser 理论 + Fermi 面嵌套 [定理，Overhauser 理论；推广为猜想，2D/多带]

```
=== SDW 的理论根基：Overhauser + Fermi 面嵌套 ===

【1D 极限的 SDW（[定理]，Overhauser 1962 [来源待验证]）：】

  1D 电子链中：
  → 自旋响应函数 χ_s(Q = 2k_F) 发散（与 CDW 的电荷响应平行）
  → 但 SDW 的驱动力来自交换相互作用（Stoner 极化），而非电声耦合
  → Stoner 准则：U × N(EF) > 1 → 铁磁（均匀）不稳定
    嵌套增强特定 Q 处的响应：U × χ_s(Q) > 1 → SDW（非均匀磁序）不稳定
    其中 U = 电子-电子 Hubbard 排斥（交换相互作用）

  SDW 形成后的能谱：
  → 自旋 ↑ 和 ↓ 子带在 k 和 k+Q 处混合（类似 CDW 能带折叠）
  → 在嵌套区域打开自旋极化能隙：
    ε_{k,±} = ε_k ± √(δε_k² + m_Q²)
    其中 m_Q 是 SDW 振幅（类比 CDW 的 Δ_CDW）
  → N(EF) 在嵌套区域降低 → 配对强度降低（CDW 压制 SC 的同类逻辑）

  SDW vs CDW 的本质区别（[定理]）：
    驱动力    CDW：电声耦合（声子软化）       SDW：电子-电子交换（无声子参与）
    畸变      CDW：晶格静态畸变              SDW：无晶格畸变（纯电子序）
    能隙类型  CDW：电荷能隙（↑ 和 ↓ 相同）   SDW：自旋极化能隙（↑ 和 ↓ 反向）
    对称破缺  CDW：平移对称性（电荷周期性）   SDW：时间反演（磁序） + 平移（部分）

【2D/多带推广（[猜想，铁基 SDW 的实际情况]）：】

  铁基超导（BaFe₂As₂ 等）中的 SDW：
  → 多带金属（不是 1D 简单嵌套）
  → Fermi 面拓扑：空穴口袋（Γ 点附近）+ 电子口袋（M 点附近，即 (π,0)/(0,π)）
  → 嵌套波矢：Q = (π,0)（连接 Γ 处空穴口袋和 M 处电子口袋）
  → 这个嵌套诱发条纹 AFM（stripe AFM）序：
    Fe 原子沿 a 轴反铁磁、沿 b 轴铁磁（与简单 Néel AFM 不同）
  → 铁基的 SDW 是金属中的 SDW（Fermi 面仍存在，未完全打开能隙）
    → 与铜氧化物 Mott 绝缘体 AFM 的根本区别 [猜想，关键区分]
```

### 1.3 SDW 与超导的关系（三种模式）[猜想，现象学]

```
=== SDW 与超导的三种关系（现象学框架）===

【模式一：竞争（SDW 压制 SC）[观察，铁基母化合物]】

  机制：
  → SDW 消耗 N(EF) → SC 配对资源减少（同 CDW 压制逻辑）
  → SDW 能隙覆盖部分 Fermi 面 → Cooper 对形成受阻
  → 铁基母化合物：T_SDW = 138K（BaFe₂As₂），无超导 → SDW 直接阻断 SC

  实验证据 [观察]：
  → 掺杂消除 SDW → SC 出现（BaFe₂As₂ 体系最清晰）[来源待验证，系统实验]
  → SDW 稳定时 Tc = 0（或极低）

【模式二：协同（SDW 涨落驱动 SC）[猜想，主流理论，B3 主战场]】

  机制（B3 详述，H5 仅总结）：
  → 长程 SDW 序消失后，短程 SDW 涨落（AFM 涨落）在最优掺杂附近仍存在
  → 这些涨落（波矢 Q = (π,0)）作为配对媒介 → 诱导 s± 配对（C2）
  → s± 配对：Γ 口袋 SC 能隙符号 = + ，M 口袋 SC 能隙符号 = −
  → 铁基 SC 的最高 Tc 出现在 SDW 残余涨落最强处（接近 SDW 量子临界点附近）[猜想]

  关键注意：这是 [猜想，主流但非唯一理论]
  → FeSe 反例：无长程磁序，但有超导（Tc = 8K 体材；单层 Tc ≈ 65-109K 估计 [来源待验证]）
  → 说明 SDW 涨落不是铁基 SC 的唯一机制（轨道涨落等也有贡献）[猜想]

【模式三：共存（SDW + SC 部分相图重叠）[观察，争议，G1 主战场]】

  铁基共存区域 [观察，争议]：
  → Co 掺杂 BaFe₂As₂：在欠掺杂区（x ≈ 0.04–0.06），中子散射 + µSR 测到
    SDW 信号和 SC 信号同时存在
  → 争议：微观均匀共存 vs 宏观相分离 → G1 详述，H5 仅总结位置关系
```

## 2. 历史关键节点

```
=== SDW 研究的历史里程碑 ===

【1959-1962 Overhauser 理论奠基】[来源待验证]
  → Overhauser（1960/1962）：预言金属中的 SDW 和 CDW 不稳定性 [来源待验证，精确年份/期刊待核]
  → 关键洞察：交换相互作用（而非电声耦合）可以驱动磁序
  → 起初被 Bardeen 等质疑，后来在铬中被实验证实

【~1959-1962 铬（Cr）的 SDW 发现】[来源待验证]
  → Cr 是第一个公认的金属 SDW 体系
  → T_SDW ≈ 311K（Néel 温度）[来源待验证]
  → 非公度 SDW（incommensurate）：Q ≈ (0, 0, 0.95) 2π/a（不完美嵌套）[来源待验证]
  → 确立了 SDW 作为金属中真实存在的量子态的地位

【1987 铜氧化物母化合物的 AFM（= Mott AFM，非金属 SDW）】
  → La₂CuO₄ 的 Néel 温度 TN ≈ 300K [来源待验证]
  → ⚠️ 铜氧化物 AFM = Mott 绝缘体中的局域磁矩序（超交换驱动）
    ≠ 铁基 SDW（金属中的巡游电子 SDW）
  → 两者都是磁序，但物理机制不同 [重要区分，见 §9 反谄媚]

【1995 Tranquada — LBCO x=1/8 条纹序（SDW + CDW 复合序）】[来源待验证]
  → La₁.₈₇₅Ba₀.₁₂₅CuO₄：自旋条纹（SDW 成分）+ 电荷条纹（CDW 成分）
  → Tc 严重抑制至 ≈ 4K（H4 已详述，H5 仅标注关联位置）[来源待验证]
  → 这是铜氧化物中 SDW 成分最清晰的案例之一

【2008 LaFeAsO₁₋ₓFₓ 发现（铁基 SC 元年）】[Kamihara 2008, JACS 130, 3296]
  → Kamihara et al.：F 掺杂 LaFeAsO 中发现 Tc = 26K 超导 [来源确认]
  → 立刻引发对母化合物 LaFeAsO 磁性的研究
  → 母化合物 LaFeAsO 的 T_SDW ≈ 137K [来源待验证，Kamihara 2008 相关实验]

【2008 BaFe₂As₂ — 铁基 SDW 的经典案例】[Rotter 2008, PRL 101, 107006]
  → Rotter et al.：BaFe₂As₂ 的结构和磁性 → T_SDW = 138K [确定，可靠来源]
  → 条纹 AFM（stripe AFM）：Q = (π, 0)，Fe 自旋沿 a 轴 AFM、沿 b 轴 FM
  → 同时发生四方 → 斜方结构相变（向列相变，H6 关联）
  → 无超导（常压下）

【2008-2010 铁基掺杂相图系统研究】[来源待验证，多组]
  → 电子掺杂（Co）：BaFe₂₋ₓCoₓAs₂，随 x 增大，T_SDW 降低，SC 穹出现
  → 空穴掺杂（K）：Ba₁₋ₓKₓFe₂As₂，类似相图
  → 同晶型替换（P）：BaFe₂（As₁₋ₓPₓ）₂，类质同象压力
  → 最优 Tc（BaFe₂₋ₓCoₓAs₂）：Tc ≈ 22K，x ≈ 0.07 [来源待验证]
  → 建立了铁基超导 SDW→SC 相图的完整图像

【2010-2015 FeSe 的特殊性确立】[来源待验证，多组]
  → FeSe 体材：Tc = 8K，无长程磁序（零场下）[来源待验证]
  → 单层 FeSe/SrTiO₃：Tc 大幅提升（ARPES 估计 65-100K 量级）[来源待验证，争议]
  → FeSe 确立了"铁基 SC 不必须依赖 SDW 涨落"的观察 [猜想→观察]
```

## 3. 铁基超导的 SDW（最重要案例）

### 3.1 BaFe₂As₂ 母化合物：SDW 的经典案例

```
=== BaFe₂As₂（Ba122）母化合物 ===

基本参数 [Rotter 2008, PRL 101, 107006]（确定）：
  T_SDW = 138K（SDW 转变温度，同时伴随结构相变）
  SDW 类型：条纹 AFM（stripe AFM）
  SDW 波矢：Q = (π, 0)（沿 a 轴方向的反铁磁嵌套波矢）
  自旋结构：Fe 原子沿 a 轴方向相邻为反铁磁（↑↓↑↓）
            沿 b 轴方向相邻为铁磁（↑↑↑↑）
            → "条纹序"因此得名（不是简单的 Néel 棋盘型 AFM）
  局域磁矩：≈ 0.87 µ_B / Fe 原子 [来源待验证]（远小于 Mott AFM 的饱和值）
            → 证明是巡游（itinerant）SDW，非局域磁矩序

同时发生的结构相变 [观察，来源待验证]：
  → 四方（I4/mmm）→ 斜方（Fmmm 或 Cmma）结构相变
  → T_SDW = T_structural ≈ 138K（同时发生，在 BaFe₂As₂ 中两者不分离）
  → a 轴和 b 轴产生约 0.3% 的差异（向列序，H6 关联）
  → 向列序 = 铁基 SDW 几乎必然的伴生序（但两者的先后和因果关系有争议，H6 §7）

无超导（常压）：
  → 常压下 BaFe₂As₂ 无超导（Tc = 0）
  → 正是 SDW 序阻断了 SC 配对（N(EF) 降低 + Fermi 面部分能隙化）
```

### 3.2 铁基掺杂相图 — SDW 与 SC 穹的关系

```
=== BaFe₂₋ₓCoₓAs₂ 掺杂相图（电子掺杂端）===

相图主要特征 [观察，系统实验，来源待验证]：

  x = 0 → T_SDW = 138K，Tc = 0（母化合物，纯 SDW）
  x ≈ 0.02-0.04 → T_SDW 开始下降，SC 开始出现（欠掺杂区，SDW+SC 共存）
  x ≈ 0.07-0.08 → T_SDW ≈ 0，Tc 最高（最优掺杂，SC 穹顶）[来源待验证]
  x > 0.12 → T_SDW = 0，Tc 下降（过掺杂区，纯 SC 衰减）

SDW 终点与 SC 穹顶的位置关系 [猜想，量子临界讨论]：
  → SDW 消失点（量子临界点 QCP？）≈ SC 最高 Tc 处
  → 这被解读为：SDW 量子临界涨落驱动 SC 配对（量子临界超导）[猜想]
  → ⚠️ 注意：相图吻合是[猜想]，不能直接等同于因果证明！
    → N(EF) 在 SDW 消失处恢复也可以解释 SC 升高 [猜想，替代解读]
    → 区分"SDW 涨落驱动"和"N(EF) 恢复 → Tc 升高"需要精细的分频谱实验 [猜想]

其他掺杂路径（K 空穴掺杂，P 同晶型替换）[观察，来源待验证]：
  → Ba₁₋ₓKₓFe₂As₂：相似相图，空穴端最优 Tc ≈ 38K（高于 Co 电子掺杂）[来源待验证]
  → BaFe₂（As₁₋ₓPₓ）₂：等价于化学压力，相图结构类似 [来源待验证]
  → FeAs 面是主战场：所有这些体系的 SDW 都源于 FeAs 层中 Fe 3d 轨道的 Fermi 面嵌套

LaFeAsO₁₋ₓFₓ 的相图 [观察，来源待验证]：
  → 母化合物 LaFeAsO：T_SDW ≈ 137K [来源待验证]
  → F 掺杂 → SDW 消失 → Tc 最高 ≈ 26K（优化掺杂）[来源待验证，Kamihara 2008]
  → 相图趋势与 Ba122 类似，但 Tc 相对较低
```

### 3.3 SDW 涨落 → s± 配对（铁基 SC 的配对机制核心争论）

```
=== SDW 涨落 → s± 超导：机制与证据 [猜想，主流理论] ===

基本图像（B3 详述，此处仅总结）：
  → SDW 长程序消失后：波矢 Q = (π,0) 附近的短程自旋涨落仍存在
  → 这些涨落作为"配对胶水"（pairing glue）：
    在 Γ 口袋（空穴型）和 M 口袋（电子型）之间散射 Cooper 对
    → s± 配对：两个 Fermi 口袋的 SC 能隙符号相反（s_+ 和 s_-）
  → 形式上类比铜氧化物的 AFM 涨落驱动 d 波配对 [猜想，类比]

支持 s± 配对的实验证据 [猜想→观察，强度不一]：
  → 中子散射：最优掺杂铁基中仍测到强 AFM 涨落（Q = (π,0) 处）[来源待验证]
  → 中子自旋共振（spin resonance）：SC 态中 ω_res ≈ 2Δ_SC
    能量和波矢与 s± 图像一致（Q = (π,0)）[来源待验证，多组]
  → 比较不同掺杂：AFM 涨落越强，Tc 越高（定性相关）[猜想]

关键限制与反例：
  → FeSe 体材：Tc = 8K，但无长程磁序，自旋涨落存在但较弱 [来源待验证]
    → FeSe 的配对机制：轨道涨落驱动？[猜想]还是 SDW 涨落驱动？[猜想]
  → 铁基 SC 的 s± 是"s±" 还是 "s++"（同号 s 波）？
    → 部分实验（准粒子干涉）支持 s++，部分支持 s± [猜想，争议持续]
  → SDW 涨落不是铁基 SC 的唯一机制，轨道涨落（多轨道 Hund 物理）也有贡献 [猜想]
```

## 4. 铬（Cr）和其他 SDW 体系（非超导，参考）

```
=== 铬（Cr）：SDW 物理的教科书案例 ===

基本参数 [来源待验证]：
  T_SDW ≈ 311K（Néel 转变温度）
  SDW 类型：非公度 SDW（incommensurate）
  SDW 波矢：Q ≈ (0, 0, 0.95) × 2π/a（接近但不等于 (0,0,1)，偏移约 5%）[来源待验证]
  → 非公度 = Fermi 面嵌套不完美（Cr 的嵌套波矢≠布里渊区边界）
  → 结果：SDW 产生"卫星"超结构衍射峰，而非简单倍周期结构

Cr 的学术价值：
  → 无超导（无 SC 干扰），是研究"纯净 SDW"物理的理想体系
  → Fermi 面嵌套机制在 Cr 中相对干净（相比铁基的多轨道复杂性）
  → Cr 掺杂 Mn：增强嵌套 → 公度 SDW（commensurate）[来源待验证]
  → Cr 掺杂 V：破坏嵌套 → T_SDW 降低 → Cr-V 合金相图研究经典 [来源待验证]

## 5. SDW 与 SC 共存的相图（G1 补充）

```
=== 铁基超导中的 SDW + SC 共存区域（G1 主战场，H5 补充视角）===

共存区域的实验观察 [观察，争议]：
  Co 掺杂 BaFe₂As₂（x ≈ 0.04–0.06 欠掺杂区）：
  → 中子散射：在同一样品中测到 Q = (π,0) 处的 AFM 超晶格峰 [来源待验证]
  → µSR：局域磁场（SDW 信号）+ 完全抗磁性（SC Meissner 效应）同时出现 [来源待验证]
  → 两种信号同时存在 → 表观"共存"

但，微观共存 vs 相分离 [猜想，G1 核心争议]：
  1. 微观均匀共存（intrinsic coexistence）：
     → SDW 和 SC 序参量在相同晶胞内共存（不同 Fermi 口袋各自形成序）
     → 可能是因为 SDW（空穴口袋 + 电子口袋部分嵌套）+ SC（剩余 Fermi 面）
     → 需要：两者占据不同的 k 空间区域，互不干扰 [猜想]
  2. 宏观相分离（phase separation）：
     → 样品中 SDW 相和 SC 相空间分离，µSR/中子是体积平均
     → 可能原因：掺杂不均匀（Co 分布非均匀）→ 局域 x 不同 → 局域相不同 [猜想]
  → 目前证据不能区分这两种情形（尺度问题）[观察，待解决]

铁基 vs 铜氧化物 SDW/AFM + SC 相图的对比：

  铁基（金属态 SDW）：
  → SDW 和 SC 穹可以直接接触/重叠（相图共存区存在）
  → SDW T_SDW 和 Tc 相差不大（138K vs 20-38K，T_SDW/Tc ≈ 4-7）
  → SDW 能隙不完整（Fermi 面仍有残余部分），SC 可以在此基础上形成 [猜想]

  铜氧化物（Mott 绝缘体 AFM）：
  → AFM 和 SC 穹之间有 p ≈ 0.05-0.08 的过渡区（两者不接触）
  → T_N（AFM）>> Tc（Tc_max ≈ 90-130K，T_N > 300K for La₂CuO₄）
  → Mott AFM 完全绝缘化 → SC 需要足够多的空穴才能出现 [猜想]

区别的物理根源（关键，B2 关联）[猜想，重要]：
  → 铁基 SDW：金属中巡游电子的磁序（SDW 能隙部分，Fermi 面仍有金属性）
    → SC 可以在残余 Fermi 面上形成 → SDW 和 SC 可以共存（即使只是部分 k 空间）
  → 铜氧化物 AFM：Mott 绝缘体的局域磁矩序（全 Fermi 面能隙化）
    → 必须用空穴掺杂"摧毁"Mott 绝缘体后，SC 才能出现
  → 这个区别预测 [猜想]：铁基中可能有真正的 SDW+SC 微观共存，
    铜氧化物中只有 AFM 和 SC 的空间分相（而非微观共存）
```

## 6. SDW vs 向列序（H6 关联）

```
=== 铁基超导中 SDW 与向列序的关系 ===

向列序（Nematic Order，H6）的定义 [观察]：
  → 铁基超导中：四方对称性（C₄）自发破缺 → 斜方对称性（C₂）
  → 宏观上表现为电阻率各向异性（ρ_a ≠ ρ_b）[来源待验证，多组]
  → 与 SDW 共存（BaFe₂As₂ 中 T_SDW = T_nematic = 138K，同时相变）[Rotter 2008，确定]

SDW 和向列序的先后关系（重要争议）[猜想，铁基核心争论之一]：

  案例一：同时发生（BaFe₂As₂ 母化合物）[观察，确定]
  → T_SDW = T_structural = 138K（在 Ba122 中，两者同一温度发生）
  → 说明在母化合物中，向列序和 SDW 是同一相变的两个方面

  案例二：向列序先于 SDW（欠掺杂区）[观察，来源待验证]
  → 在轻度掺杂（欠掺杂）的铁基体系中，某些样品：
    T_nematic > T_SDW（向列序先发生，磁序后发生）
  → 这证明：向列序不依赖 SDW 长程序，可以独立存在 [观察，重要]
  → 向列序可以是"轨道"起源或"自旋"起源 [猜想，仍有争议]

向列序的起源争议（与 SDW 的关系）[猜想，两大阵营]：
  1. 自旋驱动（SDW 涨落驱动向列序）[猜想]：
     → SDW 涨落（短程磁序）产生各向异性 → 在 T_SDW 以上驱动结构畸变
     → 向列序是 SDW 的"前驱"（precursor）[猜想]
  2. 轨道驱动（轨道序驱动向列序）[猜想]：
     → d_xz 和 d_yz 轨道的简并自发破缺 → 产生 C₄→C₂ 对称破缺
     → 向列序独立于 SDW，SDW 只是向列序的伴生相 [猜想]

区分实验：[观察，来源待验证]
  → 弹性形变（elastoresistance）：测量 C₄→C₂ 对称破缺的涨落（探测向列序敏感度）
  → Raman 散射：轨道分量 vs 自旋分量的各向异性 [来源待验证]
  → 目前实验尚未给出最终答案（争议持续至 2026）[猜想]

重要结论（H5 和 H6 的接口）：
  → 在研究铁基 SDW 时，必须同时考虑向列序（两者通常绑定出现）
  → 向列序的 C₄→C₂ 对称破缺 + SDW 的磁序，是铁基母化合物正常态最重要的两个竞争序
  → H6 详述向列序；H5 仅标注与 SDW 的关系
```

## 8. 关键反谄媚（Anti-Sycophancy）

```
⚠️ 反谄媚 H5.1：SDW 长程序 ≠ SDW 涨落（最重要！）

  流行混淆："SDW 消失了，SDW 涨落也消失了"
  实际层级：[错误！长程 SDW 序和 SDW 涨落是不同层级的物理概念]

  核心区别：
  → 长程 SDW 序（T < T_SDW）：
    中子散射出现清晰磁性超晶格峰，长程相干磁序
    µSR 测到均匀局域磁场，全体积磁序
    N(EF) 显著降低，Fermi 面重构 → SC 被强烈压制
  → SDW 涨落（短程序，T > T_SDW 或掺杂超过 T_SDW = 0 的临界点）：
    中子散射只有 Q = (π,0) 处的漫散射（没有长程衍射峰）
    相关长度 ξ_SDW 有限（几到几十晶格常数）
    N(EF) 未被大量消耗 → SC 可以发展
    涨落本身可以驱动 Cooper 对形成（B3 的核心机制）[猜想]

  正确表述：
  → "掺杂消除了 SDW 长程序，但 SDW 涨落（短程 AFM 涨落）在最优掺杂区仍然存在，
    并可能驱动 s± 超导" [猜想]
  → "SDW 长程序压制 SC；SDW 涨落可能增强 SC" → 两者效果方向相反！

⚠️ 反谄媚 H5.2：铁基 SDW ≠ 铜氧化物 AFM（虽然都是磁序）

  流行混淆："铁基的 SDW 和铜氧化物的 AFM 都是反铁磁，本质一样"
  实际层级：[猜想→观察，两者物理机制不同，不可直接类比]

  关键区别：
  维度           铁基 SDW                  铜氧化物 AFM
  ---            ----                      ----
  母化合物性质   金属（有 Fermi 面）        Mott 绝缘体（无 Fermi 面）
  磁序起源       Fermi 面嵌套（巡游电子）   超交换（局域 Cu 3d⁹ 磁矩）
  磁矩大小       0.87 µ_B / Fe（小，部分嵌套）约 0.5-1 µ_B / Cu（接近饱和）[来源待验证]
  磁序类型       条纹 AFM（Q=(π,0)）        Néel AFM（Q=(π,π)，棋盘型）
  能隙           部分 Fermi 面能隙化        完全 Mott 能隙（全绝缘）
  SC 出现方式    掺杂降低 T_SDW → SC 穹    掺杂进入 Mott 绝缘体 → SC 穹

  物理后果（重要！）：
  → 铁基：SDW 和 SC 可以（微观上）共存（各占不同 k 空间）[猜想]
  → 铜氧化物：AFM 和 SC 不能微观共存（Mott 绝缘体 + 超导矛盾）
    → 相图中 AFM 和 SC 穹中间有无 SC 的过渡区

  正确表述：
  → "铁基 SDW（金属中的巡游 SDW）和铜氧化物 AFM（Mott 绝缘体局域磁矩）
    虽然都是磁序，但物理机制根本不同，对 SC 的影响也不同" [猜想，关键区分]

⚠️ 反谄媚 H5.3：SDW 是 s± 配对的必要条件吗？

  流行表述："铁基 SC 必须有 SDW 涨落驱动，没有磁序就没有超导"
  实际层级：[猜想，FeSe 是直接反例]

  FeSe 反例 [来源待验证，观察]：
  → FeSe 体材：Tc = 8K，零场下无长程磁序
  → 磁性涨落存在但较弱（相比 Ba122）
  → 单层 FeSe/SrTiO₃：Tc 大幅提升，而界面效应（声子/极化子）可能是主要驱动
  → 结论：铁基 SC 不需要强 SDW 涨落也可以出现 [猜想→观察]

  可能的非 SDW 驱动机制（铁基中）：
  → 轨道涨落（d_xz / d_yz 轨道间的涨落）[猜想]
  → Hund 物理（多轨道强关联）[猜想]
  → 界面声子（单层 FeSe 情形）[猜想]

  正确表述：
  → "SDW 涨落是铁基 SC 的重要驱动机制之一（最优掺杂 Ba122 等），
    但不是唯一机制（FeSe 反例）"[猜想，弱必要，强充分的近似]

⚠️ 反谄媚 H5.4：SDW + SC 共存区域是"真实"微观共存还是相分离？

  流行表述："铁基欠掺杂区中子散射 + µSR 显示 SDW 和 SC 共存，证明两者可以共存"
  实际层级：[观察，但解读为猜想，微观结构未定]

  问题：
  → 中子散射和 µSR 是体积平均技术，分辨率约 10-100 nm
  → 如果 SDW 和 SC 区域在 1-10 nm 尺度分相（掺杂不均匀导致），
    这些技术无法分辨微观共存和相分离
  → 真正的微观共存需要：原子尺度的 STM 直接成像，或 NMR 短程序测量 [来源待验证]
  → 目前证据：支持"表观共存"，但不能区分真实微观共存和相分离 [猜想]

  正确表述：
  → "铁基欠掺杂区的 SDW 和 SC 信号同时存在是可靠观察，
    但其微观性质（均匀共存 vs 相分离）尚未最终确定" [猜想]
  → G1 是处理这个争议的主战场

    Tc_max ≈ 38K（空穴端） [来源待验证]

[4] SDW 与超导的三种关系：
  [观察，铁基]：长程 SDW 序压制 SC（N(EF) 机制，母化合物无超导）
  [猜想，主流]：SDW 涨落（AFM 涨落）驱动 s± 配对（最优掺杂区，B3 详述）
  [观察，争议]：SDW + SC 共存于欠掺杂区（微观均匀共存 vs 相分离，G1 详述）

[5] 四条反谄媚：
  ★ SDW 长程序 ≠ SDW 涨落（两者对 SC 的效果相反！长程序压制，涨落可能增强）
  ★ 铁基 SDW（金属，巡游）≠ 铜氧化物 AFM（Mott 绝缘体，局域磁矩）
  ★ SDW 不是 s± 配对的必要条件（FeSe 反例：无磁序，有超导）
  ★ SDW + SC 共存信号 ≠ 微观均匀共存（可能是 nm 尺度相分离）

  [定理，BCS 框架]：SDW 打开 Fermi 面能隙 → N(EF) 降低 → Tc 降低
