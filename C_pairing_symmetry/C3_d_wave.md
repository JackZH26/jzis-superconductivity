## 1. 物理定义与核心思想

### 1.1 铜氧化物选择 d-wave 的必然性

铜氧化物高温超导体（YBCO、BSCCO、LSCO 等）的超导发生在 CuO₂ 平面上。这个平面的晶格对称群是 D₄h（四方点群），配对态必须是其中某个不可约表示。

**为什么是 B₁g（d_x²-y²），而非 A₁g（s-wave）？**

三重驱动机制共同指向同一结果：

1. **Cu 轨道特征**：导电电子主要来自 Cu 的 d_x²-y² 轨道（与 O 的 p_x/p_y 形成 σ 键），天然带有 B₁g 对称性
2. **AFM 自旋涨落（B3）**：铜氧化物母态是 AFM Mott 绝缘体，Q=(π,π) 处自旋涨落极强，有效配对势在 d_x²-y² 通道为负（吸引），在 s-wave 通道为正（排斥）
3. **RVB/Mottness（B9、B14）**：强关联 t-J 模型中，近邻交换 J > 0 天然给出 d-wave 配对

**[猜想 — 强支持，理论自洽]** 在强关联 + AFM 涨落框架下，铜氧化物 d_x²-y² 配对是"必然"的，而非巧合。但从第一性原理严格推导完整 Tc 值至今未完成。

### 1.2 d_x²-y² 的 k 空间结构

**[定义]** d_x²-y² 配对振幅（动量空间）：

$$\Delta(\mathbf{k}) = \Delta_0 (\cos k_x - \cos k_y)$$

ASCII:
```
Δ(k) = Δ₀ (cos k_x - cos k_y)

参数：
  Δ₀ = 最大配对振幅（能隙幅度）
  k_x, k_y = 约化布里渊区单位（k × a，晶格常数 a≈3.8Å）
  范围：k_x, k_y ∈ [-π, π]
```

**k 空间分布（ASCII 示意）：**
```
布里渊区 Δ(k) 的分布（正负号）：

       k_y
        ↑
   (-,-)│(+,+)
   (+,-)│(-,+)   ← 各象限值（cos k_x - cos k_y 的正负）
    ────┼────→ k_x
   (-,+)│(+,-)
   (+,+)│(-,-)

实际：
  (π,0) 方向：Δ = Δ₀(cos π - cos 0) = -2Δ₀  ← 最大（负）
  (0,π) 方向：Δ = Δ₀(cos 0 - cos π) = +2Δ₀  ← 最大（正）
  (π,π) 方向：Δ = Δ₀(cos π - cos π) = 0      ← 节线（零）
  (0,0) 方向：Δ = Δ₀(1 - 1) = 0              ← 零（Γ 点）
```

### 1.3 节线位置与节点

**节线方程**（Δ(k) = 0 的轨迹）：

$$\cos k_x = \cos k_y \Rightarrow k_y = \pm k_x$$

ASCII: 节线：k_y = ±k_x（布里渊区的对角线方向，即 (π,π) 方向）

**在铜氧化物的费米面上：**
- 费米面是近似正方形（来自 Cu-O 的 σ 带）
- 节线与费米面的交点形成**节点（nodes）**：4 个点，位于 (±π/2, ±π/2) 附近
- 节点处：Δ(k_node) = 0，能隙消失

**[定理 — 对称性强制节点]** B₁g 配对的节点是**对称性强制**的（symmetry-protected），不能被杂质或外场移动（只要 B₁g 对称性保持）。这与 C2 s± 的"意外节点"有本质区别。

### 1.4 极坐标形式与角度依赖

在铜氧化物费米面上（近圆形近似），用费米面角 φ 表达：

$$\Delta(\mathbf{k}_F) \approx \Delta_0 \cos(2\phi)$$

LaTeX: $\Delta(\mathbf{k}_F) = \Delta_0 \cos(2\phi)$

ASCII: Δ(k_F) = Δ₀ cos(2φ)，φ = 费米面上从 k_x 轴量起的角度

```
角度依赖：
  φ = 0（反节点方向，(π,0) 附近）：|Δ| = Δ₀ = 最大
  φ = π/4（节点方向，(π/2,π/2)）：|Δ| = 0
  φ = π/2（反节点，(0,π) 附近）：|Δ| = Δ₀ = 最大
```

这个 cos(2φ) 角度依赖是 ARPES 实验直接验证 d-wave 的核心观测量。

### 1.5 与 s-wave 的根本对比

| 特征 | s-wave（C1，A₁g）| d-wave（C3，B₁g）|
|------|-----------------|-----------------|
| **晶格旋转 90°（C₄）** | Δ(k) → +Δ(k)（不变号）| Δ(k) → -Δ(k)（变号）|
| **节点（群论强制）** | 无 | 4 个节点（± π/2, ± π/2 附近）|
| **节线** | 无 | 沿 k_y = ±k_x 方向 |
| **时间反演** | 保持 | 保持 |
| **空间反演** | 保持 | 保持 |
| **低能态密度** | N(ω) = N₀（常数，有能隙）| N(ω) ∝ ω（线性，节点 Dirac 锥）|
| **穿透深度 T→0** | 指数饱和 | 线性 T（Hardy 1993 实验判决）|

**判断规则：** 穿透深度线性 T → 排除全能隙 s-wave。Josephson 相位测量 → 判决性区分 d-wave vs s-wave。

### 2.2 关键实验 #1：Hardy 1993 穿透深度（UBC 组）

**背景：** 若超导体有能隙，低温下准粒子热激活被指数压制，穿透深度 λ(T) 应指数饱和（s-wave 签名）。d-wave 的节线上低能准粒子未被压制 → λ(T) 应线性依赖 T。

**实验方案（[定理 — 逻辑清晰]）：**
- 材料：YBCO 高质量单晶（YBa₂Cu₃O₇₋δ，最优掺杂）
- 方法：微波技术精密测量磁场穿透深度 λ(T)
- 关键：需要达到 Å 量级精度，排除所有系统误差

**Hardy 1993 结果：**
$$\frac{\delta\lambda(T)}{\lambda(0)} \propto T \quad (T \to 0)$$

LaTeX: $\delta\lambda(T) \propto T$ as $T \to 0$

ASCII: δλ(T) ∝ T（线性！指数行为被排除）

**定量结果：**
- λ(0) ≈ 150 nm（伦敦穿透深度）
- dλ/dT|_{T→0} = 3.7 Å/K [来源: Hardy et al. 1993, PRL 70, 3999; DOI: 10.1103/PhysRevLett.70.3999]
- T 线性行为从 T ≈ 4 K 以下清晰可见，远低于 Tc = 92 K

**物理意义：**
- 线性 T 来自节点附近的线性色散准粒子（Dirac 锥）
- 全能隙 s-wave → 指数 exp(-Δ₀/k_BT) → 被排除
- anisotropic extended s-wave（"哑铃形"，无节点）→ 也应给出指数 → 也被排除
- **结论：YBCO 必须有节点，最可能是 d-wave**

**[反谄媚]** 1993 年时 Hardy 结果有争议：有人认为是表面态效应，有人认为是 YBCO 中的 CuO 链贡献。在 Tsuei-Kirtley 1994 的相位实验出来前，穿透深度证据虽然强但仍有争议空间。

**[来源：Hardy et al. 1993, PRL 70, 3999; DOI: 10.1103/PhysRevLett.70.3999]**

### 2.4 关键实验 #2：Tsuei-Kirtley 1994 三角结相位实验（**判决性**）

这是迄今为止检验配对对称性最直接、最优雅的实验，被誉为"凝聚态史上最漂亮的实验之一"。

**设计原理：**

d-wave 序参量 Δ(k) 在旋转 90° 后变号（B₁g），这意味着：如果两个约瑟夫森结（Josephson junctions）面对铜氧化物的不同晶轴方向，它们会感受到不同符号的 Δ，导致相位差 π。

**实验构型（"三角结"，tricrystal）：**

```
Tsuei-Kirtley 1994 三角结几何（示意）：

     晶体 1        晶体 2
   ┌──────────┐ ┌──────────┐
   │ a 轴→    │ │ ←a' 轴   │
   │ (YBCO)  │J│ (YBCO)   │
   └──────────┘│└──────────┘
               │ 界面 1（90° 旋转）
               │
   ┌───────────┴──────────┐
   │      晶体 3（底）      │
   │         YBCO          │
   └──────────────────────┘

Josephson 结 1（晶体 1-3）：感受 Δ(k)
Josephson 结 2（晶体 2-3）：感受 Δ(k·旋转90°) = -Δ(k)
→ 两个结之间相位差 π → 这个"π 结"会自发产生半整数磁通量子
```

**判决机制（[定理 — 拓扑论证]）：**

- **若 s-wave（A₁g）**：旋转 90° 后 Δ 不变号 → 两个结相位差 = 0 → 正常约瑟夫森行为 → 无自发磁通
- **若 d-wave（B₁g）**：旋转 90° 后 Δ 变号 → 两个结相位差 = π（"π 结"）→ 三角结交点处**自发出现半整数磁通量子 Φ₀/2**

**实验结果（Tsuei-Kirtley 1994）：**

在扫描超导量子干涉仪（scanning SQUID 显微镜）下，在三角结的中心交汇点处观测到：
$$\Phi_{\rm spontaneous} = \frac{1}{2}\Phi_0 = \frac{h}{4e}$$

ASCII: Φ_spontaneous = Φ₀/2 = h/(4e)（半整数磁通量子！）

**[来源: C.C. Tsuei et al. 1994, PRL 73, 593; DOI: 10.1103/PhysRevLett.73.593]**
**[来源: J.R. Kirtley et al. 1995, Nature 373, 225; DOI: 10.1038/373225a0]**（后续更精确实验）

**为什么这是"判决性"的：**

1. **相位是拓扑量**：磁通量子是量子化的，没有连续误差，不依赖材料参数
2. **直接测量对称性**：不测能隙大小，而是测波函数的**变换性质**（这才是配对对称性的本质）
3. **控制实验完整**：同时测量单晶（无 π 结 → 无半整数磁通）和三角结（有 π 结 → 有半整数磁通），两个结果完全按照 d-wave 预言出现
4. **旋转变体验证**：后续实验用不同旋转角度验证相位变化 → 完全符合 cos(2φ)

**[观察 — 历史评价]** 这个实验通常被视为"相位敏感实验的典范"，Tsuei 和 Kirtley 因此工作获得多个重要奖项（包括 Buckley Prize）。在 1994 年以后，d-wave 对称性在铜氧化物中被科学界广泛接受。

## 3. 数学框架

### 3.1 d-wave 能隙函数的等价形式

**[定理 — 代数等价（已严格证明）]**

$$\Delta(\mathbf{k}) = \Delta_0(\cos k_x - \cos k_y)$$

多种等价表达式：

**(1) 三角函数展开：**
$$\Delta_0(\cos k_x - \cos k_y) = 2\Delta_0 \sin\!\left(\frac{k_x}{2}\right)\sin\!\left(\frac{k_y}{2}\right) \cdot \frac{\cos k_x - \cos k_y}{2\sin(k_x/2)\sin(k_y/2)}$$

更实用的直接关系（lattice d-wave）：

ASCII:
```
Δ(k) = Δ₀ (cos k_x - cos k_y)
     = Δ₀ × 2[-sin²(k_x/2) + sin²(k_y/2)]  ← 展开后
     ≈ Δ₀(k_x² - k_y²)/2                    ← 连续极限（k 小）
     = Δ₀ k² cos(2φ)                         ← 极坐标，φ = arctan(k_y/k_x)
```

**(2) 极坐标形式（费米面近圆形时）：**
$$\Delta(k_F, \phi) \approx \Delta_0 \cos(2\phi)$$

ASCII: Δ(k_F, φ) ≈ Δ₀ cos(2φ)（ARPES 直接测量的形式）

**(3) 实空间（近邻配对振幅）：**

d-wave 对应**最近邻 Cooper 对**（lattice site i 和 j = i ± x̂，i ± ŷ）：
```
Δ_{ij} = +Δ₀  （i,j 沿 x 方向）
Δ_{ij} = -Δ₀  （i,j 沿 y 方向）
Δ_{ij} = 0    （i,j 沿对角方向）
```
这正是 d_x²-y² 的实空间直觉：x 和 y 方向的 Cooper 对振幅符号相反。

### 3.2 节点 Dirac 锥

**[定理 — d-wave 低能有效理论]**

在节点 k₀ = (π/2, π/2) 附近（其他三个节点类似），展开 BdG Hamiltonian：

$$H_{\rm eff} = v_F \delta k_F \sigma_z + v_\Delta \delta k_\Delta \sigma_x$$

ASCII:
```
H_eff = v_F δk_F σ_z + v_Δ δk_Δ σ_x

δk_F = 沿费米面方向的 k 偏离（平行费米面方向）
δk_Δ = 垂直费米面方向的 k 偏离
v_F = 费米速度（沿费米面）
v_Δ = 能隙速度（垂直费米面）= (1/ℏ)|∂Δ/∂k|_{node}
σ_z, σ_x = 2×2 Pauli 矩阵（Nambu 空间 particle-hole）
```

**准粒子色散（各向异性 Dirac cone）：**

$$E(\mathbf{k}) = \sqrt{v_F^2 \delta k_F^2 + v_\Delta^2 \delta k_\Delta^2}$$

LaTeX: $E = \sqrt{v_F^2 \delta k_F^2 + v_\Delta^2 \delta k_\Delta^2}$

ASCII: E(k) = sqrt(v_F² δk_F² + v_Δ² δk_Δ²)（各向异性 Dirac 锥）

**低能态密度（线性 DOS）：**

$$N(\omega) = \frac{2}{\pi} \frac{|\omega|}{v_F v_\Delta}$$

ASCII: N(ω) = 2|ω|/(π v_F v_Δ)（线性！来自 4 个节点各自贡献线性 DOS）

对比：s-wave 中 N(ω) = N₀（常数，有能隙，ω > Δ₀ 才有态密度）

**定量参数（YBCO）：**
- v_F ≈ 2 eV·Å [来源待验证，参考 Chiao 2000 PRB 62, 3554]
- v_Δ ≈ 0.05-0.2 eV·Å，比 v_F 小约 10-40 倍 [来源待验证]
- 各向异性比：v_F/v_Δ ≈ 14-19（YBCO）[Chiao 2000 PRB 62, 3554; DOI: 10.1103/PhysRevB.62.3554]

### 3.3 BCS 自洽方程（d-wave 版本）

**[定理 — d-wave gap equation]**

$$\Delta(\mathbf{k}) = -\frac{1}{N}\sum_{\mathbf{k}'} V(\mathbf{k}, \mathbf{k}') \frac{\Delta(\mathbf{k}')}{2E(\mathbf{k}')} \tanh\left(\frac{E(\mathbf{k}')}{2k_BT}\right)$$

ASCII:
```
Δ(k) = -(1/N) Σ_{k'} V(k,k') × Δ(k')/(2E_{k'}) × tanh(E_{k'}/2k_BT)

E_{k'} = sqrt(ξ_{k'}² + |Δ(k')|²)   （BdG 准粒子能量）
N = 系统格点数
```

**d-wave 配对势投影：**

$$V(\mathbf{k}, \mathbf{k}') = V_d \cdot f_d(\mathbf{k}) \cdot f_d(\mathbf{k}')$$

其中 d-wave 形状因子：

$$f_d(\mathbf{k}) = \cos k_x - \cos k_y$$

ASCII:
```
V(k,k') = V_d × f_d(k) × f_d(k')
f_d(k) = cos k_x - cos k_y

代入自洽方程后（自旋涨落主导时 V_d < 0，吸引）：
Δ(k) = Δ₀ (cos k_x - cos k_y) 是自洽解
→ 这确认 d-wave 形式是 gap equation 的稳定解
```

### 3.4 穿透深度线性 T 定理

**[定理 — 在干净极限（l → ∞）的 d-wave 超导中]**

$$\frac{\delta\lambda(T)}{\lambda(0)} = \frac{2\ln 2}{\pi} \cdot \frac{k_B T}{\Delta_{\max}}$$

LaTeX: $\frac{\delta\lambda(T)}{\lambda(0)} = \frac{2\ln 2}{\pi} \cdot \frac{k_B T}{\Delta_{\max}}$

ASCII:
```
δλ(T)/λ(0) = (2 ln2 / π) × (k_BT/Δ_max)

线性斜率 = 2 ln2/π × k_B/Δ_max × λ(0)
  = 0.441 × k_B/Δ_max × λ(0)

来自：节点准粒子热激活（线性 DOS → 线性 T 穿透深度）
```

**对比（脏极限，存在杂质散射时）：** 节点有限化 → δλ(T) ∝ T²（非线性，但仍非指数）

## 4. d-wave 的实验识别方法（六大判据）

### 4.1 穿透深度 λ(T) 线性（Hardy 1993，判据 #1）

**[定理 — 干净极限]**

$$\delta\lambda(T) \propto T \quad (T \to 0)$$

ASCII: δλ(T) ∝ T（区别于指数 s-wave）

- **物理来源**：节点附近线性色散 → 低温准粒子热激活正比于 T
- **排除**：全能隙 s-wave（指数饱和）、anisotropic s-wave（无节点）
- **不能排除**：其他节点超导体（如 d_{xy}，但频率不同）
- **YBCO 实测**：dλ/dT = 3.7 Å/K [Hardy et al. 1993, PRL 70, 3999]
- **注意**：杂质会把线性 T 变成 T²（但不会变成指数），所以脏样品中线性 T 消失不能否定 d-wave

### 4.2 比热 C_s ∝ T²（判据 #2）

**[定理]**

$$C_s \propto T^2 \quad (T \to 0)$$

ASCII: C_s ∝ T²（来自节点线性 DOS 积分）

- N(ω) ∝ ω → 电子比热 C = ∫ N(ω) × ∂f/∂T dω ∝ T²
- vs s-wave：C_s ∝ exp(-Δ₀/k_BT)（指数激活）
- 注意：磁场下 T² 变成 T（Volovik 效应，半经典淬火）→ 进一步确认节点
- **[观察]** 铜氧化物比热测量的困难在于晶格比热（声子 C ∝ T³）与电子 T² 分离，需要高精度数据

### 4.3 NMR 1/T₁ ∝ T³（Hebel-Slichter 峰消失，判据 #3）

**[定理]**

$$\frac{1}{T_1} \propto T^3 \quad (T \to 0)$$

ASCII: 1/T₁ ∝ T³

- s-wave 的 Hebel-Slichter 峰（Tc 以下相干峰）在 d-wave 中消失
- 两个效应叠加：
  - (a) 节点 DOS（N(ω) ∝ ω）抑制态密度 → T³ 而非 T
  - (b) 相干因子（Coherence factor）：NMR 测的是"不平凡"相干因子，d-wave 的节点抑制 Hebel-Slichter 增强
- **[反谄媚]** s± 的 Hebel-Slichter 峰也被压制（但原因不同：相干因子相消），所以 1/T₁ 行为不能区分 d-wave 和 s±。需要结合穿透深度（d-wave 线性，s± 指数）综合判断

### 4.4 热导率 κ/T → 常数（T→0，判据 #4）

**[定理 — Durst-Lee 公式（通用极限热导率）]**

在 d-wave 超导节点处，热导率有"通用"（universal）极限：

$$\frac{\kappa}{T}\bigg|_{T \to 0} = \frac{k_B^2}{3\hbar} \frac{n_{\rm node}}{\pi} \frac{v_F}{v_\Delta} \times \text{const}$$

ASCII: κ/T → 常数（T→0），与杂质浓度无关（通用极限）

- 节点 Dirac 准粒子携带热流 → κ/T 有有限极限（即使 T→0）
- vs s-wave：κ/T → 0（无低能激发）
- "通用热导率"的值只依赖于 v_F/v_Δ（基本参数），不依赖杂质
- **YBCO 实测**：Chiao 2000 精确测量节点 κ/T，与 d-wave Dirac 锥预测一致 [Chiao 2000, PRB 62, 3554]

### 4.5 三角结相位实验（判决性，判据 #5）

**已在第 2.4 节详述。**

关键点（[定理 — 拓扑论证]）：
- B₁g 配对 → 旋转 90° 变号 → π 结 → 半整数磁通量子 Φ₀/2
- 直接测量**波函数的变换性质**，而非能隙大小
- 任何全对称（A₁g）配对（无论是否各向异性）都不会产生 π 结
- **唯一的"判决性"实验**：不依赖任何材料参数，纯粹基于量子干涉

### 4.6 ARPES Δ(k) 的角度依赖（判据 #6）

**[观察 — 直接可视化]**

ARPES 直接测量费米面上各方向的能隙 |Δ(k)|：

$$|\Delta(\phi)| = |\Delta_0 \cos(2\phi)|$$

ASCII: |Δ(φ)| = |Δ₀ cos(2φ)|（ARPES 测量量）

- 反节点（φ=0, π/2）处能隙最大 → 强 ARPES 强度变化
- 节点（φ=π/4, 3π/4）处能隙→0 → 费米面上准粒子穿过
- Shen 1995（BSCCO）：首次精确测量 Δ(φ) = Δ₀|cos(2φ)|，4 个节点清晰可见
- **局限性**：ARPES 只测 |Δ|，不测相位 → 不能区分 d-wave 和 anisotropic s-wave（需要相位实验）；铜氧化物 ARPES 主要在 BSCCO 上（易解理面），其他材料相对欠缺

## 7. d-wave 内部的子结构：赝能隙 vs 超导能隙

### 7.1 赝能隙现象

**[观察 — 实验确立，起源争议]**

欠掺杂铜氧化物（x < x_optimal）中，T* > T_c 以上已出现能隙特征（赝能隙）：
- ARPES 看到 T_c < T < T* 时，反节点方向（φ≈0）的能隙仍然存在
- NMR Knight shift 在 T* 以上才恢复正常（T_c 以上 Knight shift 未完全恢复）
- 赝能隙的 k 空间结构：类 d-wave（B₁g 对称），但只在反节点区域可见

### 7.2 Fermi arc（费米弧）

**[观察]**

欠掺杂铜氧化物中，ARPES 在 T_c < T < T* 看到：
- 节点附近（φ ≈ π/4）：费米面金属性保持（Fermi arc）
- 反节点附近（φ ≈ 0）：赝能隙开口，费米面"消失"
- 整个费米弧随温度降低而缩短（T → T_c 时弧缩短为节点）

**物理图像争议（均为 [猜想]）：**

| 解释 | 支持证据 | 反对证据 |
|------|---------|---------|
| 预形成 Cooper 对（phase fluctuation）| 赝能隙 d-wave 对称，与超导能隙连续演化 | T* 与 T_c 分离太大（欠掺杂区 T* > 200K）|
| CDW/SDW 竞争序 | STM 发现 CuO₂ 面上的电荷密度波（CDW），反节点方向 | CDW 对称性（B₁g CDW）与 d-wave 赝能隙相同，不好区分 |
| Mott 绝缘体赝能隙（Mottness）| 电子 Hubbard 模型的数值模拟显示反节点赝能隙 | 微观机制不清晰 |
| 拓扑序 / 轨道流序 | 中子散射在某些铜氧化物中发现 T* 以下时间反演破缺 | 争议极大，重复性不好 |

**[观察]** 2010 年代后，越来越多的证据支持 CDW 竞争序（X 射线散射），但赝能隙完整图像仍未定论。这是铜氧化物物理中最大的未解问题之一。

## 8. 关键反谄媚发现

### 8.1 铁基不是 d-wave（重要区分）

**[观察 — 一致性强]**

铁基超导（C2 s± 主角）不是 d-wave：
- 铁基有多个 Fermi 面（C2 多带结构），其 Q=(π,π) 散射给出 s±（A₁g 跨带符号反转）
- Tsuei-Kirtley 型相位实验（如果可做）预期铁基不会给出 π 结
- ARPES 在铁基中看到的是各向同性（或轻微各向异性）的能隙，无 d-wave 节点
- **意义**：d-wave（B₁g）是铜氧化物独有的，与铁基（A₁g s±）有本质对称性差异

### 8.2 Sr₂RuO₄：d-wave 实验方法的旁证

**[观察 — 2019 翻案教训]**

Sr₂RuO₄ 长期（1994-2019）被认为是 triplet p-wave 超导体：
- 最初 Knight shift 数据（1998-2000）显示超导相中 spin-susceptibility 不变 → triplet
- 2019 年新的 Knight shift（更高场强，更仔细扫描）推翻：spin-susceptibility 实际降低 → singlet
- 2020 年后主流共识：Sr₂RuO₄ 可能是 **d-wave singlet（B₁g）**，即与铜氧化物同一对称性

**教训**：即使是被"确立"的配对对称性结论也可能因新实验被推翻。需要多重独立实验交叉验证（相位实验是最可靠的）。

### 8.3 铜氧化物过掺杂端：d-wave 减弱

**[观察]**

- 过掺杂端（p > 0.22）：Δ_max 减小，BCS-like 行为部分恢复
- ARPES 测量：Δ_max 随过掺杂单调下降（与 Tc dome 一起下降）
- **但 d-wave 对称性（节点位置）仍保持**：节点在 (π/2,π/2) 方向，不随掺杂移动
- 这与 d-wave 是 B₁g 的对称性强制一致（对称性不会被掺杂改变）

### 8.4 赝能隙的 d-wave 形式不意味着 SC 起源

**[反谄媚]**

- 赝能隙有 d-wave（B₁g）对称性，这是事实
- **错误推论**：赝能隙 = 预形成超导能隙
- **原因**：CDW 或其他竞争序也可以有 B₁g 对称性（B₁g CDW 与 d-wave SC 有相同对称性，无法仅靠对称性区分）
- **实验挑战**：STM 的 Bogoliubov 干涉（QPI）显示超导能隙有 Bogoliubov 准粒子签名，而赝能隙区 QPI 特征不同 → 支持两者性质不同

### 8.5 ARPES 测量的表面效应

**[观察]**

- 铜氧化物的 ARPES 数据 90% 来自 BSCCO（因其易解理面）
- YBCO 的 ARPES 因表面重构（CuO 链 vs 平面）较难解释
- LSCO 的 ARPES 数据相对少，因为 La 的 4f 态背景干扰
- **后果**：d-wave 能隙函数 Δ(k) 的精确形式（是否真的是 cos k_x - cos k_y，还是更高阶修正）主要由 BSCCO 数据给出，其他材料需要假设类比

## 9. 与 Paper A/E 的关联

### 9.1 Paper A（多通道框架）

**[观察 — Paper A 不直接包含铜氧化物]**

铜氧化物 d-wave 超导不在 Paper A 的多通道 BCS 微扰框架内：
- Paper A 的框架基于 BCS-like 微扰展开，适用于弱关联体系（声子/轨道/自旋涨落媒介的配对）
- 铜氧化物是**强关联 Mott 系统**（Hubbard U >> t，需要 B14 Mottness 框架）
- d-wave 从 t-J 模型（B9 RVB）涌现，不是简单的 BCS 弱耦合可以描述的

**如果 Paper A 要扩展覆盖铜氧化物**：需要加入 RVB（B9）+ d-wave 通道 + Mott 物理的修正。

### 9.2 Paper E（轨道-能隙对应原则）

**[观察 — d-wave 是 Paper E 的核心案例]**

Paper E 的"轨道-能隙对应原则"：配对能隙的对称性 = 配对轨道的对称性。

在铜氧化物中：
- 导电电子主要来自 **Cu d_x²-y² 轨道**（B₁g 表示）
- 配对能隙 **Δ(k) = Δ₀(cos k_x - cos k_y)** 也是 B₁g 表示
- **完美匹配**：轨道对称性（B₁g）= 能隙对称性（B₁g）

**Paper E 的变分原理预言（[猜想]）：** 对于 Cu d_x²-y² 轨道主导的费米面，d-wave 配对（B₁g）是能量最优的配对态，这一预言与铜氧化物的实验完全一致。

**对比铁基（C2 的关联）：** 铁基的主要轨道 = Fe d_{xz}/d_{yz}（E_g，二维 E 表示），但铁基配对 = A₁g (s±)，这意味着 Paper E 的轨道-能隙对应需要更复杂的处理（多轨道混合 + AFM 涨落媒介）。

**结论**：铜氧化物 d-wave 是 Paper E 最干净、最直接的实验支持案例。

## 附录：关键公式速查（ASCII，适配 Discord）

```
=== C3 d_x²-y² wave 核心公式速查 ===

[1] d-wave 能隙函数（三种等价形式）：
  Δ(k) = Δ₀ (cos k_x - cos k_y)         ← 晶格 BZ 形式
  Δ(k_F, φ) = Δ₀ cos(2φ)               ← 极坐标（费米面角 φ）
  Δ(k) = Δ₀(k_x² - k_y²)/k_F²          ← 连续极限
  B₁g 表示：C₄ 旋转下 Δ → -Δ（变号！）

[2] 节线与节点：
  节线：cos k_x = cos k_y → k_y = ±k_x（对角线）
  节点：(±π/2, ±π/2) 附近（费米面与节线的交点）
  4 个节点（B₁g 对称性强制，不可移动）

[3] 节点 Dirac 锥（有效低能 Hamiltonian）：
  H_eff = v_F δk_F σ_z + v_Δ δk_Δ σ_x
  E(k) = sqrt(v_F² δk_F² + v_Δ² δk_Δ²)
  v_F/v_Δ ≈ 14-19（YBCO）[Chiao 2000]

[4] 低能态密度（线性）：
  N(ω) = 2|ω| / (π v_F v_Δ)  ← 来自 4 个节点
  vs s-wave N(ω) = N₀（有能隙，全能隙下为 0）

[5] 穿透深度（干净极限 d-wave）：
  δλ(T)/λ(0) = (2 ln2/π) × (k_BT/Δ_max)  ← 线性 T！
  实测 YBCO：dλ/dT = 3.7 Å/K [Hardy 1993 PRL 70, 3999]
  vs s-wave：δλ ∝ exp(-Δ₀/k_BT)（指数饱和）

[6] 比热与 NMR：
  d-wave 比热：C_s ∝ T²（节点线性 DOS 积分）
  d-wave NMR：1/T₁ ∝ T³，Hebel-Slichter 峰消失
  s-wave 比热：C_s ∝ exp(-Δ₀/k_BT)

[7] 三角结相位实验（判决性）：
  d-wave（B₁g）：旋转 90° → Δ 变号 → π 结 → 自发 Φ₀/2
  s-wave（A₁g）：旋转 90° → Δ 不变 → 0 结 → 无自发磁通
  实验（Tsuei-Kirtley 1994）：观测到 Φ₀/2 → d-wave 无可争议

=== d-wave 实验判据优先级 ===
  1. 三角结 Φ₀/2（最判决性，拓扑量不依赖材料参数）
  2. ARPES Δ(k) = Δ₀ cos(2φ)（直接可视化节点）
  3. 穿透深度线性 T（排除全能隙，证明节点存在）
  4. 热导率 κ/T 通用极限（节点 Dirac 锥定量）
  5. 比热 T²（节点 DOS 积分）
  6. NMR 1/T₁ ∝ T³（Hebel-Slichter 峰消失）

=== 关键材料速查 ===
  YBCO（YBa₂Cu₃O₇）：Tc = 92K，λ(0)=150nm，标准材料 [Hardy 1993]
  BSCCO（Bi₂Sr₂CaCu₂O₈）：Tc = 91K，ARPES 主战场 [Shen 1995]
  LSCO（La₂₋ₓSrₓCuO₄）：Tc = 38K，三角结验证 [Tsuei 1997]
  Hg-1223：Tc = 134K（常压），151K（加压），最高 Tc [Deng 2026]

=== 反谄媚核心点 ===
  ✗ 铁基不是 d-wave：铁基 s±（A₁g），铜氧化物 d-wave（B₁g）
  ✗ Sr₂RuO₄ 翻案：2019 Knight shift 推翻 triplet → 实验要交叉验证
  ✗ 赝能隙 d-wave 对称 ≠ 预形成超导：CDW 也有 B₁g 对称性
  ✗ 过掺杂端 Δ_max 减小：BCS-like 行为恢复，但对称性（节点位置）不变
  ✗ ARPES 主要来自 BSCCO：其他铜氧化物 k 空间数据较少
  ✗ Hg-1223 d-wave 未经相位实验直接验证
```
