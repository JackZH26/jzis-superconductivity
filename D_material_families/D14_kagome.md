## 1. 物理定义与核心思想

### 1.1 Kagome 晶格的几何

**[Kagome 晶格的几何性质为 [定理]；从几何到能带的推论为 [定理]；配对机制推论为 [猜想]]**

```
Kagome 晶格（角共享三角形网络）：

  二维 Kagome 晶格示意（ASCII）：
    
    *---*   *---*   *---*
    |\ /|   |\ /|   |\ /|
    | * |   | * |   | * |
    |/ \|   |/ \|   |/ \|
    *---*---*---*---*---*
    |\ /|   |\ /|   |\ /|
    | * |   | * |   | * |
    |/ \|   |/ \|   |/ \|
    *---*   *---*   *---*
    
  几何特征：
    - 角共享三角形（corner-sharing triangles）
    - 每个格点连 4 个近邻（非 Honeycomb 的 3 个）
    - 3 个格点构成一个原胞（3 个子晶格 A/B/C）
    - 空间群：P6/mmm（含 z 方向 → 三维 AV₃Sb₅）

  Kagome 晶格的紧束缚能带（最简单模型）[定理]：

    哈密顿量（最近邻跳跃 t，化学势 μ）：
    ASCII: H = -t * sum_{<ij>} c_i^dag * c_j + h.c. - mu * sum_i n_i

    三条能带（Kagome 特有）[定理]：
    (1) 平坦带（Flat Band, FB）：  E_FB = +2t（与 k 无关！）
    (2) 上抛物带（Upper Dispersive）：E_u(k) = -t + 2t*cos(k) 型（示意）
    (3) 下色散带（Lower Dispersive）：E_l(k) = -t - 2t*cos(k) 型（示意）

    平坦带的数学来源 [定理]：
      破坏性量子干涉（destructive quantum interference）
      跳跃路径相互抵消 → 电子无法传播 → 群速度 v_g = ∂E/∂k = 0
      → 平坦带（localized eigenstates）[定理]

    Van Hove 奇点（VHS）[定理]：
      M 点处（布里渊区边界中点）色散带的鞍点
      鞍点 → DOS 发散（二维中对数发散）
      ASCII: N(E_VHS) ~ -ln|E - E_VHS| / (pi * a^2 * |det(D²E)|^{1/2})
      → N(EF) 大幅增强，有利于各种序参量（SC、CDW、磁性）

    K/K' 点的 Dirac 锥 [定理]：
      色散带在 K/K' 交叉 → Dirac 型线性色散
      （类比 Graphene，但 Kagome 有额外的平坦带）
```

### 1.2 为什么 Kagome 超导体物理极其丰富

**[各机制概述；具体材料实现标注层级]**

```
Kagome 体系的多序竞争（D14 的核心主题）：

  ① 平坦带 → 量子几何（关联 B13）：
     平坦带中 v_g = 0 → BCS 公式 Tc ~ exp(-1/(N(EF)*V)) 失效
     Peotta-Törmä 框架：Tc 由量子度规（quantum metric）控制，而非 N(EF) [猜想]
     → 平坦带 Kagome SC 的 Tc 来源与普通 BCS 机制不同 [猜想]
     ⚠️ 但 AV₃Sb₅ 的平坦带在 EF 附近（精确位置 [来源待验证]）
     ⚠️ 且偏离理想模型（见第 8 节反谄媚）

  ② VHS → 高 N(EF) → 多序竞争：
     VHS 处 N(EF) 增强 → SC、CDW、磁性序均有利 [猜想]
     → AV₃Sb₅ 中 CDW 在 94K（>> Tc=2.5K）出现，VHS 驱动 CDW [猜想]
     CDW 开口能隙 → 减少 N(EF) → SC 被削弱（见第 6 节）

  ③ 拓扑能带 → 可能的拓扑超导（关联 C7）：
     Kagome 金属有非平凡 Berry 相（Z₂ 拓扑指数或 Chern 数）[来源待验证]
     + SC → 可能的拓扑超导 [猜想]
     d+id 配对（C7）在 Kagome 对称性（C₆ + 时间反演）下自然涌现 [猜想]

  ④ CDW 竞争 → 奇异正常态（关联 B6）：
     T_CDW（94K in CsV₃Sb₅）>> Tc（2.5K）→ CDW 是正常态中的主角
     CDW 态内：可能有时间反演破缺（AHE 观测，争议中）[观察 + 猜想]
     SC 在 CDW 背景下出现：配对对称性极可能被 CDW 修改 [猜想]

  结论：AV₃Sb₅ 是当前凝聚态物理中多序竞争最丰富的体系之一
  研究价值：理解多序竞争 → 通往非常规 SC 机制的窗口 [猜想]
```

## 2. 历史关键节点

**[AV₃Sb₅ 为 2021+ 新兴热点；发现论文来源较确定；后续实验数据大量标注 [来源待验证]]**

```
★★ 2018–2019：Kagome 金属的理论预言期
  理论组（Bzdušek/Neupert 等）预言 Kagome 金属应具有拓扑能带和平坦带性质
  研究主要聚焦于 Fe₃Sn₂、FeSn 等磁性 Kagome 金属
  超导体系尚未进入主流视野 [来源待验证]
  ⚠️ 这一阶段多为理论预言，具体引用需核查

★★★ 2020 Ortiz et al.（Santa Barbara，B. R. Ortiz + S. D. Wilson 组）
  论文：CsV₃Sb₅ 的晶体合成与基本物性表征
  [Ortiz et al. 2020, Phys. Rev. Materials 4, 094801]（来源较确定）
  主要结果：
    - 发现新型 Kagome 金属 CsV₃Sb₅（V 原子形成 Kagome 网络）
    - 测量到 CDW 转变（~94K）和超导转变（Tc ≈ 0.9K）
  ⚠️ 注：Ortiz 2020 PRM 可能记录的是 Tc=0.9K；后续优化样品 Tc 升至 2.5K

★★★ 2021 AV₃Sb₅ 家族确立
  多组几乎同时：KV₃Sb₅ (Tc≈0.9K)，RbV₃Sb₅ (Tc≈0.92K) 亦被发现
  [来源待验证：多篇 2021 年 PRB/PRL 论文；精确引用需核查]
  CsV₃Sb₅ 样品优化后 Tc 确认为 ~2.5K [来源待验证]
  AV₃Sb₅（A = K, Rb, Cs）三兄弟构成完整家族
  意义：第一个 Kagome 金属超导体系，引发全球研究热潮

★★★ 2021 反常霍尔效应（AHE）发现
  CsV₃Sb₅ 在 CDW 态内（T < T_CDW ≈ 94K）观测到反常霍尔效应
  特征：无外磁场下 ρ_xy 非零（时间反演破缺信号）
  争议：AHE 来源于 CDW 诱导的轨道磁矩？还是真正的磁性序？[猜想，激烈争议]
  [来源待验证：Yang et al. 2020/2021 相关报道，精确引用需核查]

★★ 2021–2022 CDW 结构的详细表征
  X 射线/中子衍射：确认 AV₃Sb₅ 中 2×2×4 超结构（三维 CDW）
  STM：看到 2×2 周期的电荷有序 [来源待验证]
  CDW 波矢：M 点附近（Q ≈ (1/2, 0, 0) 型）[来源待验证]
  ⚠️ CDW 是 3Q（三个方向叠加）还是单 Q，以及具体超结构，不同组之间存在分歧

★★ 2022 µSR 时间反演破缺信号
  µSR 实验：CsV₃Sb₅ 在 SC 态内观测到零场 µSR 信号变化
  → 解读为 SC 态中时间反演对称（TRS）自发破缺 [猜想]
  → 支持 chiral d+id 配对 [猜想]
  [来源待验证：Mielke 等 2022，精确引用需核查]
  ⚠️ 但 µSR 信号极弱，且 CDW 态对 µSR 的贡献难以完全排除

★★ 2022 Yin et al.（清华大学 Jia-Xin Yin 组）
  STM 实验：CsV₃Sb₅ 在超导态内观测到三倍对称周期的超导有序
  → 解读为可能的配对密度波（PDW）[猜想，C8]
  [来源待验证：Yin et al. Nature 2022 或 PRX 2022，精确引用需核查]
  ⚠️ STM 三倍周期 ≠ 确认 PDW（见第 8 节反谄媚）

★ 2023 CsCr₃Sb₅（更高 Tc 的 Cr 基 Kagome SC）
  CsCr₃Sb₅（Cr 替换 V）在高压下 Tc ≈ 6K [来源待验证]
  Cr 系统可能有 AFM 涨落（关联 B3），机制与 AV₃Sb₅ 不同 [来源待验证]
  常压 Tc 很低，高压需要几 GPa [来源待验证]

★ 2023–2025 其他 Kagome 体系探索
  FeGe（磁性 Kagome + SC，Tc ≈ 1.4K）[来源待验证]
  ScV₆Sn₆（Vanadium-Sn Kagome，CDW 体系但超导不确定）[来源待验证]
  RV₆Sn₆（R = 稀土，新型 Kagome 家族）[来源待验证]
  Kagome 超导研究从 AV₃Sb₅ 扩展到更广阔的材料库
```

## 3. AV₃Sb₅ 主要体系详析

### 3.1 CsV₃Sb₅（最重要，Tc 最高）

**[Ortiz 2020 PRM 为基础来源；后续实验结论大量标注 [来源待验证]]**

```
CsV₃Sb₅（铯钒锑，the flagship Kagome superconductor）：

  晶体结构 [来源待验证，但结构分析较可靠]：
    空间群：P6/mmm（六方层状）
    层状结构：Cs 原子层 + V₃Sb₅ 层交替堆叠
    V 原子形成完美 Kagome 网络（ab 面内）
    Sb1 原子：Kagome 层中填充六角孔（蜂窝型）
    Sb2 原子：形成 Cs-Sb 层（绝缘隔离层）
    晶格常数：a ≈ 5.5 Å，c ≈ 9.3 Å（近似值）[来源待验证]

  能带结构（关键特征）[来源待验证]：
    费米能级附近：V 的 3d 轨道主导（dxz/dyz/dz²）
    Kagome 平坦带：位于 EF 以上约 0.1–0.2 eV [来源待验证，不同文献有差异]
    Van Hove 奇点（VHS）：位于 EF 以下约 80–90 meV [来源待验证]
    → VHS 附近的高 DOS 驱动 CDW 和竞争序
    Dirac 锥：K/K' 点处，但被 CDW 部分破坏 [来源待验证]
    ⚠️ 平坦带并不在 EF 处（见第 8 节反谄媚！）

  超导性质：
    Tc ≈ 2.5K（最佳样品，常压）[来源：Ortiz 系列论文，较确定但精确来源待核查]
    SC 类型 II（由上临界场 Hc₂ 和短相干长度判断）[来源待验证]
    上临界场 μ₀Hc₂(0) ≈ 1–2 T（外推值）[来源待验证]
    相干长度 ξ ≈ 10–15 nm（从 Hc₂ 估算）[来源待验证]

  CDW 性质（核心！）：
    T_CDW ≈ 94K [来源：Ortiz 相关论文，较确定]
    CDW 超结构：2×2×4（面内 2×2 + 层间 4 倍周期）[来源待验证]
    CDW 序参量复杂：可能包含多个 Q 矢量（3Q CDW）[来源待验证]
    CDW 导致能带重整 + 部分能隙开口 → 减少 N(EF)

  反常霍尔效应（AHE）[观察，来源待验证]：
    在 CDW 态内（T_CDW > T > Tc）观测到 AHE
    ρ_xy(AHE) ≈ 0.05–0.15 μΩ·cm（不同组有差异）[来源待验证]
    无外磁场、无磁性杂质 → TRS 在 CDW 态内自发破缺？[猜想]
    AHE 来源争议（三种假说，均为 [猜想]）：
      (a) 轨道磁矩（orbital magnetic moment）：CDW 形成轨道环流 → 轨道磁矩
      (b) Berry 相（CDW 诱导）：拓扑非平凡 CDW 结构 → 非零 Berry 相
      (c) 手性 CDW：时间反演破缺的 CDW 序（chiral CDW）→ AHE
```

### 3.2 KV₃Sb₅ 和 RbV₃Sb₅（三兄弟对比）

**[[来源待验证]；三者趋势较为普遍认可但精确数据需核查]**

```
AV₃Sb₅ 三兄弟对比表：

  A = K：
    Tc ≈ 0.9K [来源待验证]
    T_CDW ≈ 78K [来源待验证]
    晶格参数：a 较小（K 离子半径最小）[来源待验证]

  A = Rb：
    Tc ≈ 0.92K [来源待验证]
    T_CDW ≈ 102K [来源待验证]
    介于 K 和 Cs 之间

  A = Cs：
    Tc ≈ 2.5K（最高！）[来源待验证，但数量级可靠]
    T_CDW ≈ 94K
    Cs 离子半径最大 → 更大化学压力 → 不同的带结构细节

  三者趋势分析：
    ┌─────────┬──────┬──────────┬────────────┐
    │  A 离子 │  Tc  │  T_CDW   │  趋势解读  │
    ├─────────┼──────┼──────────┼────────────┤
    │   K     │ 0.9K │   78K    │            │
    │   Rb    │ 0.92K│  102K    │ CDW 非单调 │
    │   Cs    │ 2.5K │   94K    │ Tc 随 A 增大│
    └─────────┴──────┴──────────┴────────────┘

    关键：Cs 的 Tc 最高（2.5K），而 T_CDW 不是最高（94K < Rb 的 102K）
    → 简单的"CDW 越强 SC 越弱"规律在三者中不完全成立 [猜想]
    → 化学压力（离子大小）对 SC 和 CDW 的影响不对称 [猜想]
    → 与 D10 有机 SC（Mott-SC 相图）类比：A 离子控制化学压力，类似于外加压力

  压力效应（CsV₃Sb₅）[来源待验证]：
    常压 Tc = 2.5K
    增压 → CDW 被抑制 → Tc 先升高（~8K at optimal pressure）[来源待验证]
    继续加压 → Tc 再下降（超出 CDW 完全抑制后，其他机制？）[猜想]
    相图：CDW dome + SC dome 形成竞争相图 [来源待验证]
```

## 4. Kagome 超导配对对称性的激烈争议

**[这是 D14 最重要的未解问题；所有候选方案均为 [猜想]；实验结论相互矛盾]**

```
AV₃Sb₅ 的配对对称性：当前凝聚态物理最激烈的争议之一

配对候选方案（均为 [猜想]）：

  候选 1：s-wave（常规 BCS）[猜想]
    支持证据（[来源待验证]）：
      部分比热测量：C/T vs T² 线性（全能隙，无节点）[来源待验证]
      部分穿透深度测量：低 T 饱和（全能隙行为）[来源待验证]
    物理图像：声子介导的 BCS s-wave；CDW 只是竞争者不影响配对类型
    弱点：无法解释 µSR 的 TRS 破缺信号和 STM 的周期调制

  候选 2：d-wave（关联 C3）[猜想]
    支持证据（[来源待验证]）：
      理论：VHS 处（M 点）d-wave 的散射是主导的 → d-wave 自然涌现
      部分穿透深度：低 T 线性（节点行为）[来源待验证]
    物理图像：VHS 处的涨落（CDW 涨落？）驱动 d-wave 配对
    弱点：节点 vs 无节点实验结论互相矛盾；d-wave 无法自然给出 TRS 破缺

  候选 3：d+id（Chiral，关联 C7）[猜想]
    支持证据（[来源待验证]）：
      理论：Kagome 晶格的 C₆ 对称 → d+id = e^{i*n*pi/3} 型配对自然允许
      µSR：零场 µSR 信号变化（TRS 破缺）[来源待验证]
      AHE：CDW 态+SC 态均有 TRS 破缺信号 [来源待验证]
    物理图像：
      d+id = d_{x²-y²} + i * d_{xy}（两个简并 d-wave 分量 π/2 相移）
      → 时间反演自发破缺（C7 核心特征）
      → Chern 数 = 2 → 拓扑超导 [猜想]
    弱点：µSR 信号弱，CDW 对 µSR 的贡献难以排除；
          d+id 需要严格简并（偶然实现？）[猜想]

  候选 4：PDW（配对密度波，关联 C8）[猜想]
    支持证据（[来源待验证]）：
      Yin et al. STM：超导态内 3 倍周期超导序（CsV₃Sb₅）[来源待验证]
      PDW 的波矢与 CDW 波矢相关（CDW → PDW 锁相）[猜想]
    物理图像：有限 q 的 Cooper 对配对，SC 序参量在空间周期调制
    弱点：STM 三倍对称 ≠ 确认 PDW（可能是 CDW 表面效应）[见反谄媚第8节]
          PDW 很难与均匀 SC 共存，能量如何最优化？[猜想]

  候选 5：s±（多带，关联 C2）[猜想]
    支持证据：V 带有多个费米面（三条 Kagome 能带）→ 可能的 s± [猜想]
    但：AV₃Sb₅ 无磁性 → AFM 涨落驱动 s± 的前提缺失 [猜想]
    弱点：AV₃Sb₅ 被认为非磁性 → s± 动机弱

实验证据的混乱（核心问题）：

  ┌────────────────┬──────────────────────────────────────────────────┐
  │ 实验技术       │ 结论                                             │
  ├────────────────┼──────────────────────────────────────────────────┤
  │ 比热           │ 不同组：全能隙 vs 有节点 [来源待验证]            │
  │ 穿透深度       │ 不同组：低 T 饱和 vs 线性 [来源待验证]           │
  │ µSR            │ 部分组：TRS 破缺信号；其他组：无明显信号 [来源待验证]│

## 8. 关键反谄媚

**[AV₃Sb₅ 热门程度使其容易被过度解读；以下是关键校正]**

```
反谄媚 1：Kagome 平坦带并不在费米能级处！

  ⚠️ 常见误读：AV₃Sb₅ 有平坦带 → 量子几何驱动的 Tc 增强（B13 机制）
  ✅ 事实：CsV₃Sb₅ 的平坦带位于 EF 以上约 0.1–0.2 eV [来源待验证]
           → 平坦带并不参与超导（Tc 的 Cooper 对来自 EF 处的色散带和 VHS）
  ✅ 修正：AV₃Sb₅ 的 Tc 不能用 B13（量子几何）框架直接解释
           量子几何效应在 AV₃Sb₅ 中可能只是次要贡献 [猜想]
  ✅ 此外：AV₃Sb₅ 的平坦带偏离理想 Kagome 模型（d 轨道混合 + 三维耦合）
           所谓"平坦带"只是近似，修正很大 [来源待验证]

反谄媚 2：反常霍尔效应 ≠ 确认时间反演破缺超导

  ⚠️ 常见误读：AHE 观测 → TRS 在 SC 态破缺 → 拓扑 SC（d+id）
  ✅ 分析：
    (a) AHE 主要出现在 T_CDW > T > Tc（CDW 态），而不仅仅是 SC 态
    (b) CDW 本身（特别是轨道 CDW 或手性 CDW）完全可以产生 AHE [猜想]
    (c) AHE 的出现不区分 CDW 驱动 vs SC 驱动
  ✅ 正确结论：AHE 是 TRS 破缺的证据，但 TRS 破缺的来源（CDW？SC？两者耦合？）
               仍完全不清楚 [猜想]
  ✅ 警告：µSR 的 TRS 破缺信号也可能来自 CDW 而非 SC 配对对称性

反谄媚 3：STM 的三倍周期超导序 ≠ 确认 PDW

  ⚠️ 常见误读：Yin et al. STM 看到三倍周期 → CsV₃Sb₅ 是 PDW（C8）
  ✅ 分析：
    (a) CsV₃Sb₅ 的 CDW 超结构本身有三倍周期对称（3Q CDW）[来源待验证]
    (b) CDW 的周期调制可以在 SC 态的 STM 图像中留下调制痕迹，即使配对是均匀的
    (c) PDW 需要超导序参量本身随位置调制（振幅或相位）
        → 仅凭 STM 图像的周期调制无法区分 CDW 产物 vs 真正的 PDW
  ✅ 正确结论：三倍周期 STM 信号是 PDW 候选证据之一，但远未确认 [猜想]

反谄媚 4：AV₃Sb₅ 的 Tc 极低（2.5K），无任何实用价值

  ✅ 坦率声明：
    CsV₃Sb₅ Tc = 2.5K，远低于液氦温区（4.2K）
    → 在任何实用场景（医疗/工程/能源）中毫无价值
    → 研究价值 100% 来自基础科学：多序竞争物理、拓扑 SC 候选、新型 CDW 物理
  ✅ 不要被炒作误导："发现 Kagome SC！拓扑超导突破！"
    这些标题对应的实验 Tc 均在 1–2.5K 范围，高压才能达到 7–8K

反谄媚 5：配对对称性之争不代表物理理解的进步

  ✅ 现实：截至 2026-04，已有数十篇论文宣称支持不同的配对对称性
           s-wave、d-wave、d+id、PDW 均有团队支持
  ✅ 根本原因：样品质量差异、CDW 背景干扰、表面 vs 体效应
  ⚠️ 风险：在没有判决性实验的情况下，每组倾向于解读数据支持自己偏好的图像
  ✅ 判决性实验路径（仍未实现）：
    - 相敏干涉实验（类似铜氧化物 Tsuei-Kirtley 1994，C3）
    - 节点的角分辨热导率（旋转场测量）
    - 单晶 NMR Knight shift 温度依赖

反谄媚 6：与 MATBG 的对比 — Kagome SC 不是 "Kagome 版魔角石墨烯"

  ⚠️ 常见类比：Kagome SC（平坦带 + 强关联）≈ MATBG（平坦带 + 强关联）
  ✅ 关键区别：
    MATBG：平坦带在 EF，Mott 绝缘体母态，强关联驱动 SC（B13 + B14 核心）
    AV₃Sb₅：平坦带在 EF 以上，非 Mott 绝缘体母态，弱关联-中等关联
    → 物理机制相差很大，MATBG 的成功框架不能直接移植到 AV₃Sb₅ [猜想]
```

**[明确适用范围和框架外问题]**

```
