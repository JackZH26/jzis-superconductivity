## 1. 物理定义与核心思想

### 1.1 反铁磁序的基本物理（[定理]）

```
=== AFM 序的基本物理 [定理] ===

反铁磁（Antiferromagnet, AFM）的定义：
  → 相邻格点自旋方向相反（↑↓↑↓...），无净磁矩（M_total ≈ 0）
  → 区别于铁磁（FM，↑↑↑↑，有净磁矩）
  → 区别于顺磁（PM，随机取向，无长程序）

奈尔温度（Néel temperature, TN）[定理]：
  → AFM → PM 的相变温度（二阶相变，有序 → 无序）
  → T < TN：AFM 长程序稳定
  → T > TN：热涨落摧毁 AFM 序，进入顺磁态
  → 类比铁磁的居里温度（Tc^FM），但对应的序不同

AFM 序参量（sublattice magnetization）[定理]：
  M = ⟨S_A⟩ - ⟨S_B⟩

  其中：
    S_A = A 子格的自旋（↑ 方向）
    S_B = B 子格的自旋（↓ 方向）
    M = 0：顺磁态（T > TN）
    M > 0：AFM 长程序（T < TN）
    → M 是 AFM 的序参量，类比超导的 |ψ|（Cooper 对密度）

自旋波（magnon，[定理]）：
  → AFM 序的低能涨落（集体激发，玻色子）
  → AFM magnon 是 two-branch 的（acoustic + optical，不同于 FM 的单支）
  → 能量 ∝ |q|（线性色散，区别于 FM magnon 的 q² 色散）
  → 在 TN 以上：magnon 变为弥散的临界涨落（短程 AFM 涨落）
  → 这些临界 AFM 涨落是 B3 的核心媒介玻色子！

自旋密度波（SDW，Spin Density Wave）[定理]：
  → 巡游电子体系中的 AFM 序（区别于 Mott 绝缘体中的局域 AFM）
  → 序参量：⟨n↑(r) - n↓(r)⟩ ∝ cos(Q·r)（SDW 波矢 Q）
  → 铁基超导的 AFM 序是 SDW 型（Q = (π, 0)，条纹状）
  → 铜氧化物未掺杂的 AFM 是局域 Heisenberg 型（Mott 绝缘体）
  ← 区分 SDW（巡游）与 Heisenberg-AFM（局域）是关键！
```

### 1.2 SC + AFM 的三类关系（[物理分类，广泛接受的框架]）

```
=== SC 与 AFM 的三类关系：竞争 / 协同 / 共存 ===

⭐ 最核心的物理直觉：
  AFM 【序】 vs AFM 【涨落】 → 完全不同的角色！
  → AFM 序（长程，T < TN）：对 singlet Cooper 对有害（竞争）
  → AFM 涨落（短程，T ~ TN 或 T > TN）：可以驱动 d-wave 配对（协同）
  ← 这个区分是 G1 最重要的物理洞察，也是最常见的混淆源！

─────────────────────────────────────────────────────
A. 竞争（Competition）[观察 + 理论支持]
─────────────────────────────────────────────────────
物理机制：
  → AFM 长程序中，子格 A 自旋 ↑，子格 B 自旋 ↓
  → singlet Cooper 对（↑↓ 配对）穿越 AFM 序：感受到反向局域磁场
  → 类比 Abrikosov-Gorkov（AG）效应：磁性杂质打破时间反演 → 抑制 Tc
  → 但全局 AFM 序的效果更系统：AFM 序越强，SC 越弱

典型体现（铜氧化物）：
  → p = 0（未掺杂）：强 AFM（TN ~ 300K），零 SC
  → p 增大：AFM 减弱，SC 出现
  → 两者的穹形相图"背靠背"：AFM 与 SC 互为消长

重要约束：
  → 竞争不是绝对的，强弱取决于 AFM 序的强度和 SC 配对能量的相对大小
  → 在 AFM 序很弱时（近 TN 或 QCP 附近），竞争减弱，共存可能出现

─────────────────────────────────────────────────────
B. 协同（Synergy）[猜想，广泛接受但非严格定理]
─────────────────────────────────────────────────────
物理机制：
  → AFM 涨落（T > TN 或掺杂将 TN → 0 后的临界区）
  → 涨落的波矢 q ~ Q_AFM（反铁磁波矢）
  → 电子交换 AFM 涨落 → 有效相互作用 V(q) 在 q ~ Q_AFM 处吸引
  → 对于 d-wave 配对（节点沿对角线）：V(q~Q_AFM) 选择 d_x²-y² 对称性
  → 即：AFM 涨落是 d-wave Cooper 配对的"黏合剂"（B3 详述）

典型体现（铜氧化物最优掺杂）：
  → p ≈ 0.16：AFM 序已消失，但 AFM 涨落最强
  → Tc 最高（LSCO: 38K，YBCO: 93K）
  → 涨落介导的 d-wave SC 在 TN → 0 的 QCP 附近达到峰值

⚠️ 层级提醒：
  → 这个机制是"[猜想]"：AFM 涨落与 SC 的关联有大量实验证据（中子散射、NMR、ARPES），
    但从关联到"AFM 涨落驱动 SC"的因果推论还有争议
  → 特别是：铁基 FeSe 在没有 AFM 涨落的情况下也有高 Tc（需要 B5/B10 等其他机制）

─────────────────────────────────────────────────────
C. 共存（Coexistence）[观察，微观共存 vs 相分离争议中]
─────────────────────────────────────────────────────
物理机制：
  → 在某些材料和条件下，AFM 序和 SC 可以在同一温度、同一样品中同时存在
  → 必要条件：两种序参量相互"容忍"，不完全竞争
  → 均匀共存：同一晶胞内既有 AFM 序（M≠0）又有 SC 能隙（Δ≠0）
  → 相分离：样品不同区域分别是 AFM 和 SC（宏观混合，非真正微观共存）

典型体现：
  → 铁基 BaFe₂As₂（Co 掺杂，中间掺杂区域）：中子散射 + µSR 均见 SDW+SC [观察，争议]
  → 重费米子 CeRhIn₅（压力 ~ 1.75-2.3 GPa）：AFM + SC 微观共存 [观察，待验证]
```

## 2. 历史关键节点

```
=== SC + AFM 研究的历史发展 ===

[1957] Abrikosov-Gorkov（AG）理论（[确定，JETP 5, 1174 (1957)；Sov. Phys. JETP 12 (1961)]）：
  磁性杂质（自旋-轨道散射）抑制 SC
  关键公式（Gorkov 1961 推广）：对 s-wave SC：
    ln(Tc/Tc0) = ψ(1/2) - ψ(1/2 + ħ/(2πkBτs*Tc))
    → τs = 磁性杂质散射寿命；Tc0 = 纯 SC 转变温度
  物理图像：磁性杂质打破时间反演对称（T），配对要求 T 守恒 → 磁性杂质破坏 singlet Cooper 对
  → 为后来 AFM 序抑制 SC 的定量讨论奠定理论基础（AG 类比：AFM 子格磁场 ↔ 磁性杂质）
  ⚠️ AG 定理本身是针对磁性杂质，用于 AFM 序需要额外近似（局域场近似）

[1964] Anderson（Anderson 定理，[确定]）：
  非磁杂质不影响 s-wave SC 的 Tc（A 定理）
  → 时间反演守恒保护 singlet 配对 → 非磁散射无害
  → 反面：磁性（AFM）序打破 T → AFM 序必然影响 SC

[1970s-1980s] 重费米子超导发现（[确定]）：
  → CeCu₂Si₂（1979，Steglich）：首个重费米子 SC（Tc=0.6K）
  → 重费米子中 AFM 序和 SC 相邻共存 → AFM-SC 相图研究开始
  → 早期认识：AFM 和 SC 竞争，但 AFM 涨落可能是配对媒介 [猜想当时已提出]

[1986-1987] 铜氧化物高温超导发现（[确定，Nobel 1987，Bednorz-Müller]）：
  → La₂CuO₄（母化合物）：AFM Mott 绝缘体，TN ≈ 325K [来源待验证，综述值]
  → 掺杂（Sr/Ba 替换）→ AFM 减弱 → SC 出现 → AFM-SC 相图成为高温 SC 研究核心
  → YBCO（92K）：AFM 母态（YBa₂Cu₃O₆），掺 O → SC（YBa₂Cu₃O₇）

[1987] Anderson RVB（[来源待验证，Anderson 1987 Science 235, 1196]）：
  → RVB 图像：AFM Mott 绝缘体的基态是 singlet pair（共振价键）的叠加
  → 掺杂释放 RVB 态的相位相干 → SC
  → 即：AFM 的量子基态"预成"了 Cooper 对，掺杂只是激活相干 [猜想，B9 详述]

[1989-1991] AFM 涨落介导 d-wave SC 理论（[来源待验证，Monthoux-Scalapino 系列]）：
  → Monthoux, Balatsky, Pines（1991）[来源待验证，PRL 67, 3448]：
    AFM 自旋涨落 → 有效配对核 V(q) → d_x²-y² 对称性自然涌现
  → 定量理论：Eliashberg-类框架但媒介玻色子是 AFM magnon 而非声子
  → 预言最优掺杂附近 AFM 涨落最强 → Tc 最高（与实验吻合）[B3 详述]

[1990s] 铜氧化物相图系统测量（[确定，多组]）：
  → LSCO 相图（Keimer et al., Tranquada et al. 等）[来源待验证]：
    TN(p) vs Tc(p) 精确测量
    → AFM 穹（p = 0 → 0.05，TN ~ 300K → 0）
    → SC 穹（p = 0.06 → 0.27，Tc_max ≈ 38K at p = 0.16）
  → NCCO（电子掺杂，n 型）的对应相图：
    AFM 更稳定（x = 0 → 0.13 才消失）[来源待验证]

[2008] 铁基高温 SC 发现（[确定，Kamihara 2008 JACS 130, 3296]）：
  → LaFeAsO：有 SDW（条纹 AFM，Q=(π,0)，T_SDW ≈ 140K）
  → F 掺杂 → SDW 减弱 → SC 出现（Tc ~ 26K → 55K）
  → SDW+SC 相图与铜氧化物类似，但机制存在重要差异（D4 详述）

[2009+] 铁基 SDW+SC 共存系统研究（[观察，争议中]）：
  → BaFe₂As₂ Co 掺杂系列（x = 0.04–0.10）：
    中子散射：SDW 峰在 SC 态中仍存在 [来源待验证，Nandi 2010 等]
    µSR：在某掺杂区域，SC 和磁序共存 [来源待验证]
    → 均匀共存 vs 相分离争议至今未完全解决

[2000s] 重费米子精细 AFM+SC 研究（[观察，来源待验证]）：
  → CeRhIn₅（Hegger 2000 PRL 84, 4986）：
    常压：AFM（TN = 3.8K），无 SC
    高压：AFM → QCP → SC 穹出现
    中间压力区：AFM + SC 微观共存（NMR 证据）[来源待验证，Curro et al.]
```

## 3. 铜氧化物的 AFM-SC 相图（最重要案例）

```
=== 铜氧化物相图：AFM 与 SC 的正面对决 ===

横轴：空穴掺杂 p（holes per Cu，0 → 0.30+）

ASCII 相图示意（LSCO 型）：

  温
  度   AFM         SC
  (K)  ────        ────
  300  │TN(p=0)         │
       │  \              │
  200  │   \             │
       │    \            │
  100  │     \    ┌──┐   │
       │      \  /Tc \   │
   50  │       \/max  \  │
       │       /│      \ │
   0   └───────/─────────┘
       0    0.05  0.16   0.27
            p (空穴掺杂)
            ←AFM→ ← SC穹 →

关键数据点（以 LSCO 为例）：

  p = 0：Mott 绝缘体 + AFM，TN ≈ 325K [来源待验证]
  p = 0.02-0.05：AFM 消失（TN → 0），SC 开始出现（Tc ~ 5-15K）
  p ≈ 0.16（最优掺杂）：Tc_max ≈ 38K，AFM 序完全消失 [来源待验证]
  p > 0.20（过掺杂）：Tc 下降，费米液体行为逐渐恢复
  p ≈ 0.27+：SC 消失

边界区域（p ~ 0.05-0.08）的 AFM+SC 共存 [争议，来源待验证]：
  → 部分实验（µSR）在这一范围见到磁序和 SC 的共存信号
  → 另一解释：样品不均匀（掺杂空间分布不均 → 不同区域分别处于 AFM 或 SC）
  → 共识：至少这一区域存在某种共存或竞争，但微观机制不清

n 型铜氧化物（电子掺杂，NCCO = Nd₂-xCexCuO₄）相图的关键差异：
  → AFM 更稳定：x = 0 → 0.13 才消失（p 型是 ~ 0.05）[来源待验证]
  → SC 穹更窄（Tc_max ≈ 24K，远低于 p 型的 ~90K）[来源待验证]
  → 相图 p/n 不对称性：
    p 型（空穴掺杂）：强 SC，AFM 窄
    n 型（电子掺杂）：弱 SC，AFM 宽
    → 任何完整理论必须解释这个不对称性（[观察，反谄媚关键点！]）

为什么不对称？（[猜想，无定论]）：
  → d 轨道电子 vs 空穴的 Fermi 面拓扑不同
  → 铜 d-轨道 vs 氧 p-轨道的混合程度在 p/n 掺杂中不同
  → Mott-Hubbard 到 charge-transfer 绝缘体在 n 型中更显著
  → 以上都是[猜想]，没有定论

铜氧化物的 AFM 母态物理（[定理，Mott-Hubbard 框架]）：
  → La₂CuO₄：Cu2+ 的 d⁹（一个空穴），Hubbard U >> t（强关联）→ Mott 绝缘体
  → 超交换（superexchange）J ≈ 4t²/U（反铁磁交换，[定理，Anderson 1950]）：
    J ≈ 130 meV（LSCO）[来源待验证]
    → 极强的 AFM 超交换 → 高 TN（~ 325K）
  → 这个 J 同时是 d-wave SC 配对强度的能量标度（B9 RVB 框架）
```

## 4. 铁基 SDW + SC 共存/竞争（系统性案例）

```
=== 铁基超导的 SDW+SC：巡游 AFM 与 SC 的竞争 ===

BaFe₂As₂ 系列（122 族）的相图：
  → 母化合物 BaFe₂As₂：T_SDW = 138K（条纹 AFM，Q=(π,0)）[确定，Rotter 2008 PRL 101, 107006]
  → Co 掺杂（Ba(Fe₁₋ₓCoₓ)₂As₂）：
    x = 0：T_SDW = 138K，Tc = 0
    x = 0.04-0.06：SDW 减弱，SC 开始出现（共存区！）
    x = 0.06-0.08：SDW 消失，Tc_max ≈ 22K [来源待验证]
    x > 0.12：SC 减弱，过掺杂

  → K 掺杂（Ba₁₋ₓKₓFe₂As₂）：空穴掺杂
    x = 0.2–0.4：Tc_max ≈ 38K [来源待验证]

SDW 与铜氧化物 AFM 的关键区别：
  铜氧化物 AFM：局域自旋（Heisenberg 型），Mott 绝缘体
  铁基 SDW：巡游电子（SDW 型），金属态（有 Fermi 面）
  → 两者在相图上类似（AFM 与 SC 背靠背），但微观物理完全不同！

铁基 SDW+SC 共存区（[观察，争议中]）：

  实验证据支持共存（均匀）：
    → 中子散射：SDW 磁峰在进入 SC 态后仍存在，强度减小但非零
      [来源待验证，Nandi 2010 等相关文献]
    → SDW 波矢 Q=(π,0) 在 SC 态没有变化（↑ 支持均匀共存，因为相分离时
      SDW 区域的 Q 不应受影响）
    → NMR/NQR：部分频谱在 SC 态见到 AFM 超精细场 [来源待验证]

  实验证据支持相分离：
    → µSR：在 SC+SDW 共存区，局域磁场分布呈非均匀（双峰或宽峰）
      [来源待验证]
    → 非均匀分布 → 样品不同区域磁场不同 → 可能是宏观相分离
    → STM：局域能谱不均匀 [来源待验证]

  争议的核心：
    1. 均匀共存（同一晶胞内 M≠0 且 Δ≠0）？
    2. 纳米尺度相分离（不同 nm 区域分别是 SDW 或 SC）？
    3. 宏观相分离（mm 尺度不同区域）？
    → 中子散射是体积加权（分辨率低），µSR 对 nm 尺度磁场分布敏感 → 两者可能看到不同尺度的现象
    → 2026 年仍是开放问题

向列序（nematic order，B5 关联）：
  → 在铁基 SDW+SC 的边界，四方 → 斜方结构畸变（C4 → C2 对称性破缺）
  → 向列序出现温度 T_s > T_SDW：向列涨落比 SDW 更早出现
  → 向列涨落（B5）可能独立贡献 SC 配对 → SDW+nematic+SC 三者的纠缠
  → 对于 FeSe：有向列序（Ts≈90K）无 SDW（T_SDW=0），但有 SC（Tc=9K）
    → 说明 FeSe 中向列涨落可以在没有 SDW 的情况下驱动 SC（B5 的证据）
```

## 5. 重费米子中的 AFM+SC（精细调控体系）

```
=== 重费米子：压力调控的 AFM → QCP → SC 轨迹 ===

CeRhIn₅（Hegger 2000）[来源待验证，PRL 84, 4986 (2000)]：

  常压：AFM（TN = 3.8K），无超导
  压力调控：

    P = 0：AFM（TN = 3.8K）
    P ~ 1.75 GPa：AFM 开始减弱，SC 开始出现（Tc ~ 1K）
    P ~ 1.75-2.3 GPa：AFM + SC 微观共存！[来源待验证，Curro et al. NMR]
    P ~ 2.3 GPa：AFM → 0（QCP），SC Tc 最高
    P > 2.3 GPa：纯 SC，TN = 0

  NMR 证据：
    → 在 P = 1.75-2.3 GPa，低温比热和 NMR 驰豫同时显示：
      AFM 超精细场（磁序信号）+ SC 能隙（准粒子压制信号）共存
      [来源待验证，Curro et al. 约 2000-2005]
    → 这是重费米子 AFM+SC 微观共存的最有说服力证据之一

  物理解释（[猜想，D8 关联]）：
    → 在 QCP 附近（P ~ 2.3 GPa）：量子临界涨落最强
    → 量子临界 AFM 涨落 → 驱动重费米子 SC（类似铜氧化物的涨落介导机制）
    → 但 CeRhIn₅ 是重费米子（f 电子 Kondo 屏蔽），配对对称性可能是 d-wave
    ⚠️ [猜想]：QCP → SC 的因果链有理论支持和关联证据，但非因果证明

CeCoIn₅（[来源待验证，Petrovic 2001 J. Phys. C.]）：
  → 常压 SC（Tc = 2.3K，重费米子 Tc 记录之一）
  → 无 AFM 序，但强 AFM 涨落 [来源待验证，中子散射]
  → 可能是 AFM 涨落驱动 SC 的"干净"案例（没有 AFM 序的干扰）[猜想]
  → 磁场下高场区可能有 FFLO 态（E5 关联）[观察，争议]

其他重费米子 AFM+SC 案例：
  → CePd₂Si₂：压力下 AFM → SC（类似 CeRhIn₅）[来源待验证]
  → UPt₃：多超导相 + AFM 邻近（f-wave [猜想]，C6 详述）[来源待验证]
  → URhGe / UCoGe：铁磁 + SC 共存（G2 关联，不在 G1 范围内，仅提及）
```

## 6. 判断 AFM+SC 共存的实验工具

```
=== 实验判据：如何区分 AFM+SC 共存 vs 相分离 ===

[1] 中子散射（Neutron Scattering）
  原理：磁矩散射中子 → 磁峰（布拉格峰）的位置和强度
  判据：
    → 磁峰在 Tc 以下是否仍存在？（SDW 峰消失 → 竞争，存在 → 共存）
    → 磁峰强度 vs 温度（T 扫）：在 Tc 处是否突变？
  优势：体积探针（全样品信号）
  局限：分辨率 ~ mm，无法区分纳米尺度相分离
  适用：铁基 SDW+SC 的初步判断（BaFe₂As₂ 系列）

[2] µSR（正 muon 自旋弛豫）
  原理：注入正 muon，感受局域磁场 → 弛豫曲线 → 局域磁场分布
  判据：
    → 均匀内场（单一弛豫频率）→ 均匀磁序（支持均匀共存）
    → 宽或双峰分布 → 样品不均匀（相分离的证据）
    → 在 SC 态中弛豫信号是否变化（Tc 处是否有跳变）？
  优势：对 nm 尺度磁场分布敏感（µ 探测局域场，比中子散射更细致）
  局限：体积探针（nm 分辨，非原子尺度）
  适用：铁基共存/相分离争议的关键判据之一

[3] NMR / NQR（核磁共振/核四极共振）
  原理：核自旋探测局域磁场和超精细场 → 谱线分裂/宽化
  判据：
    → 两个谱线（AFM 区域峰 + 非磁区域峰）→ 相分离
    → 单一谱线但 Knight shift 异常 → 均匀共存（或均匀无序）
    → 1/T₁T 弛豫 → 低能自旋涨落（与 SC 能隙打开相关）
  优势：原子尺度局域探针；可以区分不同晶体学位置
  局限：需要选择性核（如 In 在 CeRhIn₅，As 在 BaFe₂As₂）
  适用：重费米子 AFM+SC 共存的最强证据（CeRhIn₅，Curro 组）

[4] 比热（Specific Heat）
  原理：相变在比热中产生跳变（二阶相变 → 比热台阶）
  判据：
    → 两个跳变（TN 处 + Tc 处）→ 两个独立相变（支持共存）
    → 一个跳变（仅 Tc 或仅 TN）→ 其中一个相变已经消失
    → SC 凝聚能 ~ γTc（BCS），AFM 凝聚能 ~ 磁热比热
  优势：热力学总量，量化两个相变的相对强度
  局限：两个跳变太近时难以区分；样品不均匀会模糊信号

[5] 输运测量（Transport）
  原理：电阻率 ρ(T)、Hall 系数 R_H(T)
  判据：
    → ρ(T) → 0：SC 转变（Tc）
    → ρ(T) 在 TN 处的斜率变化 → AFM 序打开 SDW 能隙 → 载流子减少
    → R_H 符号变化 → SDW 时 Fermi 面重构 → 空穴/电子载流子比例变化
  优势：简单，精确，可确定 Tc
  局限：宏观平均，无微观信息；AFM-SC 边界处难以区分两个相变
```

## 8. 关键反谄媚

```
⚠️ 反谄媚 G1.1：AFM 序有害 ≠ AFM 涨落有害（最重要！）

  AFM 长程序（T < TN，有序态）：
    → 局域磁矩 → 破坏 singlet Cooper 对时间反演 → 抑制 SC（AG 类比）
    → 铜氧化物 p=0：强 AFM，无 SC —— 竞争的直接证据

  AFM 短程涨落（T > TN 或 QCP 附近，无长程序）：
    → 软模涨落（magnon/paramagnon），波矢 ~ Q_AFM
    → 交换这些涨落 → d-wave 配对势 V(q~Q_AFM) 为吸引
    → 铜氧化物最优掺杂（p~0.16）：AFM 序消失，AFM 涨落仍强，Tc 最高 ← 协同

  ← 最常见混淆：看到 AFM 涨落就说"AFM 抑制 SC"
    → 反了！AFM 涨落是 d-wave SC 的"催化剂"
    → 正确说法：AFM 【序】竞争 SC，AFM 【涨落】可以驱动非常规 SC

⚠️ 反谄媚 G1.2：铁基 SDW+SC 共存问题尚无定论

  不可说："铁基超导中 SDW 和 SC 均匀共存" [已知事实，确定]
  正确说：
    → 中子散射看到 SDW 峰在 SC 态仍存在 [观察]
    → µSR 看到局域磁场非均匀分布 [观察]
    → 均匀共存 vs 纳米相分离 vs 宏观相分离 三种解释均有支持 [争议中]
    → 在不同掺杂水平、不同材料体系，答案可能不同
    → 2026 年仍是开放问题，不要用确定语气描述

⚠️ 反谄媚 G1.3：铜氧化物相图的 p/n 不对称是真实约束，不能忽略

  空穴掺杂（p 型）：
    → AFM 在 p ~ 0.05 消失，SC Tc_max ~ 90K（YBCO）
  电子掺杂（n 型）：
    → AFM 在 x ~ 0.13 消失，SC Tc_max ~ 24K（NCCO）—— 窄得多，低得多

  任何声称"统一解释高温 SC 的理论"必须解释这个不对称
  → 不对称说明：空穴和电子的粒子-空穴对称性在 Mott 绝缘体附近 broken
  → 只解释 p 型的理论是不完整的 [猜想层级，但有强实验约束]

⚠️ 反谄媚 G1.4：QCP 驱动 SC 是[猜想]，不是定理

  常见说法："AFM 在 QCP 处消失，SC 穹出现在 QCP 附近 → QCP 驱动 SC"
  逻辑问题：时间/空间关联 ≠ 因果
  → QCP 处量子临界涨落最强 → 驱动 SC 是理论预言（Hertz-Millis-Moriya）[来源待验证]
  → 但实验验证极难（QCP 附近总有其他序的干扰）
  → 反例：FeSe 有 SC 和向列序，无 SDW，无 SDW-QCP → 说明 SC 不需要 SDW-QCP
  → 正确表述："AFM QCP 附近的量子涨落可能增强 SC，但非必要条件" [猜想]

⚠️ 反谄媚 G1.5：SC + AFM 相图的"相似性" ≠ 机制相同

  铜氧化物：AFM Mott 绝缘体（局域 Heisenberg 磁性）→ 掺杂 → SC
  铁基：SDW 金属（巡游电子）→ 掺杂/压力 → SC
  两者相图形状相似（AFM 穹与 SC 穹背靠背），但：
    → 铜氧化物：J 超交换 ~ 130 meV，强局域磁矩
    → 铁基：SDW 能隙 ~ Fermi 面嵌套，巡游 AFM
    → 配对对称性：铜氧化物 d-wave [确定]，铁基 s± [猜想，FeSe 是反例]
  ← 相图类比是结构类比，不是机制证明！用铜氧化物的思路直接套铁基会出错
```

[1] AFM 基本定义 [定理]：

  A. 竞争：AFM 【序】抑制 singlet SC（AG 类比）→ AFM 穹与 SC 穹背靠背
  B. 协同：AFM 【涨落】驱动 d-wave SC（交换 magnon/paramagnon）[猜想]
  C. 共存：某些体系 AFM 序和 SC 真正微观共存 [观察，均匀 vs 相分离争议]
