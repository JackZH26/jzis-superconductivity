# Pervasive spin-triplet superconductivity in rhombohedral graphene

**arXiv:** 2511.16578v2 | **作者:** Kumar, Waleffe, Okounkova, ..., Folk, Yankowitz (UWash + UBC)
**日期:** 2025-11-20 (v2: 2026-01-09)

## 核心方法论

在菱面体七层石墨烯 (R7G) 中观测到由面内磁场诱导和稳定的超导。R7G 与底部 hBN 对齐形成 13.5nm moiré 超晶格。

**实验手段：** 电输运测量（ρ_xx vs 门电压、温度、磁场），微分电阻 (dV/dI vs I_dc)

## 关键结果

1. **磁场诱导超导：** B∥ = 0 时只有两个小的超导 pocket (SC1, SC2)，B∥ = 1T 时整条锋利电阻脊变为超导
2. **远超 Pauli 极限：** 临界磁场远超 Pauli 顺磁极限 → **spin-triplet 配对**
3. **Tc 非单调演化：** Tc 随 B∥ 先增后减，存在最优磁场
4. **超导二极管效应：** 在多个区域观测到弱超导二极管效应，暗示 valley 不平衡与超导共存
5. **与整数量子反常 Hall 态相邻：** 超导出现在 IQAH 态附近

**物理机制：** 
- 低能态集中在表面（SSH 链类比）
- spin 旋转自由度在无 SOC 时可自由对齐 → B∥ 驱动自旋极化
- 自旋极化态有利于 spin-triplet, valley-singlet 配对

## 与研究的关联

**SC 方向转型：**
- 这是非 BCS 超导的干净实验平台
- 磁场增强超导 = Eliashberg 理论无法解释的现象
- 与量子几何综述直接呼应：R7G 的平带 + 量子几何 → 非常规超导

## 启发的新想法

**假设：** 如果 spin-triplet 配对可以在石墨烯实现，类似的机制是否可能在具有重元素（强 SOC）的层状材料中实现更高 Tc？即把 graphene 的 spin-triplet 物理迁移到 transition metal dichalcogenide 平台。

## 数值/公式

- Tc(R7G, zero field) < 0.1K [SC1, SC2 pockets]
- Tc(R7G, B∥ = 1T) ~ 数百 mK（精确值需看 specific gate settings）
- Pauli limit violation ratio: >> 1 (far beyond weak-coupling limit)
- 方法学关键: 4/P = 1/A + 1/bP + 1/cP 表示中 δ,b,c 形成 Z³ 中的仿射格
