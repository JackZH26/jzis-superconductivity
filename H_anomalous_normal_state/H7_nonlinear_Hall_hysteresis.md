## 1. 物理定义与核心思想

### 1.1 非线性 Hall 效应（NLHE）的理论框架 [定理，量子几何]

```
=== 非线性 Hall 效应（NLHE）的定义 ===

【线性 Hall 效应（参考基准）】
  j_y = sigma_xy * E_x   （一阶，Berry 曲率对 BZ 的积分）
  sigma_xy = e²/hbar * ∫_BZ Omega_z(k) * f(k) dk / (2π)^d
  → sigma_xy 是 Berry 曲率的加权积分，需要时间反演破缺（或外加磁场）才能非零

【非线性 Hall 效应（NLHE）】
  j_y^(2) = chi_yxx * E_x²   （二阶，非线性响应）
  chi_yxx ∝ D_xy = ∫_BZ (∂Ω_z/∂k_x) * f(k) dk / (2π)^d

  D_xy = Berry 曲率偶极子（BCD）= Berry 曲率在 k 空间的一阶矩（偶极矩）

  物理图像：
  → 线性 AHE：Berry 曲率的"总量"（零阶矩）→ 需要时间反演破缺
  → NLHE：Berry 曲率的"空间梯度"（一阶矩）→ 需要空间反演破缺 OR 时间反演破缺
  → 关键：NLHE 在中心反演破缺的系统中不需要磁场！（区别于 Lorentz-force Hall）

【对称性要求（NLHE vs 线性 AHE 的最重要区别）】

  线性 AHE（sigma_xy ≠ 0）要求：
    时间反演破缺（T-破坏）  →  磁性材料、外磁场
    且不要求空间反演破缺

  NLHE（chi_yxx ≠ 0）要求：
    时间反演破缺（T-破坏）  OR  空间反演破缺（I-破坏），至少一个破缺
    → I-破坏：非中心对称材料（如 TMD、少层 WTe₂）中，无需磁场可测 NLHE
    → T-破坏：磁性/磁化材料（如铁基向列相、Eu²⁺ 磁矩）中可测 NLHE
    注意：I-破坏体系的 chi_yxx 在时间反演作用下会反号
       → 可通过 B 反转或磁化反转改变 NLHE 符号，作为实验特征

【实验测量方法：二次谐波霍尔（2ω Hall）】
  驱动：I_x = I₀ cos(ω t)（电流频率 ω）
  线性 Hall：V_y^(ω)  ∝ sigma_xy（在频率 ω 处）
  非线性 Hall：V_y^(2ω) ∝ chi_yxx * I₀²（在 2ω 处！）
  → 用锁相放大器测 2ω 分量 → 直接分离 NLHE 信号
  → 挑战：信号很弱（I² 依赖），需要精确校准 2ω 背景
```

### 1.2 磁阻滞回（Magnetic Hysteresis）的物理起源 [猜想，多种可能]

```
=== 正常态磁阻滞回的多种起源 ===

定义：
  → 正常态（T > Tc）的电阻率 rho(H) 或 Hall 系数 R_H(H) 在
    磁场升/降扫描中走不同路径 → 产生磁滞回线

【机制 A：磁性杂质 / 磁畴翻转 [猜想/观察]】
  物理：
  → 样品中含有局域磁矩（稀土离子、过渡金属 d 电子等）
  → 在外加磁场下，磁矩/磁畴有"翻转场"（矫顽场 H_c）
  → 升/降场时磁畴翻转路径不同 → 磁阻/Hall 系数滞回
  例子：Yang 2026 中 Eu²⁺（S=7/2）大磁矩在磁场下翻转，产生磁性行为 [来源：已精读]

【机制 B：磁序（FM/AFM）与超导的竞争中的亚稳态 [猜想]】
  物理：
  → 磁性序和超导序竞争能量 → 在相变附近形成亚稳态
  → 升/降场时进入的相不同（亚稳相 vs 稳定相）→ 滞回
  → 特别是在 FM+SC 共存区（Yang 2026 的 FM+SC 共存区，G2 关联）

【机制 C：交换场 + J-P 效应的非线性响应 [猜想]】
  物理（J-P = Jaccarino-Peter 效应）：
  → 局域磁矩产生内禀交换场 H_ex，方向与外加场 H_ext 相反
  → H_total = H_ext - H_ex
  → 当 H_ext ≈ H_ex 时：总场趋零 → Cooper 对重新配对 → 再入超导
  → 滞回来自：升降场时 Eu²⁺ 磁矩的磁化路径不对称

【机制 D：量子干涉 + 磁性散射（弱局域化） [猜想]】
  物理：
  → 磁性杂质的自旋翻转散射打破时间反演 → 破坏相干传播
  → 磁场下弱反局域化（WAL）→ 磁阻（一般无滞回）
  → 若磁矩有矫顽性 → 可能产生小的磁阻滞回
  注意：此机制在 NLHE 情形更多出现在拓扑表面态，不是 Yang 2026 的主因

结论：在超导材料中，磁阻滞回最可能来自 A+C（局域磁矩 + J-P 类效应），
而不是简单的铁磁长程序（见反谄媚 9.2）。
```

## 2. 历史关键节点

```
=== NLHE 与磁阻滞回的历史里程碑 ===

【2015 Sodemann & Fu — BCD 和 NLHE 的理论预言（奠基之作）】
  → 提出 Berry 曲率偶极子（BCD）的概念
  → 预言：在时间反演对称但空间反演破缺的系统中，BCD ≠ 0 → NLHE
  → 给出 chi_yxx = e³ tau / (2 hbar²) * epsilon_zxy * D_xy（完整表达式）
  [Sodemann & Fu 2015, PRL 115, 216806，来源待验证]

  → 材料：少层 WTe₂（非中心对称拓扑 Weyl 半金属）
  → 方法：2ω Hall 测量
  → 关键特征：NLHE 信号随 B 反转改变符号（T 对称的特征）
  → 意义：首次把 BCD 理论预言变为实验事实
  [Ma et al. 2019, Nature 565, 337，来源待验证；注意发表时间约 2019 年非 2018 年，待核实]

【2019-2023 二维材料中 NLHE 的扩展】

## 6. Berry 曲率偶极子（BCD）的材料依赖

```
=== 哪些材料满足 BCD ≠ 0 的对称性条件 ===

对称性分析（[定理，群论]）：
  BCD = ∫_BZ (∂Ω_z/∂k_x) f(k) dk
  
  若时间反演 T 保持：Ω(-k) = -Ω(k) → BCD 可以非零（反演破缺体系）
  若空间反演 I 保持：Ω(k) = Ω(-k) → BCD 积分 = 0（BCD 恒为零！）
  若 T × I 保持（PT 对称，磁群 mGray）：BCD 也恒为零
  
  结论：BCD ≠ 0 需要破坏 I，或破坏 T，或同时破坏两者（最强 BCD）
  
  BCD 非零的材料类型：
  类型 I：空间反演破缺（非中心对称），T 保持
    → TMD（MoS₂, WSe₂, WTe₂ 单层）：强 BCD，谷 NLHE [来源待验证]
    → 有机超导体（BEDT-TTF 类）：非中心对称结构 [猜想]
    → 扭转 moiré 体系（MATBG：扭转打破全局 I）：中等 BCD [来源待验证]
  
  类型 II：时间反演破缺，I 保持（磁性材料）
    → Eu²⁺ 掺杂的 Sm-nickelate（Yang 2026）：Eu²⁺ T 破缺 → BCD ≠ 0 [来源：已精读，推断]
    → 铁基超导向列相：SDW/向列序 → T-破缺 → BCD ≠ 0 [猜想，基于 Romero 2026 背景]
    → 线性 AHE 强的材料一般 BCD 也可能大（但两者不成正比）[猜想]
  
  类型 III：I 和 T 同时破缺（最一般情形）
    → Chern 绝缘体（非零 Chern 数 = 非零总 Berry 曲率 → BCD ≠ 0 可能）[猜想]
    → 磁性 TI（MnBi₂Te₄ 等）：T-破缺 + 界面破缺 I → BCD 大 [来源待验证]

【铁基超导（FeTeSe）中的 BCD 分析（向列 QCP 背景，Romero 2026 推断）】

  高温相（T > T_nem，C4 保持）：
  → I + T 均（近似）保持 → BCD ≈ 0 → NLHE ≈ 0

  向列相（T < T_nem，C4→C2）：
  → 向列序本身破坏 C4，但是否破坏 I 取决于具体的 d 轨道分布 [猜想]
  → SDW 涨落同时存在 → T-破缺分量 → BCD ≠ 0 可能性增加
  → FeTeSe 是拓扑材料（Dirac 点 + 拓扑表面态）→ 体 BCD 可能较大 [猜想]
  → 定量估算：缺乏 k 空间 Berry 曲率分布的 DFT 计算结果 [来源待验证]

【Eu-nickelate 中的 BCD 分析（Yang 2026 体系，推断）】

  正常态（T > Tc，FM 区）：
  → Eu²⁺ 的 4f 电子有大磁矩（S=7/2）→ 强 T-破缺 → BCD ≠ 0
  → 薄膜界面（LSAT 基底 / 薄膜 / 真空）→ 结构上破坏 I [猜想]
  → 预测：FM 区的 NLHE 应该比 SC dome 区更强（T-破缺更强）[猜想，未检验]

  超导穹顶（T < Tc，SC 区，x ≈ 0.22）：
  → Eu²⁺ 较少，T-破缺弱 → BCD 较小 → NLHE 弱 [猜想]
```

## 8. 关键反谄媚（Anti-Sycophancy）

```
⚠️ 反谄媚 H7.1：NLHE ≠ 线性 AHE（最核心的概念区分！）

  流行混淆："非线性 Hall 效应就是反常 Hall 效应（AHE）的一种"
  实际层级：[错误！响应阶数完全不同]

  正确物理：
  线性 AHE：j_y = sigma_xy * E_x（一阶响应，Berry 曲率积分，需 T-破缺）
  NLHE：j_y^(2) = chi_yxx * E_x²（二阶响应，BCD，需 I-破缺 OR T-破缺）
  
  关键区分点：
  → 响应阶数：AHE 在频率 ω 处，NLHE 在 2ω 处（实验可分辨）
  → 对称性要求：AHE 必须 T-破缺；NLHE 只需 I-破缺或 T-破缺之一
  → 磁场依赖：AHE 在低场下（铁磁体）可以存在；NLHE 原则上在零磁场下也可以
  → 物理起源：AHE 是 Berry 曲率的零阶矩（总量），NLHE 是一阶矩（梯度/偶极子）
  
  实验上的混淆风险：
  → 在 FM+SC 共存区（如 Yang 2026）：AHE 和 NLHE 可能同时存在
  → 若只测 DC Hall（sigma_xy）：只看到 AHE（一阶），NLHE 信号被一阶响应淹没
  → 必须用 AC 驱动 + 锁相放大器测 2ω 分量才能分离 NLHE

⚠️ 反谄媚 H7.2：磁阻滞回 ≠ 铁磁长程序 [猜想层级混淆]

  流行混淆："观察到磁阻滞回就证明样品有铁磁序"
