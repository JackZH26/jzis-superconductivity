# A3. 比热跃变（Specific Heat Jump）

**MECE 编号：** A3
**关联 MECE：** A5（能隙）、A1+A2（SC 三重判据）、B1（BCS）、B2（Eliashberg）、C1-C10（节点结构）
**精读日期：** 2026-04-27
**精读模型：** Claude Opus 4.7（high thinking）
**层级：** 综述（基于 BCS + Carbotte 1990 + 现代多带数据）

## 1. 物理定义

**比热跃变：** 当 T 从上方接近 T_c 时，等容比热 C_v(T) 出现一个**有限阶跃**：
$$\Delta C \equiv C_v(T_c^-) - C_v(T_c^+) > 0$$

**BCS 普适比值（弱耦合）：**
$$\frac{\Delta C}{\gamma T_c} = \frac{12}{7\zeta(3)} \approx 1.43$$

其中 γ 是正常态 Sommerfeld 系数 (C_v^n = γT)，ζ(3) ≈ 1.202 是 Apéry 常数。

**物理意义：** 二阶相变特征 — 序参量 |Ψ|² 在 T < T_c 时连续从 0 增长，但其温度导数在 T_c 处不连续。

## 3. 关键数值与公式

### 3.1 BCS 弱耦合普适关系

| 量 | BCS 值 | 依赖 |
|----|--------|------|
| Δ_0 / k_BT_c | 1.764 | π / e^γ_E |
| 2Δ_0 / k_BT_c | 3.528 | 同上 |
| ΔC / (γT_c) | 1.43 | 12/(7ζ(3)) |
| (Δ_0)² / [γ T_c²] | 1.054 | 推论 |

**注释：**
$$\zeta(3) = \sum_{n=1}^{\infty} \frac{1}{n^3} \approx 1.202\,056\,9 \quad \text{[Apéry 常数]}$$

### 3.2 跃变随耦合强度变化

[定理 - BCS 1957] 在严格弱耦合极限 λ → 0 时 ΔC/γT_c = 1.43

[观察 - Carbotte 1990] 强耦合修正：
| 材料 | 2Δ/kBTc | ΔC/γTc | 来源 |
|------|---------|--------|------|
| Al | 3.5 | 1.45 | 弱耦合 |
| Sn | 3.5 | 1.6 | 中等 |
| In | 3.6 | 1.7 | 中等 |
| Tl | 3.6 | 1.5 | 中等 |
| Ta | 3.6 | 1.6 | 中等 |
| V | 3.4 | 1.5 | 中等 |
| Hg | 4.6 | 2.4 | 强耦合 |
| Pb | 4.3 | 2.7 | 强耦合 |
| Nb₃Sn | ~4.3 | ~2.4 | 强耦合 |

[来源] Carbotte 1990 Rev. Mod. Phys. 62, 1027 [DOI: 10.1103/RevModPhys.62.1027]

**关键观察：** ΔC/γT_c 与 2Δ/k_BT_c 一起增长 — 两者都反映**配对强度**，是同一物理的不同侧面。

### 3.3 强耦合公式

[定理 - Marsiglio-Carbotte 1991] 在强耦合 Eliashberg 框架下：
$$\frac{\Delta C}{\gamma T_c} \approx 1.43 \left[ 1 + 53 \left(\frac{T_c}{\omega_{\log}}\right)^2 \ln\left(\frac{\omega_{\log}}{3T_c}\right) \right]$$

这给出一个解释为何 Pb（ω_log 低）显示比 Al（ω_log 高）大得多的跃变比。

## 4. 多能隙体系

### 4.1 MgB₂（双能隙典范）
[来源] Bouquet et al. 2002 Europhys. Lett. 56, 856 [DOI: 10.1209/epl/i2002-00193-6]

- σ 带（B p_x,y 带，强耦合）：Δ_σ ~ 7 meV
- π 带（B p_z 带，弱耦合）：Δ_π ~ 2 meV
- C(T) 显示两个跃变：T_c 主跃变 + 低 T 下的 π 带"额外肩部"

[观察] ΔC/γTc 总和 ~ 1.0（远低于单带 BCS 1.43）— 因为弱 π 带"稀释"了平均能隙。

### 4.2 铁基 SC
- Fe pnictides：多带，最多三个能隙
- C(T) 拟合需 multi-gap 模型
- BaFe₂(As,P)₂ 节点 SC：C(T) ∝ T² 在低 T 下

### 4.3 镍酸盐 SC（D5）
- La₃Ni₂O₇ 在压力下 80K 的体相比热数据**目前不全**（薄膜样品体积小）
- 是 D5 综述的一个缺口

## 5. 节点结构对 C(T) 的影响

| 配对 | T → 0 时 C_v | 节点物理 |
|------|---------------|---------|
| s-wave（无节点）| ∝ exp(-Δ/T) | 指数压制 |
| d-wave（4 节点）| ∝ T² | 节点附近线性 DOS |
| p-wave（2 节点）| ∝ T² | 类似 d 波 |
| f-wave（6 节点）| ∝ T² | 类似 |
| 有杂质 d-wave | ∝ T（残留线性项）| 杂质散射散开节点 |

[关键应用] C(T) 在 0-T_c/3 区间的形状是**配对对称性诊断**的重要工具，与 A5、A9、A10 联动。

## 6. 测量方法

### 6.1 弛豫法（最常用）
- 适用于小样品（< 1 mg）
- 高精度（~1%）
- 标准 PPMS 平台

### 6.2 AC 比热
- 频率响应可分离不同弛豫机制
- 高灵敏度
- 大型设备

### 6.3 绝热脉冲
- 经典方法，需大样品（克级）
- 高精度，但样品要求高

### 6.4 弹性中子散射 + 比热联合
- 提取磁性贡献
- 复杂多体体系

## 7. 已知例外 / 复杂情形

### 7.1 一阶相变
- 通常 SC 是二阶相变，但**铁磁 SC**（如 UGe₂）可能是一阶
- 一阶相变给出 latent heat（非 ΔC）

### 7.2 涨落主导的 C(T)
- 强涨落 SC（如 cuprate underdoped）：C(T) 在 T_c 上下有明显涨落贡献
- BCS 阶跃模糊或丢失

### 7.3 拓扑超导
- 边态贡献小，体相 ΔC 类似常规
- 但能隙各向异性可能影响

### 7.4 Pseudogap 与"伪跃变"
- Cuprate underdoped 区 T* 处可能有"小跃变"
- 与真正 T_c 处跃变需明确区分

### 7.5 多带 + 涨落混合
- MgB₂ 双跃变 + 涨落 → 拟合复杂
- 必须谨慎使用 BCS 普适比

### 7.6 平坦带 SC
- MATBG 体积太小，C(T) 测量困难
- 当前数据稀缺

## 8. 与 A1+A2 的三重判据

经典 SC 验证三重证据：
1. **A1 零电阻**：ρ → 0
2. **A2 Meissner**：B → 0（χ = -1）
3. **A3 比热跃变**：ΔC > 0 在 T_c

**任何 SC 声明（包括 LK-99 类伪声明）必须**这三条同时满足才被接受。

[案例] Bozovic 2016 LSCO：χ ≈ -0.5 但 ΔC 完整 → 体超导但有空腔/缺陷

## 11. 数据汇总（反幻觉确认）

[定理] BCS ΔC/γT_c = 12/(7ζ(3)) ≈ 1.43

[观察] Carbotte 1990 数据：常规金属普遍在 1.43-2.7 间

[观察] MgB₂ Bouquet 2002：双能隙体系 ΔC/γT_c < 1.43

[观察] BaFe₂(As,P)₂ 节点 SC 在低 T 显示 C ∝ T²，即 d-wave 标志

[未检验巧合] BCS 比值 2Δ_0/k_BT_c = 2π e^(-γ_E) ≈ 3.528 与 π + 1/φ² ≈ 3.524 偏差 0.114% — 详见 QSH-E7 中讨论。
